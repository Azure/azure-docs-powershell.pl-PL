# Install-AzsUpdate

## STRESZCZENIe
Stosowanie określonej aktualizacji w lokalizacji aktualizacji.

## POLECENIA

### Zastosuj (domyślnie)
```
Install-AzsUpdate -Name <String> [-Location <String>] [-ResourceGroupName <String>] [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### ApplyViaIdentity
```
Install-AzsUpdate -InputObject <IUpdateAdminIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Opis
Stosowanie określonej aktualizacji w lokalizacji aktualizacji.

## Przykłady

### Przykład 1: Install-AzsUpdate według nazwy
```powershell
PS C:\> Install-AzsUpdate -Name Microsoft1.1907.0.10
```

Unifiedgroup pozwala instalować określone aktualizacje według nazwy.
Pamiętaj, że istnieje wymóg, aby wersja aktualizacji była ściśle większa niż bieżąca wersja.
