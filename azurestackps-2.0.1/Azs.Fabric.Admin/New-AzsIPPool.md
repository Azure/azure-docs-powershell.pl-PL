# New-AzsIPPool

## STRESZCZENIe
Tworzenie puli adresów IP.
Po utworzeniu puli adresów IP nie można usunąć.

## POLECENIA

### Rozszerzenie (domyślne)
```
New-AzsIPPool -Name <String> [-Location <String>] [-ResourceGroupName <String>] [-SubscriptionId <String>]
 [-AddressPrefix <String>] [-EndIPAddress <String>]
 [-NumberOfAllocatedIPAddress <Int64>] [-NumberOfIPAddress <Int64>] [-NumberOfIPAddressesInTransition <Int64>]
 [-StartIPAddress <String>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### Tworzonych
```
New-AzsIPPool -Name <String> -Pool <IIPPool> [-Location <String>] [-ResourceGroupName <String>]
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## Opis
Tworzenie puli adresów IP.
Po utworzeniu puli adresów IP nie można usunąć.

## Przykłady

### Przykład 1:
```powershell
PS C:\> New-AzsIpPool -Name IpPool4 -StartIpAddress ***.***.***.*** -EndIpAddress ***.***.***.*** -AddressPrefix ***.***.***.***/24

```

Tworzenie nowej puli adresów IP.

  - `[AddressPrefix <String>]`: Prefiks adresu.
  - `[EndIPAddress <String>]`: Końcowy adres IP.
  - `[NumberOfAllocatedIPAddresses <Int64?>]`: Liczba obecnie przydzielonych adresów IP.
  - `[NumberOfIPAddresses <Int64?>]`: Łączna liczba adresów IP.
  - `[NumberOfIPAddressesInTransition <Int64?>]`: Bieżąca liczba adresów IP w ramach przejścia.
  - `[StartIPAddress <String>]`: Początkowy adres IP.
