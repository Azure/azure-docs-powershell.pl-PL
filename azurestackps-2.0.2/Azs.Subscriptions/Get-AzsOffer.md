# Get-AzsOffer

## STRESZCZENIe
Pobierz listę ofert publicznych dla głównego dostawcy.

## POLECENIA

```
Get-AzsOffer [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## Opis
Pobierz listę ofert publicznych dla głównego dostawcy.
