# Get-AzsBackup

## STRESZCZENIe
Zwraca kopię zapasową z lokalizacji na podstawie nazwy.

## POLECENIA

### Lista (domyślna)
```
Get-AzsBackup [-Location <String>] [-ResourceGroupName <String>] [-SubscriptionId <String[]>] [-Skip <String>]
 [-Top <String>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Pobierz
```
Get-AzsBackup -Name <String> [-Location <String>] [-ResourceGroupName <String>] [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-AzsBackup -InputObject <IBackupAdminIdentity> [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## Opis
Zwraca kopię zapasową z lokalizacji na podstawie nazwy.

## Przykłady

### Przykład 1: kopie zapasowe list
```powershell
PS C:\> Get-AzsBackup

```

Pobierz informacje sbout wszystkie kopie zapasowe stosu platformy Azure.

### Przykład 2: pobieranie określonej kopii zapasowej
```powershell
PS C:\> Get-AzsBackup -Name 'backupName'

```

Uzyskiwanie informacji na temat określonej kopii zapasowej stosu platformy Azure.
