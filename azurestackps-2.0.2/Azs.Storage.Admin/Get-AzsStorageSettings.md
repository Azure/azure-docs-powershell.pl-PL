# Get-AzsStorageSettings

## STRESZCZENIe
Zwraca ustawienia dostawcy zasobów magazynu.

## POLECENIA

```
Get-AzsStorageSettings [-Location <String>] [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## Opis
Zwraca ustawienia dostawcy zasobów magazynu.

## Przykłady

### Przykład 1:
```powershell
PS C:\> Get-AzsStorageSettings
```

Pobierz ustawienia miejsca do magazynowania.
