# Get-AzCognitiveServicesAccountKey

## STRESZCZENIe
Pobiera klucze interfejsu API dla konta.

## POLECENIA

```
Get-AzCognitiveServicesAccountKey [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzCognitiveServicesAccountKey** pobiera klucze interfejsu API dla konta usług poznawczych.
Konto usług poznawczych ma dwa klucze API: KEY1 oraz key2.
Klucze umożliwiają interakcję z punktem końcowym konta usług poznawczych.
Aby ponownie wygenerować klucz, użyj New-AzCognitiveServicesAccountKey.

## Przykłady

### Przykład 1
```powershell
PS C:\> Get-AzCognitiveServicesAccountKey -ResourceGroupName cognitive-services-resource-group -name myluis

Key1                             Key2
----                             ----
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```
