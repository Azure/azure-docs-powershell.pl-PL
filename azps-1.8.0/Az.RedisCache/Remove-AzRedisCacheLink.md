# Remove-AzRedisCacheLink

## STRESZCZENIe
Usuń łącze replikacji Geo między dwiema Redis pamięci podręcznej.

## POLECENIA

```
Remove-AzRedisCacheLink -PrimaryServerName <String> -SecondaryServerName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Usuń łącze replikacji Geo między dwiema Redis pamięci podręcznej.

## Przykłady

### Przykład 1: usuwanie linku replikacji Geo
```
PS C:\>Remove-AzRedisCacheLink -PrimaryServerName "mycache1" -SecondaryServerName "mycache2"
```

To polecenie usuwa linki replikacji geograficznej, w której pamięć podręczna Redis o nazwie mycache1 jest podstawowa, a pamięć podręczna Redis o nazwie mycache2 jest pomocnicza.
