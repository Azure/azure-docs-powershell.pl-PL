# Add-AzApplicationGatewayAuthenticationCertificate

## STRESZCZENIe
Dodaje certyfikat uwierzytelniania do bramy aplikacji.

## POLECENIA

```
Add-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Add-AzApplicationGatewayAuthenticationCertificate** dodaje certyfikat uwierzytelniania do bramy aplikacji platformy Azure.

## Przykłady

### Przykład 1: Dodawanie certyfikatu uwierzytelniania do bramy aplikacji
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Add-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway $appgw -Name "cert01" -CertificateFile "C:\cert.cer"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

Pierwsze polecenie uzyskuje bramkę Application Gateway o nazwie appGwName i zapisuje ją w zmiennej $appgw.
Drugie polecenie dodaje certyfikat uwierzytelniania o nazwie cert01 do bramy aplikacji.
W trzecim poleceniu jest aktualizowana Brama Application Gateway.
