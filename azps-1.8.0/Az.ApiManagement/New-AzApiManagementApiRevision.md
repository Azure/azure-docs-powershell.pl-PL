# New-AzApiManagementApiRevision

## STRESZCZENIe
Tworzy nową poprawkę istniejącego interfejsu API.

## POLECENIA

```
New-AzApiManagementApiRevision -Context <PsApiManagementContext> -ApiId <String> -ApiRevision <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis

Polecenie cmdlet **New-AzApiManagementApiRevision** tworzy poprawkę API dla istniejącego interfejsu API w kontekście zarządzania interfejsem API.

## Przykłady

### Przykład 1: Tworzenie poprawki interfejsu API dla interfejsu API
```powershell
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>New-AzApiManagementApiRevision -Context $ApiMgmtContext  -ApiId 5adf6fbf0faadf3ad8558065 -ApiRevision 6

```

To polecenie tworzy wersję API `2` `echo-api` interfejsu API.
