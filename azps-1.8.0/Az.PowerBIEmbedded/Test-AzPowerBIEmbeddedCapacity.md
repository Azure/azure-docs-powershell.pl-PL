# Test-AzPowerBIEmbeddedCapacity

## STRESZCZENIe
Testowanie istnienia wystąpienia z osadzonymi możliwościami usługi PowerBI.

## POLECENIA

```
Test-AzPowerBIEmbeddedCapacity [-Name] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Test-AzPowerBIEmbeddedCapacity testuje istnienie wystąpienia osadzonej pojemności usługi PowerBI

## Przykłady

### Przykład 1
```
PS C:\> Test-AzPowerBIEmbeddedCapacity -Name "testcapacity"
True
```

To polecenie zostanie sprawdzone, jeśli istnieje pojemność o nazwie testcapacity
