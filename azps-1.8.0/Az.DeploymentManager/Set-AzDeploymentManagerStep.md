# Set-AzDeploymentManagerStep

## STRESZCZENIe
Umożliwia zaktualizowanie kroku.

## POLECENIA

```
Set-AzDeploymentManagerStep [-InputObject] <PSStepResource> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzDeploymentManagerStep** aktualizuje krok z określonym obiektem kroku.
Polecenie cmdlet zwraca zaktualizowany obiekt kroku.

## Przykłady

### Przykład 1
```powershell
PS C:\> Set-AzDeploymentManagerStep -InputObject $stepObject
```

To polecenie umożliwia zaktualizowanie kroku, którego nazwa i nazwa źródła danych są zgodne z właściwościami Name i ResourceGroupName odpowiednio w $stepObject.
Krok zostanie zaktualizowany o właściwości ustawione w $stepObject.
