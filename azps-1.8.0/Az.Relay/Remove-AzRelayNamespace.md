# Remove-AzRelayNamespace

## STRESZCZENIe
Usuwa obszar nazw z określonej grupy zasobów. 

## POLECENIA

```
Remove-AzRelayNamespace [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzRelayNamespace** usuwa obszar nazw z określonej grupy zasobów.

## Przykłady

### Przykład 1
```
PS C:\> Remove-AzRelayNamespace -ResourceGroupName Default-ServiceBus-WestUS -Name TestNameSpace-Relay1
```

Usuwa obszar nazw przekaźnika `TestNameSpace-Relay1` z określonej grupy zasobów `Default-ServiceBus-WestUS` .
