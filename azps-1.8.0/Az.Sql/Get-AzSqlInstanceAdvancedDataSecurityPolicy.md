# Get-AzSqlInstanceAdvancedDataSecurityPolicy

## STRESZCZENIe
Pobiera zaawansowane zasady zabezpieczeń danych wystąpienia zarządzanego.

## POLECENIA

```
Get-AzSqlInstanceAdvancedDataSecurityPolicy [-InputObject <AzureSqlManagedInstanceModel>]
 -InstanceName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzSqlInstanceAdvancedDataSecurityPolicy** retrives zaawansowane zasady zabezpieczeń danych wystąpienia zarządzanego.

## Przykłady

### Przykład 1-pobiera zaawansowane zabezpieczenia danych wystąpienia zarządzanego
```powershell
PS C:\>  Get-AzSqlInstanceAdvancedDataSecurityPolicy `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Przykład 2 — pobiera zaawansowane zabezpieczenia danych wystąpienia zarządzanego zasobu wystąpienia
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Get-AzSqlInstanceAdvancedDataSecurityPolicy

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```
