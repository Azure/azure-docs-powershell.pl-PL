# Remove-AzSqlServerThreatDetectionPolicy

## STRESZCZENIe
Usuwa zasady wykrywania zagrożeń z serwera.

## POLECENIA

```
Remove-AzSqlServerThreatDetectionPolicy [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Remove-AzSqlServerThreatDetectionPolicy usuwa zasady wykrywania zagrożeń z programu Azure SQL Server.
Aby użyć tego polecenia cmdlet, określ parametry ResourceGroupName oraz nazwa_serwera w celu zidentyfikowania serwera, z którego to polecenie cmdlet usunie zasady.

## Przykłady

### Przykład 1: usuwanie zasad wykrywania zagrożeń dla bazy danych
```
PS C:\> Remove-AzSqlServerThreatDetectionPolicy -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
```

To polecenie usuwa zasady wykrywania zagrożeń z serwera o nazwie Server01.
