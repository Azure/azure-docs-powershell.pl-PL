# Stop-AzSqlDatabaseActivity

## STRESZCZENIe
Anuluje operację aktualizacji asynchronicznych w bazie danych.

## POLECENIA

```
Stop-AzSqlDatabaseActivity [-ServerName] <String> [-ElasticPoolName <String>] -DatabaseName <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **stop-AzSqlDatabaseActivity** anuluje operację aktualizacji asynchronicznych w bazie danych.

## Przykłady

### Przykład 1. Anuluj operację aktualizacji asynchronicznych w bazie danych.
```
PS C:\>Stop-AzSqlDatabaseActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01" -OperationId af97005d-9243-4f8a-844e-402d1cc855f5

```

To polecenie anuluje operację aktualizacji asynchronicznych w bazie danych.
