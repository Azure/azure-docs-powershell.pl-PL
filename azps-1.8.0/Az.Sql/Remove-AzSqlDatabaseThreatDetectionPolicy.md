# Remove-AzSqlDatabaseThreatDetectionPolicy

## STRESZCZENIe
Usuwa zasady wykrywania zagrożeń z bazy danych.

## POLECENIA

```
Remove-AzSqlDatabaseThreatDetectionPolicy [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzSqlDatabaseThreatDetectionPolicy** usuwa zasady wykrywania zagrożeń z bazy danych AzureAzure SQL.
Aby użyć tego polecenia cmdlet, określ parametry *ResourceGroupName* oraz *nazwa_serwera* w celu zidentyfikowania bazy danych, z której to polecenie cmdlet usunie zasady.

## Przykłady

### Przykład 1: usuwanie zasad wykrywania zagrożeń dla bazy danych
```
PS C:\>Remove-AzSqlDatabaseThreatDetectionPolicy -ResourceGroupName "ResourceGroup11" -ServerName "Server01" -DatabaseName "Database01"
```

To polecenie usuwa zasady wykrywania zagrożeń z bazy danych o nazwie Database01 na serwerze o nazwie Server01.
