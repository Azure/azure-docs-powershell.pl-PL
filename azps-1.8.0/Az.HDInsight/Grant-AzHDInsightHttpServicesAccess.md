# Grant-AzHDInsightHttpServicesAccess

## STRESZCZENIe
Przyznaje dostęp HTTP do klastra.

## POLECENIA

```
Grant-AzHDInsightHttpServicesAccess [-ClusterName] <String> [-HttpCredential] <PSCredential>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Grant-AzHDInsightHttpServicesAccess** udziela dostępu HTTP do klastra usługi Azure HDInsight przy użyciu ODBC, Ambari, Oozie i usług sieci Web.

## Przykłady

### Przykład 1: udzielenie dostępu HTTP do klastra usługi Azure HDInsight
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

PS C:\> Grant-AzHDInsightHttpServicesAccess `
            -ClusterName $clusterName `
            -HttpCredential $newClusterCreds
```

To polecenie udziela dostępu HTTP do klastra o nazwie Your-Hadoop-001.
