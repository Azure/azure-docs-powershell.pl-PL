# Get-AzExpressRouteCircuit

## STRESZCZENIe
Pobiera obwód ExpressRoute platformy Azure z platformy Azure.

## POLECENIA

```
Get-AzExpressRouteCircuit [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzExpressRouteCircuit** służy do pobierania obiektu obwodu ExpressRoute z subskrypcji. Zwracany obiekt obwodu może być wykorzystywany jako wejście dla innych poleceń cmdlet, które działają na obwodach ExpressRoute.
