# Remove-AzExpressRoutePort

## STRESZCZENIe
Usuwa ExpressRoutePort.

## POLECENIA

### ResourceNameParameterSet (domyślny)
```
Remove-AzExpressRoutePort -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzExpressRoutePort -InputObject <PSExpressRoutePort> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzExpressRoutePort -ResourceId <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzExpressRoutePort** usuwa ExpressRoutePort.

## Przykłady

### Przykład 1
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $PortName -ResourceGroupName $rg
```

Usuwa $PortName zasób ExpressRoutePort w $rg grupy zasobów w subskrypcji.

### Przykład 2
```powershell
PS C:\> Remove-AzExpressRoutePort -InputObject $erPort
```

Usuwa zasób ExpressRoutePort w polu Inputobject.

### Przykład 3
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $ResourceId $id
```

Usuwa zasób ExpressRoutePort o identyfikatorze ResourceId $id.
