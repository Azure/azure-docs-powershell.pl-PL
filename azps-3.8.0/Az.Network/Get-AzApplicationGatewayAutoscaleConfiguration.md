# Get-AzApplicationGatewayAutoscaleConfiguration

## STRESZCZENIe
Pobiera konfigurację skalowania bramy aplikacji.

## POLECENIA

```
Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzApplicationGatewayAutoscaleConfiguration umożliwia wyświetlenie** konfiguracji autoskalowania bramy aplikacji.

## Przykłady

### Przykład 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $autoscaleConfiguration = Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $autoscaleConfiguration.MinCapacity
```

Pierwsze polecenie uzyskuje bramę Application Gateway i zapisuje ją w zmiennej $gw.
Drugie polecenie wyodrębnia konfigurację autoskalowania z bramy aplikacji.
