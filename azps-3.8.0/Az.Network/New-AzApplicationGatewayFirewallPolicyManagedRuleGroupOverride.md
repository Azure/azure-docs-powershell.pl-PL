# New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride

## STRESZCZENIe
Tworzy wpis RuleGroupOverride w ManagedRuleSets dla zasad zapory.

## POLECENIA

```
New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName <String>
 -Rule <PSApplicationGatewayFirewallPolicyManagedRuleOverride[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
**Nowy-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride** tworzy wpis RuleGroupOverride w managedRuleSet dla zasad zapory.

## Przykłady

### Przykład 1
```powershell
PS C:\> $overrideEntry = New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName $ruleName -Rules $rule1,$rule2
```

Tworzy wpis RuleGroupOverride z nazwą grupy jako $ruleName i regułami $rule 1, $rule 2. Przypisuje te same $overrideEntry
