# Set-AzApplicationGatewayAutoscaleConfiguration

## STRESZCZENIe
Umożliwia zaktualizowanie konfiguracji autoskalowania bramy aplikacji.

## POLECENIA

```
Set-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway> -MinCapacity <Int32>
 [-MaxCapacity <Int32>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzApplicationGatewayAutoscaleConfiguration** modyfikuje istniejącą konfigurację autoskalowania bramy Application.

## Przykłady

### Przykład 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Set-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw -MinCapacity 5
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Pierwsze polecenie uzyskuje bramę Application Gateway i zapisuje ją w zmiennej $gw.
Drugie polecenie aktualizuje konfigurację autoskalowania od bramy aplikacji.
W trzecim poleceniu jest aktualizowana Brama Application Gateway na platformie Azure.
