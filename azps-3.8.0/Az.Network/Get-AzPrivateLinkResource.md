# Get-AzPrivateLinkResource

## STRESZCZENIe
Pobiera zasób link prywatny.

## POLECENIA

### ByPrivateLinkResourceId (domyślny)
```
Get-AzPrivateLinkResource -PrivateLinkResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzPrivateLinkResource** pobiera wszystkie zasoby linku do PrivateLinkResource.

## Przykłady

### Przykład 1
```
Get-AzPrivateLinkResource -PrivateLinkResourceId '/subscriptions/00000000-0000-0000-0000-000000000000/resourceGroups/TestResourceGroup/providers/Microsoft.Sql/servers/mySql'
```

W tym przykładzie podano listę wszystkich prywatnych zasobów linków nbelong do programu SQL Server o nazwie mySql.
