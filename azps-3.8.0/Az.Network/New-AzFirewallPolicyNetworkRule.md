# New-AzFirewallPolicyNetworkRule

## STRESZCZENIe
Utwórz nową regułę sieciową zasad zapory platformy Azure

## POLECENIA

```
New-AzFirewallPolicyNetworkRule -Name <String> [-Description <String>] -SourceAddress <String[]>
 -DestinationAddress <String[]> -DestinationPort <String[]> -Protocols <String[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **New-AzFirewallPolicyNetworkRule** tworzy regułę sieciową dla zasad zapory platformy Azure.

## Przykłady

### Przykład 1
```powershell
PS C:\> New-AzFirewallPolicyNetworkRule -Name NRC1 -Protocol "TCP" -SourceAddress "192.168.0.0/16" -DestinationAddress * -DestinationPort *
```

W tym przykładzie jest tworzona reguła sieci zawierająca adres źródłowy, protokół, adres docelowy i port docelowy.
