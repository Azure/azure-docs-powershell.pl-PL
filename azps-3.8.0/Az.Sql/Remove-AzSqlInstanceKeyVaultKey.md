# Remove-AzSqlInstanceKeyVaultKey

## STRESZCZENIe
Usuwa klucz magazynu kluczy z wystąpienia zarządzanego przez SQL

## POLECENIA

### AddAzureRmSqlManagedInstanceKeyVaultKeyDefaultParameterSet (domyślny)
```
Remove-AzSqlInstanceKeyVaultKey [-ResourceGroupName] <String> [-InstanceName] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyInputObjectParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-Instance] <AzureSqlManagedInstanceModel> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyResourceIdParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-InstanceResourceId] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Remove-AzSqlInstanceKeyVaultKey usuwa klucz magazynu kluczy z określonego wystąpienia zarządzanego. Pamiętaj, że uprawnienia wystąpienia zarządzanego przez SQL do magazynu kluczy nie są zmieniane. Aby zmienić uprawnienia, użyj ustawienia Set-AzKeyVaultAccessPolicy. Pamiętaj, że to polecenie cmdlet nie wprowadza żadnych zmian w magazynie kluczy. Aby usunąć klucz z magazynu kluczy, użyj przycisku Remove-AzureKeyVaultKey.
