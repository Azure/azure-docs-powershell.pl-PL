# Remove-AzDataLakeStoreItemAcl

## STRESZCZENIe
Czyści listę ACL pliku lub folderu w usłudze Data Lake Store.

## POLECENIA

```
Remove-AzDataLakeStoreItemAcl [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Default] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzDataLakeStoreItemAcl** czyści listę kontroli dostępu (ACL) pliku lub folderu w usłudze Data Lake Store.

## Przykłady

### Przykład 1: Usuwanie listy ACL z folderu
```
PS C:\>Remove-AzDataLakeStoreItemAcl -AccountName "ContosoADL" -Path "/"
```

To polecenie usuwa listę ACL katalogu głównego dla konta usługi ContosoADL.
