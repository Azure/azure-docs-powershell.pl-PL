# Remove-AzDataLakeStoreAccount

## STRESZCZENIe
Trwale usuwa konto Data Lake Store.

## POLECENIA

```
Remove-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzDataLakeStoreAccount** umożliwia trwałe usunięcie konta usługi Data Lake Store.

## Przykłady

### Przykład 1. Usuń konto w usłudze Data Lake Store
```
PS C:\>Remove-AzDataLakeStoreAccount -Name "ContosoADL"
```

To polecenie usuwa konto o nazwie ContosoADL z usługi Data Lake Store.
