# New-AzMapsAccountKey

## STRESZCZENIe
Generuje ponownie klucz konta.

## POLECENIA

### NameParameterSet (domyślny)
```
New-AzMapsAccountKey [-ResourceGroupName] <String> [-Name] <String> [-KeyName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
New-AzMapsAccountKey [-KeyName] <String> [-InputObject <PSMapsAccount>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
New-AzMapsAccountKey [-KeyName] <String> [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet New-AzMapsAccountKey ponownie generuje klucz interfejsu API dla konta usługi Azure Maps.

```powershell
PS C:\> New-AzMapsAccountKey -ResourceGroupName MyResourceGroup -Name MyAccount -KeyName Primary
```

Generuje ponownie podstawowy klucz interfejsu API dla konta Moje konto w grupie zasobu moja grupa zasobów.

Generuje ponownie pomocniczy klucz API dla określonego konta usługi Azure Maps.
