# New-AzCosmosDBGremlinIncludedPath

## STRESZCZENIe
Tworzy nowy obiekt typu PSIncludedPath. Może on zostać przekazany jako wartość parametru w parametrze Set-AzCosmosDBGremlinGraph.

## POLECENIA

```
New-AzCosmosDBGremlinIncludedPath [-Path <String>] [-Index <PSIndexes[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Obiekt odpowiadający IncludedPath API Gremlin.

## Przykłady

### Przykład 1
```powershell
PS C:\> $index1 = New-AzCosmosDBGremlinIncludedPathIndex -DataType String -Precision -1 -Kind Hash
New-AzCosmosDBGremlinIncludedPath -Path "/*" -Index $index1
Path Indexes
---- -------
/*   {Microsoft.Azure.Commands.CosmosDB.Models.PSIndexes}
```
