# Get-AzCosmosDBCassandraKeyspace

## STRESZCZENIe
Pobiera CosmosDB Cassandra.

## POLECENIA

### ByNameParameterSet (domyślny)
```
Get-AzCosmosDBCassandraKeyspace -ResourceGroupName <String> -AccountName <String> [-Name <String>] [-Detailed]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBCassandraKeyspace [-Name <String>] -InputObject <PSDatabaseAccount> [-Detailed]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzCosmosDBCassandraKeyspace** umożliwia utworzenie nowego lub zaktualizowanie istniejącego CosmosDB Cassandra miejsca na dysku.

## Przykłady

### Przykład 1
```powershell
PS C:\> Get-AzCosmosDBCassandraKeyspace -ResourceGroupName {resourceGroupName} -AccountName {accountName} -Name {name}

Name    Id   Resource
{name}  {id} {resourceObject}
```

Get-AzCosmosDBCassandraKeyspace pobiera właściwości istniejącego CassandraKeyspace. Możesz rozwinąć zasób, aby uzyskać _etag, _ts _rid właściwości.
