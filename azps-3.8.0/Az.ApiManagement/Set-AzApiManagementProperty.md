# Set-AzApiManagementProperty

## STRESZCZENIe
Modyfikuje Właściwość zarządzania interfejsem API.

## POLECENIA

```
Set-AzApiManagementProperty -Context <PsApiManagementContext> -PropertyId <String> [-Name <String>]
 [-Value <String>] [-Secret <Boolean>] [-Tag <String[]>] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzApiManagementProperty** modyfikuje Właściwość zarządzania interfejsem Azure API.

## Przykłady

### Przykład 1. Zmienianie tagów właściwości
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>$Tags = 'sdk', 'powershell'
PS C:\> Set-AzApiManagementProperty -Context $apimContext -PropertyId "Property11" -Tags $Tags -PassThru
```

Pierwsze polecenie powoduje przypisanie dwóch wartości do zmiennej $Tags.
Drugie polecenie modyfikuje właściwość o IDENTYFIKATORze Property11.
Polecenie przypisuje ciągi w $Tags jako znaczniki we właściwości.

### Przykład 2: modyfikowanie właściwości w celu uzyskania tajnej wartości
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Set-AzApiManagementProperty -Context $apimContext -PropertyId "Property12" -Secret $True -PassThru
```

To polecenie zmienia właściwość, która ma być szyfrowana.
