# Reset-AzAttestationPolicy

## STRESZCZENIe
Resetuje zasady od dzierżawy na platformie Azure Attestationn.}

## POLECENIA

### NameParameterSet
```
Reset-AzAttestationPolicy [-Name] <String> [-ResourceGroupName] <String> -Tee <String> [-Policy <String>]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Reset-AzAttestationPolicy [-ResourceId] <String> -Tee <String> [-Policy <String>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Reset-AzAttestationPolicy resetuje zasady zaświadczania zdefiniowane przez użytkownika od dzierżawy w zaświadczeniu platformy Azure.

## Przykłady

### Przykład 1
```powershell
PS C:\> Reset-AzAttestationPolicy -Name pshtest -ResourceGroupName psh-test-rg -Tee SgxEnclave
```

Przywróć domyślną zasadę dla *Pshtest* dostawcy zaświadczania dla typu tee *SgxEnclave*.
