# Get-AzAttestationPolicy

## STRESZCZENIe
Pobiera zasady od dzierżawy w zaświadczeniu platformy Azure.

## POLECENIA

### NameParameterSet
```
Get-AzAttestationPolicy [-Name] <String> [-ResourceGroupName] <String> -Tee <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzAttestationPolicy [-ResourceId] <String> -Tee <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Get-AzAttestationPolicy pobiera zasady od dzierżawy w zaświadczeniu platformy Azure.
