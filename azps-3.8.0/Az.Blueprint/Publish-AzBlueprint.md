# Publish-AzBlueprint

## STRESZCZENIe
Publikowanie nowej wersji programu plan.

## POLECENIA

```
Publish-AzBlueprint -Version <String> -ChangeNote <String> -Blueprint <PSBlueprint> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
Publikowanie nowej wersji definicji plany.

## Przykłady

### Przykład 1
```powershell
PS C:\> Publish-AzBlueprint -Blueprint $bp -Version 1.0
```
