# Remove-AzIntegrationAccountSchema

## STRESZCZENIe
Umożliwia usunięcie schematu konta integracji.

## POLECENIA

```
Remove-AzIntegrationAccountSchema -ResourceGroupName <String> -Name <String> -SchemaName <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzIntegrationAccountSchema** umożliwia usunięcie schematu konta integracji z grupy zasobów.
Określanie nazwy konta integracji, nazwy grupy zasobów i nazwy schematu.
Ten moduł obsługuje parametry dynamiczne.
Aby użyć parametru dynamicznego, wpisz go w poleceniu.
Aby wykryć nazwy parametrów dynamicznych, wpisz znak łącznika (-) po nazwie polecenia cmdlet, a następnie naciskaj klawisz Tab, aby przechodzić między dostępnymi parametrami.
W przypadku pominięcia wymaganego parametru szablonu polecenie cmdlet monituje o podanie wartości.

## Przykłady

### Przykład 1: usuwanie schematu konta integracji
```
PS C:\>Remove-AzIntegrationAccountSchema -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -SchemaName "IntegrationAccountSchema43"
```

To polecenie umożliwia usunięcie schematu konta integracji o nazwie IntegrationAccountSchema43.
