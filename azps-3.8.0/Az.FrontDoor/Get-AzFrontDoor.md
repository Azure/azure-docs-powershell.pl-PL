# Get-AzFrontDoor

## STRESZCZENIe
Pobierz przednią stację równoważenia obciążenia

## POLECENIA

```
Get-AzFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
Na karcie **Get-AzFrontDoor** cmdletGet są pobierane wszystkie istniejące drzwi przednie w bieżącej subskrypcji, które pasują do podanych informacji.
