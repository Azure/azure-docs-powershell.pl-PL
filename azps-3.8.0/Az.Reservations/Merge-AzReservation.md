# Merge-AzReservation

## STRESZCZENIe
Scala dwa `Reservation` s.

## POLECENIA

### Wiersz polecenia (domyślnie)
```
Merge-AzReservation -ReservationOrderId <Guid> -ReservationId <Guid[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Potokobject
```
Merge-AzReservation -Reservation <PSReservation[]> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Opis
Scal określony `Reservation` s w nowy `Reservation` . Oba `Reservation` scalane s muszą mieć te same właściwości.

## Przykłady

### Przykład 1
```
PS C:\> Merge-AzReservation -ReservationOrderId "1111aaaa-b1b2-c0c2-d0d2-00000fffff" -ReservationId "11111111-1111-1111-1111-1111111111","11111111-0000-0000-0000-1111111111"
```

Scalanie dwóch określonych liter `Reservation` s w jednej `Reservation`
