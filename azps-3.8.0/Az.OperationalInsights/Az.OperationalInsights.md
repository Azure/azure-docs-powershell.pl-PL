# AZ. OperationalInsights, moduł
## Opis
W tym temacie są wyświetlane tematy pomocy dotyczące poleceń cmdlet usługi Azure Operations Insights.

### [Disable-AzOperationalInsightsIISLogCollection](Disable-AzOperationalInsightsIISLogCollection.md)
Zatrzymuje zbieranie dzienników programu IIS na komputerach.

Zatrzymuje zbieranie niestandardowych dzienników z komputerów z systemem Linux.

### [Enable-AzOperationalInsightsLinuxCustomLogCollection](Enable-AzOperationalInsightsLinuxCustomLogCollection.md)
