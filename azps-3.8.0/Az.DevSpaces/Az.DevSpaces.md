# AZ. DevSpaces, moduł
## Opis
Polecenia do interakcji z kontrolerem usługi Azure DevSpaces.

## AZ. DevSpaces polecenia cmdlet
### [Get-AzDevSpacesController](Get-AzDevSpacesController.md)
Uzyskaj lub wystaw kontroler DevSpaces platformy Azure.

### [Nowe — AzDevSpacesController](New-AzDevSpacesController.md)
Utwórz nowy kontroler DevSpaces platformy Azure.

### [Remove-AzDevSpacesController](Remove-AzDevSpacesController.md)
Usuń kontroler DevSpaces.

### [Update-AzDevSpacesController](Update-AzDevSpacesController.md)
Zaktualizuj kontroler DevSpaces, aby dodać tagi.
