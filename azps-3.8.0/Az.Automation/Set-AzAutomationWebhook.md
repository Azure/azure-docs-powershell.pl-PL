# Set-AzAutomationWebhook

## STRESZCZENIe
Modyfikuje element webhook dla elementu Runbook automatyzacji.

## POLECENIA

```
Set-AzAutomationWebhook [-Name] <String> [-IsEnabled] <Boolean> [[-Parameters] <IDictionary>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzAutomationWebhook** modyfikuje element webhook dla elementu Runbook usługi Azure Automation.

## Przykłady

### Przykład 1. wyłączenie elementu webhook
```
PS C:\>Set-AzAutomationWebhook -Name "Webhook01" -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01" -IsEnabled $False
```

To polecenie wyłącza element webhook o nazwie Webhook01 na koncie usługi Automation o nazwie AutomationAccount01.
