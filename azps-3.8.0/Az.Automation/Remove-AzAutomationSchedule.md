# Remove-AzAutomationSchedule

## STRESZCZENIe
Usuwa harmonogram automatyzacji.

## POLECENIA

```
Remove-AzAutomationSchedule [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzAutomationSchedule** usuwa harmonogram z usługi Azure Automation.

## Przykłady

### Przykład 1. Usuń harmonogram
```
PS C:\>Remove-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -ResourceGroupName "ResourceGroup01"
```

To polecenie usuwa harmonogram o nazwie Schedule01 na koncie usługi Automation Contoso17 w grupie zasobów ResourceGroup01.
