# Remove-AzAutomationHybridWorkerGroup

## STRESZCZENIe
Usuwa grupę hybrydowych procesów roboczych z automatyzacji.

## POLECENIA

```
Remove-AzAutomationHybridWorkerGroup [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet Remove-AzAutomationHybridWorkerGroup usuwa grupę hybrydowego procesu roboczego z automatyzacji.

## Przykłady

### Przykład 1
To polecenie powoduje usunięcie hybrydowego procesu roboczego według nazwy.

```powershell
PS C:\> Remove-AzAutomationHybridWorkerGroup -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "GroupName" `
                                                  -Force
```
