# Get-AzDataShare

## SYNOPSIS
Uzyskaj informacje na temat udziałów danych.

## SKŁADNIA

### ByFieldsParameterSet (Domyślne)
```
Get-AzDataShare -ResourceGroupName <String> -AccountName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzDataShare -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzDataShare** pobiera informacje o udziałach danych w usłudze azure data share accoount.
Jeśli określisz nazwę udziału danych, to polecenie cmdlet pobiera informacje o tym udostępnie danych.
Jeśli nie określisz nazwy, to polecenie cmdlet pobiera informacje o wszystkich udziałach danych w koncie udostępniania danych platformy Azure.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie określonego udziału danych
```
PS C:\>Get-AzDataShare -ResourceGroupName "ADS" -AccountName "WikiAdsAccount" -Name "AdsShare"
Name                : AdsShare
Id                  : /subscriptions/f3ead1ff-d0ab-4cf4-9a5a-86f1661d4685/resourceGroups/ADS/providers/Microsoft.DataShare/accounts/WikiAdsAccount/shares/AdsShare
Type                : Microsoft.DataShare/shares
CreatedAt           : 6/11/2019 12:00:00 AM
CreatedBy           : Contoso ADS
ShareKind           : CopyBased
Description         : Example of description  
ProvisioningState   : Succeeded
Terms               : This should not be shared
```

To polecenie służy do wyświetlania informacji o udostępnianiu danych usługi AdsShare na koncie współużytkowania danych platformy Azure WikiAdsAccount i usłudze ADS grupy zasobów.
