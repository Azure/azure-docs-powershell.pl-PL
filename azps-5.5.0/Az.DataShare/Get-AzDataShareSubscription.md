# Get-AzDataShareSubscription

## SYNOPSIS
Otrzymuje informacje na temat udostępniania subskrypcji na koncie udostępniania danych.

## SKŁADNIA

### ByFieldsParameterSet (Domyślne)
```
Get-AzDataShareSubscription -ResourceGroupName <String> -AccountName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzDataShareSubscription -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzDataShareSubscription** dostarcza informacji o subskrypcji udostępniania na koncie udostępniania danych. Jeśli podano nazwę subskrypcji udostępniania, polecenie cmdlet udostępnia informacje o określonej subskrypcji udostępniania. W przeciwnym razie polecenie cmdlet udostępnia listę subskrypcji udostępniania na koncie udostępniania danych.

```
PS C:\> Get-AzDataShareSubscription -ResourceGroupName "ADS" -AccountName "WikiAds" -Name "AdsShareSubscription"
```

To polecenie zawiera informacje o udostępnianiu subskrypcji AdsShareSubscription w danych współużytkowania konta WikiAds.
