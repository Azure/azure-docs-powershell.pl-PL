# Remove-AzRelayHybridConnection

## SYNOPSIS
Usuwa połączenie hybrydowe z określonej przestrzeni nazw usługi HybridConnection.

## SKŁADNIA

```
Remove-AzRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzRelayHybridConnection** usuwa połączenie hybrydowe z określonej przestrzeni nazw przekazywania.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Remove-AzRelayHybridConnection -ResourceGroupName Default-ServiceBus-WestUS -Namespace TestNameSpace-Relay1 -Name TestHybridConnection
```

Usuwa połączenie hybrydowe z `TestHybridConnection` przestrzeni `TestNameSpace-Relay1` nazw.
