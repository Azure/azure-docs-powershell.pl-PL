# Get-AzUserAssignedIdentity

## SYNOPSIS
Pobiera tożsamości/tożsamości przypisane przez użytkownika.

## SKŁADNIA

### SuscriptionParameterSet (ustawienie domyślne)
```
Get-AzUserAssignedIdentity [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceGroupParameterSet
```
Get-AzUserAssignedIdentity -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Identyfikator **Get-AzUserAssignedIdentity** pobiera istniejące tożsamości przypisane do użytkownika.
