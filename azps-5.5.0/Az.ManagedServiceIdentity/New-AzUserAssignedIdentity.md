# New-AzUserAssignedIdentity

## SYNOPSIS
Tworzy nową tożsamość przypisaną do użytkownika lub aktualizuje istniejącą tożsamość przypisaną przez użytkownika.

## SKŁADNIA

```
New-AzUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzUserAssignedIdentity** tworzy nową tożsamość przypisaną do użytkownika. W przypadku korzystania z istniejącej tożsamości zaktualizowano ją.
Aby dodać tagi usługi Azure Resource Manager do tożsamości, użyj Set-AzResource cmdlet.
