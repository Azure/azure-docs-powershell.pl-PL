# Get-AzReservationCatalog

## SYNOPSIS
Pobierz katalog dostępnych rezerwacji

## SKŁADNIA

```
Get-AzReservationCatalog [-SubscriptionId <Guid>] -ReservedResourceType <String> [-Location <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Pobierz regiony i jednostki SKU dostępne do zakupu w ramach zarezerwowanego wystąpienia dla określonej subskrypcji platformy Azure.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Get-AzReservationCatalog -ReservedResourceType VirtualMachines -Location westus
```

Pobierz katalog VirtualMachines w języku zachódus dla domyślnej subskrypcji

### Przykład 2
```
PS C:\> Get-AzReservationCatalog -SubscriptionId "1111aaaa-b1b2-c0c2-d0d2-00000fffff" -ReservedResourceType SuseLinux
```

Uzyskiwanie katalogu SuseLinux dla określonej subskrypcji
