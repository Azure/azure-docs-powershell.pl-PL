# Remove-AzBatchAccount

## SYNOPSIS
Usuwa konto usługi Batch.

## SKŁADNIA

```
Remove-AzBatchAccount [-AccountName] <String> [[-ResourceGroupName] <String>] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzBatchAccount** usuwa konto usługi Azure Batch.
To polecenie cmdlet wyświetli monit przed usunięciem konta, chyba że określisz parametr *Force.*

## PRZYKŁADY

### Przykład 1. Usuwanie konta wsadowego
```
PS C:\>Remove-AzBatchAccount -AccountName "pfuller"
```

To polecenie usuwa konto batch o nazwie Pfuller.
To polecenie wyświetla monit o potwierdzenie przed usunięciem konta.
