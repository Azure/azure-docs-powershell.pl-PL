# Remove-AzBatchApplication

## SYNOPSIS
Usuwa aplikację z konta wsadowego.

## SKŁADNIA

```
Remove-AzBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [-ApplicationName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzBatchApplication** usuwa aplikację z konta usługi Azure Batch.

## PRZYKŁADY

### Przykład 1. Usuwanie aplikacji z konta wsadowego
```
PS C:\>Remove-AzBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationName "Litware"
```

To polecenie usuwa aplikację Litware z konta ContosoBatch.
To polecenie kończy się niepowodzeniem, jeśli aplikacja zawiera jakiekolwiek pakiety.
