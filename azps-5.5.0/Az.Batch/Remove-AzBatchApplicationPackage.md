# Remove-AzBatchApplicationPackage

## SYNOPSIS
Usuwa rekord pakietu aplikacji i plik binarny.

## SKŁADNIA

```
Remove-AzBatchApplicationPackage [-AccountName] <String> [-ResourceGroupName] <String>
 [-ApplicationName] <String> [-ApplicationVersion] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzBatchApplicationPackage** usuwa rekord pakietu aplikacji i plik binarny z konta usługi Azure Batch.

## PRZYKŁADY

### Przykład 1. Usuwanie pakietu aplikacji z konta wsadowego
```
PS C:\>Remove-AzBatchApplicationPackage -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationName "litware" -ApplicationVersion "1.0"
```

To polecenie usuwa wersję 1.0 aplikacji Litware z konta ContosoBatchGroup.
Polecenie usuwa zarówno rekord pakietu, jak i obiekt blob, który zawiera plik binarny pakietu.
