# Get-AzBlockchainConsortium

## SYNOPSIS
Zawiera listę dostępnych przedsiębiorstw w ramach subskrypcji.

## SKŁADNIA

```
Get-AzBlockchainConsortium -Location <String> [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## OPIS
Zawiera listę dostępnych przedsiębiorstw w ramach subskrypcji.

## PRZYKŁADY

### Przykład 1. Uzyskaj więcej organizacji.
```powershell
PS C:\> Get-AzBlockchainConsortium -Location eastus

```

To polecenie wyświetla listę organizacji w ramach subskrypcji dla określonej lokalizacji.
