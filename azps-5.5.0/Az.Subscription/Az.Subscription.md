# Moduł az.subskrypcji
## Opis
Tematy w tej sekcji dokumentuj polecenia cmdlet programu Azure PowerShell dla subskrypcji platformy Azure w ramach usługi Azure Resource Manager (ARM). Polecenia cmdlet istnieją w przestrzeni nazw Microsoft.Azure.Commands.Subscription.

## Az.Subscription Cmdlet
### [Get-AzSubscriptionAlias](Get-AzSubscriptionAlias.md)
Pobiera szczegóły aliasu subskrypcji

### [New-AzSubscriptionAlias](New-AzSubscriptionAlias.md)
Tworzy nowy alias i subskrypcję

### [Remove-AzSubscriptionAlias](Remove-AzSubscriptionAlias.md)
Usuwa alias subskrypcji.

### [Update-AzSubscription](Update-AzSubscription.md)
Aktualizacja subskrypcji platformy Azure
