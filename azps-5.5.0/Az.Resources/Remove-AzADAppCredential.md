# Remove-AzADAppCredential

## SYNOPSIS
Usuwa poświadczenia z aplikacji.

## SKŁADNIA

### ApplicationObjectIdWithKeyIdParameterSet (Domyślne)
```
Remove-AzADAppCredential -ObjectId <String> [-KeyId <Guid>] [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ApplicationIdWithKeyIdParameterSet
```
Remove-AzADAppCredential -ApplicationId <Guid> [-KeyId <Guid>] [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ApplicationDisplayNameParameterSet
```
Remove-AzADAppCredential -DisplayName <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ApplicationObjectWithKeyIdParameterSet
```
Remove-AzADAppCredential [-KeyId <Guid>] -ApplicationObject <PSADApplication> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
To Remove-AzADAppCredential cmdlet może służyć do usuwania klucza poświadczeń z aplikacji w przypadku naruszenia zabezpieczeń lub w ramach wygasania klucza poświadczeń.
Aplikacja jest identyfikowana przez podanie identyfikatora obiektu lub identyfikatora aplikacji.
Poświadczenia do usunięcia są identyfikowane za pomocą identyfikatora klucza.

## PRZYKŁADY

### Przykład 1. Usuwanie określonego poświadczenia z aplikacji

```powershell
PS C:\> Remove-AzADAppCredential -ObjectId 7663d3fb-6f86-4352-9e6d-cf9d50d5ee82 -KeyId 9044423a-60a3-45ac-9ab1-09534157ebb
```

Usuwa poświadczenia o identyfikatorze klucza "9044423a-60a3-45ac-9ab1-09534157ebb" z aplikacji o identyfikatorze obiektu '7663d3fb-6f86-4352-9e6d-cf9d50d5ee82'.

### Przykład 2. Usuwanie wszystkich poświadczeń z aplikacji

```powershell
PS C:\> Remove-AzADAppCredential -ApplicationId 4589cd6b-3d79-4bb4-93b8-a0b99f3bfc58
```

Usuwa wszystkie poświadczenia z aplikacji o identyfikatorze aplikacji '4589cd6b-3d79-4bb4-93b8-a0b99f3bfc58'.

### Przykład 3. Usuwanie wszystkich poświadczeń za pomocą rur

```powershell
PS C:\> Get-AzADApplication -ObjectId 7663d3fb-6f86-4352-9e6d-cf9d50d5ee82 | Remove-AzADAppCredential
```

Pobiera aplikację z identyfikatorem obiektu '7663d3fb-6f86-4352-9e6d-cf9d50d5ee82" i potokami, które są do polecenia cmdlet programu Remove-AzADAppCredential, i usuwa wszystkie poświadczenia z tej aplikacji.
