# Get-AzRoleDefinition

## SYNOPSIS
Lista wszystkich ról RBAC platformy Azure, które są dostępne do przypisania.

## SKŁADNIA

### Zestaw RoleDefinitionNameParameterSet (domyślny)
```
Get-AzRoleDefinition [[-Name] <String>] [-Scope <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### RoleDefinitionIdParameterSet
```
Get-AzRoleDefinition -Id <Guid> [-Scope <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### RoleDefinitionCustomParameterSet
```
Get-AzRoleDefinition [-Scope <String>] [-Custom] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Aby wyświetlić Get-AzRoleDefinition z określoną nazwą roli, użyj tego polecenia.
Aby sprawdzić poszczególne operacje, do których rola udziela dostępu, przejrzyj właściwości Akcji i NotActions roli.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Get-AzRoleDefinition -Name Reader
```

Uzyskiwanie definicji ról czytnika

### Przykład 2
```
PS C:\> Get-AzRoleDefinition
```

Lista wszystkich definicji ról RBAC
