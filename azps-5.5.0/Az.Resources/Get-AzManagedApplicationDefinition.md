# Get-AzManagedApplicationDefinition

## SYNOPSIS
Pobiera definicje aplikacji zarządzanych

## SKŁADNIA

### GetByNameAndResourceGroup (domyślne)
```
Get-AzManagedApplicationDefinition [-Name <String>] -ResourceGroupName <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetById
```
Get-AzManagedApplicationDefinition -Id <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzManagedApplicationDefinition** pobiera zarządzane definicje aplikacji

## PRZYKŁADY

### Przykład 1. Uzyskiwanie wszystkich definicji aplikacji zarządzanych w grupie zasobów
```
PS C:\>Get-AzManagedApplicationDefinition -ResourceGroupName "MyRG"
```

To polecenie pobiera definicje aplikacji zarządzanych w grupie zasobów "MyRG"

### Przykład 2. Uzyskiwanie definicji aplikacji zarządzanej
```
PS C:\>Get-AzManagedApplicationDefinition -ResourceGroupName "MyRG" -Name "myManagedAppDef"
```

To polecenie pobiera definicję aplikacji zarządzanej "myManagedAppDef" w grupie zasobów "MyRG"
