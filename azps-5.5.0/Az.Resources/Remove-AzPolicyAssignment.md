# Remove-AzPolicyAssignment

## SYNOPSIS
Usuwa przypisanie zasad.

## SKŁADNIA

### NameParameterSet (Domyślne)
```
Remove-AzPolicyAssignment -Name <String> [-Scope <String>] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSet
```
Remove-AzPolicyAssignment -Id <String> [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzPolicyAssignment -InputObject <PsPolicyAssignment> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzPolicyAssignment** usuwa przypisanie określonych zasad.

## PRZYKŁADY

### Przykład 1. Usuwanie przypisania zasad według nazwy i zakresu
```
PS C:\> $ResourceGroup = Get-AzResourceGroup -Name 'ResourceGroup11'
PS C:\> Remove-AzPolicyAssignment -Name 'PolicyAssignment07' -Scope $ResourceGroup.ResourceId -Confirm
```

Pierwsze polecenie pobiera grupę zasobów o nazwie ResourceGroup11 przy użyciu Get-AzResourceGroup cmdlet.
Polecenie przechowuje ten obiekt w $ResourceGroup zmienną.
Drugie polecenie usuwa przypisanie zasad o nazwie PolicyAssignment07, które zostało przypisane na poziomie grupy zasobów.
Właściwość **ResourceId** $ResourceGroup identyfikuje grupę zasobów.

### Przykład 2. Usuwanie przypisania zasad według identyfikatora
```
PS C:\> $ResourceGroup = Get-AzResourceGroup -Name 'ResourceGroup11' 
PS C:\> $PolicyAssignment = Get-AzPolicyAssignment -Name 'PolicyAssignment07' -Scope $ResourceGroup.ResourceId
PS C:\> Remove-AzPolicyAssignment -Id $PolicyAssignment.ResourceId -Confirm
```

Pierwsze polecenie pobiera grupę zasobów o nazwie ResourceGroup11, a następnie przechowuje ten obiekt w $ResourceGroup zmienną.
Drugie polecenie pobiera przydział zasad na poziomie grupy zasobów, a następnie przechowuje je w $PolicyAssignment zmienną.
Właściwość **ResourceId** właściwości $ResourceGroup identyfikuje grupę zasobów.
Ostatnie polecenie usuwa przypisanie zasad, które określa $PolicyAssignment **ResourceId.**
