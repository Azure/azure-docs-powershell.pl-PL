# Get-AzProviderFeature

## SYNOPSIS
Pobiera informacje o funkcjach dostawców platformy Azure.

## SKŁADNIA

### ListAvailableParameterSet (Domyślne)
```
Get-AzProviderFeature [-ProviderNamespace <String>] [-ListAvailable] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetFeature
```
Get-AzProviderFeature -ProviderNamespace <String> -FeatureName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzProviderFeature** pobiera nazwę funkcji, nazwę dostawcy i stan rejestracji funkcji dostawcy platformy Azure.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie wszystkich dostępnych funkcji
```
PS C:\>Get-AzProviderFeature -ListAvailable
```

To polecenie otrzymuje dostęp do wszystkich dostępnych funkcji.

### Przykład 2. Uzyskiwanie informacji o określonej funkcji
```
PS C:\>Get-AzProviderFeature -FeatureName "AllowPreReleaseRegions" -ProviderNamespace "Microsoft.Compute"
```

To polecenie pobiera informacje dotyczące funkcji o nazwie AllowPreReleaseRegions dla określonego dostawcy.
