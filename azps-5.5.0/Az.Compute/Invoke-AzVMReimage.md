# Invoke-AzVMReimage

## SYNOPSIS
Ponownie przeprojektuj maszynę wirtualną platformy Azure.

## SKŁADNIA

```
Invoke-AzVMReimage [-ResourceGroupName] <String> [-VMName] <String> [-TempDisk] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Invoke-AzVMReimage** ponownie projektuje maszynę wirtualną platformy Azure.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Invoke-AzVMReimage -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

To polecenie ponownie dodaje maszynę wirtualną o nazwie VirtualMachine07 w grupie ResourceGroup11.
