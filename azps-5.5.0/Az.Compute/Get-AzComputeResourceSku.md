# Get-AzComputeResourceSku

## SYNOPSIS
Lista wszystkich jednostki SKU zasobu obliczeniowego

## SKŁADNIA

```
Get-AzComputeResourceSku [[-Location] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Lista wszystkich jednostki SKU zasobu obliczeniowego

## PRZYKŁADY

### Przykład 1
```
PS C:\> Get-AzComputeResourceSku "westus";
```

Lista wszystkich jednostki SKU zasobów obliczeniowych w regionie Stanów Zjednoczonych Zachód
