# Get-AzVMImageSku

## SYNOPSIS
Pobiera jednostki SKU WIRTUALNEJ obrazów.

## SKŁADNIA

```
Get-AzVMImageSku -Location <String> -PublisherName <String> -Offer <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzVMImageSku** pobiera jednostki SKU VMImage.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie jednostki SKU VMImage
```
PS C:\> Get-AzVMImageSku -Location "Central US" -PublisherName "Fabrikam" -Offer "LinuxServer"
```

To polecenie pobiera jednostki SKU dla określonego wydawcy i ofertę.
