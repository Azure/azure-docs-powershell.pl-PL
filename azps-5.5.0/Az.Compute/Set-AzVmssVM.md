# Set-AzVmssVM

## SYNOPSIS
Modyfikuje stan wystąpienia maszyny wirtualnej.

## SKŁADNIA

### DefaultParameters (domyślne)
```
Set-AzVmssVM [-ResourceGroupName] <String> [-VMScaleSetName] <String> [-InstanceId] <String> [-Reimage]
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```


## OPIS
Polecenie **cmdlet Set-AzVmssVM** modyfikuje stan wystąpienia zestawu skal maszyny wirtualnej (VIRTUAL Machine Scale Set, MACHINESS).

## PRZYKŁADY

### Przykład 1

Modyfikuje stan wystąpienia maszyny wirtualnej. (wygenerowane automatycznie)

```powershell <!-- Aladdin Generated Example --> 
Set-AzVmssVM -InstanceId <String> -Reimage -ResourceGroupName myresourcegroup -VMScaleSetName 'VMSS001'
```
