# Get-AzGalleryImageDefinition

## SYNOPSIS
Pobierz definicje obrazów z galerii lub wybierz galerię.

## SKŁADNIA

### DefaultParameters (domyślne)
```
Get-AzGalleryImageDefinition [-ResourceGroupName] <String> [-GalleryName] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameters
```
Get-AzGalleryImageDefinition [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Pobierz definicje obrazów galerii lub pobierz je z listy.
