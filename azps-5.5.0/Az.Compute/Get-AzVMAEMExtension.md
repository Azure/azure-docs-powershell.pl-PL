# Get-AzVMAEMExtension

## SYNOPSIS
Pobiera informacje o rozszerzeniu AEM.

## SKŁADNIA

```
Get-AzVMAEMExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-Status]
 [[-OSType] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzVMAEMExtension** pobiera informacje o rozszerzeniu Azure Enhanced Monitoring (AEM).

## PRZYKŁADY

### Przykład 1. Uzyskiwanie rozszerzenia AEM
```
PS C:\> Get-AzVMAEMExtension -ResourceGroupName "ResourceGroup11" -VMName "contoso-server"
```

To polecenie pobiera informacje dotyczące rozszerzenia AEM dla maszyny wirtualnej o nazwie contoso-server.
