# Get-AzVmssSku

## SYNOPSIS
Pobiera dostępne jednostki SKU dla maszyny wirtualnej.

## SKŁADNIA

```
Get-AzVmssSku [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzVmssSku** pobiera dostępne jednostki SKU dla zestawu skal maszyny wirtualnej (VIRTUAL Machine Scale Set).

## PRZYKŁADY

### Przykład 1. Uzyskiwanie wszystkich dostępnych wersji SKU z usługi MASZYNY.WIRTUALNE
```
PS C:\> Get-AzVmssSku -ResourceGroupName "ContosoGroup" -VMScaleSetName "ContosoVMSS"
```

To polecenie pobiera wszystkie dostępne jednostki SKU z usług MASZYNY WIRTUALNE o nazwie ContosoVMSS, które należą do grupy zasobów o nazwie ContosoGroup.
