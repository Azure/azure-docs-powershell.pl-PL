# Set-AzSnapshotImageReference

## SYNOPSIS
Ustawia właściwości odwołania do obrazu w obiekcie migawki.

## SKŁADNIA

```
Set-AzSnapshotImageReference [-Snapshot] <PSSnapshot> [[-Id] <String>] [[-Lun] <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzSnapshotImageReference** ustawia właściwości odwołania do obrazu w obiekcie migawki.

## PRZYKŁADY

### Przykład 1
```
PS C:\> $snapshotconfig = New-AzSnapshotConfig -SnapshotSizeGB 10 -AccountType PremiumLRS -OsType Windows -CreateOption FromImage;
PS C:\> $image = '/subscriptions/0000000-0000-0000-0000-000000000000/resourceGroups/ResourceGroup01/providers/Microsoft.Compute/images/TestImage123';        
PS C:\> $snapshotconfig = Set-AzSnapshotImageReference -Snapshot $snapshotconfig -Id $image -Lun 0;
PS C:\> New-AzSnapshot -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01' -Snapshot $snapshotconfig;
```

Pierwsze polecenie tworzy obiekt migawki lokalnej o rozmiarze 10 GB w Premium_LRS typu konta magazynu.  Ustawia także typ systemu operacyjnego Windows.
Drugie polecenie ustawia identyfikator obrazu i jednostkę logiczną o numerze 0 obiektu migawki.
Ostatnie polecenie pobiera obiekt migawki i tworzy migawkę o nazwie "Migawka01" w grupie zasobów "Grupa Zasobów01".
