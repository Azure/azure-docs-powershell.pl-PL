# Set-AzVMBginfoExtension

## SYNOPSIS
Dodaje rozszerzenie BGInfo do maszyny wirtualnej.

## SKŁADNIA

```
Set-AzVMBginfoExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name <String>]
 [-TypeHandlerVersion <String>] [-Location <String>] [-DisableAutoUpgradeMinorVersion] [-ForceRerun <String>]
 [-NoWait] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzVMBGInfoExtension** dodaje rozszerzenie BGInfo do maszyny wirtualnej.

## PRZYKŁADY

### Przykład 1. Dodawanie rozszerzenia BGInfo dla maszyny wirtualnej
```
PS C:\> Set-AzVMBgInfoExtension -ResourceGroupName "ContosoRG" -VMName "ContosoVM" -Name "ExtensionName" -TypeHandlerVersion "2.1" -Location "West Europe"
```

To polecenie dodaje rozszerzenie BGInfo do maszyny wirtualnej o nazwie ContosoVM.
To polecenie określa grupę zasobów i lokalizację maszyny wirtualnej.
To polecenie określa nazwę i wersję rozszerzenia.
