# Remove-AzVMSecret

## SYNOPSIS
Usuwa (a) tajemnice z obiektu maszyny wirtualnej.

## SKŁADNIA

```
Remove-AzVMSecret [-VM] <PSVirtualMachine> [[-SourceVaultId] <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie Remove-AzVMSecret usuwa (a) sekrety z obiektu maszyny wirtualnej.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Get-AzVM -ResourceGroupName "rg1" -Name "vm1" | Remove-AzVMSecret | Update-AzVM
```

Usuwa wszystkie tajemnice z maszyny wirtualnej "maszyny wirtualnej "maszyny wirtualnej 1" w grupie zasobów "rg1" i zaktualizuj maszyny wirtualnej
