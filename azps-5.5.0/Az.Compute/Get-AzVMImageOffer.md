# Get-AzVMImageOffer

## SYNOPSIS
Pobiera typy ofert dla usługi VMImage.

## SKŁADNIA

```
Get-AzVMImageOffer -Location <String> -PublisherName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzVMImageOffer** pobiera typy ofert WIRTUALNEJImage.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie typów ofert dla wydawcy
```
PS C:\> Get-AzVMImageOffer -Location "Central US" -PublisherName "Fabrikam"
```

To polecenie pobiera typy ofert dla określonego wydawcy w regionie Środkowych Stanów Zjednoczonych.
