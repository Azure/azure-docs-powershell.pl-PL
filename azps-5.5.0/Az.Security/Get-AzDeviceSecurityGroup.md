# Get-AzDeviceSecurityGroup

## SYNOPSIS
Uzyskaj grupę zabezpieczeń urządzeń (zabezpieczenia centrum IoT)

## SKŁADNIA

### ResourceIdScope (domyślna)
```
Get-AzDeviceSecurityGroup -HubResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdLevelResource
```
Get-AzDeviceSecurityGroup -HubResourceId <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie Get-AzDeviceSecurityGroup zwraca grupę zabezpieczeń urządzenia zdefiniowaną w rozwiązaniu zabezpieczeń iot
