# New-AzIotSecuritySolutionRecommendationConfigurationObject

## SYNOPSIS
Utwórz nową konfigurację zaleceń dla rozwiązania zabezpieczającego iot

## SKŁADNIA

```
New-AzIotSecuritySolutionRecommendationConfigurationObject -RecommendationType <String> -Enabled <Boolean>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
AzIotSecuritySolutionRecommendationConfigurationObject tworzy nowy obiekt konfiguracji zalecenia dla rozwiązania zabezpieczeń iot.
W ten sposób jest skonfigurowany stan zalecenia, niezależnie od tego, czy zostało włączone, czy wyłączone.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> New-AzIotSecuritySolutionRecommendationConfigurationObject -RecommendationType "IoT_ACRAuthentication" -Enabled $false

RecommendationType: "IoT_ACRAuthentication"
Name: "Service prinicpal not used with ACR repository"
Status: "Disabled"
```

Utwórz nową konfigurację rekomendacji dla typu IoT_ACRAuthentication "IoT_ACRAuthentication" ze statusem ustawionym na wyłączenie
