# Disable-AzHDInsightMonitoring

## SYNOPSIS
Wyłącza monitorowanie w klastrze HDInsight i odpowiednie dzienniki przestaną napływać do obszaru roboczego monitorowania określonego podczas włączania.

## SKŁADNIA

```
Disable-AzHDInsightMonitoring [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Disable-AzHDInsightMonitoring** wyłącza monitorowanie w klastrze usługi Azure HDInsight.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster

True
```

Monitorowanie zostanie wyłączone w klastrze HDInsight, a odpowiednie dzienniki przestaną napływać do obszaru roboczego monitorowania.

### Przykład 2
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg

True
```

Monitorowanie zostanie wyłączone w klastrze HDInsight, a odpowiednie dzienniki przestaną napływać do obszaru roboczego monitorowania.
