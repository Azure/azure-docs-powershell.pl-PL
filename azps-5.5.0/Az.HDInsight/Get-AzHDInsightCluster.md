# Get-AzHDInsightCluster

## SYNOPSIS
Pobiera i wyświetla wszystkie klastry usługi Azure HDInsight skojarzone z bieżącą subskrypcją lub określoną grupą zasobów albo pobiera określony klaster.

## SKŁADNIA

```
Get-AzHDInsightCluster [[-ResourceGroupName] <String>] [[-ClusterName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzHDInsightCluster** wyświetla listę klastrów usługi Azure HDInsight dla bieżącej subskrypcji.
Użyj *parametru ClusterName,* aby uzyskać szczegółowe informacje dotyczące określonego klastra.

## PRZYKŁADY

### Przykład 1. Lista wszystkich grup usługi Azure HDInsight
```
PS C:\>Get-AzHDInsightCluster
```

To polecenie zawiera listę wszystkich klastrów usługi Azure HDInsight.
