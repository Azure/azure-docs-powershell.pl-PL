# Set-AzHDInsightClusterSize

## SYNOPSIS
Ustawia liczbę węzłów pracownicy w określonym klastrze.

## SKŁADNIA

```
Set-AzHDInsightClusterSize [-ClusterName] <String> [-TargetInstanceCount] <Int32> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzHDInsightClusterSize** ustawia liczbę węzłów Worker w określonym klastrze usługi Azure HDInsight.

## PRZYKŁADY

### Przykład 1. Ustawianie rozmiaru określonego klastra
```
PS C:\>Set-AzHDInsightClusterSize -ClusterName "your-hadoop-001" -TargetInstanceCount 6
```

To polecenie ustawia rozmiar klastra o nazwie Twoja-hadoop-001.
