# Get-AzHDInsightJob

## SYNOPSIS
Pobiera listę zadań z klastra i wyświetla je w odwrotnej kolejności chronologicznej lub pobiera określone zadanie.

## SKŁADNIA

```
Get-AzHDInsightJob [-ClusterName] <String> [-HttpCredential] <PSCredential> [[-JobId] <String>]
 [-NumOfJobs <Int32>] [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzHDInsightJob** pobiera ostatnie zadania dla określonego klastru usługi Azure HDInsight w odwrotnej kolejności chronologicznej z najnowszym zadaniem na początku listy.
Uzyskaj określone zadanie, podając parametr *JobId.*

## PRZYKŁADY

### Przykład 1. Uzyskiwanie ostatnich zadań dla określonego klastra usługi Azure HDInsight
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

# Hive job details
PS C:\> $statusFolder = "tempStatusFolder/"
PS C:\> $query = "SHOW TABLES"

PS C:\> New-AzHDInsightHiveJobDefinition -StatusFolder $statusFolder `
            -Query $query `
        | Start-AzHDInsightJob -ClusterName $clusterName `
            -ClusterCredential $clusterCreds `
        | Get-AzHDInsightJob -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

To polecenie pobiera wszystkie ostatnie zadania dla klastra o nazwie Twoja-hadoop-001.
