# Use-AzHDInsightCluster

## SYNOPSIS
Wybiera klaster, który ma być używany z Invoke-RmAzureHDInsightHiveJob cmdlet.

## SKŁADNIA

```
Use-AzHDInsightCluster [-ClusterName] <String> [-HttpCredential] <PSCredential> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Use-AzHDInsightCluster** wybiera klaster usługi Azure HDInsight dla polecenia cmdlet Invoke-AzHDInsightHiveJob, które będzie używać do przesyłania zadań w usłudze HdInsight platformy Azure.

## PRZYKŁADY

### Przykład 1. Wybieranie klastrów dla przesyłania zapytania Przez użytkownika
```
PS C:\># Cluster info
PS C:\>$clusterName = "your-hadoop-001"
PS C:\>$clusterCreds = Get-Credential

PS C:\>Use-AzHDInsightCluster `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

To polecenie wybiera klaster dla przesłania zapytania Przez użytkownika.
