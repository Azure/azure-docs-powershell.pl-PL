# Remove-AzAutoscaleSetting

## SYNOPSIS
Usuwa ustawienie autoskalowania.

## SKŁADNIA

```
Remove-AzAutoscaleSetting -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzAutoscaleSetting** usuwa ustawienie autoskalowania.
Należy określić nazwę ustawienia i nazwę grupy zasobów, do której jest ono przypisane.
To polecenie cmdlet implementuje wzorzec ShouldProcess, czyli może wymagać potwierdzenia od użytkownika przed jego utworzeniem, zmodyfikowaniem lub usunięciem.

## PRZYKŁADY

### Przykład 1

Usuwa ustawienie automatycznego skalowania. (wygenerowane automatycznie)

```powershell <!-- Aladdin Generated Example --> 
Remove-AzAutoscaleSetting -Name 'LogAlertRule1' -ResourceGroupName MyResourceGroup
```
