# Remove-AzAlertRule

## SYNOPSIS
Usuwa regułę alertu.

## SKŁADNIA

```
Remove-AzAlertRule -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzAlertRule** usuwa regułę alertu.
Należy określić nazwę reguły alertu i grupy zasobów, do której jest przypisana.
To polecenie cmdlet implementuje wzorzec ShouldProcess, czyli może wymagać potwierdzenia od użytkownika przed jego utworzeniem, zmodyfikowaniem lub usunięciem.

## PRZYKŁADY

### Przykład 1. Usuwanie reguły alertu
```
PS C:\>Remove-AzAlertRule -ResourceGroup "Default-Web-CentralUS" -Name "myalert-7da64548-214d-42ca-b12b-b245bb8f0ac8"
RequestId                                                                                                    StatusCode
---------                                                                                                    ----------
2c6c159b-0e73-4a01-a67b-c32c1a0008a3                                                                                 OK
```

To polecenie usuwa regułę alertu o nazwie myalert-7da64548-214d-42ca-b12b-b245bb8f0ac8 w grupie zasobów Default-Web-CentralUS.
