# New-AzMetricFilter

## SYNOPSIS
Tworzy filtr wymiarów metrycznych, który może być używany do tworzenia zapytań dotyczących metryk.

## SKŁADNIA

```
New-AzMetricFilter [-Dimension] <String> [-Operator] <String> [-Value] <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzMetricFilter** tworzy filtr wymiarowania metryczne, który może być używany do tworzenia zapytań dotyczących metryk.

## PRZYKŁADY

### Przykład 1. Tworzenie filtru wymiarów metrycznych
```
PS C:\>New-AzMetricFilter -Dimension City -Operator eq -Value "Seattle","New York"
```

To polecenie tworzy filtr wymiarów metrycznych w formacie "Miasto eq "Seattle" lub Miasto eq 'Nowy Jork".
