# Disable-AzOperationalInsightsLinuxCustomLogCollection

## SYNOPSIS
Zatrzymuje zbieranie niestandardowych dzienników z komputerów z systemem Linux.

## SKŁADNIA

### ByWorkspaceName (Domyślna)
```
Disable-AzOperationalInsightsLinuxCustomLogCollection [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByWorkspaceObject
```
Disable-AzOperationalInsightsLinuxCustomLogCollection [-Workspace] <PSWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Disable-AzOperationalInsightsLinuxCustomLogCollection** zatrzymuje kolekcję niestandardowych dzienników z połączonych komputerów z systemem Linux w obszarze roboczym.
