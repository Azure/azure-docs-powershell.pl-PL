# Remove-AzApiManagementNamedValue

## SYNOPSIS
Usuwa wartość nazwaną zarządzania interfejsem API.

## SKŁADNIA

```
Remove-AzApiManagementNamedValue -Context <PsApiManagementContext> -NamedValueId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzApiManagementNamedValue** usuwa wartość nazwaną usługi Azure API **Management.**

## PRZYKŁADY

### Przykład 1. Usuwanie nazwanej wartości
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementNamedValue -Context $apimContext -NamedValueId "Property11" -PassThru
```

To polecenie usuwa nazwaną wartość, która ma właściwość ID11.
