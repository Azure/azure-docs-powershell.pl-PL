# Get-AzApiManagementTenantSyncState

## SYNOPSIS
Pobiera stan ostatniej synchronizacji między bazą danych konfiguracji a repozytorium Git.

## SKŁADNIA

```
Get-AzApiManagementTenantSyncState -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzApiManagementTenantSyncState** pobiera stan ostatniej synchronizacji między bazą danych konfiguracji a repozytorium Git.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie stanu ostatniej synchronizacji
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantSyncState -Context $apimContext
```

To polecenie otrzymuje stan ostatniej synchronizacji między bazą danych konfiguracji a repozytorium Git.
