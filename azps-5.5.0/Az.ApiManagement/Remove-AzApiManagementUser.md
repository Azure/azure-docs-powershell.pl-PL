# Remove-AzApiManagementUser

## SYNOPSIS
Usuwa istniejącego użytkownika.

## SKŁADNIA

```
Remove-AzApiManagementUser -Context <PsApiManagementContext> -UserId <String> [-DeleteSubscriptions]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzApiManagementUser** usuwa istniejącego użytkownika.

## PRZYKŁADY

### Przykład 1. Usuwanie użytkownika
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementUser -Context $apimContext -UserId "0123456789" -Force
```

To polecenie usuwa istniejącego użytkownika.
