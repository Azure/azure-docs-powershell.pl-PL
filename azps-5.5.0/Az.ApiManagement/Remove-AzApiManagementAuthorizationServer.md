# Remove-AzApiManagementAuthorizationServer

## SYNOPSIS
Usuwa serwer autoryzacji.

## SKŁADNIA

```
Remove-AzApiManagementAuthorizationServer -Context <PsApiManagementContext> -ServerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzApiManagementAuthorizationServer** usuwa serwer autoryzacji usługi Azure API Management.

## PRZYKŁADY

### Przykład 1. Usuwanie serwera autoryzacji
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementAuthorizationServer -Context $ApiMgmtContext -ServerId "authserverid" -Force
```

To polecenie usuwa określony serwer autoryzacji zarządzania interfejsami API.
Parametr *Force jest* określony, dlatego potwierdzenie nie jest wymagane.
