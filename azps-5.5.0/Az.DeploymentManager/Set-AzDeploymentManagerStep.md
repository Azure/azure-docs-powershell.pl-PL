# Set-AzDeploymentManagerStep

## SYNOPSIS
Aktualizuje krok.

## SKŁADNIA

```
Set-AzDeploymentManagerStep [-InputObject] <PSStepResource> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzDeploymentManagerStep** aktualizuje krok o określonym obiekcie kroku.
Polecenie cmdlet zwraca zaktualizowany obiekt kroku.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Set-AzDeploymentManagerStep -InputObject $stepObject
```

To polecenie aktualizuje krok, którego nazwa i grupa zasobów są zgodne odpowiednio z właściwościami Name (Nazwa) $stepObject ResourceGroupName (Nazwa grupy zasobów).
Krok zostanie zaktualizowany do właściwości określonych w $stepObject.
