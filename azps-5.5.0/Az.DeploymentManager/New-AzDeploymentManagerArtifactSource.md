# New-AzDeploymentManagerArtifactSource

## SYNOPSIS
Tworzy źródło artefaktu.

## SKŁADNIA

```
New-AzDeploymentManagerArtifactSource -ResourceGroupName <String> -Name <String> -Location <String>
 -SasUri <String> [-Tag <Hashtable>] [-ArtifactRoot <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzDeploymentManagerArtifactSource** tworzy źródło artefaktu.
Określ właściwości *Name (Nazwa),* *ResourceGroupName (Nazwa* Grupy Zasobów) i wymagane.

Zwrócony obiekt można zmodyfikować lokalnie, a następnie zastosować zmiany do źródła artefaktu za pomocą Set-AzDeploymentManagerArtifactSource cmdlet.

Polecenie cmdlet zwraca obiekt ArtifactSource o identyfikatorze ResourceId, do którego można się odwołać w poleceniach cmdlet programu New-AzDeploymentManagerServiceTopology, dzięki czemu do artefaktów wymaganych dla zasobu ServiceUnit, plików Template i Parameters można odwoływać się z tej lokalizacji.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> New-AzDeploymentManagerArtifactSource -ResourceGroupName ContosoResourceGroup -Name ContosoArtifactSource -Location "Central US" -SasUri "https://ContosoStorage.blob.core.windows.net/ContosoArtifacts?sasParameters"
```

Tworzy źródło artefaktu w grupie ContosoResourceGroup z nazwą ContosoArtifactSource (Źródło ContosoArtifactSource) z lokalizacją zasobu w Środkowych Usach. Właściwość SasUri dostarcza do kontenera magazynu, w którym są przechowywane artefakty, wartość SAS Uri magazynu platformy Azure.
