# Remove-AzServiceFabricApplicationType

## SYNOPSIS
Usuń sieć szkieletową usługi z klastrów. Spowoduje to usunięcie wszystkich wersji tego typu w ramach tego zasobu. Obsługuje tylko ARM typów aplikacji wdrożonych.

## SKŁADNIA

### ByResourceGroup (Default)
```
Remove-AzServiceFabricApplicationType [-ResourceGroupName] <String> [-ClusterName] <String> [-Name <String>]
 [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzServiceFabricApplicationType -ResourceId <String> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzServiceFabricApplicationType -InputObject <PSApplicationType> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
To polecenie cmdlet powoduje usunięcie typu aplikacji z klastrem i usunięcie całej wersji typu w ramach tego zasobu, ale wszystkie aplikacje pod nim powinny zostać usunięte przed uruchomieniem tego polecenia.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $resourceGroupName = "testRG"
PS C:\> $clusterName = "testCluster"
PS C:\> $appTypeName = "testAppType"
PS C:\> Remove-AzServiceFabricApplicationType -ResourceGroupName $resourceGroupName -ClusterName $clusterName -Name $appTypeName -Verbose
```

W tym przykładzie usuniesz typ aplikacji "testAppType" i całą wersję pod nim. Jeśli istnieją aplikacje utworzone za pomocą tego typu, polecenie zgłasza wyjątek.
