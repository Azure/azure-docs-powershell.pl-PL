# Get-AzServiceFabricManagedCluster

## SYNOPSIS
Uzyskaj szczegółowe informacje dotyczące zarządzanego zasobu klastrów.

## SKŁADNIA

### BySubscription (Default)
```
Get-AzServiceFabricManagedCluster [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceGroup
```
Get-AzServiceFabricManagedCluster [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByName
```
Get-AzServiceFabricManagedCluster [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Uzyskaj szczegółowe informacje dotyczące zarządzanego zasobu klastrów.

## PRZYKŁADY

### Przykład 1
```powershell
$rgName = "testRG"
$clusterName = "testCluster"
Get-AzServiceFabricManagedCluster -ResourceGroupName $rgName -Name $clusterName
```

Uzyskaj szczegółowe informacje o klastrze.

### Przykład 2
```powershell
$rgName = "testRG"
Get-AzServiceFabricManagedCluster -ResourceGroupName $rgName
```

Uzyskiwanie listy klastrów w ramach określonej grupy zasobów.

### Przykład 3
```powershell
Get-AzServiceFabricManagedCluster
```

Uzyskaj listę klastrów w ramach bieżącej subskrypcji.
