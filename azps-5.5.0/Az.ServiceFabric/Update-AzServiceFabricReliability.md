# Update-AzServiceFabricReliability

## SYNOPSIS
Zaktualizuj warstwę niezawodności typu węzła podstawowego w klastrze.

## SKŁADNIA

```
Update-AzServiceFabricReliability [-ResourceGroupName] <String> [-Name] <String>
 -ReliabilityLevel <ReliabilityLevel> [-AutoAddNode] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## OPIS
Użyj **funkcji Update-AzServiceFabricReliability,** aby zaktualizować niezawodność typu węzła podstawowego w klastrze.

## PRZYKŁADY

### Przykład 1
```powershell
PS c:> Update-AzServiceFabricReliability -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -ReliabilityLevel Silver
```

To polecenie zmienia warstwę niezawodności typu węzła podstawowego na srebrną.

### Przykład 2

Aktualizowanie warstwy niezawodności typu węzła podstawowego w klastrze. (wygenerowane automatycznie)

<!-- Aladdin Generated Example -->
```powershell
Update-AzServiceFabricReliability -AutoAddNode -Name 'Contoso01SFCluster' -ReliabilityLevel None -ResourceGroupName 'Group1'
```
