# New-AzServiceFabricApplicationType

## SYNOPSIS
Utwórz nowy typ aplikacji sieci szkieletowej usług w ramach określonej grupy zasobów i klastrów.

## SKŁADNIA

```
New-AzServiceFabricApplicationType [-ResourceGroupName] <String> [-ClusterName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet tworzy nowy typ aplikacji sieci szkieletowej usługi w ramach określonej grupy zasobów i klastra.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $resourceGroupName = "testRG"
PS C:\> $clusterName = "testCluster"
PS C:\> $appTypeName = "testAppType"
PS C:\> $appType = New-AzServiceFabricApplicationType -ResourceGroupName $resourceGroupName -ClusterName $clusterName -Name $appTypeName -Verbose
```

W tym przykładzie zostanie określony nowy typ aplikacji "testAppType" w grupie zasobów i określonym klastrze.
