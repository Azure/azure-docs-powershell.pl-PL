# Moduł Az.Dns
## Opis
W tym temacie są wyświetlane tematy pomocy dotyczące cmdlet usługi Azure DNS.

## Az.Dns — polecenia cmdlet
### [Add-AzDnsRecordConfig](Add-AzDnsRecordConfig.md)
Dodaje rekord DNS do obiektu lokalnego zestawu rekordów.

### [Get-AzDnsRecordSet](Get-AzDnsRecordSet.md)
Pobiera zestaw rekordów DNS.

### [Get-AzDnsZone](Get-AzDnsZone.md)
Pobiera strefę DNS.

### [New-AzDnsRecordConfig](New-AzDnsRecordConfig.md)
Tworzy nowy obiekt lokalny rekordu DNS.

### [New-AzDnsRecordSet](New-AzDnsRecordSet.md)
Tworzy zestaw rekordów DNS.

### [New-AzDnsZone](New-AzDnsZone.md)
Tworzy nową strefę DNS.

### [Remove-AzDnsRecordConfig](Remove-AzDnsRecordConfig.md)
Usuwa rekord DNS z obiektu lokalnego zestawu rekordów.

### [Remove-AzDnsRecordSet](Remove-AzDnsRecordSet.md)
Usuwa zestaw rekordów.

### [Remove-AzDnsZone](Remove-AzDnsZone.md)
Usuwa strefę DNS z grupy zasobów.

### [Set-AzDnsRecordSet](Set-AzDnsRecordSet.md)
Aktualizuje zestaw rekordów DNS.

### [Set-AzDnsZone](Set-AzDnsZone.md)
Aktualizuje właściwości strefy DNS.
