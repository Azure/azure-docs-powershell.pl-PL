# Test-AzStreamAnalyticsInput

## SYNOPSIS
Sprawdza stan połączenia danych wejściowych.

## SKŁADNIA

```
Test-AzStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Test-AzStreamAnalyticsInput** sprawdza możliwość łączenia się z elementami wejściowymi przez usługę Stream Analytics.

## PRZYKŁADY

### Przykład 1. Testowanie stanu połączenia strumienia wejściowego
```powershell
PS C:\>Test-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

To sprawdza stan połączenia wejściowego strumienia Danych Wejściowych w transmisję strumieniową StreamingJob.
