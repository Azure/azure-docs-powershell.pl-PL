# Moduł Az.AnalysisServices
## Opis
W tym temacie są wyświetlane tematy pomocy dotyczące polecenia cmdlet usługi Azure AnalysisServices.

## Polecenia cmdlet az.AnalysisServices
### [Add-AzAnalysisServicesAccount](Add-AzAnalysisServicesAccount.md)
Dodaje uwierzytelnione konto do użycia na potrzeby żądań cmdlet serwera usług Azure Analysis Services.

### [Export-AzAnalysisServicesInstanceLog](Export-AzAnalysisServicesInstanceLog.md)
Eksportuje dziennik z wystąpienia serwera usług Analysis Services w środowisku obecnie zalogowanym zgodnie z Add-AzAnalysisServicesAccount programu

### [Get-AzAnalysisServicesServer](Get-AzAnalysisServicesServer.md)
Pobiera szczegółowe informacje o serwerze usług Analysis Services.

### [New-AzAnalysisServicesFirewallConfig](New-AzAnalysisServicesFirewallConfig.md)
Tworzy nową konfigurację zapory usług Analysis Services 

### [New-AzAnalysisServicesFirewallRule](New-AzAnalysisServicesFirewallRule.md)
Tworzy nową regułę zapory usług Analysis Services

### [New-AzAnalysisServicesServer](New-AzAnalysisServicesServer.md)
Tworzy nowy serwer usług Analysis Services

### [Remove-AzAnalysisServicesServer](Remove-AzAnalysisServicesServer.md)
Usuwanie wystąpienia serwera usług Analysis Services

### [Restart-AzAnalysisServicesInstance](Restart-AzAnalysisServicesInstance.md)
Ponowne uruchamianie wystąpienia serwera usług Analysis Services w obecnie zalogowanym środowisku zgodnie z Add-AzAnalysisServicesAccount programu

### [Resume-AzAnalysisServicesServer](Resume-AzAnalysisServicesServer.md)
Wznawia wystąpienie serwera usług Analysis Services

### [Set-AzAnalysisServicesServer](Set-AzAnalysisServicesServer.md)
Modyfikuje wystąpienie serwera usług Analysis Services

### [Suspend-AzAnalysisServicesServer](Suspend-AzAnalysisServicesServer.md)
Zawiesza wystąpienie serwera usług Analysis Services

### [Sync-AzAnalysisServicesInstance](Sync-AzAnalysisServicesInstance.md)
Synchronizuje określoną bazę danych na określonym wystąpieniu serwera usług Analysis Services ze wszystkimi wystąpieniami skalowania zapytania w obecnie zalogowanym środowisku zgodnie z Add-AzAnalysisServicesAccount polecenia

### [Test-AzAnalysisServicesServer](Test-AzAnalysisServicesServer.md)
Sprawdza istnienie wystąpienia serwera usług Analysis Services.
