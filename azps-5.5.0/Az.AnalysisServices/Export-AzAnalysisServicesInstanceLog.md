# Export-AzAnalysisServicesInstanceLog

## SYNOPSIS
Eksportuje dziennik z wystąpienia serwera usług Analysis Services w środowisku obecnie zalogowanym zgodnie z Add-AzAnalysisServicesAccount programu

## SKŁADNIA

```
Export-AzAnalysisServicesInstanceLog -OutputPath <String> [-Force] [-Instance] <String> [-PassThru] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie Export-AzAnalysisServicesInstance eksportuje dziennik z wystąpienia serwera usług Azure Analysis Services do pliku

## PRZYKŁADY

### Przykład 1
```
PS C:\>Export-AzAnalysisServicesInstanceLog -Instance testserver -OutputPath C:\path\to\log\testserver.log
```

To polecenie wyeksportuje dziennik z serwera "serwer testowy" w środowisku określonym za pomocą polecenia Add-AzAnalysisServicesAccount i zapisze go w pliku określonym w pliku outputPath 'C:\path\to\log\testserver.log"
