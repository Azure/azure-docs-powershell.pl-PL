# Get-AzApplicationGatewayCustomError

## SYNOPSIS
Pobiera błędy niestandardowe z bramy aplikacji.

## SKŁADNIA

```
Get-AzApplicationGatewayCustomError [-StatusCode <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzApplicationGatewayCustomError** pobiera błędy niestandardowe z bramy aplikacji.

## PRZYKŁADY

### Przykład 1. Zwraca błąd niestandardowy w bramie aplikacji
```powershell
PS C:\> $ce = Get-AzApplicationGatewayCustomError -ApplicationGateway $appgw -StatusCode HttpStatus502
```

To polecenie pobiera i zwraca niestandardowy błąd kodu stanu http 502 z bramy $appgw.

### Przykład 2. Pobiera listę wszystkich błędów niestandardowych w bramie aplikacji
```powershell
PS C:\> $ces = Get-AzApplicationGatewayCustomError -ApplicationGateway $appgw
```

To polecenie pobiera i zwraca listę wszystkich błędów niestandardowych z listy błędów $appgw.
