# New-AzVpnServerConfiguration

## SYNOPSIS
Utwórz nową konfigurację VpnServerConfiguration dla połączeń punktowych z witryną.

## SKŁADNIA

```
New-AzVpnServerConfiguration -ResourceGroupName <String> -Name <String> -Location <String>
 [-VpnProtocol <String[]>] [-VpnAuthenticationType <String[]>] [-VpnClientRootCertificateFilesList <String[]>]
 [-VpnClientRevokedCertificateFilesList <String[]>] [-RadiusServerAddress <String>]
 [-RadiusServerSecret <SecureString>] [-RadiusServerList <PSRadiusServer[]>]
 [-RadiusServerRootCertificateFilesList <String[]>] [-RadiusClientRootCertificateFilesList <String[]>]
 [-AadTenant <String>] [-AadAudience <String>] [-AadIssuer <String>] [-VpnClientIpsecPolicy <PSIpsecPolicy[]>]
 [-Tag <Hashtable>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **New-AzVpnServerConfiguration** umożliwia utworzenie nowej konfiguracji VpnServerConfiguration z różnymi parametrami VpnProtocols, VpnAuthenticationTypes, IpsecPolicies oraz ustawienie wybranych parametrów powiązanych z uwierzytelnianiem sieci vpn zgodnie z wymaganiami klienta w zakresie łączności typu Point to site.
