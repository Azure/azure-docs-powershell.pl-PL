# New-AzP2sVpnGateway

## SYNOPSIS
Utwórz nową bramę P2SVpnGateway w obszarze VirtualHub, aby wskazać łączność z witryną.

## SKŁADNIA

 -VirtualHubName <String> [-VpnServerConfiguration <PSVpnServerConfiguration>] -VpnClientAddressPool <String[]>
 [-CustomDnsServer <String[]>] [-RoutingConfiguration <PSRoutingConfiguration>] [-EnableInternetSecurityFlag]

## OPIS
Polecenie cmdlet New-AzP2sVpnGateway umożliwia utworzenie nowej aplikacji P2SVpnGateway w obszarze VirtualHub for Point na łączność z witryną z klientów usługi Point do klientów witryny do usługi Azure VirtualWan.
