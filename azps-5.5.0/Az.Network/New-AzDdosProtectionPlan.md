# New-AzDdosProtectionPlan

## SYNOPSIS
Tworzy plan ochrony przed DDoS.

## SKŁADNIA

```
New-AzDdosProtectionPlan -ResourceGroupName <String> -Name <String> -Location <String> [-Tag <Hashtable>]
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie New-AzDdosProtectionPlan cmdlet tworzy plan ochrony przed DDoS.

## PRZYKŁADY

### Przykład 1. Tworzenie i kojarzenie planu ochrony przed DDoS z nową siecią wirtualną
```
D:\> $ddosProtectionPlan = New-AzDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlanName -Location "West US"
D:\> $subnet = New-AzVirtualNetworkSubnetConfig -Name SubnetName -AddressPrefix 10.0.1.0/24
D:\> $vnet = New-AzVirtualNetwork -Name VnetName -ResourceGroupName ResourceGroupName -Location "West US" -AddressPrefix 10.0.0.0/16 -DnsServer 8.8.8.8 -Subnet $subnet -EnableDdoSProtection -DdosProtectionPlanId $ddosProtectionPlan.Id
```

Najpierw tworzymy nowy plan ochrony DDoS za pomocą **polecenia New-AzDdosProtectionPlan.**
Następnie tworzymy nową sieć wirtualną za pomocą **funkcji New-AzVirtualNetwork** i określamy identyfikator nowo utworzonego planu w parametrze **DdosProtectionPlanId.** W tym przypadku, ponieważ kojarzymy sieć wirtualną z planem, możemy również określić parametr **EnableDdoSProtection.**

### Przykład 2. Tworzenie i kojarzenie planu ochrony przed DDoS z istniejącą siecią wirtualną

D:\> $ddosProtectionPlan = New-AzDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlanName -Location "West US"

D:\> $vnet.DdosProtectionPlan = New-Object Microsoft.Azure.Commands.Network.Models.PSResourceId

Po drugie, otrzymuję najbardziej zaktualizowaną wersję sieci wirtualnej, którą chcemy skojarzyć z planem. Aktualizujemy właściwość **DdosProtectionPlan** za pomocą obiektu **PSResourceId zawierającego** odwołanie do identyfikatora nowo utworzonego planu. W takim przypadku, jeśli skojarzymy sieć wirtualną z planem ochrony przed DDoS, możemy również ustawić dla flagi **EnableDdosProtection** wartość true.
Na koniec utrzymujemy nowy stan, ustawiając zmienną lokalną do **Set-AzVirtualNetwork.**
