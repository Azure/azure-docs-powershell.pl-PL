# Set-AzApplicationGatewayPrivateLinkConfiguration

## SYNOPSIS
Modyfikuje konfigurację privatelinku dla bramy aplikacji.

## SKŁADNIA

```
Set-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 -IpConfiguration <PSApplicationGatewayPrivateLinkIpConfiguration[]> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzApplicationGatewayPrivateLinkConfiguration** modyfikuje konfigurację privateLink bramy aplikacji Azure.

## PRZYKŁADY

### Przykład 1. Ustawianie konfiguracji privatelinku
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway $AppGw -Name "privateLinkConfig01" -IpConfiguration $privateLinkIpConfiguration01
```

Pierwsze polecenie pobiera bramę aplikacji o nazwie ApplicationGateway01, która należy do grupy zasobów o nazwie ResourceGroup01, i przechowuje ją w zmiennej $AppGw zasobów.
Drugie polecenie ustawia konfigurację privateLink o nazwie privateLinkConfig01 na używanie konfiguracji adresów IP przechowywanej w u $privateLinkIpConfiguration 01
