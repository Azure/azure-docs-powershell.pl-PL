# Remove-AzApplicationGatewayFrontendPort

## SYNOPSIS
Usuwa port frontonie z bramy aplikacji.

## SKŁADNIA

```
Remove-AzApplicationGatewayFrontendPort -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzApplicationGatewayFrontendPort** usuwa port frontoronu z bramy aplikacji platformy Azure.

## PRZYKŁADY

### Przykład: usuwanie portu frontoronu z bramy aplikacji
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayFrontendPort -ApplicationGateway $AppGw -Name "FrontEndPort02"
```

Pierwsze polecenie otrzymuje bramę aplikacji o nazwie ApplicationGateway01, która należy do grupy zasobów o nazwie ResourceGroup01 i przechowuje bramę w $AppGw zmienną.
Drugie polecenie usuwa z bramy aplikacji port o nazwie FrontEndPort02.
