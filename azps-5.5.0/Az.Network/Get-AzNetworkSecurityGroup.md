# Get-AzNetworkSecurityGroup

## SYNOPSIS
Otrzymuje grupę zabezpieczeń sieciowych.

## SKŁADNIA

### NoExpand
```
Get-AzNetworkSecurityGroup [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Rozwiń
```
Get-AzNetworkSecurityGroup -Name <String> -ResourceGroupName <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzNetworkSecurityGroup** otrzymuje grupę zabezpieczeń sieci platformy Azure.
