# New-AzLoadBalancerProbeConfig

## SYNOPSIS
Tworzy konfigurację konfiguracji sprzętu do równoważenia obciążenia.

## SKŁADNIA

```
New-AzLoadBalancerProbeConfig -Name <String> [-Protocol <String>] -Port <Int32> -IntervalInSeconds <Int32>
 -ProbeCount <Int32> [-RequestPath <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzLoadBalancerProbeConfig** tworzy konfigurację konfiguracji dla narzędzia do równoważenia obciążenia platformy Azure.

## PRZYKŁADY

### Przykład 1. Tworzenie konfiguracji
```powershell
PS C:\>New-AzLoadBalancerProbeConfig -Name "MyProbe" -Protocol "http" -Port 80 -IntervalInSeconds 15 -ProbeCount 15
```

To polecenie tworzy konfigurację o nazwie MyProbe przy użyciu protokołu HTTP.
Nowa usługa połączy się z usługą równoważenia obciążenia na porcie 80.

### Przykład 2

Tworzy konfigurację konfiguracji konfiguracji dla równoważenia obciążenia. (wygenerowane automatycznie)

<!-- Aladdin Generated Example -->
```powershell
New-AzLoadBalancerProbeConfig -IntervalInSeconds 15 -Name 'MyProbe' -Port 80 -ProbeCount 15 -Protocol 'http' -RequestPath 'healthcheck.aspx'
```
