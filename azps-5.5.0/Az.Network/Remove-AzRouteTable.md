# Remove-AzRouteTable

## SYNOPSIS
Usuwa tabelę tras.

## SKŁADNIA

```
Remove-AzRouteTable -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzRouteTable** usuwa tabelę trasy platformy Azure.

## PRZYKŁADY

### Przykład 1. Usuwanie tabeli trasy
```
PS C:\>Remove-AzRouteTable -ResourceGroupName "ResourceGroup11 -Name "RouteTable01"
Confirm
Are you sure you want to remove resource 'RouteTable01'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

To polecenie usuwa tabelę tras o nazwie RouteTable01 z grupy zasobów o nazwie ResourceGroup11.
Polecenie cmdlet wyświetli monit o potwierdzenie przed usunięciem tabeli.
