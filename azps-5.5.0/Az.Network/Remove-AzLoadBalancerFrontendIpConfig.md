# Remove-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
Usuwa konfigurację frontoronu IP z urządzenia do równoważenia obciążenia.

## SKŁADNIA

```
Remove-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzLoadBalancerFrontendIpConfig** usuwa konfigurację frontoronu IP z narzędzia do równoważenia obciążenia platformy Azure.

## PRZYKŁADY

### Przykład 1. Usuwanie konfiguracji frontoronu ip z urządzenia do równoważenia obciążenia
```
PS C:\>$loadbalancer = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:> Remove-AzLoadBalancerFrontendIpConfig -Name "frontendName" -LoadBalancer $loadbalancer
```

Pierwsze polecenie pobiera równoważenie obciążenia skojarzone z konfiguracją frontonu adresu IP, którą chcesz usunąć, a następnie zapisuje je w $loadbalancer sieci.
Drugie polecenie usuwa skojarzoną konfigurację frontonu IP z równoważenia obciążenia w $loadbalancer.
