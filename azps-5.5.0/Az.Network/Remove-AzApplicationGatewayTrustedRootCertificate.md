# Remove-AzApplicationGatewayTrustedRootCertificate

## SYNOPSIS
Usuwa zaufany certyfikat główny z bramy aplikacji.

## SKŁADNIA

```
Remove-AzApplicationGatewayTrustedRootCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzApplicationGatewayTrustedRootCertificate** usuwa certyfikat trusted root z istniejącej bramy aplikacji.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayTrustedRootCertificate -ApplicationGateway $gw -Name "myRootCA"
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Pierwsze polecenie pobiera bramę aplikacji i przechowuje je w $gw aplikacji.
Drugie polecenie usuwa zaufany certyfikat główny o nazwie myRootCA z bramy aplikacji przechowywanej w $gw.
Trzecie polecenie aktualizuje bramę aplikacji na platformie Azure.
