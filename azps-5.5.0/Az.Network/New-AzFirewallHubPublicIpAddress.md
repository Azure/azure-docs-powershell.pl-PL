# New-AzFirewallHubPublicIpAddress

## SYNOPSIS
Publiczny adres IP assoicated do zapory w centrum wirtualnym

## SKŁADNIA

```
New-AzFirewallHubPublicIpAddress [-Count <Int32>] [-Addresses <PSAzureFirewallPublicIpAddress[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Publiczny adres IP assoicated do zapory w centrum wirtualnym

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> New-AzFirewallHubPublicIpAddress -Count 2
```

Spowoduje to utworzenie 2 publicznych adresów IP na zaporze dołączonej do centrum wirtualnego. Spowoduje to utworzenie adresu IP w zaplecza. Nie możemy jawnie udostępnić tychdresses dla nowej zapory.

### Przykład 2
```powershell
PS C:\> $publicIp1 = New-AzFirewallPublicIpAddress -Address 10.2.3.4
PS C:\> $publicIp2 = New-AzFirewallPublicIpAddress -Address 20.56.37.46
PS C:\> New-AzFirewallHubPublicIpAddress -Count 3 -Addresses $publicIp1, $publicIp2
```

Spowoduje to utworzenie 1 nowego publicznego adresu IP w zaporze przez zachowanie $publicIp 1, $publicIp 2 istniejących już w zaporze.
