# Get-AzLoadBalancer

## SYNOPSIS
Pobiera równoważenie obciążenia.

## SKŁADNIA

### NoExpand (Domyślne)
```
Get-AzLoadBalancer [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Rozwiń
```
Get-AzLoadBalancer -ResourceGroupName <String> -Name <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzLoadBalancer** pobiera co najmniej jedno narzędzie do równoważenia obciążenia platformy Azure, które są zawarte w grupie zasobów.
