# New-AzApplicationGatewayRewriteRuleHeaderConfiguration

## SYNOPSIS
Tworzy konfigurację nagłówka reguły ponownego w bramie aplikacji.

## SKŁADNIA

```
New-AzApplicationGatewayRewriteRuleHeaderConfiguration -HeaderName <String> [-HeaderValue <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
**Polecenie cmdlet AzApplicationGatewayRewriteRuleHeaderConfiguration** tworzy zestaw akcji reguły ponownego w przypadku bramy aplikacji azure.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $hc = New-AzApplicationGatewayRewriteRuleHeaderConfiguration -HeaderName abc -HeaderValue def
```

To polecenie tworzy konfigurację nagłówka reguły ponownego tworzenia i zapisuje wynik w zmiennej o nazwie $hc.
