# Get-AzPrivateEndpoint

## SYNOPSIS
Uzyskiwanie prywatnego punktu końcowego

## SKŁADNIA

### NoExpand (Domyślne)
```
Get-AzPrivateEndpoint [-Name <String>] [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Rozwiń
```
Get-AzPrivateEndpoint -Name <String> -ResourceGroupName <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzPrivateEndpoint** pobiera co najmniej jeden prywatny punkt końcowy.
