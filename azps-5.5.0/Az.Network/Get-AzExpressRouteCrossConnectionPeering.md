# Get-AzExpressRouteCrossConnectionPeering

## SYNOPSIS
Pobiera konfigurację komunikacji równorzędnej połączenia krzyżowego usługi ExpressRoute.

## SKŁADNIA

```
Get-AzExpressRouteCrossConnectionPeering [-Name <String>]
 -ExpressRouteCrossConnection <PSExpressRouteCrossConnection> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzExpressRouteCrossConnectionPeering** pobiera konfigurację relacji komunikacji równorzędnej dla połączenia krzyżowego usługi ExpressRoute.

## PRZYKŁADY

### Przykład 1. Wyświetlanie konfiguracji komunikacji równorzędnej dla połączenia krzyżowego usługi ExpressRoute
```
$cc = Get-AzExpressRouteCrossConnection -Name $CrossConnectionName -ResourceGroupName $RG
Get-AzExpressRouteCrossConnectionPeering -Name "AzurePrivatePeering" -ExpressRouteCrossConnection $cc
```
