# Remove-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Usuwa konfigurację automatycznego skalowania z bramy aplikacji.

## SKŁADNIA

```
Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzApplicationGatewayAutoscaleConfiguration** usuwa konfigurację skalowania automatycznego z istniejącej bramy aplikacji.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Pierwsze polecenie pobiera bramę aplikacji i przechowuje ją w $gw zmienną.
Drugie polecenie usuwa konfigurację automatycznego skalowania z bramy aplikacji.
Trzecie polecenie aktualizuje bramę aplikacji na platformie Azure.
