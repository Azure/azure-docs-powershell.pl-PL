# Remove-AzNetworkInterface

## SYNOPSIS
Usuwa interfejs sieciowy.

## SKŁADNIA

```
Remove-AzNetworkInterface -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzNetworkInterface** usuwa interfejs sieciowy platformy Azure.

## PRZYKŁADY

### Przykład 1. Usuwanie interfejsu sieciowego
```
PS C:\>Remove-AzNetworkInterface -Name "NetworkInterface1" -ResourceGroup "ResourceGroup1"
```

To polecenie usuwa interfejs sieciowy NetworkInterface1 w grupie zasobów ResourceGroup1.
Ponieważ parametr *Force* nie jest używany, użytkownik zostanie wyświetlony monit o potwierdzenie tej akcji.

### Przykład 2. Usuwanie interfejsu sieciowego
```
PS C:\>Get-AzNetworkInterface -ResourceGroupName "ResourceGroup1" | Remove-AzNetworkInterface -Force
```

To polecenie usuwa wszystkie interfejsy sieciowe w grupie zasobów ResourceGroup1.
Parametr *Force jest* używany, dlatego użytkownik nie jest monitowany o potwierdzenie.
