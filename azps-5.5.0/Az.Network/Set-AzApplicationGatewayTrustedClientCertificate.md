# Set-AzApplicationGatewayTrustedClientCertificate

## SYNOPSIS
Modyfikuje łańcuch certyfikatów zaufanego urzędu certyfikacji klienta bramy aplikacji.

## SKŁADNIA

```
Set-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzApplicationGatewayTrustedClientCertificate** modyfikuje łańcuch certyfikatów zaufanego urzędu certyfikacji klienta bramy aplikacji.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Set-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name $certName --CertificateFile ".\clientCAUpdated.cer"
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

W powyższych przykładowych scenariuszach pokazano, jak zaktualizować istniejący obiekt łańcucha certyfikatów zaufanego urzędu certyfikacji klienta. Pierwsze polecenie pobiera bramę aplikacji i przechowuje je w $gw aplikacji. Drugie polecenie modyfikuje istniejący obiekt łańcucha certyfikatów zaufanego urzędu certyfikacji klienta przy użyciu nowego pliku łańcucha certyfikatów urzędu certyfikacji. Trzecie polecenie aktualizuje bramę aplikacji na platformie Azure.
