# Get-AzApplicationGatewayProbeConfig

## SYNOPSIS
Pobiera istniejącą konfigurację kondycji z bramy aplikacji.

## SKŁADNIA

```
Get-AzApplicationGatewayProbeConfig [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie Get-AzApplicationGatewayProbeConfig pobiera istniejącą konfigurację kondycji z bramy aplikacji.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie istniejącego pliku z bramy aplikacji
```
PS C:\>Get-AzApplicationGatewayProbeConfig -ApplicationGateway Gateway -Name "Probe02"
```

To polecenie pobiera nazwę kondycji o nazwie Łowe02 z bramy aplikacji o nazwie Brama.
