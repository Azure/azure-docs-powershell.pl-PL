# Remove-AzApplicationGatewayIdentity

## SYNOPSIS
Usuwa tożsamość z bramy aplikacji.

## SKŁADNIA

```
Remove-AzApplicationGatewayIdentity -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Remove-AzApplicationGatewayIdentity** usuwa tożsamość z bramy aplikacji.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $appgw = Remove-AzApplicationGatewayIdentity -ApplicationGateway $appgw
PS C:\> $updatedgateway = Set-AzApplicationGateway -ApplicationGateway $appgw
```

W tym przykładzie usuwamy tożsamość z istniejącej bramy aplikacji.
Uwaga: Jeśli brama odwołuje się do klucza tajnego, ważne jest również usunięcie tych odwołań do certyfikatów SSL podczas tej operacji.
