# New-AzPrivateLinkServiceIpConfig

## SYNOPSIS
Utwórz konfigurację adresu IP prywatnej usługi linków.

## SKŁADNIA

```
New-AzPrivateLinkServiceIpConfig -Name <String> [-PrivateIpAddressVersion <String>]
 [-PrivateIpAddress <String>] [-PublicIpAddress <PSPublicIpAddress>]
 [-Subnet <PSSubnet>] [-Primary]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzPrivateLinkServiceIpConfig** tworzy konfigurację ip usługi łączy prywatnych. Ta konfiguracja jest używana dla źródłowego punktu końcowego nat-ing ruchu przychodzącego z prywatnego punktu końcowego. 

## PRZYKŁADY

### Przykład 1
```powershell
New-AzPrivateLinkServiceIpConfig -Name $IpConfigurationName -PrivateIpAddress "10.0.0.5" -Primary
```

W tym przykładzie utworzysz konfigurację adresu IP prywatnej usługi łączy w pamięci.

### Przykład 2

Utwórz konfigurację adresu IP prywatnej usługi linków. (wygenerowane automatycznie)

<!-- Aladdin Generated Example -->
```powershell
New-AzPrivateLinkServiceIpConfig -Name 'IP-Config' -PrivateIpAddress '10.0.0.5' -Subnet <PSSubnet>
```
