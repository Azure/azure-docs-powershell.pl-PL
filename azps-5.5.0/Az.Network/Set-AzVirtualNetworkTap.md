# Set-AzVirtualNetworkTap

## SYNOPSIS
Aktualizuje dotknięcie sieci wirtualnej.

## SKŁADNIA

```
Set-AzVirtualNetworkTap -VirtualNetworkTap <PSVirtualNetworkTap> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzVirtualNetworkTap** aktualizuje naciśnięcie sieci wirtualnej.

## PRZYKŁADY

### Przykład 1. Konfigurowanie naciśnięcia sieci wirtualnej
```
PS C:\>$vTap = Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
PS C:\>$vTap.DestinationNetworkInterfaceIPConfiguration = $newDestinationNic.IpConfigurations[0]
PS C:\>Set-AzVirtualNetworkTap -VirtualNetworkTap $vTap
```

Polecenie aktualizuje docelową konfigurację IpConfiguration i aktualizuje naciśnięcie sieci wirtualnej.
Jeśli istnieją jakiekolwiek konfiguracje typu tap odwołujące się do niego, po aktualizacji nie rozpocznie się dublowanie całego ruchu źródłowego do nowej konfiguracji adresu IP lokalizacji docelowej.
