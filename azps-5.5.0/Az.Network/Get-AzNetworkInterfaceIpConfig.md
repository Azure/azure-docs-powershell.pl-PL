# Get-AzNetworkInterfaceIpConfig

## SYNOPSIS
Pobiera konfigurację adresu IP interfejsu sieciowego dla interfejsu sieciowego.

## SKŁADNIA

```
Get-AzNetworkInterfaceIpConfig [-Name <String>] -NetworkInterface <PSNetworkInterface>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzNetworkInterfaceIPConfig** pobiera konfigurację adresu IP interfejsu sieciowego z interfejsu sieciowego platformy Azure.

## PRZYKŁADY

### 1. Uzyskiwanie konfiguracji protokołu IP interfejsu sieciowego
```
$nic1 = Get-AzNetworkInterface -Name mynic -ResourceGroupName $myrg
Get-AzNetworkInterfaceIpConfig -Name ipconfig1 -NetworkInterface $nic1
```

Pierwsze polecenie uzyskuje istniejący interfejs sieciowy o nazwie mynic i przechowuje je w zmiennej $nic 1. Drugie polecenie pobiera konfigurację protokołu IP o nazwie ipconfig1 tego interfejsu sieciowego.
