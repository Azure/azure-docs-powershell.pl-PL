# New-AzApplicationGatewayFirewallDisabledRuleGroupConfig

## SYNOPSIS
Tworzy nową wyłączono konfigurację grupy reguł.

## SKŁADNIA

```
New-AzApplicationGatewayFirewallDisabledRuleGroupConfig -RuleGroupName <String> [-Rules <Int32[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzApplicationGatewayFirewallDisabledRuleGroupConfig** powoduje utworzenie nowej wyłączonej konfiguracji grupy reguł.

## PRZYKŁADY

### Przykład 1
```
PS C:\> $disabledRuleGroup1 = New-AzApplicationGatewayFirewallDisabledRuleGroupConfig -RuleGroupName "REQUEST-942-APPLICATION-ATTACK-SQLI" -Rules 942130,942140
```

To polecenie tworzy nową konfigurację wyłączonej grupy reguł dla grupy reguł o nazwie "REQUEST-942-APPLICATION-ATTACK-SQLI" z wyłączonymi regułami 942130 i regułą 942140. Nowa wyłączona konfiguracja grupy reguł zostanie zapisana w $disabledRuleGroup 1.
