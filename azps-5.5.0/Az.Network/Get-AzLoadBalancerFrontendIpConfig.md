# Get-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
Pobiera konfigurację frontoronu IP w równoważeniu obciążenia.

## SKŁADNIA

```
Get-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzLoadBalancerFrontendIpConfig** otrzymuje konfigurację frontoronu IP lub listę konfiguracji frontoronu IP w równoważeniu obciążenia.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie konfiguracji frontoronu ip w ładowarce
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerFrontendIpConfig -Name "MyFrontEnd" -LoadBalancer $slb
```

Pierwsze polecenie pobiera równoważenie obciążenia o nazwie MyLoadBalancer, a następnie przechowuje je w zmiennym $slb.
Drugie polecenie pobiera skojarzoną z tym równoważeniem obciążenia konfigurację frontonu adresu IP.
