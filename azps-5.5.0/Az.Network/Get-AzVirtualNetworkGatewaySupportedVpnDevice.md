# Get-AzVirtualNetworkGatewaySupportedVpnDevice

## SYNOPSIS
To polecenie commandlet zwraca listę obsługiwanych marek urządzeń VPN, modeli i wersji oprogramowania układowego.

## SKŁADNIA

```
Get-AzVirtualNetworkGatewaySupportedVpnDevice -Name <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
To polecenie commandlet zwraca listę obsługiwanych marek urządzeń VPN, modeli i wersji oprogramowania układowego.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Get-AzVirtualNetworkGatewaySupportedVpnDevice -ResourceGroupName TestRG -Name TestGateway 
<?xml version="1.0" encoding="utf-8"?>
<RpVpnDeviceList version="1.0">
  <Vendor name="Cisco-Test">
    <DeviceFamily name="IOS-Test">
       <FirmwareVersion name="20" />
    </DeviceFamily>
  </Vendor>
</RpVpnDeviceList>
```

Zwraca listę obsługiwanych marek urządzeń SIECI VPN, modeli i wersji oprogramowania układowego:
<?xml version="1.0" encoding="utf-8"?>
<RpVpnDeviceList version="1.0">
  <Vendor name="Cisco-Test">
    <DeviceFamily name="IOS-Test">
       <FirmwareVersion name="20" />
    </DeviceFamily>
  </Vendor>
</RpVpnDeviceList>
