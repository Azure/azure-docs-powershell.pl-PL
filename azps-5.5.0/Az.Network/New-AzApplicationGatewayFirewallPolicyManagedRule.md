# New-AzApplicationGatewayFirewallPolicyManagedRule

## SYNOPSIS
Utwórz usługi ManagedRules dla zasad zapory.

## SKŁADNIA

```
New-AzApplicationGatewayFirewallPolicyManagedRule
 [-ManagedRuleSet <PSApplicationGatewayFirewallPolicyManagedRuleSet[]>]
 [-Exclusion <PSApplicationGatewayFirewallPolicyExclusion[]>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Reguła **New-AzApplicationGatewayFirewallPolicyManagedRule** tworzy reguły zarządzane dla zasad zapory.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $condition = New-AzApplicationGatewayFirewallPolicyManagedRule -ManagedRuleSet $managedRuleSet -Exclusion $exclusion1,$exclusion2
```

Polecenie tworzy reguły zarządzane, listę zestawu ManagedRuleSet z $managedRuleSet i listę wykluczeń z wpisami jako $exclusion 1, $exclusion 2.
