# Get-AzEffectiveRouteTable

## SYNOPSIS
Pobiera skuteczną tabelę tras w interfejsie sieciowym.

## SKŁADNIA

```
Get-AzEffectiveRouteTable -NetworkInterfaceName <String> [-ResourceGroupName <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzEffectiveRouteTable** zwraca efektywną tabelę trasy, która jest stosowana w interfejsie sieciowym.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie efektywnej tabeli tras w interfejsie sieciowym
```
PS C:\>Get-AzEffectiveRouteTable -NetworkInterfaceName "MyNetworkInterface" -ResourceGroupName "MyResourceGroup"
```

To polecenie pobiera efektywną tabelę tras skojarzoną z interfejsem sieciowym o nazwie MyNetworkInterface w grupie zasobów o nazwie MyResourceGroup.
