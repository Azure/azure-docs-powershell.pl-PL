# Get-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Pobiera konfigurację automatycznego skalowania bramy aplikacji.

## SKŁADNIA

```
Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzApplicationGatewayAutoscaleConfiguration** pobiera polecenie Konfiguracji automatycznego skalowania bramy aplikacji.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $autoscaleConfiguration = Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $autoscaleConfiguration.MinCapacity
```

Pierwsze polecenie pobiera bramę aplikacji i przechowuje ją w $gw zmienną.
Drugie polecenie wyodrębnia konfigurację automatycznego skalowania z bramy aplikacji.
