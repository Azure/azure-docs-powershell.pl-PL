# Set-AzApplicationGatewayBackendAddressPool

## SYNOPSIS
Aktualizuje pulę adresów końcowych dla bramy aplikacji.

## SKŁADNIA

```
Set-AzApplicationGatewayBackendAddressPool -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-BackendIPAddresses <String[]>] [-BackendFqdns <String[]>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzApplicationGatewayBackendAddressPool** aktualizuje pulę adresów zaplecza dla bramy aplikacji platformy Azure.
Adresy końcowe można określić jako adresy IP, w pełni kwalifikowane nazwy domen (FQDN) lub identyfikatory konfiguracji adresów IP.

## PRZYKŁADY

### Przykład 1. Ustawianie puli adresów back-end przy użyciu sieci FQDN
```
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayBackendAddressPool -ApplicationGateway $AppGw -Name "Pool02" -BackendFqdns "contoso1.com", "contoso2.com"
```

Pierwsze polecenie pobiera bramę aplikacji o nazwie ApplicationGateway01 w grupie zasobów o nazwie ResourceGroup01 i zapisuje ją w $AppGw zmienną.
Drugie polecenie aktualizuje pulę adresów back-end bramy aplikacji w programie $AppGw przy użyciu sieci FQNS.

### Przykład 2. Ustawianie puli adresów zaplecza przy użyciu adresów IP serwera zaplecza
```
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayBackendAddressPool -ApplicationGateway $AppGw -Name "Pool02" -BackendIPAddresses "10.10.10.10", "10.10.10.11"
```

Pierwsze polecenie pobiera bramę aplikacji o nazwie ApplicationGateway01 w grupie zasobów o nazwie ResourceGroup01 i zapisuje ją w $AppGw zmienną.
Drugie polecenie aktualizuje pulę adresów końcowych bramy aplikacji w programie $AppGw przy użyciu adresów IP.
