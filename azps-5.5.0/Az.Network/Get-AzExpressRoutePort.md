# Get-AzExpressRoutePort

## SYNOPSIS
Pobiera zasób Azure ExpressRoutePort.

## SKŁADNIA

### ResourceNameParameterSet (Domyślne)
```
Get-AzExpressRoutePort [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzExpressRoutePort -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzExpressRoutePort** służy do pobierania obiektu ExpressRoutePort z subskrypcji. Zwrócony obiekt expressrouteport może być używany jako dane wejściowe do innych cmdlet, które operują na expressRoutePort.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Get-AzExpressRoutePort -Name $PortName -ResourceGroupName $rg
```

Pobiera obiekt ExpressRoutePort o nazwie $PortName w grupie zasobów $rg w subskrypcji.

### Przykład 2
```powershell
PS C:\> Get-AzExpressRoutePort -Name test*
```

Pobiera wszystkie obiekty expressRoutePort, których nazwa zaczyna się od "test".

### Przykład 3
```powershell
PS C:\> Get-AzExpressRoutePort -ResourceId $id
```

Pobiera obiekt ExpressRoutePort z właściwością ResourceId $id.
