# New-AzFirewallHubIpAddress

## SYNOPSIS
Adresy IP assoicated do zapory w centrum wirtualnym

## SKŁADNIA

```
New-AzFirewallHubIpAddress [-PrivateIPAddress <String>] [-PublicIPs <PSAzureFirewallHubPublicIpAddresses>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Adresy IP assoicated do zapory w centrum wirtualnym. Mogą to być adresy publiczne i prywatne

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $fwpips = New-AzFirewallHubPublicIpAddress -Count 2
PS C:\> New-AzFirewallHubIpAddress -PublicIPs $fwpips
```

W tym przykładzie jest tworzenie obiektu adresu IP Centrum z zliczaną 2 publicznymi adresami IP. Obiekt HubIPAddress jest skojarzony z zaporą w centrum wirtualnym.
