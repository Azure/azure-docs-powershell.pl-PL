# Add-AzVirtualHubRouteTable

## SYNOPSIS
Tworzy zasób tabeli tras wirtualnego centrum, który jest podrzędnym zasobem aplikacji VirtualHub.

## SKŁADNIA

```
Add-AzVirtualHubRouteTable -Name <String> -Route <PSVirtualHubRoute[]> -Connection <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Zasób tabeli tras wirtualnego centrum zawiera listę tras oraz listę połączeń, do których można go dołączyć, i służy do rozsyłania ruchu w centrum wirtualnym.
