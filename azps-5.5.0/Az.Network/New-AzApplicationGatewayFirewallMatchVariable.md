# New-AzApplicationGatewayFirewallMatchVariable

## SYNOPSIS
Tworzy zmienną dopasowania dla warunku zapory.

## SKŁADNIA

```
New-AzApplicationGatewayFirewallMatchVariable -VariableName <String> [-Selector <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
**New-AzApplicationGatewayFirewallMatchVariable** tworzy zmienną dopasowania dla warunku zapory.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $variable = New-AzApplicationGatewayFirewallMatchVariable -VariableName RequestHeaders -Selector Content-Length
```

Polecenie tworzy nową zmienną dopasowania z nazwami nagłówków żądań, a selektor to pole Content-Length (Długość zawartości). Nowa zmienna dopasowania jest zapisywana w $variable.
