# New-AzLoadBalancerBackendAddressPoolConfig

## SYNOPSIS
Tworzy konfigurację puli adresów zaplecza dla równoważenia obciążenia.

## SKŁADNIA

```
New-AzLoadBalancerBackendAddressPoolConfig -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzLoadBalancerBackendAddressPoolConfig** tworzy konfigurację puli adresów zaplecza dla narzędzia do równoważenia obciążenia platformy Azure.

## PRZYKŁADY

### Przykład 1. Tworzenie konfiguracji puli adresów zaplecza dla równoważenia obciążenia
```
PS C:\>New-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02"
```

To polecenie tworzy konfigurację puli adresów zaplecza o nazwie BackendAddressPool02 dla równoważenia obciążenia.
