# New-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Tworzy konfigurację automatycznego skalowania dla bramy aplikacji.

## SKŁADNIA

```
New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity <Int32> [-MaxCapacity <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzApplicationGatewayAutoscaleConfiguration** tworzy konfigurację autoskalowania dla bramy aplikacji platformy Azure.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $autoscaleConfig = New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity 3
PS C:\> $gw = New-AzApplicationGateway -Name $appgwName -ResourceGroupName $rgname ..  -AutoscaleConfiguration $autoscaleConfig
```

Pierwsze polecenie tworzy konfigurację skalowania automatycznego o minimalnej pojemności 3.
Drugie polecenie tworzy bramę aplikacji z konfiguracją skalowania automatycznego.
