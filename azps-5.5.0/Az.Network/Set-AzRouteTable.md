# Set-AzRouteTable

## SYNOPSIS
Aktualizuje tabelę tras.

## SKŁADNIA

```
Set-AzRouteTable -RouteTable <PSRouteTable> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzRouteTable** aktualizuje tabelę trasy.

## PRZYKŁADY

### Przykład 2. Modyfikowanie tabeli tras

```
PS C:\> $rt = Get-AzRouteTable -ResourceGroupName "rgName" -Name "rtName"
PS C:\> $rt.DisableBgpRoutePropagation
False
PS C:\> $rt.DisableBgpRoutePropagation = $true
PS C:\> Set-AzRouteTable -RouteTable $rt
PS C:\> $rt = Get-AzRouteTable -ResourceGroupName "rgName" -Name "rtName"
PS C:\> $rt.DisableBgpRoutePropagation
True
```

Pierwsze polecenie pobiera tabelę trasy o nazwie rtName i zapisuje ją w $rt zmienną.
Drugie polecenie wyświetla wartość właściwości DisableBgpRoutePropagation.
Trzecie polecenie aktualizuje wartość właściwości DisableBgpRoutePropagation.
Czwarte polecenie aktualizuje tabelę trasy na serwerze.
Piąte polecenie zostanie zaktualizowane w tabeli trasy i będzie przechowywane w $rt danych.
Szóste polecenie wyświetla wartość właściwości DisableBgpRoutePropagation.
