# New-AzApplicationGatewayCustomError

## SYNOPSIS
Tworzy błąd niestandardowy z kodem stanu http i niestandardowym adresem URL strony błędu 

## SKŁADNIA

```
New-AzApplicationGatewayCustomError -StatusCode <String> -CustomErrorPageUrl <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzApplicationGatewayCustomError** powoduje utworzenie błędu niestandardowego.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $customError403Url = "https://mycustomerrorpages.blob.core.windows.net/errorpages/403-another.htm"
PS C:\> $ce = New-AzApplicationGatewayCustomError -StatusCode HttpStatus403 -CustomErrorPageUrl $customError403Url
```

To polecenie tworzy niestandardowy błąd o kodzie stanu http 403.
