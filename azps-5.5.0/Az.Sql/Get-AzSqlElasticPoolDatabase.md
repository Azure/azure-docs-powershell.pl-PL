# Get-AzSqlElasticPoolDatabase

## SYNOPSIS
Pobiera elastyczne bazy danych w elastycznej puli i wartości ich właściwości.

## SKŁADNIA

```
Get-AzSqlElasticPoolDatabase [-ElasticPoolName] <String> [-DatabaseName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzSqlElasticPoolDatabase** pobiera elastyczne bazy danych w elastycznej puli i wartości ich właściwości.
Możesz określić nazwę elastycznej bazy danych w usłudze Azure SQL Database, aby wyświetlić wartości właściwości tylko dla tej bazy danych.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie wszystkich baz danych w elastycznej puli
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

To polecenie pobiera wszystkie bazy danych w elastyczną pulę o nazwie Poszybowe Bufor01.

### Przykład 2. Korzystanie z filtrowania we wszystkich bazach danych w elastycznej puli
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01" -DatabaseName "Database*"
```

To polecenie pobiera wszystkie bazy danych w elastycznej puli o nazwie Poszybowej01, która zaczyna się od "Bazy danych".
