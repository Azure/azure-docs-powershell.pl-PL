# Stop-AzSqlSyncGroupSync

## SYNOPSIS
Zatrzymuje synchronizację grupy synchronizacji.

## SKŁADNIA

```
Stop-AzSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Stop-AzSqlSyncGroupSync** zatrzymuje synchronizację grupy synchronizacji.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Stop-AzSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

To polecenie zatrzymuje synchronizację, która jest w toku dla mojej grupy synchronizacji.
