# Invoke-AzSqlInstanceFailover

## SYNOPSIS
Failovers an Azure SQL Managed Instance.

## SKŁADNIA

```
Invoke-AzSqlInstanceFailover [-Name] <String> [-AsJob] [-PassThru] [-Force] [-ReadableSecondary]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Invoke-AzSqlInstanceFailover failovers** an Azure SQL Managed Instance.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Invoke-AzSqlInstanceFailover -ResourceGroupName "ResourceGroup01" -Name "ManagedInstance01"
```

To polecenie będzie trybu failover podstawowej repliki wystąpienia o nazwie "ManagedInstance01".

### Przykład 2
```powershell
PS C:\> Invoke-AzSqlInstanceFailover -ResourceGroupName "ResourceGroup01" -Name "ManagedInstance01" -ReadableSecondary
```

To polecenie będzie trybu failover czytelnej repliki pomocniczej zarządzanego wystąpienia "ManagedInstance01".
