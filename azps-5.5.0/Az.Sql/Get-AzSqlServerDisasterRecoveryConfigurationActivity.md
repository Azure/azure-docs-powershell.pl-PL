# Get-AzSqlServerDisasterRecoveryConfigurationActivity

## SYNOPSIS
Pobiera aktywność w przypadku konfiguracji odzyskiwania systemu serwera bazy danych.

## SKŁADNIA

```
Get-AzSqlServerDisasterRecoveryConfigurationActivity [-ServerName] <String>
 -ServerDisasterRecoveryConfigurationName <String> [-OperationId <Guid>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzSqlServerDisasterRecoveryConfigurationActivity** pobiera aktywność dla konfiguracji odzyskiwania systemu serwera bazy danych SQL.
