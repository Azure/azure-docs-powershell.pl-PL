# Remove-AzSqlInstanceKeyVaultKey

## SYNOPSIS
Usuwa klucz magazynu kluczy z wystąpienia zarządzanego przez język SQL.

## SKŁADNIA

### AddAzureRmSqlManagedInstanceKeyVaultKeyDefaultParameterSet (Domyślne)
```
Remove-AzSqlInstanceKeyVaultKey [-ResourceGroupName] <String> [-InstanceName] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyInputObjectParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-Instance] <AzureSqlManagedInstanceModel> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyResourceIdParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-InstanceResourceId] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
To Remove-AzSqlInstanceKeyVaultKey cmdlet usuwa klucz magazynu kluczy z określonego zarządzanego wystąpienia. Zwróć uwagę, że uprawnienia wystąpienia zarządzanego przez sql do magazynu klucza nie są zmieniane. Aby zmienić uprawnienia, użyj ustawienia Set-AzKeyVaultAccessPolicy. To polecenie cmdlet nie wprowadza żadnych zmian w magazynie kluczy. Aby usunąć klucz z magazynu kluczy, użyj klawisza Remove-AzureKeyVaultKey.
