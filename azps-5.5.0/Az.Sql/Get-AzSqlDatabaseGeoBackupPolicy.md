# Get-AzSqlDatabaseGeoBackupPolicy

## SYNOPSIS
Pobiera zasady tworzenia kopii geograficznej bazy danych.

## SKŁADNIA

```
Get-AzSqlDatabaseGeoBackupPolicy [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzSqlDatabaseGeoBackupPolicy** pobiera zasady tworzenia kopii zapasowej geolokalizacji zarejestrowane w tej bazie danych.
Jest to zasób usługi Azure Backup, który służy do definiowania zasad przechowywania kopii zapasowej.

## PRZYKŁADY

### Przykład 1

Pobiera zasady tworzenia kopii geograficznej bazy danych. (wygenerowane automatycznie)

```powershell
<!-- Aladdin Generated Example --> 
Get-AzSqlDatabaseGeoBackupPolicy -DatabaseName db1 -ResourceGroupName MyResourceGroup -ServerName s1
```
