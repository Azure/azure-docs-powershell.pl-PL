# Get-AzSqlElasticPoolAdvisor

## SYNOPSIS
Otrzymuje co najmniej jednego doradcę dla puli elastycznej języka Azure SQL.

## SKŁADNIA

```
Get-AzSqlElasticPoolAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -ElasticPoolName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzSqlElasticPoolAdvisor** pobiera co najmniej jednego doradcę puli elastycznej języka Azure SQL dla puli elastycznej języka Azure SQL.
