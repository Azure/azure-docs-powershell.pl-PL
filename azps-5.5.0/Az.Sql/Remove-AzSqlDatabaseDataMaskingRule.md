# Remove-AzSqlDatabaseDataMaskingRule

## SYNOPSIS
Usuwa regułę maskowania danych z bazy danych.

## SKŁADNIA

```
Remove-AzSqlDatabaseDataMaskingRule [-PassThru] [-Force] -SchemaName <String> -TableName <String>
 -ColumnName <String> [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzSqlDatabaseDataMaskingRule** usuwa określoną regułę maskowania danych z bazy danych Azure SQL.
Regułę maskowania danych można usunąć, używając parametrów *ResourceGroupName,* *ServerName,* *DatabaseName* i *RuleId* w celu zidentyfikowania reguły usunięcia tego polecenia cmdlet.
To polecenie cmdlet jest również obsługiwane przez usługę SQL Server Stretch Database na platformie Azure.

## PRZYKŁADY

### Przykład 1. Usuwanie reguły maskowania danych bazy danych
```
PS C:\>Remove-AzSqlDatabaseDataMaskingRule -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01" -SchemaName "dbo" -TableName  "table1" -ColumnName "column1"
```

To polecenie usuwa nazwę reguły Rule01 zdefiniowaną dla bazy danych Database01.
Baza danych znajduje się na serwerze Server01 i jest przypisana do grupy zasobów ResourceGroup01.
