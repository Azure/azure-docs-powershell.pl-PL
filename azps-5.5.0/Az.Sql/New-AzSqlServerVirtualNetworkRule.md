# New-AzSqlServerVirtualNetworkRule

## SYNOPSIS
Tworzy regułę wirtualnej sieci programu Azure SQL Server. 

## SKŁADNIA

```
New-AzSqlServerVirtualNetworkRule -VirtualNetworkRuleName <String> -VirtualNetworkSubnetId <String>
 [-IgnoreMissingVnetServiceEndpoint] [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Tworzy regułę wirtualnej sieci programu Azure SQL Server. Reguły sieci wirtualnej służą do łączenia programu Azure SQL Server z określoną siecią wirtualną w celu ograniczenia dostępu w usłudze Azure SQL Server tylko do sieci wirtualnej. 

## PRZYKŁADY

### Przykład 1
```
PS C:\> $virtualNetworkRule = New-AzSqlServerVirtualNetworkRule -ResourceGroupName rg -ServerName serverName -VirtualNetworkRuleName virtualNetworkRuleName -VirtualNetworkSubnetId virtualNetworkSubnetId
```

Tworzy regułę sieci wirtualnej programu Azure SQL Server
