# Remove-AzSqlSyncGroup

## SYNOPSIS
Usuwa grupę synchronizacji usługi Azure SQL Database.

## SKŁADNIA

```
Remove-AzSqlSyncGroup [-Name] <String> [-Force] [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzSqlSyncGroup** usuwa grupę synchronizacji usługi Azure SQL Database.

## PRZYKŁADY

### Przykład 1. Usuwanie grupy synchronizacji
```
PS C:\>Remove-AzSqlSyncGroup -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "database01" -Name "syncGroup01"
```

To polecenie usuwa grupę synchronizacji bazy danych Azure SQL o nazwie syncGroup01.
