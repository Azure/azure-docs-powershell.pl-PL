# Get-AzSqlDatabaseExpanded

## SYNOPSIS
Pobiera bazę danych i jej rozszerzone wartości właściwości.

## SKŁADNIA

```
Get-AzSqlDatabaseExpanded [-ServerName] <String> [[-DatabaseName] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzSqlDatabaseExpanded** pobiera bazę danych i jej rozszerzone wartości właściwości.
To polecenie cmdlet jest również obsługiwane przez usługę SQL Server Stretch Database na platformie Azure.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie obiektu bazy danych z informacjami doradcy warstwy usługi
```
PS C:\> Get-AzSqlDatabaseExpanded -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

To polecenie zwraca bazę danych z rozwiniętymi właściwościami, które zawierają informacje doradcy warstwy usługi.

### Przykład 2. Obiekty bazy danych listy przy użyciu filtrowania
```
PS C:\> Get-AzSqlDatabaseExpanded -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database*"
```

To polecenie zwraca rozwinięty obiekt bazy danych dla wszystkich baz danych w programie Server01, których nazwa rozpoczyna się od "Bazy danych".
