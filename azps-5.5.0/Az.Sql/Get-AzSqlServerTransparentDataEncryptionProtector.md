# Get-AzSqlServerTransparentDataEncryptionProtector

## SYNOPSIS
Pobiera szyfrowanie przezroczystych danych (TDE, Transparent Data Encryption)

## SKŁADNIA

```
Get-AzSqlServerTransparentDataEncryptionProtector [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie Get-AzSqlServerTransparentDataEncryptionProtector cmdlet pobiera informacje na temat TDE.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie transparentnego szyfrowania danych (TDE, Transparent Data Encryption)
```
PS C:\> Get-AzSqlServerTransparentDataEncryptionProtector -ServerName 'ContosoServer' -ResourceGroup 'ContosoResourceGroup'
```

To polecenie pobiera nazwę TDE serwera o nazwie ContosoServer w grupie zasobów o nazwie ContosoResourceGroup.
ResourceGroupName ServerName Type ServerKeyVaultKeyName
----------------- ----------                   ---- ---------------------
ContosoResourceGroup ContosoServer ServiceManaged ServiceManaged
