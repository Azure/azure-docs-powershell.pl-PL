# Get-AzPolicyRemediation

## SYNOPSIS
Pobiera działania naprawcze dotyczące zasad.

## SKŁADNIA

### SubscriptionScope (domyślne)
```
Get-AzPolicyRemediation [-Top <Int32>] [-Filter <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByName
```
Get-AzPolicyRemediation -Name <String> [-Scope <String>] [-ManagementGroupName <String>]
 [-ResourceGroupName <String>] [-Top <Int32>] [-IncludeDetail] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GenericScope
```
Get-AzPolicyRemediation -Scope <String> [-Top <Int32>] [-Filter <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ManagementGroupScope
```
Get-AzPolicyRemediation -ManagementGroupName <String> [-Top <Int32>] [-Filter <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceGroupScope
```
Get-AzPolicyRemediation -ResourceGroupName <String> [-Top <Int32>] [-Filter <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceId
```
Get-AzPolicyRemediation -ResourceId <String> [-Top <Int32>] [-IncludeDetail]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzPolicyRemediation** pobiera wszystkie działania naprawcze dotyczące zasad w zakresie lub w określonym działaniach naprawczych.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie wszystkich działań naprawczych w ramach bieżącej subskrypcji zasad
```
PS C:\> Select-AzSubscription -Subscription "My Subscription"
PS C:\> Get-AzPolicyRemediation
```

To polecenie pobiera wszystkie działania naprawcze utworzone w ramach subskrypcji o nazwie "Moja subskrypcja" lub pod nią.

### Przykład 2. Uzyskiwanie określonych środków zaradczych dotyczących zasad i szczegółów wdrożenia
```
PS C:\> Get-AzPolicyRemediation -ResourceGroupName "myResourceGroup" -Name "remediation1" -IncludeDetail
```

To polecenie pobiera działania naprawcze o nazwie "remediation1" z grupy zasobów "myResourceGroup". Zostaną uwzględnione szczegóły zasobów, których działania naprawcze zostaną uwzględnione.

### Przykład 3. Uzyskiwanie 10 środków zaradczych zasad w grupie zarządzania z filtrami opcjonalnymi
```
PS C:\> Get-AzPolicyRemediation -ManagementGroupName "mg1" -Top 10 -Filter "PolicyAssignmentId eq '/providers/Microsoft.Management/managementGroups/mg1/providers/Microsoft.Authorization/policyAssignments/pa1'"
```

To polecenie otrzymuje maksymalnie 10 działań naprawczych w zasadach od grupy zarządzania o nazwie "mg1". Zostaną pobrane tylko działania naprawcze dotyczące zasad dla danego przydziału zasad.
