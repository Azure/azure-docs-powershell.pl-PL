# Remove-AzPolicyRemediation

## SYNOPSIS
Usuwa działania naprawcze dotyczące zasad.

## SKŁADNIA

### ByName (Default)
```
Remove-AzPolicyRemediation -Name <String> [-Scope <String>] [-ManagementGroupName <String>]
 [-ResourceGroupName <String>] [-AllowStop] [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzPolicyRemediation -ResourceId <String> [-AllowStop] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzPolicyRemediation -InputObject <PSRemediation> [-AllowStop] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzPolicyRemediation** usuwa działania naprawcze dotyczące zasad.

## PRZYKŁADY

### Przykład 1. Usunięcie działania naprawczego zasad w zakresie grupy zasobów
```
PS C:\> Remove-AzPolicyRemediation -ResourceGroupName "myRG" -Name "remediation1"
```

To polecenie usuwa działania naprawcze o nazwie "remediacja1" w grupie zasobów "mójRG".

### Przykład 2. Usuwanie środków zaradczych grupy zarządzania za pośrednictwem funkcji pipingu
```
PS C:\> $remediation = Get-AzPolicyRemediation -ManagementGroupName "mg1" -Name "remediation1"
PS C:\> $remediation | Remove-AzPolicyRemediation -Confirm
```

To polecenie usuwa działania naprawcze o nazwie "remediacja1" z grupy zarządzania "mg1". Przed usunięciem zasobu zostanie wyświetlony monit o potwierdzenie.

### Przykład 3. Anulowanie i usuwanie działań naprawczych zasad
```
PS C:\> Remove-AzPolicyRemediation -ResourceGroupName "myRG" -Name "remediation1" -AllowStop
```

To polecenie usuwa działania naprawcze o nazwie "remediacja1" w grupie zasobów "mójRG". Jeśli działania naprawcze są w toku, zostaną one anulowane przed usunięciem.
