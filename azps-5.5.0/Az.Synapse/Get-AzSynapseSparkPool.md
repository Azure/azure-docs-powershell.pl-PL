# Get-AzSynapseSparkPool

## SYNOPSIS
Pobiera pulę przebiegu w czasie analizy Synapse.

## SKŁADNIA

### GetByNameParameterSet (domyślne)
```
Get-AzSynapseSparkPool [-ResourceGroupName <String>] -WorkspaceName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByParentObjectParameterSet
```
Get-AzSynapseSparkPool [-Name <String>] -WorkspaceObject <PSSynapseWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzSynapseSparkPool -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzSynapseSparkPool** pobiera informacje o puli przebiegu w czasie usługi Azure Synapse Analytics.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Get-AzSynapseSparkPool -WorkspaceName ContosoWorkspace
```

To polecenie pobiera wszystkie pule przebiegu w przebiegu w obszarze roboczym.

### Przykład 2
```powershell
PS C:\> Get-AzSynapseSparkPool -WorkspaceName ContosoWorkspace -Name ContosoSparkPool
```

To polecenie pobiera pulę przebiegu w przebiegu w obszarze roboczym ContosoWorkspace o nazwie ContosoSparkPool.

### Przykład 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseSparkPool
```

To polecenie pobiera wszystkie pule przebiegu w przebiegu w obszarze roboczym w ramach potoku.

### Przykład 4
```powershell
PS C:\> Get-AzSynapseSparkPool -ResourceId "/subscriptions/21686af7-58ec-4f4d-9c68-f431f4db4edd/resourceGroups/ContosoResourceGroup/providers/Microsoft.Synapse/workspaces/ContosoWorkspace/bigDataPools/ContosoSparkPool"
```

To polecenie pobiera pulę przebiegu w czasie z określonym identyfikatorem zasobu.
