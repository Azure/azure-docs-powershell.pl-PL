## OPIS
Polecenie **cmdlet Set-AzSynapseIntegrationRuntime** aktualizuje środowisko uruchomieniowe integracji o określone parametry.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Set-AzSynapseIntegrationRuntime -WorkspaceName ContosoWorkspace -Name 'test-selfhost-ir' -Description 'New description'
```

Polecenie cmdlet aktualizuje opis środowiska uruchomieniowego integracji o nazwie "test-selfhost-ir".

### Przykład 2
```powershell
PS C:\> Set-AzSynapseIntegrationRuntime -WorkspaceName ContosoWorkspace -Name 'test-selfhost-ir' `
                                        -SharedIntegrationRuntimeResourceId '/subscriptions/b3ee3a7f-7614-4644-ad07-afa832620b4b/resourceGroups/rg-test-dfv2/providers/Microsoft.DataFactory/factories/test-df-eu2/integrationruntimes/test-selfhost-ir' -Type "SelfHosted"
```

Polecenie cmdlet dodaje obszar roboczy, aby korzystać z udostępnionego środowiska uruchomieniowego integracji. Podczas `-SharedIntegrationRuntimeResourceId` używania parametru `-Type` musi być również uwzględniony. Zwróć uwagę, że obszar roboczy musi mieć przyznane uprawnienie do używania środowiska uruchomieniowego integracji przed uruchomieniem polecenia cmdlet.
