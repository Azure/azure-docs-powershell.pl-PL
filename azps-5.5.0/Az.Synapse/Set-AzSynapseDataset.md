# Set-AzSynapseDataset

## SYNOPSIS
Tworzy lub aktualizuje zestaw danych w obszarze roboczym.

## SKŁADNIA

### SetByName (Domyślna)
```
Set-AzSynapseDataset -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapseDataset -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzSynapseDataset** tworzy zestaw danych lub aktualizuje istniejący zestaw danych w obszarze roboczym.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Set-AzSynapseDataset -WorkspaceName ContosoWorkspace -Name ContosoDataset -DefinitionFile "C:\\samples\\Dataset.json"
```

To polecenie tworzy w obszarze roboczym zestaw danych ContosoDataset o nazwie ContosoWorkspace.
Polecenie bazuje na danych w pliku Dataset.jsdanych.
