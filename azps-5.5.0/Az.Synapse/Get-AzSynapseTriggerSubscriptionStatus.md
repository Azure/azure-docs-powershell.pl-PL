# Get-AzSynapseTriggerSubscriptionStatus

## SYNOPSIS
Uzyskiwanie stanu subskrypcji wyzwalacza zdarzenia dla określonych zdarzeń usługi zewnętrznej.

## SKŁADNIA

### GetByName (Domyślna)
```
Get-AzSynapseTriggerSubscriptionStatus -WorkspaceName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseTriggerSubscriptionStatus -WorkspaceObject <PSSynapseWorkspace> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByInputObject
```
Get-AzSynapseTriggerSubscriptionStatus -InputObject <PSTriggerResource>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzSynapseTriggerSubscriptionStatus** pobiera stan subskrypcji wyzwalacza zdarzenia do określonych zdarzeń usługi zewnętrznej. Wyzwalacza nie można uruchomić, dopóki nie zostanie zwrócony stan "Włączony".

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Get-AzSynapseTriggerSubscriptionStatus -WorkspaceName ContosoWorkspace -Name ContosoTrigger
```

To polecenie pobierze stan subscribtion wyzwalacza o nazwie ContosoTrigger do zdarzeń usługi zewnętrznej.

### Przykład 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseTriggerSubscriptionStatus -Name ContosoTrigger
```

To polecenie pobierze stan subscribtion wyzwalacza o nazwie ContosoTrigger do zdarzeń usługi zewnętrznej za pośrednictwem potoku.

### Przykład 3
```powershell
PS C:\> $trigger = Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
PS C:\> $trigger | Get-AzSynapseTriggerSubscriptionStatus
```

To polecenie pobierze stan subscribtion wyzwalacza o nazwie ContosoTrigger do zdarzeń usługi zewnętrznej za pośrednictwem potoku.
