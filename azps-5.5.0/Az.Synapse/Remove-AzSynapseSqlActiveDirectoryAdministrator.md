# Remove-AzSynapseSqlActiveDirectoryAdministrator

## SYNOPSIS
Usuwa administratora usługi Azure AD dla obszaru roboczego analizy Synapse.

## SKŁADNIA

### RemoveByNameParameterSet (Domyślne)
```
Remove-AzSynapseSqlActiveDirectoryAdministrator [-ResourceGroupName <String>] -WorkspaceName <String>
 [-PassThru] [-AsJob] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### RemoveByInputObjectParameterSet
```
Remove-AzSynapseSqlActiveDirectoryAdministrator -InputObject <PSSynapseWorkspace> [-PassThru] [-AsJob] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByResourceIdParameterSet
```
Remove-AzSynapseSqlActiveDirectoryAdministrator -ResourceId <String> [-PassThru] [-AsJob] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Remove-AzSynapseSqlActiveDirectoryAdministrator** usuwa administratora usługi Azure Active Directory (Azure AD) dla obszaru roboczego analizy Synapse.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Remove-AzSynapseSqlActiveDirectoryAdministrator -WorkspaceName ContosoWorkspace
```

To polecenie usuwa administratora usługi Azure AD dla obszaru roboczego o nazwie ContosoWorkspace.
