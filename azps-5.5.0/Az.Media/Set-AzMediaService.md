# Set-AzMediaService

## SYNOPSIS
Modyfikuje określone właściwości istniejącej usługi multimedialnej.

## SKŁADNIA

```
Set-AzMediaService [-ResourceGroupName] <String> [-AccountName] <String> [-Tag <Hashtable>]
 [-StorageAccounts <PSStorageAccount[]>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzMediaService** modyfikuje określone właściwości istniejącej usługi multimediów.

## PRZYKŁADY

### Przykład 1. Modyfikowanie istniejącej usługi multimediów
```
PS C:\>$Tags = @{"tag1" = "value1"; "tag2" = "value2"}
Set-AzMediaService -ResourceGroupName "ResourceGroup123" -AccountName "MediaService001" -Tag $Tags -StorageAccounts $StorageAccounts
```

Pierwsze polecenie tworzy serię tagów i przechowuje je w zmiennej o nazwie $Tags.
Drugie polecenie aktualizuje usługę multimediów o nazwie MediaService001, która należy do grupy zasobów o nazwie ResourceGroup123, przy użyciu tagów przechowywanych w $Tags zmienną.
To polecenie korzysta również z tablicy obiektów konfiguracji magazynu przechowywanych w $StorageAccounts danych.
