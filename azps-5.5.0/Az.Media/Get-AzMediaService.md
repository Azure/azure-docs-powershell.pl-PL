# Get-AzMediaService

## SYNOPSIS
Pobiera informacje o usłudze multimediów.

## SKŁADNIA

### ResourceGroupParameterSet
```
Get-AzMediaService [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### AccountNameParameterSet
```
Get-AzMediaService [-ResourceGroupName] <String> [-AccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzMediaService** pobiera informacje o usłudze multimediów.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie wszystkich usług multimedialnych w grupie zasobów
```
PS C:\>Get-AzMediaService -ResourceGroupName "ResourceGroup001"
```

To polecenie pobiera właściwości wszystkich usług multimedialnych w grupie zasobów o nazwie ResourceGroup001.

### Przykład 2. Uzyskiwanie właściwości usługi multimediów
```
PS C:\>Get-AzMediaService -ResourceGroupName "ResourceGroup002" -AccountName "MediaService1"
```

To polecenie pobiera właściwości usługi multimediów o nazwie MediaService1, która należy do grupy zasobów o nazwie ResourceGroup002.
