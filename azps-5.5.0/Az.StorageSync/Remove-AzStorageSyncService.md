# Remove-AzStorageSyncService

## SYNOPSIS
To polecenie spowoduje usunięcie określonej usługi synchronizacji przestrzeni dyskowej.

## SKŁADNIA

### StringParameterSet (Domyślne)
```
Remove-AzStorageSyncService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzStorageSyncService [-InputObject] <PSStorageSyncService> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzStorageSyncService [-ResourceId] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
To polecenie spowoduje usunięcie określonej usługi synchronizacji przestrzeni dyskowej. Usługę synchronizacji przestrzeni dyskowej można usunąć tylko wtedy, gdy najpierw zostaną usunięte wszystkie zawarte w niej grupy synchronizacji i zarejestrowane serwery.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Remove-AzStorageSyncService -Force -ResourceGroupName "myResourceGroup" -Name "myStorageSyncServiceName"
```

To polecenie spowoduje usunięcie usługi synchronizacji przestrzeni dyskowej.
