# Set-AzStorageSyncService

## SYNOPSIS
To polecenie ustawia usługę synchronizacji magazynu w grupie zasobów.

## SKŁADNIA

```
Set-AzStorageSyncService [-ResourceGroupName] <String> [-Name] <String> [-Location] <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Usługa synchronizacji przestrzeni dyskowej to zasób najwyższego poziomu dla usługi Azure File Sync. To polecenie ustawia usługę synchronizacji magazynu w grupie zasobów. Zalecamy utworzenie jak najbędszej niezbędnej do odróżnienia różnych grup serwerów w organizacji. Usługa synchronizacji przestrzeni dyskowej zawiera grupy synchronizacji, a także działa jako element docelowy w celu zarejestrowania serwerów. Serwer można zarejestrować tylko w jednej usłudze synchronizacji przestrzeni dyskowej. Jeśli serwery zawsze muszą uczestniczyć w synchronizacji tego samego zestawu plików, zarejestruj je w tej samej usłudze synchronizacji przestrzeni dyskowej.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Set-AzStorageSyncService -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName" -IncomingTrafficPolicy "AllowAllTraffic"
```

To polecenie skonfiguruje usługę synchronizacji przestrzeni dyskowej.
