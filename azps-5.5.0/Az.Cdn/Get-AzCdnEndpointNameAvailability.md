# Get-AzCdnEndpointNameAvailability

## SYNOPSIS
Pobiera stan dostępności punktu końcowego sieci CDN.

## SKŁADNIA

```
Get-AzCdnEndpointNameAvailability -EndpointName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzCdnEndpointNameAvailability** uzyskuje stan dostępności punktu końcowego azure dostarczania zawartości (CDN, Content Delivery Network) platformy Azure.
