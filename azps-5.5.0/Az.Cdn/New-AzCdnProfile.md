# New-AzCdnProfile

## SYNOPSIS
Tworzy profil sieci CDN.

## SKŁADNIA

```
New-AzCdnProfile -ProfileName <String> -Location <String> -Sku <PSSkuName> -ResourceGroupName <String>
 [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzCdnProfile** tworzy profil usługi Azure Content Delivery Network (CDN).

## PRZYKŁADY

### Przykład 1

Tworzy profil sieci CDN. (wygenerowane automatycznie)

<!-- Aladdin Generated Example -->


```powershell
New-AzCdnProfile -Location westus -ProfileName <String> -ResourceGroupName myresourcegroup -Sku Standard_Verizon
```
