# Get-AzCdnProfileSsoUrl

## SYNOPSIS
Pobiera adres URL logowania pojedynczego profilu sieci CDN.

## SKŁADNIA

### ByFieldsParameterSet (Domyślne)
```
Get-AzCdnProfileSsoUrl -ProfileName <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByObjectParameterSet
```
Get-AzCdnProfileSsoUrl -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzCdnProfileSsoUrl** pobiera adres URL logowania jednokrotnego profilu usługi Azure Content Delivery Network (CDN).
Ten adres URL umożliwia użytkownikom łączenie się z portalem uzupełniającym i korzystanie z dodatkowych funkcji sieci CDN.
