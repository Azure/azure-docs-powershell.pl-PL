# Moduł Az.ResourceGraph
## Opis
Microsoft Azure PowerShell: polecenia cmdlet ResourceGraph

## Polecenia cmdlet Az.ResourceGraph
### [Get-AzResourceGraphQuery](Get-AzResourceGraphQuery.md)
Pobierz jedno zapytanie wykresu według jego nazwy zasobu.

### [New-AzResourceGraphQuery](New-AzResourceGraphQuery.md)
Utwórz nowe zapytanie wykresu.

### [Remove-AzResourceGraphQuery](Remove-AzResourceGraphQuery.md)
Usuwanie zapytania wykresu.

### [Search-AzGraph](Search-AzGraph.md)
Wysyła zapytania do zasobów zarządzanych przez usługę Azure Resource Manager.

### [Update-AzResourceGraphQuery](Update-AzResourceGraphQuery.md)
Aktualizuje zapytanie wykresu, które już zostało dodane.
