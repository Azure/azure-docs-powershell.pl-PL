# Set-AzDtlAllowedVMSizesPolicy

## SYNOPSIS
Ustawia zasady dozwolonego rozmiaru maszyn wirtualnych laboratorium w laboratorium testów deweloperów.

## SKŁADNIA

### Włącz (domyślne)
```
Set-AzDtlAllowedVMSizesPolicy [[-VmSizes] <String[]>] [-Enable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Wyłącz
```
Set-AzDtlAllowedVMSizesPolicy [[-VmSizes] <String[]>] [-Disable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzDtlAllowedVMSizesPolicy** ustawia zasady dozwolonych rozmiarów maszyn wirtualnych, określającą listę dozwolonych rozmiarów maszyn wirtualnych w laboratorium.
Polecenie cmdlet używa określonej grupy zasobów i nazwy laboratorium do ustawienia zasad.
