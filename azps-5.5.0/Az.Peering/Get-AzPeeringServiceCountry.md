# Get-AzPeeringServiceCountry

## SYNOPSIS
Zawiera listę krajów dostępnych do obsługi komunikacji równorzędnej.

## SKŁADNIA

```
Get-AzPeeringServiceCountry [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Zawiera listę krajów dostępnych do obsługi komunikacji równorzędnej.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Get-AzPeeringServiceCountry
```

Lista krajów dostępnych do obsługi komunikacji równorzędnej.
