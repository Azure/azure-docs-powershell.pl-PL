# Get-AzAppServicePlan

## SYNOPSIS
Pobiera plan usługi azure app service w określonej grupie zasobów.

## SKŁADNIA

### S1
```
Get-AzAppServicePlan [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Get-AzAppServicePlan [-Location] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzAppServicePlan** pobiera plan usługi Azure App Service w określonej grupie zasobów.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie planu usługi aplikacji z grupy zasobów
```
PS C:\>Get-AzAppServicePlan -ResourceGroupName "Default-Web-WestUS" -Name "ContosoASP"
```

To polecenie pobiera plan usługi aplikacji o nazwie ContosoASP należący do grupy zasobów o nazwie Default-Web-WestUS.

### Przykład 2. Uzyskiwanie wszystkich planów usługi aplikacji w lokalizacji
```
PS C:\>Get-AzAppServicePlan -Location "West US"
```

To polecenie pobiera wszystkie plany usługi aplikacji znajdujące się w regionie "Zachód".
