# Get-AzDeletedWebApp

## SYNOPSIS
Usuwa aplikacje sieci Web z subskrypcji.

## SKŁADNIA

```
Get-AzDeletedWebApp [[-ResourceGroupName] <String>] [[-Name] <String>] [[-Slot] <String>] [-Location <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzDeletedWebApp** zwraca wszystkie usunięte aplikacje sieci Web w subskrypcji. Usunięte aplikacje można opcjonalnie filtrować według grupy zasobów, nazwy i miejsca. Istnieje więcej niż jedna usunięta aplikacja o tej samej nazwie i grupie zasobów. Sprawdź czas usunięcia, aby odróżnić usunięte aplikacje o tej samej nazwie.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Get-AzDeletedWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

To polecenie pobiera usunięte aplikacje o nazwie ContosoSite należące do grupy zasobów Default-Web-WestUS.
