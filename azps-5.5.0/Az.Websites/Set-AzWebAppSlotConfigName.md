# Set-AzWebAppSlotConfigName

## SYNOPSIS
Ustawianie nazw konfiguracji slotów aplikacji sieci Web

## SKŁADNIA

### S1
```
Set-AzWebAppSlotConfigName [[-AppSettingNames] <String[]>] [[-ConnectionStringNames] <String[]>]
 [-RemoveAllAppSettingNames] [-RemoveAllConnectionStringNames] [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Set-AzWebAppSlotConfigName [[-AppSettingNames] <String[]>] [[-ConnectionStringNames] <String[]>]
 [-RemoveAllAppSettingNames] [-RemoveAllConnectionStringNames] [-WebApp] <PSSite>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzWebAppSlotConfigName** oznacza ustawienia aplikacji i parametry połączenia jako ustawienia miejsca

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Set-AzWebAppSlotConfigName -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp" -RemoveAllAppSettingNames -RemoveAllConnectionStringNames
```

To polecenie usuwa wszystkie ustawienia i parametry połączenia aplikacji Sieci Web ContosoWebApp skojarzonej z grupą zasobów Default-Web-WestUS

### Przykład 2

Ustaw nazwy konfiguracji slotów aplikacji sieci Web. (wygenerowane automatycznie)

```powershell <!-- Aladdin Generated Example --> 
Set-AzWebAppSlotConfigName -AppSettingNames <String[]> -Name 'ContosoWebApp' -ResourceGroupName 'Default-Web-WestUS'
```
