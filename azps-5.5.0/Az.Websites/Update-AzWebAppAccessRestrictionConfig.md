# Update-AzWebAppAccessRestrictionConfig

## SYNOPSIS
Aktualizuje dziedziczenie konfiguracji usługi Restiction programu Access w witrynie programu SCM dla aplikacji Azure Web App.

## SKŁADNIA

### InputValuesParameterSet (Domyślne)
```
Update-AzWebAppAccessRestrictionConfig [-ResourceGroupName] <String> [-Name] <String>
 [-ScmSiteUseMainSiteRestrictionConfig] [-SlotName <String>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Update-AzWebAppAccessRestrictionConfig [-PassThru] -InputObject <PSAccessRestrictionConfig>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Update-AzWebAppAccessRestrictionConfig** aktualizuje konfigurację ograniczeń dostępu dla aplikacji Azure Web App.

## PRZYKŁADY

### Przykład 1. Aktualizowanie witryny usługi SCM aplikacji sieci Web w celu stosowania ograniczeń dostępu z witryny głównej

W poniższym przykładzie aplikacja Sieci Web o nazwie IpRule należąca do grupy zasobów MyResourceGroup używa konfiguracji ograniczeń dostępu witryny głównej w witrynie scm.

```powershell <!-- Aladdin Generated Example --> 
Update-AzWebAppAccessRestrictionConfig -Name IpRule -ResourceGroupName MyResourceGroup -ScmSiteUseMainSiteRestrictionConfig
```
