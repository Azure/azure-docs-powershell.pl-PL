# Get-AzAutomationAccount

## SYNOPSIS
Pobiera konta automatyzacji w grupie zasobów.

## SKŁADNIA

### ByAll (Domyślne)
```
Get-AzAutomationAccount [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByAutomationAccountName
```
Get-AzAutomationAccount [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzAutomationAccount** pobiera konta usługi Azure Automation w grupie zasobów.
Aby uzyskać więcej informacji na temat kont automatyzacji, zobacz New-AzAutomationAccount cmdlet.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie wszystkich kont
```
PS C:\>Get-AzAutomationAccount -ResourceGroupName "ResourceGroup03"
```

To polecenie pobiera wszystkie konta automatyzacji w grupie zasobów o nazwie ResourceGroup03.

### Przykład 2. Uzyskiwanie konta
```
PS C:\>Get-AzAutomationAccount -ResourceGroupName "ResourceGroup03" -Name "ContosoAutomationAccount"
```

To polecenie pobiera konto automatyzacji o nazwie ContosoAutomationAccount w grupie zasobów o nazwie ContosoResourceGroup.
