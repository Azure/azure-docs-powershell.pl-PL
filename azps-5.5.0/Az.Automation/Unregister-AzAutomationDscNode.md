# Unregister-AzAutomationDscNode

## SYNOPSIS
Usuwa węzeł dsc z zarządzania za pomocą konta automatyzacji.

## SKŁADNIA

```
Unregister-AzAutomationDscNode -Id <Guid> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Unregister-AzAutomationDscNode** usuwa węzeł APS Desired State Configuration (DSC) z zarządzania za pomocą konta usługi Azure Automation.

## PRZYKŁADY

### Przykład 1. Usuwanie węzła dsc platformy Azure z zarządzania za pomocą konta automatyzacji
```
PS C:\>Unregister-AzAutomationDscNode -AutomationAccountName "Contoso17" -ResourceGroupName "ResourceGroup01" -Id 064a8929-c98b-25e4-80hh-111ca86067j8
```

To polecenie usuwa węzeł DSC, który ma określony identyfikator GUID, z zarządzania przez konto automatyzacji o nazwie Contoso17.
