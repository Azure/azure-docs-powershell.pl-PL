# Remove-AzAutomationVariable

## SYNOPSIS
Usuwa zmienną automatyzacji.

## SKŁADNIA

```
Remove-AzAutomationVariable [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzAutomationVariable** usuwa zmienną z automatyzacji platformy Azure.

## PRZYKŁADY

### Przykład 1. Usuwanie zmiennej
```
PS C:\>Remove-AzAutomationVariable -AutomationAccountName "Contoso17" -Name "StringVariable22" -Force -ResourceGroupName "ResourceGroup01"
```

To polecenie usuwa zmienną o nazwie StringVariable22 z konta automatyzacji o nazwie Contoso17.
To polecenie określa *parametr Force.*
Dlatego nie jest wyświetlany monit o potwierdzenie.
