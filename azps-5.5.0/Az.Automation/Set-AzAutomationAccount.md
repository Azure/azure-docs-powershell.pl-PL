# Set-AzAutomationAccount

## SYNOPSIS
Modyfikuje konto automatyzacji.

## SKŁADNIA

```
Set-AzAutomationAccount [-ResourceGroupName] <String> [-Name] <String> [-Plan <String>] [-Tags <IDictionary>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzAutomationAccount** modyfikuje konto usługi Azure Automation.
Aby uzyskać więcej informacji na temat kont automatyzacji, zobacz New-AzAutomationAccount cmdlet.

## PRZYKŁADY

### Przykład 1. Ustawianie tagów dla konta automatyzacji
```
PS C:\>$Tags = @{"tag01"="value01";"tag02"="value02"}
PS C:\> Set-AzAutomationAccount -Name "AutomationAccount01" -ResourceGroupName "ResourceGroup01" -Tags $Tags
```

Pierwsze polecenie przypisuje dwie pary klucz/wartość do zmiennej $Tags zmienną.
Drugie polecenie ustawia tagi dla $Tags Automatyzacja o nazwie AutomationAccount01.

### Przykład 2. Zmienianie planu dla konta automatyzacji
```
PS C:\>Set-AzAutomationAccount -Name "AutomationAccount01" -ResourceGroupName "ResourceGroup01" -Plan Basic
```

To polecenie zmienia plan na Podstawowy dla konta automatyzacji o nazwie AutomationAccount01.
