# Set-AzAutomationRunbook

## SYNOPSIS
Modyfikuje podręcznik.

## SKŁADNIA

```
Set-AzAutomationRunbook [-Name] <String> [-Description <String>] [-Tag <IDictionary>] [-LogProgress <Boolean>]
 [-LogVerbose <Boolean>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzAutomationRunbook** modyfikuje konfigurację podręcznika azure automation w usłudze APS.

## PRZYKŁADY

### Przykład 1. Włączanie szczegółowego rejestrowania dla podręcznika uruchamiania
```
PS C:\>Set-AzAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbook02" -LogVerbose $True -ResourceGroupName "ResourceGroup01"
```

To polecenie umożliwia szczegółowe rejestrowanie zadań określonego podręcznika na koncie automatyzacji platformy Azure o nazwie Contoso17.
