# Remove-AzAutomationConnectionType

## SYNOPSIS
Usuwa typ połączenia automatyzacji.

## SKŁADNIA

```
Remove-AzAutomationConnectionType [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzAutomationConnectionType** usuwa typ połączenia z automatyzacji platformy Azure.
Wszystkie połączenia skojarzone z usuniętym typem połączenia stają się bezużytelne.
Usuń je, chyba że utworzysz nowy typ połączenia spełniający następujące kryteria: 
- Typ ma taką samą nazwę jak oryginalny typ połączenia. 
- Typ ma te same definicje pól, co oryginalny typ połączenia.
Może też mieć dodatkowe pola.

## PRZYKŁADY

### Przykład 1. Usuwanie typu połączenia
```
PS C:\>Remove-AzAutomationConnectionType -AutomationAccountName "Contoso17" -Name "ContosoConnectionType" -ResourceGroupName "ResourceGroup01"
```

To polecenie usuwa typ połączenia o nazwie ContosoConnectionType z konta automatyzacji o nazwie Contoso17.
