# Stop-AzDataLakeAnalyticsJob

## SYNOPSIS
Anulowanie zadania.

## SKŁADNIA

```
Stop-AzDataLakeAnalyticsJob [-Account] <String> [-JobId] <Guid> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Stop-AzDataLakeAnalyticsJob** anuluje zadanie usługi Azure Data Lake Analytics.

## PRZYKŁADY

### Przykład 1. Anulowanie zadania
```
PS C:\>Stop-AzDataLakeAnalyticsJob -Account "ContosoAdlAccout" -JobId "a0a78d72-3fa8-4564-9b18-6becb3fda48a"
```

To polecenie anuluje zadanie z określonym identyfikatorem.
