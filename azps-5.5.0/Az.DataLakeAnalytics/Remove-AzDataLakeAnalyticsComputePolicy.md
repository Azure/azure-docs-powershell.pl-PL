# Remove-AzDataLakeAnalyticsComputePolicy

## SYNOPSIS
Usuwa określone zasady obliczeniowe usługi Azure Data Lake Analytics

## SKŁADNIA

```
Remove-AzDataLakeAnalyticsComputePolicy [-ResourceGroupName <String>] [-Account] <String> [-Name] <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Ustawienie **Remove-AzDataLakeAnalyticsComputePolicy** usuwa określone zasady obliczeniowe usługi Azure Data Lake Analytics.

## PRZYKŁADY

### Przykład 1. Usuwanie zasad obliczania
```
PS C:\>Remove-AzDataLakeAnalyticsComputePolicy -Account "contosoadla" -Name myPolicy
```

To polecenie usuwa określone zasady obliczeniowe o nazwie "myPolicy" na koncie "contosoadla".
