# Set-AzDataLakeAnalyticsFirewallRule

## SYNOPSIS
Aktualizuje regułę zapory na koncie usługi Data Lake Analytics.

## SKŁADNIA

```
Set-AzDataLakeAnalyticsFirewallRule [-Account] <String> [-Name] <String> [[-StartIpAddress] <String>]
 [[-EndIpAddress] <String>] [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzDataLakeAnalyticsFirewallRule** aktualizuje regułę zapory na koncie usługi Azure Data Lake Analytics.

## PRZYKŁADY

### Przykład 1. Aktualizowanie reguły zapory
```
PS C:\>Set-AzDataLakeAnalyticsFirewallRule -Account "ContosoAdlAcct" -Name "My firewall rule" -StartIpAddress 127.0.0.1 -EndIpAddress 127.0.0.10
```

To polecenie aktualizuje regułę zapory o nazwie "reguła mojej zapory" w koncie "ContosoAdlAcct", aby nowy zakres adresów IP: 127.0.0.1 – 127.0.0.10
