# Add-AzDataLakeAnalyticsFirewallRule

## SYNOPSIS
Dodaje regułę zapory do konta usługi Data Lake Analytics.

## SKŁADNIA

```
Add-AzDataLakeAnalyticsFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Add-AzDataLakeAnalyticsFirewallRule** dodaje regułę zapory do konta usługi Azure Data Lake Analytics.

## PRZYKŁADY

### Przykład 1. Dodawanie reguły zapory
```
PS C:\>Add-AzDataLakeAnalyticsFirewallRule -Account "ContosoAdlAcct" -Name "My firewall rule" -StartIpAddress 127.0.0.1 -EndIpAddress 127.0.0.10
```

To polecenie dodaje regułę zapory o nazwie "reguła zapory" z konta "ContosoAdlAcct" z zakresem adresów IP: 127.0.0.1 – 127.0.0.10
