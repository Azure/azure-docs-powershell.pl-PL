# Get-AzCosmosDBSqlContainer

## SYNOPSIS
Pobiera kontener sql DossDB.

## SKŁADNIA

### ByNameParameterSet (Domyślne)
```
Get-AzCosmosDBSqlContainer -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] -AccountName <String> -DatabaseName <String> [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBSqlContainer [-Name <String>] -ParentObject <PSSqlDatabaseGetResults>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Get-AzCosmosDBSqlContainer** pobiera listę wszystkich istniejących kontenerów sql Dla organizacji ResourceGroupName, AccountName i DatabaseName oraz otrzymuje jeden kontener sql dla nazwa_grupy zasobów, nazwa_konta i nazwa_bazy danych dla danej nazwy ResourceGroupName, AccountName, DatabaseName i ContainerName.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Get-AzCosmosDBSqlContainer -AccountName {accountName} -ResourceGroupName {resourceGroupName} -DatabaseName {databaseName}

Name                     : {containerName1}
Id                       : Id
Resource                 : Microsoft.Azure.Commands.CosmosDB.Models.PSSqlContainerGetPropertiesResource
```
