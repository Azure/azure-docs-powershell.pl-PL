# Remove-AzTrafficManagerProfile

## SYNOPSIS
Usuwa profil Menedżera ruchu.

## SKŁADNIA

### Pola
```
Remove-AzTrafficManagerProfile -Name <String> -ResourceGroupName <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Obiekt
```
Remove-AzTrafficManagerProfile -TrafficManagerProfile <TrafficManagerProfile> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzTrafficManagerProfile** usuwa profil usługi Azure Traffic Manager.
Określ profil do usunięcia, używając parametrów *Name (Nazwa)* i *ResourceGroupName (Nazwa* grupy zasobów).
Możesz również określić obiekt **TrafficManagerProfile** przy użyciu parametru *TrafficManagerProfile* lub użyć potoku.

## PRZYKŁADY

### Przykład 1. Usuwanie profilu określonego przez nazwę
```
PS C:\>Remove-AzTrafficManagerProfile -Name "ContosoProfile" -ResourceGroupName "ResourceGroup11"
```

To polecenie usuwa profil o nazwie ContosoProfile w grupie Zasobów11.
Polecenie wyświetli monit o potwierdzenie.

### Przykład 2. Usuwanie profilu przy użyciu potoku
```
PS C:\>Get-AzTrafficManagerProfile -Name "ContosoProfile" -ResourceGroupName "ResourceGroup11" | Remove-AzTrafficManagerProfile -Force
```

To polecenie pobiera profil o nazwie ContosoProfile w grupie Zasobów11.
Następnie polecenie przekazuje ten profil do polecenia cmdlet **Remove-AzTrafficManagerProfile** przy użyciu operatora potoku.
To polecenie cmdlet usuwa ten profil.
Polecenie określa parametr *Force.*
Dlatego nie jest wyświetlany monit o potwierdzenie.
