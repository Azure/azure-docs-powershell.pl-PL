# Get-AzWvdRegistrationInfo

## SYNOPSIS
Uzyskaj informacje dotyczące rejestracji pulpitu wirtualnego systemu Windows.

## SKŁADNIA

```
Get-AzWvdRegistrationInfo -HostPoolName <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## OPIS
Uzyskaj informacje dotyczące rejestracji pulpitu wirtualnego systemu Windows.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie tokenu rejestracji pulpitu wirtualnego systemu Windows
```powershell
PS C:\> Get-AzWvdRegistrationInfo -ResourceGroupName ResourceGroupName -HostPoolName HostPoolName
```

To polecenie otrzymuje token rejestracji pulpitu wirtualnego systemu Windows w puli hostów.
