# Restart-AzMigrateServerReplication

## SYNOPSIS
Powoduje ponowne uruchomienie replikacji dla określonego serwera.

## SKŁADNIA

### ByIDVBlockCbt (domyślne)
```
Restart-AzMigrateServerReplication -TargetObjectID <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### ByInputObjectVCbt
```
Restart-AzMigrateServerReplication -InputObject <IMigrationItem> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## OPIS
Polecenie Restart-AzMigrateServerReplication naprawia replikację dla określonego serwera.
