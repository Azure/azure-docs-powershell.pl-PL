# Start-AzMigrateTestMigrationCleanup

## SYNOPSIS
Czyści migrację testów dla serwera replikowania.

## SKŁADNIA

### ByIDVBlockCbt (domyślne)
```
Start-AzMigrateTestMigrationCleanup -TargetObjectID <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### ByInputObjectVCbt
```
Start-AzMigrateTestMigrationCleanup -InputObject <IMigrationItem> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## OPIS
Polecenie Start-AzMigrateTestMigrationCleanup cmdlet inicjuje oczyszczanie migracji testowej dla serwera replikowania.
