# Az.Communication Module
## Opis
Microsoft Azure PowerShell: polecenia cmdlet komunikacji

## Az.Communication Cmdlet
### [Get-AzCommunicationService](Get-AzCommunicationService.md)
Uzyskaj usługę CommunicationService i jej właściwości.

### [Get-AzCommunicationServiceKey](Get-AzCommunicationServiceKey.md)
Uzyskaj klucze dostępu zasobu CommunicationService.

### [New-AzCommunicationService](New-AzCommunicationService.md)
Utwórz nową usługę CommunicationService lub zaktualizuj istniejącą usługę CommunicationService.

### [New-AzCommunicationServiceKey](New-AzCommunicationServiceKey.md)
Klucz dostępu generuj ponownie usługę CommunicationService.
KluczPodstawowy i Klucz Pomocniczy nie można ponownie wygenerować jednocześnie.

### [Remove-AzCommunicationService](Remove-AzCommunicationService.md)
Operacja usunięcia usługi CommunicationService.

### [Set-AzCommunicationServiceNotificationHub](Set-AzCommunicationServiceNotificationHub.md)
Łączy Centrum powiadomień Azure z tą usługą komunikacji.

### [Update-AzCommunicationService](Update-AzCommunicationService.md)
Operacja aktualizacji istniejącej usługi CommunicationService.
