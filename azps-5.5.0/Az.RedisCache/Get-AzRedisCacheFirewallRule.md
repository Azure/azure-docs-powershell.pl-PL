# Get-AzRedisCacheFirewallRule

## SYNOPSIS
Pobierz reguły zapory ustawione w pamięci podręcznej Redis.

## SKŁADNIA

```
Get-AzRedisCacheFirewallRule [-ResourceGroupName <String>] -Name <String> [-RuleName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Jeśli **parametr RuleName** jest podany, polecenie **cmdlet Get-AzRedisCacheFirewallRule** pobiera szczegółowe informacje na temat określonej reguły zapory w usłudze Azure Redis Cache. Jeśli zostanie **określona tylko** nazwa, ta operacja uzyskuje dostęp do wszystkich reguł zapory w tej pamięci podręcznej Redis.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie reguły pojedynczej zapory
```
PS C:\>Get-AzRedisCacheFirewallRule -Name "mycache" -RuleName "ruleone"

        ResourceGroupName : myGroup
        Name              : mycache
        FirewallRuleId    : /subscriptions/a559b6fd-3a84-40bb-a450-b0db5ed37dfe/resourceGroups/myGroup/providers/Microsoft.Cache/Redis/mycache/firewallRules/ruleone
        RuleName          : ruleone
        Type              : Microsoft.Cache/Redis/firewallRules
        StartIP           : 10.0.0.1
        EndIP             : 10.0.0.32
```

To polecenie pobiera regułę zapory o nazwie ruleone z pamięci podręcznej Redis Cache o nazwie mycache.

### Przykład 2. Uzyskiwanie wszystkich reguł zapory
```
PS C:\>Get-AzRedisCacheFirewallRule -Name "mycache"

        ResourceGroupName : myGroup
        Name              : mycache
        FirewallRuleId    : /subscriptions/a559b6fd-3a84-40bb-a450-b0db5ed37dfe/resourceGroups/myGroup/providers/Microsoft.Cache/Redis/mycache/firewallRules/ruleone
        RuleName          : ruleone
        Type              : Microsoft.Cache/Redis/firewallRules
        StartIP           : 10.0.0.1
        EndIP             : 10.0.0.32

        ResourceGroupName : myGroup
        Name              : mycache
        FirewallRuleId    : /subscriptions/a559b6fd-3a84-40bb-a450-b0db5ed37dfe/resourceGroups/myGroup/providers/Microsoft.Cache/Redis/mycache/firewallRules/ruletwo
        RuleName          : ruletwo
        Type              : Microsoft.Cache/Redis/firewallRules
        StartIP           : 10.0.0.33
        EndIP             : 10.0.0.64
```

To polecenie pobiera wszystkie reguły zapory z pamięci podręcznej Redis Cache o nazwie mycache.
