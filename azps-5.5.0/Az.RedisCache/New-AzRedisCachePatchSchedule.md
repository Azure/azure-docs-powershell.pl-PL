# New-AzRedisCachePatchSchedule

## SYNOPSIS
Dodaje harmonogram poprawek.

## SKŁADNIA

```
New-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> -Entries <PSScheduleEntry[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzRedisCachePatchSchedule** dodaje harmonogram poprawek do pamięci podręcznej w usłudze Azure Redis Cache.

## PRZYKŁADY

### Przykład 1. Tworzenie i dodawanie harmonogramu poprawek w pamięci podręcznej
```
PS C:\>New-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -Entries @(New-AzRedisCacheScheduleEntry -DayOfWeek "Weekend" -StartHourUtc 2 -MaintenanceWindow "06:00:00")
```

To polecenie dodaje harmonogram poprawek do pamięci podręcznej o nazwie RedisCache06.
Parametr Entries przyjmuje jako wartość polecenie, które tworzy harmonogram za pomocą polecenia **New-AzRedisCacheScheduleEntry.**
