# Moduł Az.RedisCache
## Opis
W tym temacie są wyświetlane tematy pomocy dotyczące polecenia cmdlet usługi Azure Redis Cache.

## Az.RedisCache Cmdlet
### [Export-AzRedisCache](Export-AzRedisCache.md)
Eksportuje dane z pamięci podręcznej Azure Redis Cache do kontenera.

### [Get-AzRedisCache](Get-AzRedisCache.md)
Pobiera pamięć podręczną Redis.

### [Get-AzRedisCacheFirewallRule](Get-AzRedisCacheFirewallRule.md)
Pobierz reguły zapory ustawione w pamięci podręcznej Redis.

### [Get-AzRedisCacheKey](Get-AzRedisCacheKey.md)
Pobiera klawisze dostępu dla pamięci podręcznej Redis.

### [Get-AzRedisCacheLink](Get-AzRedisCacheLink.md)
Uzyskaj link replikacji geograficznej dla funkcji Redis Cache.

### [Get-AzRedisCachePatchSchedule](Get-AzRedisCachePatchSchedule.md)
Otrzymuje harmonogram poprawek.

### [Import-AzRedisCache](Import-AzRedisCache.md)
Importuje dane z obiektów blob do usługi Azure Redis Cache.

### [New-AzRedisCache](New-AzRedisCache.md)
Tworzy pamięć podręczną Redis.

### [New-AzRedisCacheFirewallRule](New-AzRedisCacheFirewallRule.md)
Utwórz regułę zapory w pamięci podręcznej Redis.

### [New-AzRedisCacheKey](New-AzRedisCacheKey.md)
Ponownie generuje klucz dostępu pamięci podręcznej Redis.

### [New-AzRedisCacheLink](New-AzRedisCacheLink.md)
Utwórz link replikacji geograficznej między dwoma pamięciami podręcznmi redis.

### [New-AzRedisCachePatchSchedule](New-AzRedisCachePatchSchedule.md)
Dodaje harmonogram poprawek.

### [New-AzRedisCacheScheduleEntry](New-AzRedisCacheScheduleEntry.md)
Tworzy wpis harmonogramu.

### [Remove-AzRedisCache](Remove-AzRedisCache.md)
Usuwa pamięć podręczną Redis.

### [Remove-AzRedisCacheDiagnostic](Remove-AzRedisCacheDiagnostic.md)
Wyłącza diagnostykę w usłudze Azure Redis Cache.

### [Remove-AzRedisCacheFirewallRule](Remove-AzRedisCacheFirewallRule.md)
Usuwanie reguły zapory z pamięci podręcznej programu Redis.

### [Remove-AzRedisCacheLink](Remove-AzRedisCacheLink.md)
Usuwanie linku replikacji geograficznej między dwoma pamięciami podręcznmi programu Redis.

### [Remove-AzRedisCachePatchSchedule](Remove-AzRedisCachePatchSchedule.md)
Usuwa harmonogram poprawek.

### [Reset-AzRedisCache](Reset-AzRedisCache.md)
Powoduje ponowne uruchomienie węzłów pamięci podręcznej.

### [Set-AzRedisCache](Set-AzRedisCache.md)
Modyfikuje pamięć podręczną Redis.

### [Set-AzRedisCacheDiagnostic](Set-AzRedisCacheDiagnostic.md)
Włącza diagnostykę w usłudze Azure Redis Cache.
