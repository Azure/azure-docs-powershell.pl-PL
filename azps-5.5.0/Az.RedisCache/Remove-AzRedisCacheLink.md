# Remove-AzRedisCacheLink

## SYNOPSIS
Usuwanie łącza replikacji geograficznej między dwoma pamięciami podręczni redis.

## SKŁADNIA

```
Remove-AzRedisCacheLink -PrimaryServerName <String> -SecondaryServerName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Usuwanie łącza replikacji geograficznej między dwoma pamięciami podręczni redis.

## PRZYKŁADY

### Przykład 1. Usuwanie linku replikacji geograficznej
```
PS C:\>Remove-AzRedisCacheLink -PrimaryServerName "mycache1" -SecondaryServerName "mycache2"
```

To polecenie usuwa linki replikacji geograficznej, gdzie funkcja Redis Cache o nazwie mycache1 jest podstawowa, a funkcja Redis Cache o nazwie mycache2 jest pomocnicza.
