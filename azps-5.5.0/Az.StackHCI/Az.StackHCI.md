# Moduł Az.StackHCI
## Opis
Program Microsoft Azure PowerShell: polecenia cmdlet rejestracji HCI usługi Azure Stack

## Polecenia cmdlet Az.StackHCI
### [Register-AzStackHCI](Register-AzStackHCI.md)
Register-AzStackHCI utworzy zasób w chmurze Microsoft.AzureStackHCI reprezentujący klaster lokalnym i zarejestruje klastr lokalnego na platformie Azure.

### [Test-AzStackHCIConnection](Test-AzStackHCIConnection.md)
Test-AzStackHCIConnection sprawdza łączność lokalnych węzłów grupowanych z usługami Azure wymaganymi przez usługę Azure Stack HCI.

### [Wyrejestrzanie-AzStackHCI](Unregister-AzStackHCI.md)
Unregister-AzStackHCI usunąć zasób chmury Microsoft.AzureStackHCI reprezentujący klaster lokalnego i wyrejestruje klaster lokalnego z platformą Azure.
Zarejestrowane informacje dostępne w klastrze są używane do wyrejestrowania klastrów, jeśli nie zostaną przekazane żadne parametry.
