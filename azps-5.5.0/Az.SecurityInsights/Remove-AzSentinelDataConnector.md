# Remove-AzSentinelDataConnector

## SYNOPSIS
Usuwanie łącznika danych.

## SKŁADNIA

### DataConnectorId (domyślna)
```
Remove-AzSentinelDataConnector -ResourceGroupName <String> -WorkspaceName <String> -DataConnectorId <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzSentinelDataConnector -InputObject <PSSentinelDataConnector> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzSentinelDataConnector** trwale usuwa łącznik danych z określonego obszaru roboczego.
Obiekt **DataConnector** można przekazać za pomocą operatora potoku lub ewentualnie określić wymagane parametry.
Możesz użyć parametru Confirm i $ConfirmPreference zmienną programu Windows PowerShell, aby określić, czy polecenie cmdlet monituje o potwierdzenie.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Remove-AzSentinelDataConnector -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -DataConnectorId "MyDataConnectorId"
```

To polecenie usuwa program DataConnector z obszaru roboczego.
