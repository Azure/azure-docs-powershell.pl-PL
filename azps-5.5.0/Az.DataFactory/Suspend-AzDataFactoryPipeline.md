# Suspend-AzDataFactoryPipeline

## SYNOPSIS
Zawiesza potok w usłudze Azure Data Factory.

## SKŁADNIA

### ByFactoryName (Default)
```
Suspend-AzDataFactoryPipeline [-Name] <String> [-DataFactoryName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Suspend-AzDataFactoryPipeline [-Name] <String> [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Suspend-AzDataFactoryPipeline** zawiesza potok w usłudze Azure Data Factory.
Potok można wznowić za pomocą polecenia cmdlet Resume-AzDataFactoryPipeline cmdlet.

## PRZYKŁADY

### Przykład 1. Zawieszanie potoku
```
PS C:\>Suspend-AzDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikiSample" -DataFactoryName "WikiADF"
Confirm
Are you sure you want to suspend pipeline 'DPWikisample' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

To polecenie zawiesza potok o nazwie DPWikiSample w zakładzie danych o nazwie WikiADF.
Polecenie zwróci wartość $True.
