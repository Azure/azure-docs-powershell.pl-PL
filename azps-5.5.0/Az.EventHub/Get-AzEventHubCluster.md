# Get-AzEventHubCluster

## SYNOPSIS
Pobiera szczegółowe informacje dotyczące pojedynczego klastra centrum zdarzeń lub listę klastrów centrum zdarzeń.

## SKŁADNIA

```
Get-AzEventHubCluster [-ResourceGroupName] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie Get-AzEventHubCluster zwraca szczegóły klastra Centrum zdarzeń lub listę wszystkich klastrów centrum zdarzeń w danej grupie zasobów.
Jeśli podano nazwę klastra, są zwracane szczegóły jednego klastra.
Jeśli nazwa klastra nie jest podana, jest zwracana lista wszystkich klastrów w określonej grupie zasobów.
