# Disable-AzStorageStaticWebsite

## SYNOPSIS
Wyłącz statyczną witrynę internetową dla konta magazynu platformy Azure.

## SKŁADNIA

```
Disable-AzStorageStaticWebsite [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Disable-AzStorageWebsite** wyłącza statyczną witrynę internetową dla konta usługi Azure Storage.

## PRZYKŁADY

### Przykład 1. Wyłączenie statycznej witryny sieci Web dla konta magazynu platformy Azure
```
C:\PS>Disable-AzStorageStaticWebsite
```

To polecenie wyłącza statyczną witrynę internetową dla konta magazynu platformy Azure.
