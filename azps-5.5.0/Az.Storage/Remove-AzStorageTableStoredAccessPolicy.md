# Remove-AzStorageTableStoredAccessPolicy

## SYNOPSIS
Usuwa przechowywane zasady dostępu z tabeli magazynu platformy Azure.

## SKŁADNIA

```
Remove-AzStorageTableStoredAccessPolicy [-Table] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzStorageTableStoredAccessPolicy** usuwa przechowywane zasady dostępu z tabeli magazynu platformy Azure.

## PRZYKŁADY

### Przykład 1. Usuwanie przechowywanych zasad dostępu z tabeli magazynu
```
PS C:\>Remove-AzStorageTableStoredAccessPolicy -Table "MyTable" -Policy "Policy05"
```

To polecenie usuwa zasady o nazwie Zasady05 z tabeli magazynu o nazwie Moja tabela.
