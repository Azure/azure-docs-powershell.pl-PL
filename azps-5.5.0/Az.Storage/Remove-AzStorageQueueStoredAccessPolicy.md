# Remove-AzStorageQueueStoredAccessPolicy

## SYNOPSIS
Usuwa przechowywane zasady dostępu z kolejki magazynu platformy Azure.

## SKŁADNIA

```
Remove-AzStorageQueueStoredAccessPolicy [-Queue] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzStorageQueueStoredAccessPolicy** usuwa zasady dostępu przechowywanego z kolejki magazynu platformy Azure.

## PRZYKŁADY

### Przykład 1. Usuwanie przechowywanych zasad dostępu z kolejki magazynu
```
PS C:\>Remove-AzStorageQueueStoredAccessPolicy -Queue "MyQueue" -Policy "Policy04"
```

To polecenie usuwa z kolejki magazynu zasady dostępu o nazwie Zasady04 o nazwie Moje kolejkowanie.
