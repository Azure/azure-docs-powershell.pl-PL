# New-AzStorageQueueStoredAccessPolicy

## SYNOPSIS
Tworzy przechowywane zasady dostępu dla kolejki magazynu platformy Azure.

## SKŁADNIA

```
New-AzStorageQueueStoredAccessPolicy [-Queue] <String> [-Policy] <String> [-Permission <String>]
 [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzStorageQueueStoredAccessPolicy** tworzy zasady dostępu przechowywanego dla kolejki magazynu platformy Azure.

## PRZYKŁADY

### Przykład 1. Tworzenie zasad dostępu przechowywanego w kolejce magazynu
```
PS C:\>New-AzStorageQueueStoredAccessPolicy -Queue "MyQueue" -Policy "Policy01"
```

To polecenie tworzy zasady dostępu o nazwie Zasady01 w kolejce magazynu o nazwie Moje kolejkowanie.
