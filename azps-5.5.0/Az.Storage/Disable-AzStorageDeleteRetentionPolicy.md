# Disable-AzStorageDeleteRetentionPolicy

## SYNOPSIS
Wyłącz zasady przechowywania usuwania dla usługi obiektów blob magazynu platformy Azure.

## SKŁADNIA

```
Disable-AzStorageDeleteRetentionPolicy [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Disable-AzStorageDeleteRetentionPolicy** wyłącza zasady przechowywania usuwania dla usługi obiektów blob magazynu platformy Azure.

## PRZYKŁADY

### Przykład 1. Wyłączanie zasad przechowywania usuwania dla usługi obiektów blob
```
C:\PS>Disable-AzStorageDeleteRetentionPolicy
```

To polecenie wyłącza zasady przechowywania usuwania dla usługi obiektów blob.
