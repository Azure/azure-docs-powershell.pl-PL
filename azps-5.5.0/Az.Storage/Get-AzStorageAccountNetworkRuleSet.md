# Get-AzStorageAccountNetworkRuleSet

## SYNOPSIS
Uzyskiwanie właściwości NetWorkRule konta magazynu

## SKŁADNIA

```
Get-AzStorageAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzStorageAccountNetworkRuleSet** pobiera właściwość NetworkRule konta magazynu

## PRZYKŁADY

### Przykład 1. Uzyskiwanie właściwości NetworkRule określonego konta magazynu
```
PS C:\> Get-AzStorageAccountNetworkRuleSet  -ResourceGroupName "rg1" -AccountName "mystorageaccount"
```

To polecenie pobiera właściwość NetworkRule określonego konta magazynu
