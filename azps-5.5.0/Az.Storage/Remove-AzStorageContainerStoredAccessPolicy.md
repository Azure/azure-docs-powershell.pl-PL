# Remove-AzStorageContainerStoredAccessPolicy

## SYNOPSIS
Usuwa przechowywane zasady dostępu z kontenera magazynu platformy Azure.

## SKŁADNIA

```
Remove-AzStorageContainerStoredAccessPolicy [-Container] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzStorageContainerStoredAccessPolicy** usuwa przechowywane zasady dostępu z kontenera magazynu platformy Azure.

## PRZYKŁADY

### Przykład 1. Usuwanie przechowywanych zasad dostępu z kontenera magazynu
```
PS C:\>Remove-AzStorageContainerStoredAccessPolicy -Container "MyContainer" -Policy "Policy03"
```

To polecenie usuwa zasady dostępu o nazwie Zasady03 z przechowywanego kontenera o nazwie MyContainer.

Określona wartość jest wartością bezwzględną i nie jest mnożona przez liczbę podstawową.
