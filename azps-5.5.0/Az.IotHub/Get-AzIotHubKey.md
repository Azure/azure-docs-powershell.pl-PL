# Get-AzIotHubKey

## SYNOPSIS
Pobiera klucz aplikacji IotHub.

## SKŁADNIA

### ResourceSet (Default)
```
Get-AzIotHubKey [-ResourceGroupName] <String> [-Name] <String> [[-KeyName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdSet
```
Get-AzIotHubKey [-HubId] <String> [[-KeyName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Pobiera klucz aplikacji IotHub.
Możesz wyświetlić listę wszystkich kluczy lub przefiltrować listę według konkretnej nazwy klucza.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie wszystkich kluczy
```
PS C:\> Get-AzIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Pobiera wszystkie klucze dla usługi IotHub o nazwie "myiothub"

### Przykład 2. Uzyskiwanie informacji dotyczących określonego klucza
```
PS C:\> Get-AzIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub" -KeyName "iothubowner"
```

Pobiera informacje dotyczące klucza o nazwie "iothubowner" dla usługi IotHub o nazwie "myiothub"
