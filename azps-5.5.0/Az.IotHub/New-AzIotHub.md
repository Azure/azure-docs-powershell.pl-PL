# New-AzIotHub

## SYNOPSIS
Tworzy nową iotHub.

## SKŁADNIA

```
New-AzIotHub -ResourceGroupName <String> -Name <String> -SkuName <PSIotHubSku> -Units <Int64>
 -Location <String> [-Properties <PSIotHubInputProperties>] [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Tworzy nową iotHub.
Możesz utworzyć usługę IotHub przy użyciu właściwości domyślnych lub określić właściwości wprowadzania.

## PRZYKŁADY

### Przykład 1. Tworzenie nowej aplikacji IotHub z właściwościami domyślnymi
```
PS C:\> $tags = @{}
PS C:\> $tags.Add('key1','value1')
PS C:\> New-AzIotHub -ResourceGroupName "myresourcegroup" -Name "myiothub" -SkuName "S1" -Units 1 -Location "northeurope" -Tag $tags
```

Tworzy nową iotHub o nazwie "myiothub" z sku "S1", pojemności 1 i lokalizacji "northeurope" dołączonej do tagów.

### Przykład 2. Tworzenie nowej aplikacji IotHub z ustawieniem MaxDeliveryCount kolejki CloudToDevice na wartość 20
```
PS C:\> New-AzIotHub -ResourceGroupName "myresourcegroup" -Name "myiothub" -SkuName "S1" -Units 1 -Location "northeurope" -Properties $properties
```

Tworzy nową usługę IotHub o nazwie "myiothub" dla sKU "S1", wydajności 1 i lokalizacji "northeurope" z zaawansowanymi właściwościami wejściowymi reprezentowanymi przez $properties.
$psCloudToDeviceProperties = New-Object Microsoft.Azure.Commands.Management.IotHub.Models.PSCloudToDeviceProperties -Property @{MaxDeliveryCount=20} $properties = New-Object Microsoft.Azure.Commands.Management.IotHub.Models.PSIotHubInputProperties -Property @{CloudToDevice=$psCloudToDeviceProperties} New-AzIotHub -ResourceGroupName "myresourcegroup" -Name "myiothub" -SkuName "S1" -Units 1 -Location "northeurope" -Properties $properties
