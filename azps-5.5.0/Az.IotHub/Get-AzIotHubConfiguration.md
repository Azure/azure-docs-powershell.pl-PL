# Get-AzIotHubConfiguration

## SYNOPSIS
Wyświetla listę wszystkich lub określonej konfiguracji automatycznego zarządzania urządzeniami przez usługę IoT.

## SKŁADNIA

### ResourceSet (Default)
```
Get-AzIotHubConfiguration [-ResourceGroupName] <String> [-IotHubName] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### InputObjectSet
```
Get-AzIotHubConfiguration [-InputObject] <PSIotHub> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdSet
```
Get-AzIotHubConfiguration [-ResourceId] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Uzyskaj szczegółowe informacje na temat konfiguracji automatycznego zarządzania urządzeniami IoT lub listy konfiguracji automatycznego zarządzania urządzeniami IoT w Centrum IoT.
Aby https://docs.microsoft.com/azure/iot-hub/iot-hub-automatic-device-management uzyskać więcej informacji, zobacz te informacje.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Get-AzIotHubConfiguration -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "config1"
```

Uzyskaj szczegółowe informacje na temat konfiguracji automatycznego zarządzania urządzeniami przez IoT.

### Przykład 2
```powershell
PS C:\> Get-AzIotHubConfiguration -ResourceGroupName "myresourcegroup" -IotHubName "myiothub"
```

Lista konfiguracji automatycznego zarządzania urządzeniami IoT w Centrum IoT.
