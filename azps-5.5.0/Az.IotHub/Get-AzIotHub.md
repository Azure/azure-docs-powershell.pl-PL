# Get-AzIotHub

## SYNOPSIS
Pobiera informacje o usłudze IotHub w ramach subskrypcji.

## SKŁADNIA

### ListIotHubsByResourceGroup (domyślna)
```
Get-AzIotHub [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetIotHubByName
```
Get-AzIotHub [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Pobiera informacje o usłudze IotHub w ramach subskrypcji.
Możesz wyświetlić wszystkie wystąpienia aplikacji IotHub w subskrypcji lub filtrować wyniki według grupy zasobów lub określonej nazwy usługi IotHub.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Get-AzIotHub
```

Pobiera wszystkie usługi IotHubs w subskrypcji.

### Przykład 2
```
PS C:\> Get-AzIotHub -ResourceGroupName "myresourcegroup"
```

Pobiera wszystkie usługi IotHub z subskrypcji należącej do grupy zasobów o nazwie "myresourcegroup".

### Przykład 3
```
PS C:\> Get-AzIotHub -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Pobiera informacje o usłudze IotHub o nazwie "myiothub".
