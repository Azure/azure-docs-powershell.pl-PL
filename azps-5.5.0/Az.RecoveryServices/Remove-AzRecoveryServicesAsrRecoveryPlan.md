# Remove-AzRecoveryServicesAsrRecoveryPlan

## SYNOPSIS
Usuwa określony plan odzyskiwania asr z magazynu usług odzyskiwania.

## SKŁADNIA

### ByObject (Domyślne)
```
Remove-AzRecoveryServicesAsrRecoveryPlan -InputObject <ASRRecoveryPlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByName
```
Remove-AzRecoveryServicesAsrRecoveryPlan -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzRecoveryServicesAsrRecoveryPlan** usuwa określony plan odzyskiwania z magazynu usług odzyskiwania.

## PRZYKŁADY

### Przykład 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrRecoveryPlan -RecoveryPlan $RP
```

Rozpoczyna usuwanie określonego planu odzyskiwania i zwraca zadanie asr służące do śledzenia operacji.
