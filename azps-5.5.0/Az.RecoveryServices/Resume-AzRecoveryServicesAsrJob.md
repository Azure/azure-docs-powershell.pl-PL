# Resume-AzRecoveryServicesAsrJob

## SYNOPSIS
Wznawia zawieszone zadanie odzyskiwania witryny platformy Azure.

## SKŁADNIA

### ByObject (Domyślne)
```
Resume-AzRecoveryServicesAsrJob -InputObject <ASRJob> [-Comment <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByName
```
Resume-AzRecoveryServicesAsrJob -Name <String> [-Comment <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Resume-AzRecoveryServicesAsrJob** wznawia zawieszone zadanie odzyskiwania witryny platformy Azure.

## PRZYKŁADY

### Przykład 1
```
PS C:\> $currentJob = Resume-AzRecoveryServicesAsrJob -Job $Job
```

Wznów określone zadanie, jeśli znajduje się w stanie oczekiwania lub zawieszenia, i zwróć zaktualizowany obiekt zadania asr odpowiadający zadaniu asr.
