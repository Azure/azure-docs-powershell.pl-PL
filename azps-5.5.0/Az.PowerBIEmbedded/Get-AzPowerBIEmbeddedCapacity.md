# Get-AzPowerBIEmbeddedCapacity

## SYNOPSIS
Pobiera szczegółowe informacje o osadzonej pojemności usługi PowerBI.

## SKŁADNIA

### ByCapacityOrResourceGroupOrSubscription (Default)
```
Get-AzPowerBIEmbeddedCapacity [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceId
```
Get-AzPowerBIEmbeddedCapacity -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie Get-AzPowerBIEmbeddedCapacity cmdlet pobiera szczegółowe informacje dotyczące osadzonej pojemności usługi PowerBI.
