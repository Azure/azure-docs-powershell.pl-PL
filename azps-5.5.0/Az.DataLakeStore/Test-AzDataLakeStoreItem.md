# Test-AzDataLakeStoreItem

## SYNOPSIS
Sprawdza istnienie pliku lub folderu w sklepie Data Lake Store.

## SKŁADNIA

```
Test-AzDataLakeStoreItem [-Account] <String> [-Path] <DataLakeStorePathInstance> [[-PathType] <PathType>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Test-AzDataLakeStoreItem** sprawdza istnienie pliku lub folderu w magazynie Data Lake Store.

## PRZYKŁADY

### Przykład 1. Testowanie pliku
```powershell
PS C:\>Test-AzDataLakeStoreItem -AccountName "ContosoADL" -Path "/MyFiles/Test.csv"
```

To polecenie sprawdza, czy plik Test.csv istnieje na koncie ContosoADL.

### Przykład 2

Sprawdza istnienie pliku lub folderu w sklepie Data Lake Store. (wygenerowane automatycznie)

<!-- Aladdin Generated Example -->
```powershell
Test-AzDataLakeStoreItem -Account 'ContosoADL' -Path '/MyFiles/Test.csv' -PathType Any
```
