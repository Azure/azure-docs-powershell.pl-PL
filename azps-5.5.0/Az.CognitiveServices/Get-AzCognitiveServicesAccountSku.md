# Get-AzCognitiveServicesAccountSku

## SYNOPSIS
Pobiera dostępne jednostki SKU dla konta.

## SKŁADNIA

```
Get-AzCognitiveServicesAccountSku [-Type <String>] [-Location <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzCognitiveServicesAccountSku** pobiera dostępne jednostki SKU dla konta usług Cognitive Services.
SKU to plan warstwy dla konta.
Definiuje cenę, limit połączeń i stawkę dla konta.
SKU F0 to bezpłatna warstwa.
Płatne warstwy to między innymi S0, S1, S2 i tak dalej.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> (Get-AzCognitiveServicesAccountSku -Type 'TextAnalytics' -Location "westus").Value | Select-Object -E
xpandProperty Sku;

Name     Tier
----     ----
F0       Free
S0   Standard
```
