# Remove-AzureStorageQueue

## STRESZCZENIe
Usuwa kolejkę magazynu.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureStorageQueue [-Name] <String> [-Force] [-PassThru] [-Context <IStorageContext>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureStorageQueue** umożliwia usunięcie kolejki magazynu.

## Przykłady

### Przykład 1: usuwanie kolejki magazynu według nazwy
```
PS C:\>Remove-AzureStorageQueue "ContosoQueue01"
```

To polecenie umożliwia usunięcie kolejki o nazwie ContosoQueue01.

### Przykład 2: usuwanie wielu kolejek magazynowania
```
PS C:\>Get-AzureStorageQueue "Contoso*" | Remove-AzureStorageQueue
```

To polecenie usuwa wszystkie kolejki o nazwach rozpoczynających się od contoso.
