# Stop-AzureStorageBlobCopy

## STRESZCZENIe
Zatrzymuje operację kopiowania.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

### NamePipeline (domyślny)
```
Stop-AzureStorageBlobCopy [-Blob] <String> [-Container] <String> [-Force] [-CopyId <String>]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### BlobPipeline
```
Stop-AzureStorageBlobCopy -CloudBlob <CloudBlob> [-Force] [-CopyId <String>] [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>] [-ConcurrentTaskCount <Int32>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ContainerPipeline
```
Stop-AzureStorageBlobCopy -CloudBlobContainer <CloudBlobContainer> [-Blob] <String> [-Force] [-CopyId <String>]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **stop-AzureStorageBlobCopy** zatrzymuje operację kopiowania do określonego docelowego obiektu BLOB.

## Przykłady

### Przykład 1: zatrzymywanie operacji kopiowania według nazwy
```
PS C:\>Stop-AzureStorageBlobCopy -Container "ContainerName" -Blob "BlobName" -CopyId "CopyID"
```

To polecenie zatrzymuje operację kopiowania według nazwy.

### Przykład 2: zatrzymywanie operacji kopiowania za pomocą potoku
```
PS C:\>Get-AzureStorageContainer container* | Stop-AzureStorageBlobCopy -Blob "BlobName"
```

To polecenie zatrzymuje operację kopiowania, przesyłając kontener w rurociągu, korzystając z polecenia **Get-AzureStorageContainer**.

### Przykład 3: zatrzymywanie operacji kopiowania za pomocą potoku i Get-AzureStorageBlob
```
PS C:\>Get-AzureStorageBlob -Container "ContainerName" | Stop-AzureStorageBlobCopy -Force
```

W tym przykładzie zatrzymano operację kopiowania, przesyłając kontener w potoku za pomocą polecenia cmdlet Get-AzureStorageBlob.
