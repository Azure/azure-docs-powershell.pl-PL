# Stop-AzureRmDataLakeAnalyticsJob

## STRESZCZENIe
Anuluje zadanie.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Stop-AzureRmDataLakeAnalyticsJob [-Account] <String> [-JobId] <Guid> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **stop-AzureRmDataLakeAnalyticsJob** anuluje zadanie usługi Azure Data Lake Analytics.

## Przykłady

### Przykład 1: Anulowanie zadania
```
PS C:\>Stop-AzureRmDataLakeAnalyticsJob -Account "ContosoAdlAccout" -JobId "a0a78d72-3fa8-4564-9b18-6becb3fda48a"
```

To polecenie anuluje zadanie z określonym IDENTYFIKATORem.
