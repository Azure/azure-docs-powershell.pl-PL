# Remove-AzureRmDisk

## STRESZCZENIe
Usuwa dysk.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmDisk [-ResourceGroupName] <String> [-DiskName] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmDisk** usuwa dysk.

## Przykłady

### Przykład 1
```
PS C:\> Remove-AzureRmDisk -ResourceGroupName 'ResourceGroup01' -DiskName 'Disk01' -Force;
```

To polecenie usuwa dysk o nazwie "Disk01" w grupie zasobów "ResourceGroup01".
