# Set-AzureRmVMAccessExtension

## STRESZCZENIe
Dodaje rozszerzenie VMAccess do maszyny wirtualnej.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Set-AzureRmVMAccessExtension [-UserName <String>] [-Password <String>] [-ResourceGroupName] <String>
 [-VMName] <String> [-Name <String>] [-TypeHandlerVersion <String>] [-Location <String>]
 [-DisableAutoUpgradeMinorVersion] [-ForceRerun <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzureRmVMAccessExtension** umożliwia dodanie rozszerzenia maszyny wirtualnej dostępu do maszyny wirtualnej (VMAccess) do maszyny wirtualnej.
VMAccess może zresetować nazwę użytkownika i hasło maszyny wirtualnej.

## Przykłady

### Przykład 1: Dodawanie rozszerzenia VMAccess
```
PS C:\> Set-AzureRmVMAccessExtension -ResourceGroupName "ResrouceGroup11" -Location "Central US" -VMName "VirtualMachine07" -Name "ContosoTest" -TypeHandlerVersion "2.0" -UserName "PFuller" -Password "Password"
```

To polecenie umożliwia dodanie rozszerzenia VMAccess dla maszyny wirtualnej o nazwie VirtualMachine07 w ResrouceGroup11.
To polecenie określa nazwę i typ programu obsługi VMAccess.
