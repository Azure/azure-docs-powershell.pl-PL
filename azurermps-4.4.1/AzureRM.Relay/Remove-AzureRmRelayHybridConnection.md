# Remove-AzureRmRelayHybridConnection

## STRESZCZENIe
Usuwa HybridConnection z określonej przestrzeni nazw HybridConnection.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmRelayHybridConnection** usuwa HybridConnection z określonego obszaru nazw Relay.

## Przykłady

### Przykład 1
```
PS C:\> Remove-AzureRmRelayHybridConnection -ResourceGroupName Default-ServiceBus-WestUS -Namespace TestNameSpace-Relay1 -Name TestHybridConnection
```

Usuwa HybridConnection `TestHybridConnection` z obszaru nazw `TestNameSpace-Relay1` .
