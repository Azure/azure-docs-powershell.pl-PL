# Clear-AzureRmContext

## STRESZCZENIe
Usuwanie wszystkich poświadczeń Azure, kont i informacji o subskrypcji.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Clear-AzureRmContext -Scope <ContextModificationScope> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Usuwanie wszystkich poświadczeń Azure, kont i informacji o subskrypcji.

## Przykłady

### Wyczyść kontekst globalny
```
PS C:\> Clear-AzureRmContext -Scope CurrentUser
```

Usuń wszystkie informacje dotyczące konta, abonamentu i poświadczeń dla dowolnej sesji programu PowerShell.
