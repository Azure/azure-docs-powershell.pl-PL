# Update-AzureRmServiceFabricReliability

## STRESZCZENIe
Aktualizowanie warstwy niezawodności typu węzła podstawowego w klastrze.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Update-AzureRmServiceFabricReliability [-ResourceGroupName] <String> [-Name] <String>
 -ReliabilityLevel <ReliabilityLevel> [-AutoAddNode] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Opis
W celu zaktualizowania niezawodności podstawowego typu węzła w klastrze Użyj funkcji **Update-AzureRmServiceFabricReliability** .

## Przykłady

### Przykład 1
```
PS c:> Add-AzureRmServiceFabricReliability -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -ReliabilityLevel Silver
```

To polecenie powoduje zmianę warstwy niezawodności typu węzła podstawowego na srebrny.
