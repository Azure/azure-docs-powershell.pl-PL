# New-AzureRmSiteRecoverySite

## STRESZCZENIe
Tworzy witrynę odzyskiwania witryny.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
New-AzureRmSiteRecoverySite -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **New-AzureRmSiteRecoverySite** tworzy witrynę Azure Site Recovery w bieżącym magazynie.
