# Remove-AzureRmDataLakeStoreAccount

## STRESZCZENIe
Trwale usuwa konto Data Lake Store.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmDataLakeStoreAccount** umożliwia trwałe usunięcie konta usługi Data Lake Store.

## Przykłady

### Przykład 1. Usuń konto w usłudze Data Lake Store
```
PS C:\>Remove-AzureRmDataLakeStoreAccount -Name "ContosoADL"
```

To polecenie usuwa konto o nazwie ContosoADL z usługi Data Lake Store.
