# Test-AzureRmDataLakeStoreAccount

## STRESZCZENIe
Testowanie istnienia konta usługi Data Lake Store.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Test-AzureRmDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **test-AzureRmDataLakeStoreAccount** sprawdza istnienie konta usługi Data Lake Store.

## Przykłady

### Przykład 1: testowanie konta
```
PS C:\>Test-AzureRmDataLakeStoreAccount -Name "ContosoADL"
```

To polecenie sprawdza, czy konto o nazwie ContosoADL istnieje.
