# Get-AzureRmSqlElasticPoolAdvisor

## STRESZCZENIe
Pobiera jeden lub więcej doradców dla puli elastycznej SQL Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Get-AzureRmSqlElasticPoolAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -ElasticPoolName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureRmSqlElasticPoolAdvisor** pobiera co najmniej jeden Doradca puli elastycznej SQL platformy Azure dla puli elastycznej SQL Azure.
