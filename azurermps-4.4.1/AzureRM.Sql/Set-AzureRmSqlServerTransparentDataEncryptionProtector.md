# Set-AzureRmSqlServerTransparentDataEncryptionProtector

## STRESZCZENIe
Ustawia funkcję ochrony przezroczystego szyfrowania danych (TDE) dla programu SQL Server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Set-AzureRmSqlServerTransparentDataEncryptionProtector [-Type] <EncryptionProtectorType> [[-KeyId] <String>]
 [-Force] [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Set-AzureRmSqlServerTransparentDataEncryptionProtector ustawia ochronę TDE dla programu SQL Server.
Zmiana typu funkcji ochrony TDE spowoduje obrócenie funkcji ochrony.

## Przykłady

### --------------------------Przykład 1: Ustawianie typu funkcji Ochrona danych przezroczystego szyfrowania (TDE) na wartość servicemanage--------------------------
```
PS C:\> Set-AzureRmSqlServerTransparentDataEncryptionProtector -Type ServiceManaged -ServerName 'ContosoServer' -ResourceGroup 'ContosoResourceGroup'
```

To polecenie aktualizuje typ funkcji Ochrona TDE serwera na wartość zarządzana przez usługę.

ResourceGroupName nazwa_serwera typ ServerKeyVaultKeyName
----------------- ----------                   ---- ---------------------
ContosoResourceGroup ContosoServer servicemanage

### --------------------------Przykład 2: Ustaw typ ochrony szyfrowania danych przezroczystych na magazyn kluczy platformy Azure--------------------------
```
PS C:\> Set-AzureRmSqlServerTransparentDataEncryptionProtector -Type AzureKeyVault -KeyId 'https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901' -ServerName 'ContosoServer' -ResourceGroup 'ContosoResourceGroup'
```

To polecenie aktualizuje serwer w celu użycia klucza magazynu kluczy serwera o identyfikatorze " https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901 " jako funkcji ochrony TDE.

ResourceGroupName nazwa_serwera typ ServerKeyVaultKeyName
----------------- ----------                   ---- ---------------------
ContosoResourceGroup ContosoServer AzureKeyVault contoso_contosokey_01234567890123456789012345678901
