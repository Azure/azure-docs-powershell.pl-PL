# Set-AzureRmSqlServerVirtualNetworkRule

## STRESZCZENIe
Modyfikuje konfigurację reguły sieci wirtualnej usługi Azure SQL Server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Set-AzureRmSqlServerVirtualNetworkRule -VirtualNetworkRuleName <String> -VirtualNetworkSubnetId <String>
 -ServerName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Opis
To polecenie modyfikuje konfigurację reguły sieci wirtualnej usługi Azure SQL Server.


Aby sterować zestawem reguł sieci wirtualnej na serwerze, użyj polecenia "Add-AzureRmSqlServerVirtualNetworkRule" i "Remove-AzureRmSqlServerVirtualNetworkRule".

## Przykłady

### Przykład 1
```
PS C:\> $virtualNetworkRule = Set-AzureRmSqlServerVirtualNetworkRule -ResourceGroupName rg -ServerName serverName -VirtualNetworkRuleName virtualNetworkRuleName -VirtualNetworkSubnetId virtualNetworkSubnetId
```

Modyfikuje istniejącą regułę sieci wirtualnej o identyfikator nowej podsieci sieci wirtualnej, który zawiera informacje o nowej sieci wirtualnej.
