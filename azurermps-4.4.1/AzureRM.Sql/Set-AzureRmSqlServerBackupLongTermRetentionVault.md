# Set-AzureRmSqlServerBackupLongTermRetentionVault

## STRESZCZENIe
Ustawia magazyn długoterminowy przechowywania serwera.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Set-AzureRmSqlServerBackupLongTermRetentionVault -ResourceId <String> [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzureRmSqlServerBackupLongTermRetentionVault** umożliwia ustawienie magazynu długoterminowego przechowywania zarejestrowanego na tym serwerze.
Magazyn to zasób kopii zapasowej platformy Azure służący do przechowywania danych kopii zapasowej.
