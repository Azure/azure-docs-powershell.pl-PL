# Suspend-AzureRmDataFactoryPipeline

## STRESZCZENIe
Wstrzymuje rurociąg w usłudze Azure Data Factory.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

### ByFactoryName (domyślny)
```
Suspend-AzureRmDataFactoryPipeline [-Name] <String> [-DataFactoryName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Suspend-AzureRmDataFactoryPipeline [-Name] <String> [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Suspend-AzureRmDataFactoryPipeline** zawiesza rurociąg w fabryce Azure Data Factory.
Możesz wznowić rurociąg, korzystając z polecenia cmdlet Resume-AzureRmDataFactoryPipeline.

## Przykłady

### Przykład 1: Wstrzymywanie rurociągu
```
PS C:\>Suspend-AzureRmDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikiSample" -DataFactoryName "WikiADF"
Confirm
Are you sure you want to suspend pipeline 'DPWikisample' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

To polecenie zawiesza potok o nazwie DPWikiSample w fabryce danych o nazwie WikiADF.
Polecenie zwraca wartość $True.
