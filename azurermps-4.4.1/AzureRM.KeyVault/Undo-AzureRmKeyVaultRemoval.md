# Undo-AzureRmKeyVaultRemoval

## STRESZCZENIe
Odkrycie usuniętego magazynu kluczy w stanie aktywnym.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Undo-AzureRmKeyVaultRemoval [-VaultName] <String> [-ResourceGroupName] <String> [-Location] <String>
 [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Undo-AzureRmKeyVaultRemoval** powoduje odzyskanie uprzednio usuniętego magazynu kluczy. Odzyskany magazyn będzie aktywny po odzyskaniu

## Przykłady

### Przykład 1
```
PS C:\> Undo-AzureRmKeyVaultRemoval -VaultName 'MyKeyVault' -ResourceGroupName 'MyResourceGroup' -Location 'eastus2' -Tag @{"x"= "y"}
```

To polecenie odzyska Magazyn kluczy "MyKeyVault", który został wcześniej usunięty z obszaru eastus2 i grupy zasobów "moja grupa zasobów", w stan aktywny i dostępny. Znaczniki są również zastępowane nowym znacznikiem.
