# Start-AzureRmCdnEndpoint

## STRESZCZENIe
Rozpoczynanie punktu końcowego sieci CDN.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

### Parametr ustawiony dla parametrów pól (domyślnie)
```
Start-AzureRmCdnEndpoint -EndpointName <String> -ProfileName <String> -ResourceGroupName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Zestaw parametrów dla parametrów obiektu
```
Start-AzureRmCdnEndpoint -CdnEndpoint <PSEndpoint> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Start-AzureRmCdnEndpoint** umożliwia uruchomienie punktu końcowego sieci dostarczania zawartości Azure (CDN).
