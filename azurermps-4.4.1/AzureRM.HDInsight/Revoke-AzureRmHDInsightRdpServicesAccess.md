# Revoke-AzureRmHDInsightRdpServicesAccess

## STRESZCZENIe
Wyłącza dostęp RDP do klastra systemu Windows.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Revoke-AzureRmHDInsightRdpServicesAccess [-ClusterName] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **REVOKE-AzureRmHDInsightRdpServicesAccess** wyłącza dostęp do protokołu RDP (Remote Desktop Protocol) do klastra usługi Azure HDInsight opartego na systemie Windows.

## Przykłady

### Przykład 1: wyłączanie dostępu RDP do określonego klastra
```
PS C:\>Revoke-AzureRmHDInsightRdpServicesAccess -ClusterName "your-hadoop-001"
```

To polecenie odwołuje dostęp RDP do klastra o nazwie Your-Hadoop-001.
