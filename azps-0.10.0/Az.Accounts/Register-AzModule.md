# Register-AzModule

## STRESZCZENIe
Tylko do użytku wewnętrznego — zapewnianie obsługi środowiska cmdlet dla funkcji przygenerowania

## POLECENIA

```
Register-AzModule [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Tylko do użytku wewnętrznego — zapewnianie obsługi środowiska cmdlet dla funkcji przygenerowania

## Przykłady

### Przykład 1
```powershell
PS C:\> Register-AzModule
```
