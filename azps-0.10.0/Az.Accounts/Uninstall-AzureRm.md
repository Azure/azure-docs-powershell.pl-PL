# Uninstall-AzureRm

## STRESZCZENIe
Usuwa wszystkie moduły AzureRm z komputera.

## POLECENIA

```
Uninstall-AzureRm [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Usuwa wszystkie moduły AzureRm z komputera.

## Przykłady

### Przykład 1
```
PS C:\> Uninstall-AzureRm
```

Uruchomienie tego polecenia spowoduje usunięcie wszystkich modułów AzureRm z komputera dla wersji programu PowerShell, w której jest uruchamiane polecenie cmdlet.
