# Stop-AzKeyVaultCertificateOperation

## STRESZCZENIe
Anuluje operację certyfikatu w magazynie kluczy.

## POLECENIA

```
Stop-AzKeyVaultCertificateOperation [-VaultName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **stop-AzKeyVaultCertificateOperation** anuluje operację certyfikatu w usłudze Magazyn kluczy Azure.
