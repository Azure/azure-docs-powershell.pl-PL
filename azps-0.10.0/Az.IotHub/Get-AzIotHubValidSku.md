# Get-AzIotHubValidSku

## STRESZCZENIe
Pobiera wszystkie prawidłowe jednostki SKU, do których ta IotHub może przejść.

## POLECENIA

```
Get-AzIotHubValidSku [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
Pobiera wszystkie prawidłowe jednostki SKU, do których ta IotHub może przejść.
IotHub nie może przechodzić między bezpłatnymi i płatnymi jednostkami SKU oraz odwrotnie. Jeśli chcesz to zrobić, musisz usunąć i ponownie utworzyć iothub.

## Przykłady

### Przykład 1 pobieranie prawidłowych wersji SKU
```
PS C:\> Get-AzIotHubValidSku -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Pobiera listę wszystkich wersji SKU dla IotHub o nazwie "myiothub".
