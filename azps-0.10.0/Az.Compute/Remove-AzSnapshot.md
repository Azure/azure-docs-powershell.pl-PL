# Remove-AzSnapshot

## STRESZCZENIe
Usuwa migawkę.

## POLECENIA

```
Remove-AzSnapshot [-ResourceGroupName] <String> [-SnapshotName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzSnapshot** umożliwia usunięcie migawki.

## Przykłady

### Przykład 1
```
PS C:\> Remove-AzSnapshot -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01' -Force;
```

To polecenie usuwa migawkę o nazwie "Snapshot01" w grupie zasobów "ResourceGroup01".
