# Remove-AzVmssDiagnosticsExtension

## STRESZCZENIe
Usuwa rozszerzenie diagnostyczne z VMSS.

## POLECENIA

```
Remove-AzVmssDiagnosticsExtension [-VirtualMachineScaleSet] <VirtualMachineScaleSet> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzVmssDiagnosticsExtension** usuwa rozszerzenie diagnostyczne z zestawu skali maszyny wirtualnej (VMSS).

## Przykłady

### Przykład 1: Usuwanie rozszerzenia diagnostycznego z VMSS
```
PS C:\> Remove-AzVmssDiagnosticsExtension -VirtualMachineScaleSet $VMSS -Name $extName
```

To polecenie usuwa rozszerzenie diagnostyczne z VMSS.
