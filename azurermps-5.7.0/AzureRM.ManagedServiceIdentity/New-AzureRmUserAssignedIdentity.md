# New-AzureRmUserAssignedIdentity

## STRESZCZENIe
Tworzy nową tożsamość użytkownika o przypisanej tożsamości.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
New-AzureRmUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **New-AzureRmUserAssignedIdentity** powoduje utworzenie nowej tożsamości przypisanej do użytkownika. Gdy jest używana z już istniejącą tożsamością, zaktualizowała tożsamość.
Aby dodać znaczniki Menedżera zasobów platformy Azure do tożsamości, użyj polecenia cmdlet Set-AzureRmResource.
