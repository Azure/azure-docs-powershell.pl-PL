# Get-AzureRmUserAssignedIdentity

## STRESZCZENIe
Pobiera tożsamość/tożsamości przypisane do użytkownika.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

### SuscriptionParameterSet (domyślny)
```
Get-AzureRmUserAssignedIdentity [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceGroupParameterSet
```
Get-AzureRmUserAssignedIdentity -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
**AzureRmUserAssignedIdentity** pobiera istniejące tożsamości przypisane użytkownikowi.
