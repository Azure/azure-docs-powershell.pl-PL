# Remove-AzureRmSqlSyncAgent

## STRESZCZENIe
Usuwa agenta synchronizacji usługi Azure SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmSqlSyncAgent [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmSqlSyncAgent** usuwa agenta synchronizacji Azure SQL Sync.

## Przykłady

### Przykład 1: usuwanie agenta synchronizacji
```
PS C:\>Remove-AzureRmSqlSyncAgent -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -Name "syncAgent01"
```

To polecenie usuwa agenta synchronizacji SQL Azure o nazwie syncAgent01.
