# Set-AzureRmSqlServerRecommendedActionState

## STRESZCZENIe
Aktualizuje stan zalecanej akcji programu Azure SQL Server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Set-AzureRmSqlServerRecommendedActionState -RecommendedActionName <String> -State <RecommendedActionState>
 -ServerName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Stan aktualizacji polecenia cmdlet **Set-AzureRmSqlServerRecommendedActionState** dla zalecanej akcji programu Azure SQL Server.
To polecenie cmdlet dotyczy, przywraca lub odrzuca zalecaną akcję na podstawie nowego stanu.
