# Disable-AzureStorageDeleteRetentionPolicy

## STRESZCZENIe
Wyłącz zasadę usuwania zasad przechowywania dla usługi blob magazynu Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Disable-AzureStorageDeleteRetentionPolicy [-PassThru] [-Context <IStorageContext>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **disable-AzureStorageDeleteRetentionPolicy** wyłącza zasady dotyczące przechowywania obiektów BLOB usługi Azure Storage.

## Przykłady

### Przykład 1. Wyłącz zasady usuwania zasad przechowywania dla usługi BLOB
```
C:\PS>Disable-AzureStorageDeleteRetentionPolicy
```

To polecenie wyłącza zasady usuwania zasad przechowywania dla usługi BLOB.
