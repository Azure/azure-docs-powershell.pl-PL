# Start-AzureRmSiteRecoveryTestFailoverJob

## STRESZCZENIe
Rozpoczyna test pracy awaryjnej jednostki ochrony przed odzyskiwaniem witryny.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Opis
Polecenie cmdlet **Start-AzureRmSiteRecoveryTestFailoverJob** rozpoczyna test pracy awaryjnej jednostki ochrony usługi Azure Site Recovery lub planu odzyskiwania.
Możesz sprawdzić, czy zadanie powiodło się przy użyciu Get-AzureRmSiteRecoveryJob polecenia cmdlet.
