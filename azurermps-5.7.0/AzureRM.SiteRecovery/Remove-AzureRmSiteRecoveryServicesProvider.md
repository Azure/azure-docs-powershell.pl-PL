# Remove-AzureRmSiteRecoveryServicesProvider

## STRESZCZENIe
Usuwa dostawcę usług Azure Site Recovery Services.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmSiteRecoveryServicesProvider -ServicesProvider <ASRRecoveryServicesProvider> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmSiteRecoveryServicesProvider** Usuwa dostawcę usług Azure Site Recovery Services z magazynu.
