# Get-AzureRmSiteRecoveryVaultSettings

## STRESZCZENIe
Pobiera informacje o ustawieniach bieżącego magazynu odzyskiwania witryny.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Get-AzureRmSiteRecoveryVaultSettings [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureRmSiteRecoveryVaultSettings** pobiera informacje o ustawieniach związanych z bieżącym magazynem odzyskiwania witryny platformy Azure.
