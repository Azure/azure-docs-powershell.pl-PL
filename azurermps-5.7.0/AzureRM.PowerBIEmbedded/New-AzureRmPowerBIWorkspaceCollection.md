# New-AzureRmPowerBIWorkspaceCollection

## STRESZCZENIe
Tworzy kolekcję obszarów roboczych usługi Power BI.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
New-AzureRmPowerBIWorkspaceCollection [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-Location] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **New-AzureRmPowerBIWorkspaceCollection** tworzy kolekcję obszarów roboczych usługi Power BI dla subskrypcji platformy Azure w określonej grupie zasobów i lokalizacji.

## Przykłady

### Przykład 1. Tworzenie kolekcji obszarów roboczych
```
PS C:\>New-AzureRmPowerBIWorkspaceCollection -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11" -Location "Japan West"
```

To polecenie tworzy kolekcję obszarów roboczych o nazwie WCN11 w określonej lokalizacji w grupie zasobów.
