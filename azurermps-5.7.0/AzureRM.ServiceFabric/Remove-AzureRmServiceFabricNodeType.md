# Remove-AzureRmServiceFabricNodeType

## STRESZCZENIe
Usuwanie kompletnego typu węzła z klastra.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmServiceFabricNodeType [-ResourceGroupName] <String> [-Name] <String> -NodeType <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Usuń wszystkie węzły z określonego typu węzła i typu węzła z klastra za pomocą narzędzia **Remove-AzureRmServiceFabricNodeType** . Tego polecenia nie można użyć w celu usunięcia podstawowego typu węzła.

## Przykłady

### Przykład 1
```
PS c:> Remove-AzureRmServiceFabricNodeType -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeTypeName 'nt1'
```

To polecenie usunie NodeType "NT1" z klastra.
