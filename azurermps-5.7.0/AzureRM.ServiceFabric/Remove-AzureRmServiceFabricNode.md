# Remove-AzureRmServiceFabricNode

## STRESZCZENIe
Usuwanie węzłów z określonego typu węzła z poziomu klastra.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmServiceFabricNode -NumberOfNodesToRemove <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Za pomocą narzędzia **Remove-AzureRmServiceFabricNode** można usuwać węzły z określonego typu węzła z poziomu klastra. Usuwanie jest kontynuowane tylko wtedy, gdy spełnia on metryki kondycji klastra.

## Przykłady

### Przykład 1
```
PS c:> Remove-AzureRmServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeTypeName 'nt1' -NumberOfNodesToRemove 2
```

To polecenie usunie 2 węzły z NodeType ' NT1 '.
