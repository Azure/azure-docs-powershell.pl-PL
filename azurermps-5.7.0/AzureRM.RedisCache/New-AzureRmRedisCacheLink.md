# New-AzureRmRedisCacheLink

## STRESZCZENIe
Utwórz łącze replikacji geograficznej między dwiema Redis pamięci podręcznej.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
New-AzureRmRedisCacheLink -PrimaryServerName <String> -SecondaryServerName <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Utwórz łącze replikacji geograficznej między dwiema Redis pamięci podręcznej.

## Przykłady

### Przykład 1. Tworzenie łącza między dwiema buforami
```
PS C:\>New-AzureRmRedisCacheLink -PrimaryServerName "mycache1" -SecondaryServerName "mycache2"

        PrimaryServerName   : mycache1
        SecondaryServerName : mycache2
        ProvisioningState   : Creating
```

To polecenie tworzy link replikacji geograficznej między Redis Cache mycache1 i mycache2.
