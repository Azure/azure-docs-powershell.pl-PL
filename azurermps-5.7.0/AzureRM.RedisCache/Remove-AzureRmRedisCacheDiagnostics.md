# Remove-AzureRmRedisCacheDiagnostics

## STRESZCZENIe
Wyłącza diagnostykę w pamięci podręcznej platformy Azure Redis.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmRedisCacheDiagnostics [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmRedisCacheDiagnostics** wyłącza diagnostykę w pamięci podręcznej platformy Azure Redis.

## Przykłady

### Przykład 1: wyłączanie diagnostyki
```
PS C:\>Remove-AzureRmRedisCacheDiagnostics -ResourceGroupName "ContosoResourceGroup" -Name "PeakCache" -Force
```

To polecenie wyłącza diagnostykę w określonej pamięci podręcznej platformy Azure Redis.

Spowoduje to wyłączenie diagnostyki wszystkich pamięci podręcznych usługi Azure Redis w tym samym regionie dla subskrypcji.
