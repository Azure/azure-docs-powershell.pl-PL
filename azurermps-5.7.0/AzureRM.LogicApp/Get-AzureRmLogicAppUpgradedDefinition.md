# Get-AzureRmLogicAppUpgradedDefinition

## STRESZCZENIe
Pobiera uaktualnioną definicję dla aplikacji logicznej.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Get-AzureRmLogicAppUpgradedDefinition -ResourceGroupName <String> -Name <String> -TargetSchemaVersion <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureRmLogicAppUpgradedDefinition** pobiera uaktualnioną definicję dla wersji schematu i aplikacji logicznej z grupy zasobów.
To polecenie cmdlet zwraca obiekt reprezentujący definicję uaktualnionej aplikacji logicznej.
Określ nazwę grupy zasobów, nazwę aplikacji logicznej i wersję schematu docelowego.

Ten moduł obsługuje parametry dynamiczne.
Aby użyć parametru dynamicznego, wpisz go w poleceniu.
Aby wykryć nazwy parametrów dynamicznych, wpisz znak łącznika (-) po nazwie polecenia cmdlet, a następnie naciskaj klawisz Tab, aby przechodzić między dostępnymi parametrami.
W przypadku pominięcia wymaganego parametru szablonu polecenie cmdlet monituje o podanie wartości.
