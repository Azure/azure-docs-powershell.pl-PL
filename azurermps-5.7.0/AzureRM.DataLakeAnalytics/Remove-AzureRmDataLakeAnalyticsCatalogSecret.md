# Remove-AzureRmDataLakeAnalyticsCatalogSecret

## STRESZCZENIe
Usuwa klucz tajny Data Lake Analytics.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmDataLakeAnalyticsCatalogSecret [-Account] <String> [-DatabaseName] <String> [[-Name] <String>]
 [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmDataLakeAnalyticsCatalogSecret** usuwa hasło wykazu usługi Azure Data Lake Analytics.

## Przykłady

### Przykład 1. Usuń klucz tajny
```
PS C:\>Remove-AzureRmDataLakeAnalyticsCatalogSecret -Account "ContosoAdla" -DatabaseName "DatabaseName" -Name "SecretName"
```

To polecenie usuwa określone hasło wykazu usługi Data Lake Analytics.
