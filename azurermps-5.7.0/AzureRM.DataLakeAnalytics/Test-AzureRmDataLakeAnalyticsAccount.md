# Test-AzureRmDataLakeAnalyticsAccount

## STRESZCZENIe
Sprawdza, czy jest konto usługi Data Lake Analytics.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Test-AzureRmDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **test-AzureRmDataLakeAnalyticsAccount** sprawdza, czy istnieje konto usługi Data Lake Analytics.

## Przykłady

### Przykład 1: Sprawdzanie, czy konto istnieje
```
PS C:\>Test-AzureRmDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

To polecenie sprawdza, czy konto o nazwie ContosoAdlAccount istnieje.
