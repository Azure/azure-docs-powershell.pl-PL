# Remove-AzureRmPublicIpAddress

## STRESZCZENIe
Usuwa publiczny adres IP.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmPublicIpAddress -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmPublicIpAddress** usuwa publiczny adres IP platformy Azure.

## Przykłady

### 1: usuwanie zasobu publicznego adresu IP
```
Remove-AzureRmPublicIpAddress -Name $publicIpName -ResourceGroupName $rgName
```

To polecenie usuwa zasób publiczny adres IP o nazwie $publicIpName w grupie zasób $rgName.
