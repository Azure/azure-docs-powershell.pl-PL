# Set-AzureRmRouteTable

## STRESZCZENIe
Ustawia stan celu dla tabeli tras.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Set-AzureRmRouteTable -RouteTable <PSRouteTable> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzureRmRouteTable** ustawia stan celu dla tabeli routingu platformy Azure.
