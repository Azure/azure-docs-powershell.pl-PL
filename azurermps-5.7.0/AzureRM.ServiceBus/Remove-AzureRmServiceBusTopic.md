# Remove-AzureRmServiceBusTopic

## STRESZCZENIe
Usuwa temat z określonego obszaru nazw usługi Service Bus.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmServiceBusTopic [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmServiceBusTopic** usuwa temat z określonego obszaru nazw usługi Service Bus.

## Przykłady

### Przykład 1
```
PS C:\> Remove-AzureRmServiceBusTopic -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -TopicName SB-Topic_exampl1
```

Usuwa temat `SB-Topic_exampl1` z obszaru nazw `SB-Example1` .
