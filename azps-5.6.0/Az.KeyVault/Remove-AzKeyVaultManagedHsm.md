# Remove-AzKeyVaultManagedHsm

## SYNOPSIS
Usuwa zarządzany moduł HSM.

## SKŁADNIA

### RemoveManagedHsmByName (Domyślna)
```
Remove-AzKeyVaultManagedHsm [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveManagedHsmByInputObject
```
Remove-AzKeyVaultManagedHsm [-InputObject] <PSManagedHsm> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveManagedHsmByResourceId
```
Remove-AzKeyVaultManagedHsm [-ResourceId] <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzKeyVaultManagedHsm** usuwa określony zarządzany moduł HSM.
Usuwa też wszystkie klucze zawarte w tym wystąpieniu.
Pamiętaj, że mimo że w przypadku tego polecenia cmdlet określenie grupy zasobów jest opcjonalne, należy to zrobić, aby uzyskać lepszą wydajność.

## PRZYKŁADY

### Przykład 1. Usunięcie zarządzanego hsm
```powershell
PS C:\> Remove-AzKeyVaultManagedHsm -HsmName 'myhsm' -Force

True
```

To polecenie usuwa zarządzane polecenie hsm o nazwie myhsm z bieżącej subskrypcji.

### Przykład 2. Usunięcie zarządzanego hsm z określonej grupy zasobów
```powershell
PS C:\> Remove-AzKeyVaultManagedHsm -HsmName 'myhsm' -ResourceGroupName "myrg1" -PassThru

True
```

To polecenie usuwa zarządzane hsm o nazwie myhsm z grupy zasobów o nazwie myrg1.
Jeśli nazwa grupy zasobów nie zostanie wyokreślona, polecenie cmdlet wyszuka nazwany zarządzany moduł HSM do usunięcia w bieżącej subskrypcji.
