# Add-AzKeyVaultCertificateContact

## SYNOPSIS
Dodaje kontakt do powiadomień o certyfikatach.

## SKŁADNIA

### Interakcyjna (domyślna)
```
Add-AzKeyVaultCertificateContact [-VaultName] <String> [-EmailAddress] <String[]> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByObject
```
Add-AzKeyVaultCertificateContact [-InputObject] <PSKeyVault> [-EmailAddress] <String[]> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Add-AzKeyVaultCertificateContact [-ResourceId] <String> [-EmailAddress] <String[]> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Add-AzKeyVaultCertificateContact** dodaje kontakt dla magazynu kluczy dla powiadomień o certyfikatach w magazynie kluczy platformy Azure.
Kontakt otrzymuje aktualizacje dotyczące zdarzeń, takich jak wygaśnięcie certyfikatu, odnowienie certyfikatu itp.
Te zdarzenia są określane przez zasady certyfikatów.

## PRZYKŁADY

### Przykład 1. Dodawanie kontaktu certyfikatu magazynu kluczy
```powershell
PS C:\> Add-AzKeyVaultCertificateContact -VaultName "ContosoKV01" -EmailAddress "patti.fuller@contoso.com" -PassThru

Email                    VaultName
-----                    ---------
patti.fuller@contoso.com ContosoKV01
```

To polecenie dodaje Patti Fuller jako kontakt certyfikatu dla magazynu kluczy ContosoKV01 i zwraca listę kontaktów dla magazynu "ContosoKV01".
