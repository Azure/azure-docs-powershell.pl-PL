# Set-AzIntegrationAccountCertificate

## SYNOPSIS
Modyfikuje certyfikat konta integracji.

## SKŁADNIA

```
Set-AzIntegrationAccountCertificate -ResourceGroupName <String> -Name <String> -CertificateName <String>
 [-KeyName <String>] [-KeyVersion <String>] [-KeyVaultId <String>] [-PublicCertificateFilePath <String>]
 [-Metadata <Object>] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzIntegrationAccountCertificate** modyfikuje certyfikat konta integracji.
To polecenie cmdlet zwraca obiekt reprezentujący certyfikat konta integracji.
Określanie nazwy konta integracji, nazwy grupy zasobów i nazwy certyfikatu.
Ten moduł obsługuje parametry dynamiczne.
Aby użyć parametru dynamicznego, wpisz go w poleceniu.
Aby wykryć nazwy parametrów dynamicznych, wpisz łącznik (-) po nazwie polecenia cmdlet, a następnie naciskaj klawisz Tab, aby przechodzić przez dostępne parametry.
Jeśli pominiesz wymagany parametr szablonu, polecenie cmdlet wyświetli monit o wartość.

To polecenie modyfikuje certyfikat konta integracji w określonej grupie zasobów.
