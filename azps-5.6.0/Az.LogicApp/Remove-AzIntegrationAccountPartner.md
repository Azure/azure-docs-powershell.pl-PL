# Remove-AzIntegrationAccountPartner

## SYNOPSIS
Usuwa partnera konta integracji.

## SKŁADNIA

```
Remove-AzIntegrationAccountPartner -ResourceGroupName <String> -Name <String> -PartnerName <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzIntegrationAccountPartner** usuwa partnera konta integracji z grupy zasobów.
Określ nazwę konta integracji, nazwę grupy zasobów i nazwę partnera.
Ten moduł obsługuje parametry dynamiczne.
Aby użyć parametru dynamicznego, wpisz go w poleceniu.
Aby wykryć nazwy parametrów dynamicznych, wpisz łącznik (-) po nazwie polecenia cmdlet, a następnie naciskaj klawisz Tab, aby przechodzić przez dostępne parametry.
Jeśli pominiesz wymagany parametr szablonu, polecenie cmdlet wyświetli monit o wartość.

## PRZYKŁADY

### Przykład 1. Usuwanie partnera konta integracji
```
PS C:\>Remove-AzIntegrationAccountPartner -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -PartnerName "IntegrationAccountPartner1"
```

To polecenie usuwa partnera konta integracji o nazwie IntegrationAccountPartner1.
