# Get-AzIntegrationAccountCallbackUrl

## SYNOPSIS
Pobiera adres URL zwrotnego wywołania zwrotnego konta integracji.

## SKŁADNIA

```
Get-AzIntegrationAccountCallbackUrl -ResourceGroupName <String> -Name <String> [-NotAfter <DateTime>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzIntegrationAccountCallbackUrl** pobiera adres URL zwrotnego konta integracji z grupy zasobów.
To polecenie cmdlet zwraca obiekt **CallbackUrl** reprezentujący adres URL zwrotnego konta integracji.
Określ nazwę konta integracji i nazwę grupy zasobów.
Ten moduł obsługuje parametry dynamiczne.
Aby użyć parametru dynamicznego, wpisz go w poleceniu.
Aby wykryć nazwy parametrów dynamicznych, wpisz łącznik (-) po nazwie polecenia cmdlet, a następnie naciskaj klawisz Tab, aby przechodzić przez dostępne parametry.
Jeśli pominiesz wymagany parametr szablonu, polecenie cmdlet wyświetli monit o wartość.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie adresu URL zwrotnego zwrotnego konta integracji
```
PS C:\>Get-AzIntegrationAccountCallbackUrl -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -NotAfter "03/25/2016 18:23:22"
CallBackUrl : https://<baseurl>/integrationAccounts/8811f0155a364b5e9618ba28f7180601?api-version=2015-08-01-preview&se=2016-03
              -25T18%3A23%3A22.0000000Z&sp=%2F%2Fread&sv=1.0&sig=<value>
```

To polecenie otrzymuje adres URL wywołania zwrotnego konta integracji.
