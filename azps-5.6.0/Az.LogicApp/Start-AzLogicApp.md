# Start-AzLogicApp

## SYNOPSIS
Uruchamia aplikację logiczną w grupie zasobów.

## SKŁADNIA

```
Start-AzLogicApp -ResourceGroupName <String> -Name <String> [-Parameters <Object>] -TriggerName <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Start-AzLogicApp** uruchamia aplikację logiczną przy użyciu funkcji aplikacje logic.
Określanie nazwy, grupy zasobów i wyzwalacza.
Ten moduł obsługuje parametry dynamiczne.
Aby użyć parametru dynamicznego, wpisz go w poleceniu.
Aby wykryć nazwy parametrów dynamicznych, wpisz łącznik (-) po nazwie polecenia cmdlet, a następnie naciskaj klawisz Tab, aby przechodzić przez dostępne parametry.
Jeśli pominiesz wymagany parametr szablonu, polecenie cmdlet wyświetli monit o wartość.

## PRZYKŁADY

### Przykład 1. Uruchamianie aplikacji logiki
```
PS C:\>Start-AzLogicApp -ResourceGroupName "ResourceGroup11" -Name "LogicApp03" -TriggerName "Trigger22"
```

To polecenie uruchamia aplikację logiki w grupie zasobów o nazwie Grupa Zasobów11.
