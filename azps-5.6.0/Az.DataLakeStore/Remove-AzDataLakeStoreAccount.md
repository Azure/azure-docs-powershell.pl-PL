# Remove-AzDataLakeStoreAccount

## SYNOPSIS
Trwale usuwa konto w sklepie Data Lake Store.

## SKŁADNIA

```
Remove-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzDataLakeStoreAccount** trwale usuwa konto w sklepie Data Lake Store.

## PRZYKŁADY

### Przykład 1. Usuwanie konta w sklepie Data Lake Store
```
PS C:\>Remove-AzDataLakeStoreAccount -Name "ContosoADL"
```

To polecenie usuwa konto o nazwie ContosoADL z magazynu Data Lake Store.
