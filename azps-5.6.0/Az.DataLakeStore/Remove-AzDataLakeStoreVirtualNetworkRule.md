# Remove-AzDataLakeStoreVirtualNetworkRule

## SYNOPSIS
Usuwa określoną regułę sieci wirtualnej w określonym magazynie Data Lake Store.

## SKŁADNIA

```
Remove-AzDataLakeStoreVirtualNetworkRule [-Account] <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzDataLakeStoreVirtualNetworkRule** usuwa określoną regułę sieci wirtualnej w określonym magazynie Data Lake Store.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Remove-AzDataLakeStoreVirtualNetworkRule -Account "dls" -Name "myVNET"
```

Usuwa regułę sieci wirtualnej "myVNET" z konta "dls"
