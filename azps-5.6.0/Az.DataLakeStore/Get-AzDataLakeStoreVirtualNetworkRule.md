# Get-AzDataLakeStoreVirtualNetworkRule

## SYNOPSIS
Pobiera określone reguły sieci wirtualnej w określonym magazynie Data Lake Store.
Jeśli nie zostanie określona żadna reguła sieci wirtualnej, zostanie wymieniona lista wszystkich reguł sieci wirtualnej dla konta.

## SKŁADNIA

```
Get-AzDataLakeStoreVirtualNetworkRule [-Account] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie Get-AzDataLakeStoreVirtualNetworkRule pobiera określone reguły sieci wirtualnej w określonym magazynie Data Lake Store.
Jeśli nie zostanie określona żadna reguła sieci wirtualnej, zostanie wymieniona lista wszystkich reguł sieci wirtualnej dla konta.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Get-AzDataLakeStoreVirtualNetworkRule -Account "dls" -Name "myVNET"

ResourceGroupName                :
AccountName                      :
VirtualNetworkRuleName           : myVNET
VirtualNetworkSubnetId           : /subscriptions/<subscriptionId>/resourceGroups/<resourceGroup>/providers/Microsoft.Network/virtualNetworks/myVNET/subnets/testId
IgnoreMissingVnetServiceEndpoint :
State                            :
```

Zwraca regułę sieci wirtualnej o nazwie "myVNET" z konta "dls".
