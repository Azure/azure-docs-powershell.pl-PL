# Test-AzDataLakeStoreAccount

## SYNOPSIS
Sprawdza istnienie konta w sklepie Data Lake Store.

## SKŁADNIA

```
Test-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Test-AzDataLakeStoreAccount** sprawdza istnienie konta w sklepie Data Lake Store.

## PRZYKŁADY

### Przykład 1. Testowanie konta
```
PS C:\>Test-AzDataLakeStoreAccount -Name "ContosoADL"
```

To polecenie sprawdza, czy istnieje konto o nazwie ContosoADL.
