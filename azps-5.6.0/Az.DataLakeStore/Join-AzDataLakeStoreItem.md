# Join-AzDataLakeStoreItem

## SYNOPSIS
Dołącza jeden lub więcej plików, aby utworzyć jeden plik w sklepie Data Lake Store.

## SKŁADNIA

```
Join-AzDataLakeStoreItem [-Account] <String> [-Paths] <DataLakeStorePathInstance[]>
 [-Destination] <DataLakeStorePathInstance> [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Join-AzDataLakeStoreItem** łączy jeden lub więcej plików w celu utworzenia jednego pliku w magazynie Data Lake Store.

## PRZYKŁADY

### Przykład 1. Sprzężenia dwóch elementów
```
PS C:\>Join-AzDataLakeStoreItem -AccountName "ContosoADL" -Paths "/MyFiles/File01.txt","/MyFiles/File02.txt" -Destination "/MyFiles/CombinedFile.txt"
```

To polecenie łączy File01.txt i File02.txt, aby utworzyć plik CombinedFile.txt.
