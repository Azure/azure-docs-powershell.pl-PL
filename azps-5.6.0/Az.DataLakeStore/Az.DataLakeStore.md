# Moduł Az.DataLakeStore
## Opis
Tematy w tej sekcji dokumentuj polecenia cmdlet programu Azure PowerShell dla usługi Azure Data Lake Store w ramach usługi Azure Resource Manager (ARM). Polecenia cmdlet istnieją w przestrzeni nazw Microsoft.Azure.Commands.DataLakeStore. Te polecenia cmdlet działają tylko w przypadku kont Usługi Azure Data Lake Storage Gen1.

## Az.DataLakeStore Cmdlet
### [Add-AzDataLakeStoreFirewallRule](Add-AzDataLakeStoreFirewallRule.md)
Dodaje regułę zapory do określonego konta usługi Data Lake Store.

### [Add-AzDataLakeStoreItemContent](Add-AzDataLakeStoreItemContent.md)
Dodaje zawartość do elementu w sklepie Data Lake Store.

### [Add-AzDataLakeStoreTrustedIdProvider](Add-AzDataLakeStoreTrustedIdProvider.md)
Dodaje zaufanego dostawcę tożsamości do określonego konta usługi Data Lake Store.

### [Add-AzDataLakeStoreVirtualNetworkRule](Add-AzDataLakeStoreVirtualNetworkRule.md)
Dodaje regułę sieci wirtualnej do określonego konta usługi Data Lake Store.

### [Enable-AzDataLakeStoreKeyVault](Enable-AzDataLakeStoreKeyVault.md)
Próbuje włączyć zarządzany przez użytkownika magazyn kluczy w celu szyfrowania określonego konta usługi Data Lake Store.

### [Export-AzDataLakeStoreChildItemProperty](Export-AzDataLakeStoreChildItemProperty.md)
Eksportuje właściwości (Użycie dysku i Acl) dla całego drzewa z określonej ścieżki do ścieżki wyjściowej.
