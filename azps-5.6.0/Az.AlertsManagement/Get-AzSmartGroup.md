# Get-AzSmartGroup

## SYNOPSIS
Pobiera informacje o grupach inteligentnych

## SKŁADNIA

### SmartGroupsListByFilter (domyślne)
```
Get-AzSmartGroup [-SortBy <String>] [-SortOrder <String>] [-TimeRange <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SmartGroupById
```
Get-AzSmartGroup -SmartGroupId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
**Polecenie cmdlet Get-AzSmartGroup** pobiera informacje o grupach inteligentnych.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Get-AzSmartGroup -TimeRange "1h"
```

Lista wszystkich grup inteligentnych tworzyć w ciągu ostatniej godziny. Użyj Format-List, aby uzyskać pełne szczegóły każdej grupy inteligentnej na liście.

### Przykład 2
```powershell
PS C:\> Get-AzSmartGroup -SmartGroupId "afbf1b3a-0a6c-4f19-9c9b-644ccd7b1529" | Format-List
```

Uzyskiwanie szczegółów grupy inteligentnej według identyfikatora (GUID) lub identyfikatora zasobu (Identyfikator ARM zasobów)
