# Get-AzKustoDatabasePrincipal

## SYNOPSIS
Zwraca listę głównych użytkowników bazy danych danego klastra Kusto i bazy danych.

## SKŁADNIA

```
Get-AzKustoDatabasePrincipal -ClusterName <String> -DatabaseName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## OPIS
Zwraca listę głównych użytkowników bazy danych danego klastra Kusto i bazy danych.

## PRZYKŁADY

### Przykład 1. Lista głównych podmiotów bazy danych danego klastru i bazy danych Kusto
```powershell
PS C:\> Get-AzKustoDatabasePrincipal -ResourceGroupName testrg -ClusterName testnewkustocluster -DatabaseName mykustodatabase

AppId Email                   Fqn                                                                               Name        Role  TenantName Type
----- -----                   ---                                                                               ----        ----  ---------- ----
      someuser@microsoft.com  aaduser=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx;xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx Some User   Admin Microsoft  User
      otheruser@microsoft.com aaduser=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx;xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx Other User  Admin Microsoft  User
```

Powyższe polecenie zwraca listę głównych użytkowników bazy danych danego klastra Kusto i bazy danych.
