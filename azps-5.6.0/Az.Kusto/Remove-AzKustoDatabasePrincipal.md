# Remove-AzKustoDatabasePrincipal

## SYNOPSIS
Usuń uprawnienia głównych użytkowników bazy danych.

## SKŁADNIA

### RemoveExpanded (Default)
```
Remove-AzKustoDatabasePrincipal -ClusterName <String> -DatabaseName <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-Value <IDatabasePrincipal[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### RemoveViaIdentityExpanded
```
Remove-AzKustoDatabasePrincipal -InputObject <IKustoIdentity> [-Value <IDatabasePrincipal[]>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## OPIS
Usuń uprawnienia głównych użytkowników bazy danych.

## PRZYKŁADY

### Przykład 1. Usuwanie uprawnień głównych użytkowników bazy danych
```powershell
PS C:\> Remove-AzKustoDatabasePrincipal -ResourceGroupName testrg -ClusterName testnewkustocluster -DatabaseName mykustodatabase -Value (@{Name="Some User"; Role="Admin"; Type="User"; Email="someuser@microsoft.com"})
```

VALUE <IDatabasePrincipal[]>: Lista głównych podmiotów bazy danych Kusto.
  - `Name <String>`: główna nazwa bazy danych.
  - `Role <DatabasePrincipalRole>`: Rola podmiotu bazy danych.
  - `Type <DatabasePrincipalType>`: typ podmiotu bazy danych.
  - `[AppId <String>]`: Identyfikator aplikacji — odpowiedni tylko dla typu podmiotu zabezpieczeń aplikacji.
  - `[Email <String>]`: Główny adres e-mail bazy danych, jeśli istnieje.
  - `[Fqn <String>]`: W pełni kwalifikowana nazwa podmiotu zabezpieczeń bazy danych.
