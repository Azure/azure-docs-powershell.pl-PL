# Remove-AzRedisCache

## SYNOPSIS
Usuwa pamięć podręczną Redis.

## SKŁADNIA

```
Remove-AzRedisCache [-ResourceGroupName <String>] -Name <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzRedisCache** usuwa pamięć podręczną Azure Redis Cache.

## PRZYKŁADY

### Przykład 1. Usunięcie pamięci podręcznej Redis i zwrócenie wyniku
```
PS C:\>Remove-AzRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force -PassThru
True
```

To polecenie usuwa pamięć podręczną Redis i wyświetla, czy operacja się powiedzie.

### Przykład 2. Usunięcie pamięci podręcznej Redis i nie wyświetlanie wyniku
```
PS C:\>Remove-AzRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force
```

To polecenie usuwa pamięć podręczną Redis.
Parametr *PassThru* nie jest określony, dlatego wynik operacji nie jest wyświetlany.
