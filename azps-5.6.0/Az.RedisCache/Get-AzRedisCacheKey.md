# Get-AzRedisCacheKey

## SYNOPSIS
Pobiera klawisze dostępu dla pamięci podręcznej Redis Cache.

## SKŁADNIA

```
Get-AzRedisCacheKey [-ResourceGroupName <String>] -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzRedisCacheKey** pobiera klucze dostępu dla funkcji Azure Redis Cache.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie klawiszy dostępu dla pamięci podręcznej Redis
```
PS C:\>Get-AzRedisCacheKey -ResourceGroupName "MyResourceGroup" -Name "MyCacheKey"
PrimaryKey        : pJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
SecondaryKey      : sJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
```

To polecenie pobiera klawisze dostępu o nazwie MyCacheKey.
