# Moduł Az.DevSpaces
## Opis
Polecenia umożliwiające interakcję z kontrolerem Azure DevSpaces.

## Polecenia cmdlet Az.DevSpaces
### [Get-AzDevSpacesController](Get-AzDevSpacesController.md)
Pobierz lub przygotuj listę kontrolera Azure DevSpaces.

### [New-AzDevSpacesController](New-AzDevSpacesController.md)
Utwórz nowy kontroler Azure DevSpaces.

### [Remove-AzDevSpacesController](Remove-AzDevSpacesController.md)
Usuń kontroler DevSpaces.

### [Update-AzDevSpacesController](Update-AzDevSpacesController.md)
Zaktualizuj kontroler DevSpaces, aby dodać tagi.
