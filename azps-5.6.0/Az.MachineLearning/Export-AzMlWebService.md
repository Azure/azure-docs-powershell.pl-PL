# Export-AzMlWebService

## SYNOPSIS
Eksportuje obiekt definicji usługi sieci Web jako ciąg sformatowany JSON.

## SKŁADNIA

### ExportToFile
```
Export-AzMlWebService -WebService <WebService> -OutputFile <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ExportToJSON
```
Export-AzMlWebService -WebService <WebService> [-ToJsonString] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Eksportuje obiekt definicji dla określonej usługi sieci Web jako ciąg sformatowany JSON.
Możesz od razu zwrócić ten ciąg lub zapisać go w pliku.

## PRZYKŁADY

### Przykład 1. Eksportowanie jako ciąg
```
Export-AzMlWebService -WebService $svc -ToJsonString
```

### Przykład 2. Eksportowanie do pliku
```
Export-AzMlWebService -WebService $svc -OutputFile "C:\mlservice.json"
```
