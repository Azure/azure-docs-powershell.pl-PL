# Update-AzMlCommitmentPlan

## SYNOPSIS
Aktualizuje właściwości istniejącego zasobu planu zobowiązania.

## SKŁADNIA

```
Update-AzMlCommitmentPlan -ResourceGroupName <String> -Name <String> -SkuName <String> -SkuTier <String>
 [-SkuCapacity <Int32>] [-Tag <Hashtable>] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## OPIS
Aktualizuje istniejący zasób planu zobowiązania. Zwróć uwagę, że większość właściwości planu zobowiązania jest niezmienialna i nie można ich modyfikować. Właściwości, które można modyfikować, to między innymi SKU (umożliwiająca migrowanie planu zobowiązania z jednej wersji SKU do drugiej) i Tagi.

## PRZYKŁADY

### Przykład 1. Aktualizowanie planu zobowiązania
```
Update-AzMlCommitmentPlan -ResourceGroupName "MyResourceGroup" -Name "MyCommitmentPlanName" -Tags @{'MyTagKey'='MyTagValue'}
```
