# New-AzDiskAccess

## SYNOPSIS
Tworzy zasób dostępu do dysku

## SKŁADNIA

```
New-AzDiskAccess [-ResourceGroupName] <String> [-Name] <String> [-Location] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzAccess** tworzy zasób dostępu do dysku

## PRZYKŁADY

### Przykład 1
```
PS C:\> New-AzDiskAccess -ResourceGroupName "ResourceGroup01" -Name "DiskAccess01" -Location "NorthCentralUS"
```

To polecenie spowoduje utworzenie dostępu do dysku o podanych właściwościach.
