# New-AzVmssIpTagConfig

## SYNOPSIS
Tworzy obiekt tagu IP dla interfejsu sieciowego maszyny wirtualnej.

## SKŁADNIA

```
New-AzVmssIpTagConfig [-IpTagType] <String> [-Tag <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzVmssIpTagConfig** tworzy obiekt konfiguracji tagu IP dla interfejsu sieciowego zestawu skal maszyn wirtualnych (VIRTUAL Machine Scale Set).
Określ konfigurację tego polecenia cmdlet jako parametr *IPTag* New-AzVmssIpConfig cmdlet.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $iptag = New-AzVmssIpTagConfig -IpTagType 'FirstPartyUsage' -Tag 'Sql'
PS C:\> $ipCfg = New-AzVmssIPConfig -Name 'test' -SubnetId $subnetId -IpTag $ipTag;
```

To polecenie tworzy obiekt lokalny tagu IP z typem "FirstPartyUsage" i tagiem "Sql", a następnie tworzy konfigurację adresu IP z tym tagiem IP.
