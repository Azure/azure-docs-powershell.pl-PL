# Get-AzSnapshot

## SYNOPSIS
Pobiera właściwości migawki

## SKŁADNIA

```
Get-AzSnapshot [[-ResourceGroupName] <String>] [[-SnapshotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzSnapshot** pobiera właściwości migawki.
