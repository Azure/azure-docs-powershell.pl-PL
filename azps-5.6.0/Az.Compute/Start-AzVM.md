# Start-AzVM

## SYNOPSIS
Uruchamia maszynę wirtualną platformy Azure.

## SKŁADNIA

### ResourceGroupNameParameterSetName (Domyślna)
```
Start-AzVM [-Name] <String> [-NoWait] [-ResourceGroupName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Start-AzVM [-NoWait] [-Id] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Start-AzVM** uruchamia maszynę wirtualną platformy Azure.

## PRZYKŁADY

### Przykład 1. Uruchamianie maszyny wirtualnej
```
PS C:\> Start-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

To polecenie uruchamia maszynę wirtualną o nazwie VirtualMachine07 w grupie Zasobów11.
