# Remove-AzSnapshot

## SYNOPSIS
Usuwa migawkę.

## SKŁADNIA

```
Remove-AzSnapshot [-ResourceGroupName] <String> [-SnapshotName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzSnapshot** usuwa migawkę.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Remove-AzSnapshot -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01' -Force;
```

To polecenie usuwa migawkę o nazwie "Migawka01" z grupy zasobów "ResourceGroup01".
