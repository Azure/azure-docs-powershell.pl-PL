# Remove-AzVmssDataDisk

## SYNOPSIS
Usuwa dysk danych z maszyny wirtualnej.

## SKŁADNIA

### NameParameterSet
```
Remove-AzVmssDataDisk [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ŚleParameterSet
```
Remove-AzVmssDataDisk [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [-Lun] <Int32>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzVmssDataDataS** usuwa dysk danych z wystąpienia zestawu skal maszyny wirtualnej (VIRTUAL Machine Scale Set, VIRTUALSS).

## PRZYKŁADY

### Przykład 1
```
PS C:\> Remove-AzVmssDataDisk -VirtualMachineScaleSet $vmss -Name 'DataDisk1'
```

To polecenie usuwa dysk danych o nazwie "Data Jednak1" z obiektu MASZYNY.WIRTUALNE.

### Przykład 2
```
PS C:\> Remove-AzVmssDataDisk -VirtualMachineScaleSet $vmss -Lun 0
```

To polecenie usuwa dysk danych DEFRAGMENT 0 z obiektu MASZYNY.WIRTUALNE.
