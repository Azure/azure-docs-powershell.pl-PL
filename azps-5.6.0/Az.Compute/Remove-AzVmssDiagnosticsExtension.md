# Remove-AzVmssDiagnosticsExtension

## SYNOPSIS
Usuwa rozszerzenie diagnostyki z usługi MASZYNY.WIRTUALNE.

## SKŁADNIA

```
Remove-AzVmssDiagnosticsExtension [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzVmssDiagnosticsExtension** usuwa rozszerzenie diagnostyczne z zestawu skal maszyny wirtualnej (VIRTUAL Machine Scale Set).

## PRZYKŁADY

### Przykład 1. Usunięcie rozszerzenia diagnostyki z usługi VMSS
```
PS C:\> Remove-AzVmssDiagnosticsExtension -VirtualMachineScaleSet $VMSS -Name $extName
```

To polecenie usuwa rozszerzenie diagnostyki z usługi MASZYNY.WIRTUALNE.
