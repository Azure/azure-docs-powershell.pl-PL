# Get-AzVMSqlServerExtension

## SYNOPSIS
Pobiera ustawienia rozszerzenia programu SQL Server na komputerze wirtualnym.

## SKŁADNIA

```
Get-AzVMSqlServerExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzVMSqlServerExtension** pobiera ustawienia infrastruktury programu SQL Server jako agenta usługi (IaaS) na maszyny wirtualnej.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie ustawień rozszerzenia programu SQL Server na komputerze wirtualnym
```
PS C:\> Get-AzVMSqlServerExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM07"
ExtensionName        : SqlIaaSAgent
Publisher            : Microsoft.SqlServer.Management
Version              : 1.0
State                : Enable
RoleName             : VMName
AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

To polecenie pobiera ustawienia rozszerzenia programu SQL Server na komputerze wirtualnym o nazwie ContosoVM07.

### Przykład 2. Uzyskiwanie ustawień przy użyciu potoku
```
PS C:\> Get-AzVM -ServiceName "Service08" -Name "ContosoVM22" | Get-AzVMSqlServerExtension
ExtensionName        : SqlIaaSAgent
Publisher            : Microsoft.SqlServer.Management
Version              : 1.0
State                : Enable
RoleName             : VMName
AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

To polecenie pobiera maszynę wirtualną o nazwie ContosoVM22 w usłudze Service08 przy użyciu Get-AzVM cmdlet.
Polecenie przekazuje wyniki do bieżącego polecenia cmdlet przy użyciu operatora potoku.
Bieżące polecenie pobiera ustawienia agenta IaaS programu SQL Server na tej maszyny wirtualnej.

### Przykład 3. Uzyskiwanie ustawień konkretnej wersji programu SQL Server
```
PS C:\> Get-AzVMSqlServerExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM07" -Version "1.0"
ExtensionName        : SqlIaaSAgent
Publisher            : Microsoft.SqlServer.Management
Version              : 1.0
State                : Enable
RoleName             : VMName
AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

To polecenie pobiera ustawienia wersji 1.0 rozszerzenia programu SQL Server na komputerze wirtualnym o nazwie ContosoVM07.
