# Remove-AzImageDataDisk

## SYNOPSIS
Usuwa dysk danych z obiektu obrazu.

## SKŁADNIA

```
Remove-AzImageDataDisk [-Image] <PSImage> [-Lun] <Int32> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzImageDataData Można** usunąć dysk danych z obiektu obrazu.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Get-AzImage -ResourceGroupName 'ResourceGroup01' -ImageName 'Image01' | Remove-AzImageDataDisk -Lun 1 | Update-AzImage;
```

To polecenie usuwa dysk danych jednostki logicznej o numerze 1 z istniejącego obrazu "Obraz01" w grupie zasobów "Grupa Zasobów01".
