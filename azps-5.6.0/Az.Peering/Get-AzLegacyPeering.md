# Get-AzLegacyPeering

## SYNOPSIS
Służy do konwertowania starszych zasobów komunikacji równorzędnej na zasoby usługi Azure Resource Management (ARM). 

## SKŁADNIA

```
Get-AzLegacyPeering [-PeeringLocation] <String> [-Kind] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
To polecenie służy do wyświetlania starszych zasobów komunikacji równorzędnej, które wszystkie ty konwertujesz na zasoby usługi Azure Resource Management (ARM).
