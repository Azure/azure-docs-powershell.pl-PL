# Add-AzIotHubEventHubConsumerGroup

## SYNOPSIS
Tworzy grupę klientów z aplikacji EventHub.

## SKŁADNIA

```
Add-AzIotHubEventHubConsumerGroup [-ResourceGroupName] <String> [-Name] <String>
 [-EventHubConsumerGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Tworzy grupę klienta w witrynie Eventhub skojarzoną z określoną usługą IotHub.

## PRZYKŁADY

### Przykład 1. Dodawanie grupy konsumentów do aplikacji Eventhub telemetrii
```
PS C:\> Add-AzIotHubEventHubConsumerGroup -ResourceGroupName "myresourcegroup" -Name "myiothub" -EventHubConsumerGroupName "myconsumergroup"
```

Dodaje nową grupę klientów o nazwie "myconsumergroup" do aplikacji Eventhub na wypadek zdarzeń telemetrii w aplikacji iothub o nazwie "myiothub"
