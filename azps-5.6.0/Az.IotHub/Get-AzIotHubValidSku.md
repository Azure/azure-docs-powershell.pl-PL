# Get-AzIotHubValidSku

## SYNOPSIS
Pobiera wszystkie prawidłowe jednostki SKU, do których może zostać przejście w tej usłudze IotHub.

## SKŁADNIA

```
Get-AzIotHubValidSku [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Pobiera wszystkie prawidłowe jednostki SKU, do których może zostać przejście w tej usłudze IotHub.
Aplikacja IotHub nie może przechodzić między jednostkami SKU bezpłatnymi i płatnymi i odwrotnie. Aby osiągnąć ten cel, musisz usunąć i ponownie utworzyć tę kartę.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie prawidłowych jednostki SKU
```
PS C:\> Get-AzIotHubValidSku -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Pobiera listę wszystkich jednostki SKU dla aplikacji IotHub o nazwie "myiothub"
