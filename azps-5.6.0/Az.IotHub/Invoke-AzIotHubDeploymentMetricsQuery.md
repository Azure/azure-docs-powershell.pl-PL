# Invoke-AzIotHubDeploymentMetricsQuery

## SYNOPSIS
Wywołaj zapytanie metryczne wdrożenia IoT Edge.

## SKŁADNIA

### Zestaw zasobów (domyślny)
```
Invoke-AzIotHubDeploymentMetricsQuery [-ResourceGroupName] <String> [-IotHubName] <String> -Name <String>
 -MetricName <String> [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Invoke-AzIotHubDeploymentMetricsQuery [-InputObject] <PSIotHub> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ResourceIdSet
```
Invoke-AzIotHubDeploymentMetricsQuery [-ResourceId] <String> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Oceń docelową metrykę niestandardową lub metrykę systemu zdefiniowaną we wdrożeniu IoT Edge.
Istnieją wstępnie zdefiniowane metryki systemu, które są obliczane przez centrum Iot i nie można ich dostosować.
- "Kierowane" pokazuje urządzenia Edge IoT, które są zgodne z warunkiem określania docelowych celów wdrażania.
- "Zastosowane" pokazuje kierowane urządzenia Edge IoT, które nie są kierowane przez inne wdrożenie o wyższym priorytecie.
- "Raportowanie sukcesu" pokazuje urządzenia Edge IoT, które zgłosiły, że moduły zostały wdrożone pomyślnie.
- "Błąd raportowania" pokazuje urządzenia edge IoT, które zgłosiły, że co najmniej jeden moduł nie został pomyślnie wdrożony. 
  Aby dokładniej zbadać błąd, połącz się zdalnie z tymi urządzeniami i wyświetl pliki dziennika.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Invoke-AzIotHubDeploymentMetricsQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myDeploy1" -MetricName "warningLimit"
```

Oceń metrykę niestandardowego zdefiniowanego parametru "warningLimit".

### Przykład 2
```powershell
PS C:\> Invoke-AzIotHubDeployMetric -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myDeploy1" -MetricName "Reporting Success" -MetricType "system"
```

Oceń metrykę "Raportowanie sukcesu".
