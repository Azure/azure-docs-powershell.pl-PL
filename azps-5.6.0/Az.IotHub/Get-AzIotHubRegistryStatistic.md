# Get-AzIotHubRegistryStatistic

## SYNOPSIS
Pobiera statystykę rejestru dla aplikacji IotHub.

## SKŁADNIA

```
Get-AzIotHubRegistryStatistic [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Pobiera statystykę rejestru dla aplikacji IotHub.
Informacje na temat łącznej liczby urządzeń włączonych i wyłączonych w aplikacji IotHub.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie statystyki rejestru
```
PS C:\> Get-AzIotHubRegistryStatistic -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Pobiera statystykę rejestru dla usługi IotHub o nazwie "myiothub"
