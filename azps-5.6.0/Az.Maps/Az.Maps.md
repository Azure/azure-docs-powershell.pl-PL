# Moduł Az.Maps
## Opis
W tym temacie jest wyświetlana pomoc dla polecenia cmdlet do usługi Azure Maps.

## Polecenia cmdlet az.maps
### [Get-AzMapsAccount](Get-AzMapsAccount.md)
Pobiera konto.

### [Get-AzMapsAccountKey](Get-AzMapsAccountKey.md)
Pobiera klucze interfejsu API dla konta.
Te klucze to mechanizm uwierzytelniania używany w kolejnych wywołaniach do usługi Azure Maps.

### [New-AzMapsAccount](New-AzMapsAccount.md)
Tworzy konto usługi Azure Maps.

### [New-AzMapsAccountKey](New-AzMapsAccountKey.md)
Ponownie generuje klucz konta.

### [Remove-AzMapsAccount](Remove-AzMapsAccount.md)
Usuwa konto usługi Azure Maps.
