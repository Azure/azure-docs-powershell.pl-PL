# Remove-AzWcfRelay

## SYNOPSIS
Usuwa plik WcfRelay z określonej przestrzeni nazw przekazywania.

## SKŁADNIA

```
Remove-AzWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzWcfRelay** usuwa plik WcfRelay z określonej przestrzeni nazw przekazywania.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Remove-AzWcfRelay -ResourceGroupName Default-ServiceBus-WestUS -NamespaceName TestNameSpace-Relay1 -Name TestWCFRelay1
```

Usuwa wcfRelay `TestWCFRelay1` z przestrzeni `TestNameSpace-Relay1` nazw.
