# Remove-AzRelayNamespace

## SYNOPSIS
Usuwa przestrzeń nazw z określonej grupy zasobów. 

## SKŁADNIA

```
Remove-AzRelayNamespace [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzRelayNamespace** usuwa przestrzeń nazw z określonej grupy zasobów.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Remove-AzRelayNamespace -ResourceGroupName Default-ServiceBus-WestUS -Name TestNameSpace-Relay1
```

Usuwa przestrzeń nazw Przekazywanie `TestNameSpace-Relay1` z określonej grupy `Default-ServiceBus-WestUS` zasobów.
