# Test-AzSynapseSparkPool

## SYNOPSIS
Sprawdza, czy istnieje pula przebiegu w czasie analizy Synapse.

## SKŁADNIA

### TestByNameParameterSet (domyślne)
```
Test-AzSynapseSparkPool [-ResourceGroupName <String>] -WorkspaceName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### TestByParentObjectParameterSet
```
Test-AzSynapseSparkPool -Name <String> -WorkspaceObject <PSSynapseWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Test-AzSynapseSparkPool** sprawdza, czy istnieje pula przebiegu w czasie analizy Synapse.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Test-AzSynapseSparkPool -WorkspaceName ContosoWorkspace -Name ContosoSparkPool
```

To polecenie sprawdza istnienie określonej puli przebiegu w czasie.
