# Test-AzSynapseWorkspace

## SYNOPSIS
Sprawdza, czy istnieje obszar roboczy analizy Synapse.

## SKŁADNIA

```
Test-AzSynapseWorkspace [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Test-AzSynapseWorkspace** sprawdza, czy istnieje obszar roboczy analizy Synapse.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Test-AzSynapseWorkspace -Name ContosoWorkspace
```

To polecenie sprawdza istnienie obszaru roboczego analizy Synapse.
