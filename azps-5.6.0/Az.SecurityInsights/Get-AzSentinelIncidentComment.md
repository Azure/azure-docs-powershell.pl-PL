# Get-AzSentinelIncidentComment

## SYNOPSIS
Pobierz komentarz zdarzenia.

## SKŁADNIA

### IncidentId (Default)
```
Get-AzSentinelIncidentComment -ResourceGroupName <String> -WorkspaceName <String> -IncidentId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IncidentCommentId
```
Get-AzSentinelIncidentComment -ResourceGroupName <String> -WorkspaceName <String> -IncidentId <String>
 -IncidentCommentId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId
```
Get-AzSentinelIncidentComment -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzSentinelIncidentComment** pobiera komentarz zdarzenia z określonego obszaru roboczego.
Jeśli określisz parametry *IncidentCommentId* i *IncidentId,* zostanie zwrócony jeden obiekt **IncidentComment.**
Jeśli parametr *IncidentCommentId* nie zostanie określony, zostanie zwrócona tablica zawierająca wszystkie komentarze dotyczące zdarzenia dla określonego zdarzenia w określonym obszarze roboczym.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $IncidentComments = Get-AzSentinelIncidentComment -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -IncidentId "MyIncidentId"
```

W tym przykładzie wszystkie wartości **Zdarzenia** dotyczące określonego zdarzenia są przechowywane w określonym obszarze roboczym, a następnie są przechowywane w $IncidentComments obszarze roboczym.

### Przykład 2
```powershell
PS C:\> $IncidentComment = Get-AzSentinelIncidentComment -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -IncidentId "MyIncidentId" -IncidentCommentId "MyIncidentCommentId"
```

Ten przykład pobiera wartość **"IncidentComment"** dla określonego zdarzenia w określonym obszarze roboczym, a następnie zapisuje ją w $IncidentComment obszarze roboczym.
