# Get-AzP2sVpnGatewayVpnProfile

## SYNOPSIS
Generuje i zwraca adres URL SAS klienta, aby pobrać profil VPN, aby wskazać konfigurację klienta witryny, która ma wskazać łączność z witryną P2SVpnGateway.

## SKŁADNIA

### ByP2SVpnGatewayName (domyślna)
```
Get-AzP2sVpnGatewayVpnProfile [-Name <String>] -ResourceGroupName <String> [-AuthenticationMethod <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByP2SVpnGatewayObject
```
Get-AzP2sVpnGatewayVpnProfile -InputObject <PSP2SVpnGateway> [-AuthenticationMethod <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByP2SVpnGatewayResourceId
```
Get-AzP2sVpnGatewayVpnProfile -ResourceId <String> [-AuthenticationMethod <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Get-AzP2sVpnGatewayVpnProfile** umożliwia generowanie i pobieranie adresu URL SAS klienta w celu pobrania profilu vpn w celu przekierowania konfiguracji klienta witryny do łączności z witryną P2SVpnGateway. Wskaż konfigurację klienta witryny przy użyciu tego profilu vpn, który może połączyć się tylko z tą konkretną aplikacją P2SVpnGateway.
