# Remove-AzDdosProtectionPlan

## SYNOPSIS
Usuwa plan ochrony przed DDoS.

## SKŁADNIA

```
Remove-AzDdosProtectionPlan -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie Remove-AzDdosProtectionPlan cmdlet usuwa plan ochrony przed DDoS.

## PRZYKŁADY

### Przykład 1. Usuwanie pustego planu ochrony przed DDoS
```
D:\> Remove-AzDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlan
```

W tym przypadku usuwamy plan ochrony przed DDoS zgodnie z określonymi.

### Przykład 2. Usuwanie planu ochrony przed DDoS skojarzonego z siecią wirtualną
```
D:\> $vnet = Get-AzVirtualNetwork -Name VnetName -ResourceGroupName ResourceGroupName
D:\> $vnet.DdosProtectionPlan = $null
```

Planów ochrony przed DDoS nie można usunąć, jeśli są skojarzone z siecią wirtualną. Pierwszym krokiem jest więc nieskojarzenie obu obiektów. Tutaj jest skojarzona najbardziej zaktualizowana wersja sieci wirtualnej skojarzonej z planem i ustawiamy dla właściwości **DdosProtectionPlan** wartość pustą i flagę **EnableDdosProtection** (ta flaga nie może zostać spełniona bez planu).
Następnie utrzymujemy nowy stan, rurując zmienną lokalną do **Set-AzVirtualNetwork.** W tym momencie plan nie jest już skojarzony z siecią wirtualną.
Jeśli jest to ostatni plan skojarzony z planem, możemy usunąć plan ochrony DDoS za pomocą polecenia Remove-AzDdosProtectionPlan.
