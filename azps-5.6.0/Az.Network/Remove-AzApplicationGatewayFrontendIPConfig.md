# Remove-AzApplicationGatewayFrontendIPConfig

## SYNOPSIS
Usuwa konfigurację frontoronu ip z bramy aplikacji.

## SKŁADNIA

```
Remove-AzApplicationGatewayFrontendIPConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzApplicationGatewayFrontendIPConfig** usuwa adres IP frontendu z bramy aplikacji platformy Azure.

## PRZYKŁADY

### Przykład 1. Usuwanie konfiguracji frontoronu adresu IP
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayFrontendIPConfig -ApplicationGateway $AppGw -Name "FrontEndIP02"
```

Pierwsze polecenie pobiera bramę aplikacji o nazwie ApplicationGateway01 i przechowuje ją w $AppGw zmienną.
Drugie polecenie usuwa konfigurację frontoronu IP o nazwie FrontEndIP02 z bramy aplikacji przechowywanej w $AppGw.
