# Get-AzRouteFilterRuleConfig

## SYNOPSIS
Pobiera regułę filtrowania trasy w filtrze trasy.

## SKŁADNIA

```
Get-AzRouteFilterRuleConfig [-Name <String>] -RouteFilter <PSRouteFilter>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzRouteFilterRuleConfig** pobiera regułę filtrowania trasy lub listę reguł filtrowania trasy w filtrze trasy.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $rf = Get-AzRouteFilter -Name "MyRouteFilter" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzRouteFilterRuleConfig -RouteFilter $rf -Name "Rule01"
PS C:\> Get-AzRouteFilterRuleConfig -RouteFilter $rf
```

Pierwsze polecenie pobiera filtr trasy o nazwie MyRouteFilter, a następnie zapisuje go w zmiennym $rf.
Drugie polecenie pobiera regułę filtrowania tras o nazwie Reguła01 skojarzoną z tym filtrem trasy.
Trzecie polecenie otrzymuje listę reguł filtrowania tras skojarzonych z tym filtrem trasy.
