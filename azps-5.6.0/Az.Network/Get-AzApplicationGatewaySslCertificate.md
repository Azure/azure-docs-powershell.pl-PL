# Get-AzApplicationGatewaySslCertificate

## SYNOPSIS
Otrzymuje certyfikat SSL dla bramy aplikacji.

## SKŁADNIA

```
Get-AzApplicationGatewaySslCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzApplicationGatewaySslCertificate** otrzymuje certyfikat SSL dla bramy aplikacji.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie określonego certyfikatu SSL
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Cert = Get-AzApplicationGatewaySslCertificate -Name "Cert01" -ApplicationGateway $AppGW
```

Pierwsze polecenie pobiera bramę aplikacji o nazwie ApplicationGateway01 i przechowuje wynik w zmiennej o nazwie $AppGW.
Drugie polecenie pobiera certyfikat SSL o nazwie Cert01 z bramy aplikacji przechowywanej w zmiennej o nazwie $AppGW.
Polecenie przechowuje certyfikat w zmiennej o nazwie $Cert.

### Przykład 2. Uzyskiwanie listy certyfikatów SSL
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Certs = Get-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW
```

Pierwsze polecenie pobiera bramę aplikacji o nazwie ApplicationGateway01 i przechowuje wynik w zmiennej o nazwie $AppGW.
Drugie polecenie pobiera listę certyfikatów SSL z bramy aplikacji przechowywaną w zmiennej o nazwie $AppGW.
Następnie polecenie przechowuje wyniki w zmiennej o nazwie $Certs.
