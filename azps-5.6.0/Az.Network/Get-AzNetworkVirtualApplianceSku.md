# Get-AzNetworkVirtualApplianceSku

## SYNOPSIS
Pobierz lub na liście dostępne jednostki SKU wirtualnego urządzenia sieciowego w spisie.

## SKŁADNIA

```
Get-AzNetworkVirtualApplianceSku [-SkuName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Ten Get-AzNetworkVirtualApplianceSku pobiera lub wyświetla dostępne jednostki SKU wirtualnego urządzenia sieciowego w spisie produktów platformy Microsoft Azure.
