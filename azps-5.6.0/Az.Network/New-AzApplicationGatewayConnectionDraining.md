# New-AzApplicationGatewayConnectionDraining

## SYNOPSIS
Tworzy nową konfigurację opróżniania połączenia dla ustawień protokołu HTTP na zawczasu.

## SKŁADNIA

```
New-AzApplicationGatewayConnectionDraining -Enabled <Boolean> -DrainTimeoutInSec <Int32>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzApplicationGatewayConnectionDraining** tworzy nową konfigurację opróżniania połączenia dla końcowych ustawień HTTP.

## PRZYKŁADY

### Przykład 1
```
PS C:\> $connectionDraining = New-AzApplicationGatewayConnectionDraining -Enabled $True -DrainTimeoutInSec 42
```

To polecenie tworzy nową konfigurację rozładowania połączenia z włączonymi wartościami Prawda i DrainTimeoutInSec ustawioną na 42 sekundy i zapisuje ją w $connectionDraining.
