# Get-AzRouteTable

## SYNOPSIS
Pobiera tabele tras.

## SKŁADNIA

### NoExpand (Domyślne)
```
Get-AzRouteTable [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Rozwiń
```
Get-AzRouteTable -ResourceGroupName <String> -Name <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzRouteTable** pobiera tabele tras platformy Azure.
Możesz pobrać jedną tabelę trasy lub pobrać wszystkie tabele tras w grupie zasobów lub w ramach subskrypcji.
