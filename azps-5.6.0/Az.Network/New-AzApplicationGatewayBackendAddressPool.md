# New-AzApplicationGatewayBackendAddressPool

## SYNOPSIS
Tworzy pulę adresów back-end dla bramy aplikacji.

## SKŁADNIA

```
New-AzApplicationGatewayBackendAddressPool -Name <String> [-BackendIPAddresses <String[]>]
 [-BackendFqdns <String[]>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzApplicationGatewayBackendAddressPool** tworzy pulę adresów zaplecza dla bramy aplikacji platformy Azure.
Adres końcowy można określić jako adres IP, w pełni kwalifikowaną nazwę domeny (FQDN) lub identyfikator konfiguracji adresu IP.

## PRZYKŁADY

### Przykład 1. Tworzenie puli adresów back-end przy użyciu domeny FQDN serwera back-end
```
PS C:\>$Pool = New-AzApplicationGatewayBackendAddressPool -Name "Pool01" -BackendFqdns "contoso1.com", "contoso2.com"
```

To polecenie tworzy pulę adresów końcowych o nazwie Pool01 przy użyciu nazw FQN serwerów końcowych i przechowuje ją w $Pool danych.

### Przykład 2. Tworzenie puli adresów back-end przy użyciu adresu IP serwera back-end
```
PS C:\>$Pool = New-AzApplicationGatewayBackendAddressPool -Name "Pool02" -BackendFqdns "10.10.10.10", "10.10.10.11"
```

To polecenie tworzy pulę adresów back-end o nazwie Pool02 przy użyciu adresów IP serwerów końcowych i zapisuje ją w $Pool danych.
