# Get-AzNetworkUsage

## SYNOPSIS
Lista użycia sieci dla subskrypcji.

## SKŁADNIA

```
Get-AzNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie Get-AzNetworkUsage pobiera limity i bieżące użycie zasobów sieciowych.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Get-AzNetworkUsage -Location westcentralus

Limit        : 50
```

Pobiera dane użycia zasobów w regionie westcentralus
