# New-AzVirtualNetworkTap

## SYNOPSIS
Tworzy zasób VirtualNetworkTap.

## SKŁADNIA

### SetByResource (Domyślne)
```
New-AzVirtualNetworkTap -ResourceGroupName <String> -Name <String> [-DestinationPort <Int32>]
 [-Location <String>] [-Tag <Hashtable>]
 [-DestinationNetworkInterfaceIPConfiguration <PSNetworkInterfaceIPConfiguration>]
 [-DestinationLoadBalancerFrontEndIPConfiguration <PSFrontendIPConfiguration>] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByResourceId
```
New-AzVirtualNetworkTap -ResourceGroupName <String> -Name <String> [-DestinationPort <Int32>]
 [-Location <String>] [-Tag <Hashtable>] [-DestinationNetworkInterfaceIPConfigurationId <String>]
 [-DestinationLoadBalancerFrontEndIPConfigurationId <String>] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzVirtualNetworkTap** tworzy zasób usługi Azure Virtual Network Tap.

## PRZYKŁADY

### Przykład 1. Tworzenie dotknięcia sieci wirtualnej platformy Azure
```
PS C:\>New-AzVirtualNetworkTap -Name "VirtualNetworkTap1" -ResourceGroupName "ResourceGroup1" -Location "centralus" -DestinationPort 8888 -DestinationNetworkInterfaceIPConfiguration $destinationNic.IpConfigurations[0]
```

To polecenie tworzy naciśnięcie sieci wirtualnej o nazwie "VirtualNetworkTap1", które ma szczegóły konfiguracji docelowej maszyny wirtualnej (DestinationIpConfiguration, DestinationPort).
Cały ruch maszyny wirtualnej skonfigurowany do naciśnięcia źródła zostanie rozsyłany do tego adresu IP + portu miejsca docelowego.

### Przykład 2. Tworzenie naciśnięcia sieci wirtualnej platformy Azure przy użyciu adresu IP narzędzia LoadBalancer
```
# Create LoadBalancer
PS C:\>$frontend = New-AzLoadBalancerFrontendIpConfig -Name $frontendName -PublicIpAddress $publicip
PS C:\>New-AzVirtualNetworkTap -Name "VirtualNetworkTap1" -ResourceGroupName "ResourceGroup1" -Location "centralus" -DestinationLoadBalancerFrontEndIPConfiguration $frontend
```

To polecenie tworzy naciśnięcie sieci wirtualnej o nazwie "VirtualNetworkTap1", które ma szczegóły konfiguracji docelowej maszyny wirtualnej (FrontEndIpConfiguration).
Cały ruch maszyny wirtualnej skonfigurowany przez naciśnięcie źródła zostanie przekierowyny do tej konfiguracji IpConfiguration protokołu LoadBalancer. Domyślny port docelowy to 4789.
