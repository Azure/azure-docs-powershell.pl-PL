# Remove-AzExpressRouteCircuit

## SYNOPSIS
Usuwa obwód w układzie ExpressRoute.

## SKŁADNIA

```
Remove-AzExpressRouteCircuit -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzExpressRouteCircuit** usuwa obwód expressRoute.

## PRZYKŁADY

### Przykład 1. Usuwanie obwodu w układzie ExpressRoute
```
Remove-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg
```

### Przykład 2. Usuwanie obwodu usługi ExpressRoute przy użyciu potoku
```
Get-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg | Remove-AzExpressRouteCircuit
```
