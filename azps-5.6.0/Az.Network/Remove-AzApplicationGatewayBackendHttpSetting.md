# Remove-AzApplicationGatewayBackendHttpSetting

## SYNOPSIS
Usuwa ustawienia protokołu HTTP z bramy aplikacji.

## SKŁADNIA

```
Remove-AzApplicationGatewayBackendHttpSetting -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie Remove-AzApplicationGatewayBackendHttpSetting usuwa ustawienia protokołu HTTP (Hypertext Transfer Protocol) z bramy aplikacji Azure.

## PRZYKŁADY

### Przykład 1. Usuwanie ustawień protokołu HTTP z bramy aplikacji
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayBackendHttpSetting -ApplicationGateway $AppGw -Name "BackEndSetting02"
```

Pierwsze polecenie otrzymuje bramę aplikacji o nazwie ApplicationGateway01 należącą do grupy zasobów o nazwie ResourceGroup01 i przechowuje ją w zmiennej $AppGw zasobów.
Drugie polecenie usuwa ustawienie zaplecza HTTP o nazwie BackEndSetting02 z bramy aplikacji przechowywanej w $AppGw.
