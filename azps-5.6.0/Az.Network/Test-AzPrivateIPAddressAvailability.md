# Test-AzPrivateIPAddressAvailability

## SYNOPSIS
Przetestuj dostępność prywatnego adresu IP w sieci wirtualnej.

## SKŁADNIA

### TestByResource
```
Test-AzPrivateIPAddressAvailability -VirtualNetwork <PSVirtualNetwork> -IPAddress <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### TestByResourceId
```
Test-AzPrivateIPAddressAvailability -ResourceGroupName <String> -VirtualNetworkName <String>
 -IPAddress <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Test-AzPrivateIPAddressAvailability** sprawdza, czy określony prywatny adres IP jest dostępny w sieci wirtualnej.
To polecenie cmdlet zwraca listę dostępnych prywatnych adresów IP, jeśli żądany prywatny adres IP zostanie podany.

## PRZYKŁADY

### Przykład 1. Testowanie, czy adres IP jest dostępny przy użyciu potoku
```
PS C:\>Get-AzVirtualNetwork -Name $vnetName -ResourceGroupName $rgname | Test-AzPrivateIPAddressAvailability -IPAddress "10.0.1.10"
```

To polecenie pobiera sieć wirtualną i używa operatora potoku w celu przekazania go do **testów Test-AzPrivateIPAddressAvailability,** co sprawdza, czy określony prywatny adres IP jest dostępny.
