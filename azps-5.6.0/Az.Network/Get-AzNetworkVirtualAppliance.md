# Get-AzNetworkVirtualAppliance

## SYNOPSIS
Pobierz lub wymień wirtualne urządzenia sieciowe.

## SKŁADNIA

### ResourceNameParameterSet (Domyślne)
```
Get-AzNetworkVirtualAppliance [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzNetworkVirtualAppliance -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie Get-AzNetworkVirtualAppliance pobiera lub wyświetla zasoby urządzenia wirtualnego sieci.
