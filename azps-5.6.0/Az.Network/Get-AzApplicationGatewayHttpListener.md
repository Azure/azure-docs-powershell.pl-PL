# Get-AzApplicationGatewayHttpListener

## SYNOPSIS
Pobiera słuchacza HTTP bramy aplikacji.

## SKŁADNIA

```
Get-AzApplicationGatewayHttpListener [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzApplicationGatewayHttpListener** pobiera słuchacza HTTP bramy aplikacji.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie określonego słuchacza HTTP
```
PS C:\>$Appgw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Listener = Get-AzApplicationGatewayHttpListener -Name "Listener01" -ApplicationGateway $Appgw
```

To polecenie pobiera słuchacza HTTP o nazwie Słuchacz01.

### Przykład 2. Uzyskiwanie listy słuchaczy HTTP
```
PS C:\>$Appgw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Listeners = Get-AzApplicationGatewayHttpListener -ApplicationGateway $Appgw
```

To polecenie pobiera listę słuchaczy HTTP.
