# Get-AzLoadBalancerRuleConfig

## SYNOPSIS
Pobiera konfigurację reguły dla równoważenia obciążenia.

## SKŁADNIA

```
Get-AzLoadBalancerRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzLoadBalancerRuleConfig** pobiera jedną lub więcej konfiguracji reguł dla równoważenia obciążenia.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie konfiguracji reguły równoważenia obciążenia
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerRuleConfig -Name "MyLBrulename" -LoadBalancer $slb
```

Pierwsze polecenie pobiera równoważenie obciążenia o nazwie MyLoadBalancer, a następnie przechowuje je w zmiennym $slb.
Drugie polecenie pobiera skojarzoną konfigurację reguł o nazwie MyLBrulename z równoważenia obciążenia w $slb.
