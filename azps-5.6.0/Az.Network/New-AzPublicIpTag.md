# New-AzPublicIpTag

## SYNOPSIS
Tworzy znacznik IP.

## SKŁADNIA

```
New-AzPublicIpTag -IpTagType <String> -Tag <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzPublicIpTag** tworzy tag IP.

## PRZYKŁADY

### Przykład 1. Tworzenie nowego tagu IP
```powershell
$ipTag = New-AzPublicIpTag -IpTagType $ipTagType -Tag $tag
```

To polecenie tworzy nowy tag IP z takim tagiem jak "FirstPartyUsage" i tagiem, takim jak "/Sql". Jest to używane w przypadku tworzenia publicIpAddress z tymi określonymi tagami do przydzielenia.
