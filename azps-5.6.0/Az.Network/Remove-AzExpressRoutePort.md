# Remove-AzExpressRoutePort

## SYNOPSIS
Usuwa port ExpressRoutePort.

## SKŁADNIA

### ResourceNameParameterSet (Domyślne)
```
Remove-AzExpressRoutePort -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzExpressRoutePort -InputObject <PSExpressRoutePort> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzExpressRoutePort -ResourceId <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzExpressRoutePort** usuwa port ExpressRoutePort.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $PortName -ResourceGroupName $rg
```

Usuwa $PortName ExpressRoutePort w $rg grupy zasobów w subskrypcji.

### Przykład 2
```powershell
PS C:\> Remove-AzExpressRoutePort -InputObject $erPort
```

Usuwa zasób ExpressRoutePort z inputObject.

### Przykład 3
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $ResourceId $id
```

Usuwa zasób ExpressRoutePort przy użyciu $id ResourceId.
