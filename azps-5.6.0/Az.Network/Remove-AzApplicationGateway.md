# Remove-AzApplicationGateway

## SYNOPSIS
Usuwa bramę aplikacji.

## SKŁADNIA

```
Remove-AzApplicationGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzApplicationGateway** usuwa bramę aplikacji.

## PRZYKŁADY

### Przykład 1. Usuwanie określonej bramy aplikacji
```
PS C:\>Remove-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
```

To polecenie usuwa bramę aplikacji o nazwie ApplicationGateway01 z grupy zasobów o nazwie ResourceGroup01.
