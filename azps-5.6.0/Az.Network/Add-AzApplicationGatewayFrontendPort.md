# Add-AzApplicationGatewayFrontendPort

## SYNOPSIS
Dodaje port frontoronowy do bramy aplikacji.

## SKŁADNIA

```
Add-AzApplicationGatewayFrontendPort -ApplicationGateway <PSApplicationGateway> -Name <String> -Port <Int32>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Add-AzApplicationGatewayFrontendPort** dodaje port front end do bramy aplikacji.

## PRZYKŁADY

### Przykład 1. Dodawanie portu frontoronu do bramy aplikacji
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Add-AzApplicationGatewayFrontendPort -ApplicationGateway $AppGw -Name "FrontEndPort01" -Port 80
```

Pierwsze polecenie pobiera bramę aplikacji o nazwie ApplicationGateway01, która należy do grupy zasobów o nazwie ResourceGroup01, i przechowuje ją w zmiennej $AppGw zasobów.
Drugie polecenie dodaje port 80 jako port fronto end bramy aplikacji przechowywanej w programie $AppGw i nadaje nazwę portowi FrontEndPort01.
