# Remove-AzLoadBalancer

## SYNOPSIS
Usuwa równoważenie obciążenia.

## SKŁADNIA

```
Remove-AzLoadBalancer -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzLoadBalancer** usuwa narzędzie do równoważenia obciążenia platformy Azure.

## PRZYKŁADY

### Przykład 1. Usuwanie równoważenia obciążenia
```
PS C:\>Remove-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
```

To polecenie usuwa równoważenie obciążenia o nazwie MyLoadBalancer z grupy zasobów o nazwie MyResourceGroup.
