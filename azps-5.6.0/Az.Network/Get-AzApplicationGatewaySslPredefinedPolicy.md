# Get-AzApplicationGatewaySslPredefinedPolicy

## SYNOPSIS
Pobiera wstępnie zdefiniowane zasady SSL udostępniane przez bramę aplikacji.

## SKŁADNIA

```
Get-AzApplicationGatewaySslPredefinedPolicy [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzApplicationGatewaySslPredefinedPolicy** pobiera wstępnie zdefiniowane zasady SSL udostępniane przez bramę aplikacji.
