# Get-AzLoadBalancerBackendAddressPoolConfig

## SYNOPSIS
Pobiera konfigurację puli adresów zaplecza dla równoważenia obciążenia.

## SKŁADNIA

```
Get-AzLoadBalancerBackendAddressPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzLoadBalancerBackendAddressPoolConfig** pobiera jedną pulę adresów zaplecza lub listę pul adresów zaplecza w ramach równoważenia obciążenia.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie puli adresów zaplecza
```
PS C:\>$loadbalancer = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02" -LoadBalancer $loadbalancer
```

Pierwsze polecenie pobiera istniejące polecenie równoważenia obciążenia o nazwie MyLoadBalancer w grupie zasobów o nazwie MyResourceGroup, a następnie przechowuje je w $loadbalancer danych.
Drugie polecenie pobiera skojarzoną konfigurację puli adresów zaplecza o nazwie BackendAddressPool02 dla równoważenia obciążenia w $loadbalancer.
