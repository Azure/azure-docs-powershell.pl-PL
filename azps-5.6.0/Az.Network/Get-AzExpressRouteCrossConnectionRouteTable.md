# Get-AzExpressRouteCrossConnectionRouteTable

## SYNOPSIS
Pobiera tabelę trasy z połączenia krzyżowego expressRoute.

## SKŁADNIA

### SpecifyByParameterValues
```
Get-AzExpressRouteCrossConnectionRouteTable -ResourceGroupName <String> -CrossConnectionName <String>
 -PeeringType <String> -DevicePath <DevicePathEnum> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### SpecifyByReference
```
Get-AzExpressRouteCrossConnectionRouteTable -ExpressRouteCrossConnection <PSExpressRouteCrossConnection>
 -PeeringType <String> -DevicePath <DevicePathEnum> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzExpressRouteCrossConnectionRouteTable** pobiera szczegółową tabelę tras obwodu usługi ExpressRoute. W tabeli tras będą wyświetlane wszystkie trasy lub można je filtrować w celu pokazania tras dla określonego typu komunikacji równorzędnej. Za pomocą tabeli trasy możesz sprawdzić konfigurację komunikacji równorzędnej i łączność.

## PRZYKŁADY

### Przykład 1. Wyświetlanie tabeli trasy dla ścieżki podstawowej
```
Get-AzExpressRouteCrossConnectionRouteTable -ResourceGroupName $RG -ExpressRouteCrossConnectionName $CircuitName -DevicePath 'Primary'
```
