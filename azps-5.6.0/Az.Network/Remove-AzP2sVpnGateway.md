# Remove-AzP2sVpnGateway

## SYNOPSIS
Usuwa istniejącą bramę P2SVpnGateway.

## SKŁADNIA

### ByP2SVpnGatewayName (domyślna)
```
Remove-AzP2sVpnGateway -ResourceGroupName <String> -Name <String> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByP2SVpnGatewayObject
```
Remove-AzP2sVpnGateway -InputObject <PSP2SVpnGateway> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByP2SVpnGatewayResourceId
```
Remove-AzP2sVpnGateway -ResourceId <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzP2sVpnGateway** umożliwia usunięcie istniejącej aplikacji P2SVpnGateway w obszarze VirtualHub. Wszystkie point to site clients connectivity will fail after P2SVpnGateway is removed.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Remove-AzP2sVpnGateway -Name 683482ade8564515aed4b8448c9757ea-westus-gw-ResourceGroupName P2SCortexGATesting -Force -PassThru
```

Polecenie **cmdlet Remove-AzP2sVpnGateway** umożliwia usunięcie istniejącej aplikacji P2SVpnGateway w obszarze VirtualHub.
