# Get-AzRouteConfig

## SYNOPSIS
Pobiera trasy z tabeli tras.

## SKŁADNIA

```
Get-AzRouteConfig -RouteTable <PSRouteTable> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzRouteConfig** pobiera trasy z tabeli tras platformy Azure.
Trasę można określić według nazwy.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie tabeli tras
```
PS C:\>Get-AzRouteTable -ResourceGroupName "ResourceGroup11" -Name "RouteTable01" | Get-AzRouteConfig -Name "Route07"
Name              : route07
Id                : 
Etag              : 
ProvisioningState : 
AddressPrefix     : 10.1.0.0/16
NextHopType       : VnetLocal
NextHopIpAddress  :
```

To polecenie pobiera tabelę tras o nazwie RouteTable01 przy użyciu polecenia cmdlet **Get-AzRouteTable.**
Polecenie przekazuje tabelę do bieżącego polecenia cmdlet przy użyciu operatora potoku.
Bieżące polecenie cmdlet pobiera trasę o nazwie Route07 w tabeli tras o nazwie RouteTable01.
