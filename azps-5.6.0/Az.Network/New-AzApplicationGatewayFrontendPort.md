# New-AzApplicationGatewayFrontendPort

## SYNOPSIS
Tworzy port frontoronowy dla bramy aplikacji.

## SKŁADNIA

```
New-AzApplicationGatewayFrontendPort -Name <String> -Port <Int32> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzApplicationGatewayFrontendPort** tworzy port front end dla bramy aplikacji platformy Azure.

## PRZYKŁADY

### Przykład 1. Przykład1. Tworzenie portu frontoronu
```powershell
PS C:\>$FrontEndPort = New-AzApplicationGatewayFrontendPort -Name "FrontEndPort01" -Port 80
```

To polecenie tworzy port frontonu o nazwie FrontEndPort01 na porcie 80 i zapisuje wynik w zmiennej o nazwie $FrontEndPort.
