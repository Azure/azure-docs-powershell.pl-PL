# Remove-AzApplicationGatewayProbeConfig

## SYNOPSIS
Usuwa kondycję z istniejącej bramy aplikacji.

## SKŁADNIA

```
Remove-AzApplicationGatewayProbeConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie Remove-AzApplicationGatewayProbeConfig usuwa element cieplny z istniejącej bramy aplikacji.

## PRZYKŁADY

### Przykład 1. Usuwanie pliku kondycji z istniejącej bramy aplikacji
```
PS C:\>$Gateway = Remove-AzApplicationGatewayProbeConfig -ApplicationGateway Gateway -Name "Probe04"
```

To polecenie usuwa z bramy aplikacji o nazwie Brama kondycję o nazwie Pochłoń04.
