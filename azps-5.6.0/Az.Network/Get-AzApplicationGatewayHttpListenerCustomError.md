# Get-AzApplicationGatewayHttpListenerCustomError

## SYNOPSIS
Pobiera błędy niestandardowe od słuchacza http bramy aplikacji.

## SKŁADNIA

```
Get-AzApplicationGatewayHttpListenerCustomError [-StatusCode <String>]
 -HttpListener <PSApplicationGatewayHttpListener> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzApplicationGatewayCustomError** pobiera błędy niestandardowe od słuchacza http bramy aplikacji.

## PRZYKŁADY

### Przykład 1. Zwraca błąd niestandardowy w słuchaczu http
```powershell
PS C:\> $ce = Get-AzApplicationGatewayCustomError -HttpListener $listener01 -StatusCode HttpStatus502
```

To polecenie pobiera i zwraca niestandardowy błąd kodu stanu http 502 od odbiorcy http $listener 01.

### Przykład 2. Pobiera listę wszystkich błędów niestandardowych w słuchaczu http
```powershell
PS C:\> $ces = Get-AzApplicationGatewayCustomError -HttpListener $listener01
```

To polecenie pobiera i zwraca listę wszystkich błędów niestandardowych z grupy $listener 01 http.
