# Get-AzConfluentMarketplaceAgreement

## SYNOPSIS
Lista confluent marketplace umów w ramach subskrypcji.

## SKŁADNIA

```
Get-AzConfluentMarketplaceAgreement [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## OPIS
Lista confluent marketplace umów w ramach subskrypcji.

## PRZYKŁADY

### Przykład 1. Lista wszystkich umowy dotyczącej witryny Confluent Marketplace w ramach subskrypcji
```powershell
PS C:\> Get-AzConfluentMarketplaceAgreement

Name        Type
----        ----
marketplace Microsoft.Confluent/agreements
confluent   Microsoft.Confluent/offertypes
```

To polecenie zawiera listę wszystkich umów na sklep confluent w ramach subskrypcji.
