# Enter-AzWebAppContainerPSSession

## SYNOPSIS
Otwiera zdalną sesję programu PowerShell w kontenerze systemu Windows określonym w określonej witrynie lub przedziale i w danej grupie zasobów.

## SKŁADNIA

### S1 (domyślne)
```
Enter-AzWebAppContainerPSSession [-PassThru] [-Force] [[-SlotName] <String>] [-ResourceGroupName] <String>
 [-Name] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Enter-AzWebAppContainerPSSession [-PassThru] [-Force] [-WebApp] <PSSite>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Otwiera zdalną sesję programu PowerShell w kontenerze systemu Windows określonym w określonej witrynie lub przedziale i danej grupie zasobów.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Enter-AzWebAppContainerPSSession -ResourceGroupName "Default-Web-WestUS" -Name "ContosoASP"
```

To polecenie powoduje otwarcie zdalnej sesji programu PowerShell w aplikacji kontenera systemu Windows ContosoASP
