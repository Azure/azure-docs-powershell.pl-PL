# Get-AzAppServiceEnvironment

## SYNOPSIS
Pobiera środowisko usługi aplikacji. Jeśli zostanie określona tylko grupa zasobów, zwróci ona listę ase w grupie zasobów.

## SKŁADNIA

```
Get-AzAppServiceEnvironment [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzAppServiceEnvironment** zwraca ase pasujące do zapytania.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Get-AzAppServiceEnvironment -ResourceGroupName MyResourceGroup -Name MyAseName
```

Zwraca określone środowisko usługi aplikacji <MyAseName> o nazwie <MyResourceGroup>
