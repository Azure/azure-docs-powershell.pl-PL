# Update-AzWebAppTrafficRouting

## SYNOPSIS
Zaktualizuj regułę routingu do gniazda.

## SKŁADNIA

```
Update-AzWebAppTrafficRouting -ResourceGroupName <String> -WebAppName <String> -RoutingRule <Hashtable>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Update-AzWebAppTrafficRouting** aktualizuje konfigurację reguły routingu dla gniazda aplikacji Azure Web App.

## PRZYKŁADY

### Przykład 1. Aktualizowanie reguły routingu w celu przeniesienia 15% ruchu produkcyjnego do gniazda Stg
```powershell
PS C:\>Update-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
- RoutingRule @{AtionHostName='XXXX.azurewebsites.net';ReroutePercentage=15;Name='Stg'}
```

To polecenie aktualizuje regułę routingu w celu przeniesienia 15% ruchu produkcyjnego do gniazda Stg.

### Przykład 2. Aktualizowanie reguły routingu w celu przeniesienia ruchu produkcyjnego do zakresów slotów Stg z zakresu od 50% do 90% w sposób przyrostowy.
```powershell
PS C:\>Update-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-RoutingRule @{ActionHostName='XXXX.azurewebsites.net';ReroutePercentage=50;ChangeIntervalInMinutes=1;
MinReroutePercentage=50;MaxReroutePercentage=90;Name='Stg';ChangeStep=10}
```

To polecenie aktualizuje regułę routingu w celu przeniesienia ruchu produkcyjnego do zakresów slotów Stg z zakresu od 50% do 90% w sposób przyrostowy.
