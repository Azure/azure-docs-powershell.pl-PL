# Restart-AzWebApp

## SYNOPSIS
Ponowne uruchomienie aplikacji Azure Web App.

## SKŁADNIA

### S1
```
Restart-AzWebApp [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### S2
```
Restart-AzWebApp [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Restart-AzWebApp** zatrzymuje się, a następnie uruchamia aplikację Azure Web App.
Jeśli aplikacja Web App znajduje się w stanie zatrzymania, użyj Start-AzWebApp cmdlet.

## PRZYKŁADY

### Przykład 1. Ponowne uruchamianie aplikacji sieci Web
```
PS C:\>Restart-AzWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

To polecenie powoduje zatrzymanie aplikacji Azure Web App o nazwie ContosoSite należącej do grupy zasobów o nazwie Default-Web-WestUS, a następnie ponowne uruchomienie tej aplikacji.
