# Get-AzWebAppSnapshot

## SYNOPSIS
Pobiera migawki dostępne dla aplikacji sieci Web.

## SKŁADNIA

### FromResourceName
```
Get-AzWebAppSnapshot [-UseDisasterRecovery] [-ResourceGroupName] <String> [-Name] <String> [[-Slot] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### FromWebApp
```
Get-AzWebAppSnapshot [-UseDisasterRecovery] [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzWebAppSnapshot** zwraca wszystkie migawki aplikacji sieci Web. Migawki to automatyczne kopie zapasowe plików i ustawień aplikacji sieci Web. Migawkę można przywrócić za pomocą polecenia cmdlet **Restore-AzWebAppSnapshot.**

## PRZYKŁADY

### Przykład 1
```
PS C:\> Get-AzWebAppSnapshot -ResourceGroupName "Default-Web-WestUS" -Name "ContosoApp" -Slot "Staging"
```

Uzyskiwanie migawek aplikacji sieci Web o nazwie "ContosoApp" z slotem o nazwie "Tymczasowe" w grupie zasobów "Default-Web-WestUS"
