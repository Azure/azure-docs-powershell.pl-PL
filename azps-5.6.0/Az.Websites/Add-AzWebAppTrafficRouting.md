# Add-AzWebAppTrafficRouting

## SYNOPSIS
Dodaj regułę routingu do gniazda.

## SKŁADNIA

```
Add-AzWebAppTrafficRouting -ResourceGroupName <String> -WebAppName <String> -RoutingRule <Hashtable>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Add-AzWebAppTrafficRouting** dodaje regułę routingu do gniazda aplikacji Azure Web App.

## PRZYKŁADY

### Przykład 1. Dodawanie reguły routingu w celu przeniesienia 15% ruchu produkcyjnego do gniazda Stg
```powershell
PS C:\>Add-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-RoutingRule @{ActionHostName='XXXX.azurewebsites.net';ReroutePercentage=15;Name='Stg'}
```

To polecenie dodaje regułę routingu w celu przeniesienia 15% ruchu produkcyjnego do gniazda Stg

### Przykład 2 Dodawanie reguły routingu w celu przenoszenia ruchu produkcyjnego do zakresów slotów Stg z zakresu od 50% do 90% w sposób przyrostowy.
```powershell
PS C:\>Add-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-RoutingRule @{ActionHostName='XXXX.azurewebsites.net';ReroutePercentage=50;ChangeIntervalInMinutes=1;
MinReroutePercentage=50;MaxReroutePercentage=90;Name='Stg';ChangeStep=10}
```

To polecenie dodaje regułę routingu w celu przenoszenia ruchu produkcyjnego do zakresów przedziałów stg z zakresu od 50% do 90% w sposób przyrostowy.
