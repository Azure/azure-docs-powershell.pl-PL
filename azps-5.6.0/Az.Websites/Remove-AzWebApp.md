# Remove-AzWebApp

## SYNOPSIS
Usuwa aplikację Azure Web App.

## SKŁADNIA

### S1
```
Remove-AzWebApp [-Force] [-AsJob] [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Remove-AzWebApp [-Force] [-AsJob] [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzWebApp** usuwa aplikację Azure Web App podaną nazwę grupy zasobów i aplikacji sieci Web.
To polecenie cmdlet domyślnie usuwa również wszystkie gniazda i metryki.

## PRZYKŁADY

### Przykład 1. Usuwanie aplikacji sieci Web
```
PS C:\>Remove-AzWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

To polecenie usuwa aplikację Azure Web App o nazwie ContosoSite należącą do grupy zasobów o nazwie Default-Web-WestUS.
