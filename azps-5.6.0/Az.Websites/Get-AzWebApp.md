# Get-AzWebApp

## SYNOPSIS
Pobiera aplikacje Azure Web Apps do określonej grupy zasobów.

## SKŁADNIA

### S1
```
Get-AzWebApp [[-ResourceGroupName] <String>] [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### S2
```
Get-AzWebApp [-AppServicePlan] <PSAppServicePlan> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### S3
```
Get-AzWebApp [-Location] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzWebApp** pobiera informacje o aplikacji Azure Web App.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie aplikacji Sieci Web z grupy zasobów
```
PS C:\>Get-AzWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

To polecenie pobiera nazwę aplikacji Sieci Web o nazwie ContosoSite należącą do grupy zasobów Default-Web-WestUS.
