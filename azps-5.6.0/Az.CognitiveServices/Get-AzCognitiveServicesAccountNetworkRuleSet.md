# Get-AzCognitiveServicesAccountNetworkRuleSet

## SYNOPSIS
Uzyskiwanie właściwości NetworkRule konta usług Cognitive Services

## SKŁADNIA

```
Get-AzCognitiveServicesAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzCognitiveServicesAccountNetworkRuleSet** pobiera właściwość NetworkRule konta usług Cognitive Services

## PRZYKŁADY

### Przykład 1. Uzyskiwanie właściwości NetworkRule określonego konta usług Cognitive Services
```
PS C:\> Get-AzCognitiveServicesAccountNetworkRuleSet  -ResourceGroupName "rg1" -Name "myaccount"
```

To polecenie pobiera właściwość NetworkRule określonego konta usług Cognitive Services
