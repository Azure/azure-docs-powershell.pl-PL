# Remove-AzRecoveryServicesAsrProtectionContainerMapping

## SYNOPSIS
Usuwa wskazane mapowanie kontenera ochrony usługi Azure Site Recovery.

## SKŁADNIA

```
Remove-AzRecoveryServicesAsrProtectionContainerMapping -InputObject <ASRProtectionContainerMapping> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzRecoveryServicesAsrProtectionContainerMapping** usuwa wskazane mapowanie kontenera ochrony usługi Azure Site Recovery.

## PRZYKŁADY

### Przykład 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrProtectionContainerMapping -ProtectionContainerMapping $ProtectionContainerMapping
```

Rozpoczyna usuwanie określonego mapowania kontenera ochrony i zwraca zadanie asr służące do śledzenia operacji.
