# Start-AzRecoveryServicesAsrTestFailoverJob

## SYNOPSIS
Rozpoczyna testowanie operacji trybu failover.

## SKŁADNIA

```
 -Direction <String> [-CloudServiceCreationOption <String>] [-DataEncryptionPrimaryCertFile <String>]
 [-DataEncryptionSecondaryCertFile <String>] [-RecoveryPoint <ASRRecoveryPoint>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Start-AzRecoveryServicesAsrTestFailoverJob** uruchamia test trybu failover elementu chronionego replikacją usługi Azure Site Recovery lub planu odzyskiwania.
Możesz sprawdzić, czy zadanie zakończyło się pomyślnie za pomocą Get-AzRecoveryServicesAsrJob cmdlet.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> $currentJob = Start-AzRecoveryServicesAsrTestFailoverJob -RecoveryPlan $RP -Direction PrimaryToRecovery -VMNetwork $TestRecoveryNetwork
```

Rozpoczyna testowanie operacji trybu failover planu odzyskiwania z określonymi parametrami i zwraca zadanie funkcji ASR służące do śledzenia operacji.

### Przykład 2

Rozpoczyna testowanie operacji trybu failover. (wygenerowane automatycznie)

```powershell <!-- Aladdin Generated Example --> 
Start-AzRecoveryServicesAsrTestFailoverJob -AzureVMNetworkId <String> -Direction PrimaryToRecovery -RecoveryPlan $RP
```
