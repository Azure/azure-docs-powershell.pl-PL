# New-AzRecoveryServicesAsrStorageClassificationMapping

## SYNOPSIS
Tworzy mapowanie klasyfikacji magazynu asr w magazynie usług odzyskiwania.

## SKŁADNIA

```
New-AzRecoveryServicesAsrStorageClassificationMapping -Name <String>
 -PrimaryStorageClassification <ASRStorageClassification>
 -RecoveryStorageClassification <ASRStorageClassification> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzRecoveryServicesAsrStorageClassificationMapping** tworzy klasyfikację magazynu mapowania magazynu usług odzyskiwania.

## PRZYKŁADY

### Przykład 1
```
PS C:\> $currentJob = New-AzRecoveryServicesAsrStorageClassificationMapping -Name $StorageClassificationMappingName -PrimaryStorageClassification $PrimaryStorageClassification -RecoveryStorageClassification $RecoveryStorageClassification
```

Rozpoczyna operację tworzenia mapowania klasyfikacji magazynu z określonymi parametrami i zwraca zadanie asr służące do śledzenia operacji.
