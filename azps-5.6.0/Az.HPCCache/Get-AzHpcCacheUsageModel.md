# Get-AzHpcCacheUsageModel

## SYNOPSIS
Pobiera wszystkie wartości usageModels for NFS Storage Target(Pobiera wszystkie wartości usageModels for NFS Storage Target).

## SKŁADNIA

```
Get-AzHpcCacheUsageModel [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzHpcCacheUsageModel** zwraca listę modeli użycia dla obiektu docelowego magazynu systemu NFS.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Get-AzHpcCacheUsageModel
```
