# Get-AzHpcCacheSku

## SYNOPSIS
Pobiera wszystkie jednostki SKU dostępne w subskrypcji.

## SKŁADNIA

```
Get-AzHpcCacheSku [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzHpcCacheSku** zwraca listę wersji SKU dostępnych w subskrypcji.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Get-AzHpcCacheSku
```
