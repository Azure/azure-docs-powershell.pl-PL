# Get-AzDataMigrationProject

## SYNOPSIS
Pobiera właściwości projektu migracji bazy danych Azure.

## SKŁADNIA

### ComponentNameParameterSet (Domyślne)
```
Get-AzDataMigrationProject -ResourceGroupName <String> -ServiceName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ComponentObjectParameterSet
```
Get-AzDataMigrationProject [-InputObject] <PSDataMigrationService> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzDataMigrationProject [-ResourceId] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie Get-AzDataMigrationProject pobiera właściwości projektu migracji bazy danych Azure.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Get-AzDataMigrationProject -ServiceName testService -Name testProject -ResourceGroup testResourceGroup
```

Powyższy przykład pobiera projekt migracji bazy danych Azure o nazwie TestProject do grupy zasobów o nazwie testResourceGroup i w ramach usługi o nazwie testService

### Przykład 2
```
PS C:\> Get-AzDataMigrationProject -InputObject $myService
```

Powyższy przykład pobiera projekt migracji bazy danych Azure na podstawie przekazywanego parametru wprowadzania obiektu PSProject.
