# Stop-AzDataMigrationService

## SYNOPSIS
Zatrzymuje wystąpienie usługi migracji bazy danych Azure, które znajduje się w stanie uruchomionym.

## SKŁADNIA

### ComponentNameParameterSet (Domyślne)
```
Stop-AzDataMigrationService -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ComponentObjectParameterSet
```
Stop-AzDataMigrationService [-InputObject] <PSDataMigrationService> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Stop-AzDataMigrationService [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie Stop-AzDataMigrationService zatrzymuje wystąpienie usługi migracji bazy danych Azure, które znajduje się w stanie uruchomionym.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Stop-AzDataMigrationService -ResourceGroupName MyResourceGroup -ServiceName TestService
```

W powyższym przykładzie zatrzymano wystąpienie usługi migracji bazy danych Azure o nazwie TestService na podstawie nazwy usługi przekazanej jako parametr wejściowy

### Przykład 2
```
PS C:\> Stop-AzDataMigrationService -InputObject $TestService
```

W powyższym przykładzie zatrzymane jest wystąpienie usługi migracji bazy danych Azure na podstawie obiektu PSDataMigrationService przekazywanego jako parametr wejściowy.
