# New-AzDataFactoryGateway

## SYNOPSIS
Tworzy bramę dla usługi Azure Data Factory.

## SKŁADNIA

### ByFactoryName (Default)
```
New-AzDataFactoryGateway [-DataFactoryName] <String> [-Name] <String> [[-Description] <String>]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFactoryObject
```
New-AzDataFactoryGateway [-DataFactory] <PSDataFactory> [-Name] <String> [[-Description] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzDataFactoryGateway** tworzy bramę w usłudze Azure Data Factory.

## PRZYKŁADY

### Przykład 1. Tworzenie bramy
