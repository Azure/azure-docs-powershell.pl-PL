# New-AzDataFactory

## SYNOPSIS
Tworzy fabryczne dane.

## SKŁADNIA

```
New-AzDataFactory [-Name] <String> [-Location] <String> [[-Tag] <Hashtable>] [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzDataFactory** tworzy grupę danych o nazwie i lokalizacji określonej grupy zasobów.
Wykonaj te operacje w następującej kolejności: 
- Tworzenie fabryki danych. 
- Tworzenie połączonych usług. 
- Tworzenie zestawów danych. 
- Tworzenie potoku.

## PRZYKŁADY

### Przykład 1. Tworzenie fabryki danych
```
PS C:\>New-AzDataFactory -ResourceGroupName "ADF" -Name "WikiADF" -Location "WestUS"
DataFactoryName   : WikiADF
ResourceGroupName : ADF
Location          : WestUS
Tags              : {}
Properties        : Microsoft.WindowsAzure.Commands.Utilities.PSDataFactoryConfiguration
```

To polecenie umożliwia utworzenie w grupie zasobów O nazwie ADF w lokalizacji WestUS fabryki danych o nazwie WikiADF.
