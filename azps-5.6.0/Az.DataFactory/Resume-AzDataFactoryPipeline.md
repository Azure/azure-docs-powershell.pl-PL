# Resume-AzDataFactoryPipeline

## SYNOPSIS
Wznawia zawieszony potok w układzie Data Factory.

## SKŁADNIA

### ByFactoryName (Default)
```
Resume-AzDataFactoryPipeline [-Name] <String> [-DataFactoryName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Resume-AzDataFactoryPipeline [-Name] <String> [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Resume-AzDataFactoryPipeline** wznawia zawieszony potok w usłudze Azure Data Factory.

## PRZYKŁADY

### Przykład 1. Wznawianie potoku
```powershell
PS C:\>Resume-AzDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikisample" -DataFactoryName "WikiADF"
Confirm
Are you sure you want to resume pipeline 'DPWikisample' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

To polecenie wznawia potok o nazwie DPWikisample w zakładzie danych o nazwie WikiADF.
Użyj polecenia cmdlet **Suspend-AzDataFactoryPipeline,** aby zawiesić potok.
Polecenie zwróci wartość $True.

### Przykład 2

Wznawia zawieszony potok w układzie Data Factory. (wygenerowane automatycznie)

```powershell <!-- Aladdin Generated Example --> 
Resume-AzDataFactoryPipeline -DataFactory $DataFactory -Name 'DPWikisample'
```
