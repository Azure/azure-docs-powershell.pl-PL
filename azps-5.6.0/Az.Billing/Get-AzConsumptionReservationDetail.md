# Get-AzConsumptionReservationDetail

## SYNOPSIS
Uzyskaj szczegóły rezerwacji dla podanego zakresu dat.

## SKŁADNIA

```
Get-AzConsumptionReservationDetail -StartDate <DateTime> -EndDate <DateTime> -ReservationOrderId <String>
 [-ReservationId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzConsumptionReservationDetail** pobiera szczegóły rezerwacji dla podanego zakresu dat.
