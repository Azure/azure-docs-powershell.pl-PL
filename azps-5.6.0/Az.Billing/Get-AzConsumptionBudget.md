# Get-AzConsumptionBudget

## SYNOPSIS
Uzyskiwanie listy budżetów w ramach subskrypcji lub grupy zasobów.

## SKŁADNIA

```
Get-AzConsumptionBudget [-DefaultProfile <IAzureContextContainer>] [-ResourceGroupName <String>]
 [-Name <String>] [<CommonParameters>]
```

## OPIS
Polecenie Get-AzConsumptionBudget cmdlet pobiera listę budżetów w ramach subskrypcji lub grupy zasobów.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie listy budżetów na poziomie subskrypcji

PS C:\> Get-AzConsumptionBudget
