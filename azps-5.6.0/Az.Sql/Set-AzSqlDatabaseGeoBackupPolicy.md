# Set-AzSqlDatabaseGeoBackupPolicy

## SYNOPSIS
Ustawia zasady tworzenia kopii geograficznej bazy danych.

## SKŁADNIA

```
Set-AzSqlDatabaseGeoBackupPolicy -State <GeoBackupPolicyState> [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzSqlDatabaseGeoBackupPolicy** ustawia zasady tworzenia kopii zapasowej geolokalizacji zarejestrowane w bazie danych.
Jest to zasób usługi Azure Backup, który służy do definiowania zasad przechowywania kopii zapasowej.
