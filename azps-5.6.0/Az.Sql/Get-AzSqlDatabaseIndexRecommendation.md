# Get-AzSqlDatabaseIndexRecommendation

## SYNOPSIS
Pobiera zalecane operacje indeksowania dla serwera lub bazy danych.

## SKŁADNIA

```
Get-AzSqlDatabaseIndexRecommendation -ServerName <String> [-DatabaseName <String>] [-TableName <String>]
 [-IndexRecommendationName <String>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzSqlDatabaseIndexRecommendation** pobiera zalecane operacje indeksowania dla serwera lub bazy danych Azure SQL Database.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie zaleceń dotyczących indeksu dla wszystkich baz danych na serwerze
```
PS C:\>Get-AzSqlDatabaseIndexRecommendation -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

To polecenie zwraca zalecenia dotyczące indeksu dla wszystkich baz danych na serwerze server01.

### Przykład 2. Uzyskiwanie zaleceń dotyczących indeksu dla określonej bazy danych
```
PS C:\>Get-AzSqlDatabaseIndexRecommendation -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

To polecenie zwraca zalecenia dotyczące indeksu dla konkretnej bazy danych.

### Przykład 3. Uzyskiwanie zalecenia dotyczącego pojedynczego indeksu według nazwy
```
PS C:\>Get-AzSqlDatabaseIndexRecommendation -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01" -IndexRecommendationName "INDEX_NAME"
```

To polecenie zwraca zalecenia dotyczące pojedynczego indeksu według nazwy.
