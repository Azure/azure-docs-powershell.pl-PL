# Set-AzSqlServerRecommendedActionState

## SYNOPSIS
Aktualizuje stan zalecanej akcji w programie Azure SQL Server.

## SKŁADNIA

```
Set-AzSqlServerRecommendedActionState -RecommendedActionName <String> -State <RecommendedActionState>
 -ServerName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzSqlServerRecomctionState** aktualizuje stan zalecanej akcji programu Azure SQL Server.
To polecenie cmdlet stosuje, przywraca lub odrzuca zalecane działanie na podstawie nowego stanu.
