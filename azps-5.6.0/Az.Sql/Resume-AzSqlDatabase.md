# Resume-AzSqlDatabase

## SYNOPSIS
Wznawia pracę z bazą danych programu SQL Data Warehouse.

## SKŁADNIA

```
Resume-AzSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Resume-AzSqlDatabase** wznawia bazę danych usługi Azure SQL Data Warehouse.

## PRZYKŁADY

### Przykład 1. Wznawia korzystanie z bazy danych azure SQL Data Warehouse
```
PS C:\>Resume-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

To polecenie wznawia zawieszoną bazę danych usługi Azure SQL Data Warehouse.
