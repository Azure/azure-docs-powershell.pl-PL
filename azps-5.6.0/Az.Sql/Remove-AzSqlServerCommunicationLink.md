# Remove-AzSqlServerCommunicationLink

## SYNOPSIS
Usuwa link komunikacji dla elastycznych transakcji bazy danych między dwoma serwerami.

## SKŁADNIA

```
Remove-AzSqlServerCommunicationLink [-LinkName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Remove-AzSqlServerCommunicationLink** usuwa link do komunikacji między serwerami w celu elastycznej transakcji bazy danych w usłudze Azure SQL Database.

## PRZYKŁADY

### Przykład 1. Usuwanie linku do komunikacji
```
PS C:\>Remove-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01"
```

To polecenie usuwa link do komunikacji między serwerami o nazwie Link01 w usłudze ContosoServer17.
