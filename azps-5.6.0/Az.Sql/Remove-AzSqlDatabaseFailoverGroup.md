# Remove-AzSqlDatabaseFailoverGroup

## SYNOPSIS
Usuwa grupę trybu failover usługi Azure SQL Database.

## SKŁADNIA

```
Remove-AzSqlDatabaseFailoverGroup [-ServerName] <String> [-FailoverGroupName] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
To polecenie usuwa grupę trybu failover o określonej nazwie, pozostawiając wszystkie bazy danych i relacje replikacji bez zmian. Punkt końcowy słuchacza nie zostanie wyrejestrowany z systemu DNS.
Do wykonania polecenia należy użyć serwera podstawowego grupy trybu failover.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Remove-AzSqlDatabaseFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg
```

Usuwanie grupy trybu failover.
