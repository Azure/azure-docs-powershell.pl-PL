# Suspend-AzSqlDatabase

## SYNOPSIS
Zawiesza bazę danych PROGRAMU SQL Data Warehouse.

## SKŁADNIA

```
Suspend-AzSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Suspend-AzSqlDatabase** zawiesza bazę danych azure SQL Data Warehouse.

## PRZYKŁADY

### Przykład 1. Zawiesza bazę danych usługi Azure SQL Data Warehouse
```
PS C:\>Suspend-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

To polecenie zawiesza aktywną bazę danych azure SQL Data Warehouse.
