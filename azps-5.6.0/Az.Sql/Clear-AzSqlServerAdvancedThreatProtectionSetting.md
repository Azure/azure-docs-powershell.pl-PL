# Clear-AzSqlServerAdvancedThreatProtectionSetting

## SYNOPSIS
Usuwa z serwera zaawansowane ustawienia ochrony przed zagrożeniami.

## SKŁADNIA

```
Clear-AzSqlServerAdvancedThreatProtectionSetting [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie Clear-AzSqlServerAdvancedThreatProtectionSetting cmdlet usuwa zaawansowane ustawienia ochrony przed zagrożeniami z serwera Azure SQL.
Aby użyć tego polecenia cmdlet, określ parametry ResourceGroupName i ServerName w celu określenia serwera, z którego to polecenie cmdlet usuwa ustawienia.

## PRZYKŁADY

### Przykład 1. Usuwanie zaawansowanych ustawień ochrony przed zagrożeniami dla bazy danych
```
PS C:\> Clear-AzSqlServerAdvancedThreatProtectionSetting -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
```

To polecenie usuwa zaawansowane ustawienia ochrony przed zagrożeniami z serwera o nazwie Server01.
