# New-AzSqlServerDnsAlias

## SYNOPSIS
To polecenie umożliwia utworzenie nowego aliasu DNS programu Azure SQL Server.

## SKŁADNIA

```
New-AzSqlServerDnsAlias -Name <String> [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Tworzy nowy alias DNS programu Azure SQL Server, który wskaże określony serwer.

## PRZYKŁADY

### Przykład 1
```
PS C:\> $serverDNSAlias = New-AzSqlServerDnsAlias -ResourceGroupName rg -ServerName serverName -DnsAliasName aliasName

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
```

To polecenie tworzy alias DNS programu Azure SQL Server z nazwą aliasname, która wskazuje nazwę serwera
