# Get-AzSqlInstanceDatabaseBackupLongTermRetentionPolicy

## SYNOPSIS
Pobiera długoterminowe zasady przechowywania zarządzanej bazy danych

## SKŁADNIA

```
Get-AzSqlInstanceDatabaseBackupLongTermRetentionPolicy [-InstanceName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzSqlInstanceDatabaseBackupLongTermRetentionPolicy** pobiera zasady przechowywania danych długoterminowych zarejestrowane w tej zarządzanej bazie danych.
Zasady są zasobem kopii zapasowej azure używanym do definiowania zasad przechowywania kopii zapasowych.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Get-AzSqlInstanceDatabaseBackupLongTermRetentionPolicy -ResourceGroupName testResourceGroup -InstanceName testInstance -DatabaseName test


ResourceGroupName   : testResourceGroup
ManagedInstanceName : testInstance
DatabaseName        : test
WeeklyRetention     : P2W
MonthlyRetention    : PT0S
YearlyRetention     : PT0S
WeekOfYear          : 0
Location            :
```

Pobiera bieżącą wersję długoterminowych zasad przechowywania dla bazy danych
