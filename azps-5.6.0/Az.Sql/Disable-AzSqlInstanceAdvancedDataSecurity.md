# Disable-AzSqlInstanceAdvancedDataSecurity

## SYNOPSIS
Wyłącza zaawansowane zabezpieczenia danych w wystąpieniu zarządzanym.

## SKŁADNIA

```
Disable-AzSqlInstanceAdvancedDataSecurity [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Disable-AzSqlInstanceAdvancedDataSecurity** wyłącza zaawansowane zabezpieczenia danych w wystąpieniu zarządzanym.

## PRZYKŁADY

### Przykład 1. Wyłączanie zaawansowanego zabezpieczeń danych zarządzanego wystąpienia
```powershell
PS C:\>  Disable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```

### Przykład 2. Wyłączanie zaawansowanego zabezpieczeń danych serwera z zasobu zarządzanych wystąpień
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Disable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```
