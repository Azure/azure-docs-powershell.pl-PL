# Get-AzSqlInstanceDatabaseGeoBackup

## SYNOPSIS
Zwraca informacje o nadmiarowej kopii zapasowej bazy danych zarządzanej przez sql azure.

## SKŁADNIA

```
Get-AzSqlInstanceDatabaseGeoBackup [[-Name] <String>] [-InstanceName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzSqlInstanceDatabaseGeoBackup** pobiera co najmniej jedną nadmiarową kopię zapasową baz danych Azure SQL z zarządzanego wystąpienia usługi Azure SQL Database.
