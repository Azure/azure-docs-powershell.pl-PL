# Get-AzSqlDatabaseBackupLongTermRetentionPolicy

## SYNOPSIS
Otrzymuje zasady przechowywania długoterminowych baz danych.

## SKŁADNIA

```
Get-AzSqlDatabaseBackupLongTermRetentionPolicy [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzSqlDatabaseBackupLongTermRetentionPolicy** pobiera do tej bazy danych zasady przechowywania przez długi czas.
Zasady są zasobem kopii zapasowej azure używanym do definiowania zasad przechowywania kopii zapasowych.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie bieżącej wersji zasad przechowywania długoterminowych
```powershell
PS C:\> Get-AzSqlDatabaseBackupLongTermRetentionPolicy -ResourceGroupName resourcegroup01 -ServerName server01 -DatabaseName database01


ResourceGroupName                      : resourcegroup01
ServerName                             : server01
DatabaseName                           : database01
WeeklyRetention                        : P2W
MonthlyRetention                       : PT0S
YearlyRetention                        : PT0S
WeekOfYear                             : 0
State                                  :
RecoveryServicesBackupPolicyResourceId :
Location                               :
```

To polecenie pobiera bieżącą wersję zasad przechowywania długoterminowego dla bazy danych 01
