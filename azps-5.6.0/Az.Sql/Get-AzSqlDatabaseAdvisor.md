# Get-AzSqlDatabaseAdvisor

## SYNOPSIS
Otrzymuje co najmniej jednego doradcę w usłudze Azure SQL Database.

## SKŁADNIA

```
Get-AzSqlDatabaseAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -DatabaseName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzSqlDatabaseAdvisor** pobiera co najmniej jednego doradcę usługi Azure SQL Database Database dla usługi Azure SQL Database.
