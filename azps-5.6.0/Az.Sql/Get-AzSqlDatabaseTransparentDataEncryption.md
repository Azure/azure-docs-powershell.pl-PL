# Get-AzSqlDatabaseTransparentDataEncryption

## SYNOPSIS
Pobiera stan TDE dla bazy danych.

## SKŁADNIA

```
Get-AzSqlDatabaseTransparentDataEncryption [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzSqlDatabaseTransparentDataEncryption** pobiera stan transparentnego szyfrowania danych (TDE, Transparent Data Encryption) dla bazy danych Azure SQL.
Aby uzyskać więcej informacji, zobacz przezroczyste szyfrowanie danych przy użyciu usługi Azure SQL Database https://msdn.microsoft.com/library/dn948096 https://msdn.microsoft.com/library/dn948096) (w bibliotece Microsoft Developer Network Library).
To polecenie cmdlet uzyskuje bieżący stan TDE, ale zarówno szyfrowanie, jak i odszyfrowywanie mogą być operacjami długotrwałym.
Aby wyświetlić postęp skanowania szyfrowania, uruchom Get-AzSqlDatabaseTransparentDataEncryptionActivity cmdlet.
To polecenie cmdlet jest również obsługiwane przez usługę SQL Server Stretch Database na platformie Azure.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie stanu TDE dla bazy danych
```
PS C:\>Get-AzSqlDatabaseTransparentDataEncryption -ServerName "server01" -ResourceGroupName "resourcegroup01" -DatabaseName "database01"
ResourceGroupName             ServerName                    DatabaseName                                          State
-----------------             ----------                    ------------                                          -----
resourcegroup01               server01                      database01                                            Disabled
```

To polecenie pobiera stan TDE dla bazy danych o nazwie Database01 na serwerze o nazwie serwer01.
