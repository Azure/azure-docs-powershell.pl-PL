# Remove-AzSqlServerDnsAlias

## SYNOPSIS
Usuwa alias DNS programu Azure SQL Server.

## SKŁADNIA

### Usuwanie aliasu DNS serwera z parametrów wejściowych polecenia cmdlet
```
Remove-AzSqlServerDnsAlias -Name <String> -ServerName <String> [-ResourceGroupName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Usuwanie aliasu DNS serwera z definicji wystąpienia AzureSqlServerDnsAliasModel
```
Remove-AzSqlServerDnsAlias -InputObject <AzureSqlServerDnsAliasModel> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Usuwanie aliasu DNS serwera z identyfikatora zasobu platformy Azure
```
Remove-AzSqlServerDnsAlias -ResourceId <String> [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Te polecenia usuwają alias DNS programu Azure SQL Server z serwera, pozostawiając serwer bez zmian.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Remove-AzSqlServerDnsAlias -DnsAliasName aliasName -ServerName serverName -ResourceGroupName rg
```

Usuwa alias DNS programu Azure SQL Server z nazwą aliasName z serwera z nazwą nazwa_serwera
