# Get-AzFrontDoor

## SYNOPSIS
Uzyskaj równoważenie obciążenia drzwi przednich

## SKŁADNIA

```
Get-AzFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzFrontDoor** pobiera wszystkie istniejące drzwi frontowe w bieżącej subskrypcji, które są takie, jak podane informacje.
