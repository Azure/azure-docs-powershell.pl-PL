# Uninstall-AzureRm

## SYNOPSIS
Usuwa wszystkie moduły usługi AzureRm z komputera.

## SKŁADNIA

```
Uninstall-AzureRm [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Usuwa wszystkie moduły usługi AzureRm z komputera.

## PRZYKŁADY

### Przykład 1
```
PS C:\> Uninstall-AzureRm
```

Uruchomienie tego polecenia spowoduje usunięcie z komputera wszystkich modułów usługi AzureRm dla wersji programu PowerShell, w której jest uruchamiane polecenie cmdlet.
