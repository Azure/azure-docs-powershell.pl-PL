# Get-AzSupportService

## SYNOPSIS
Uzyskaj usługi, dla których jest dostępna pomoc techniczna. 

## SKŁADNIA

### ListParameterSet (domyślne)
```
Get-AzSupportService [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByNameParameterSet
```
Get-AzSupportService -Id <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Pobiera bieżącą listę usług platformy Azure, dla których jest dostępna pomoc techniczna. Każda usługa może zawierać co najmniej jedną informację o typie zasobów menedżera zasobów platformy Azure (ARM). Typy zasobów (na przykład " microsoft.compute/virtualmachines") mogą być przydatne do zamapowania odpowiedniego produktu i zasobu ARM podczas tworzenia biletu pomocy technicznej. Każda usługa Azure ma własny zestaw kategorii nazywany klasyfikacją problemu. Pobierz bieżącą listę klasyfikacji problemów dla usługi za pomocą funkcji Get-AzSupportProblemClassification. Za pomocą identyfikatora GUID klasyfikacji usługi i problemu możesz utworzyć nowy bilet pomocy technicznej przy użyciu new-azSupportTicket.

Identyfikatory GUID klasyfikacji usługi i problemu zawsze należy stosować programowo. Takie rozwiązanie zapewnia, że masz najnowszy zestaw usług i identyfikatory GUID klasyfikacji problemów do tworzenia biletu pomocy technicznej.
