# Get-AzEventHub

## SYNOPSIS
Pobiera szczegółowe informacje dotyczące pojedynczego centrum zdarzeń lub listę Centrum zdarzeń.

## SKŁADNIA

```
Get-AzEventHub [-ResourceGroupName] <String> [-Namespace] <String> [[-Name] <String>] [-MaxCount <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie Get-AzEventHub zwraca szczegóły centrum zdarzeń lub listę wszystkich centrum zdarzeń w bieżącej przestrzeni nazw.
Jeśli podano nazwę Centrum zdarzeń, zostaną zwrócone szczegóły pojedynczego Centrum zdarzeń.
Jeśli nazwa Centrum zdarzeń nie jest podana, jest zwracana lista wszystkich centrum zdarzeń w określonej przestrzeni nazw.

## PRZYKŁADY

### Przykład 1: określono usługę EventHub
```powershell
PS C:\> Get-AzEventHub -ResourceGroupName MyResourceGroupName -NamespaceName MyNamespaceName -EventHubName MyEventHubName
```

Zwraca szczegóły nazwy \` MyEventHubName centrum \` zdarzeń.

### Przykład 2. Lista aplikacji EventHub w określonej przestrzeni nazw
```powershell
PS C:\> Get-AzEventHub -ResourceGroup MyResourceGroupName -NamespaceName MyNamespaceName
```

Zwraca listę centrum zdarzeń w przestrzeni nazw \` MyNamespaceName. \`
