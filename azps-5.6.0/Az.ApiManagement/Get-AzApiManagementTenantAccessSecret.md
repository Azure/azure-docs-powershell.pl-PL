# Get-AzApiManagementTenantAccessSecret

## SYNOPSIS
Pobiera klucze konfiguracji dostępu dla dzierżawy.

## SKŁADNIA

```
Get-AzApiManagementTenantAccessSecret -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Pobiera klucze konfiguracji dostępu dla dzierżawy.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie kluczy konfiguracji dostępu do dzierżawy
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantAccessSecret -Context $apimContext
```

To polecenie pobiera klucze konfiguracji dostępu dzierżawy w określonym kontekście.
