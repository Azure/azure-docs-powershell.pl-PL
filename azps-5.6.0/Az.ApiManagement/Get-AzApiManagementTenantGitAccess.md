# Get-AzApiManagementTenantGitAccess

## SYNOPSIS
Pobiera konfigurację dostępu Git dla dzierżawy.

## SKŁADNIA

```
Get-AzApiManagementTenantGitAccess -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzApiManagementTenantGitAccess** pobiera konfigurację dostępu Git dla dzierżawy.
Klucze nie zostaną uwzględnione w szczegółach wyników. Aby uzyskać klucz tajny klienta, użyj funkcji **Get-AzApiManagementTenantGitAccessSecret.**

## PRZYKŁADY

### Przykład 1. Uzyskiwanie konfiguracji dostępu do dzierżawy
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantGitAccess -Context $apimContext
```
