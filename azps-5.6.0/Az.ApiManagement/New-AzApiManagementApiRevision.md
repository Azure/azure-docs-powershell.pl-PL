# New-AzApiManagementApiRevision

## SYNOPSIS
Tworzy nową wersję istniejącego interfejsu API.

## SKŁADNIA

```
New-AzApiManagementApiRevision -Context <PsApiManagementContext> -ApiId <String> -ApiRevision <String>
 [-ApiRevisionDescription <String>] [-SourceApiRevision <String>] [-ServiceUrl <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS

Polecenie **cmdlet New-AzApiManagementApiRevision** tworzy poprawkę interfejsu API dla istniejącego interfejsu API w kontekście zarządzania interfejsem API.

## PRZYKŁADY

### Przykład 1. Tworzenie pustej poprawki interfejsu API dla interfejsu API
```powershell
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>New-AzApiManagementApiRevision -Context $context -ApiId "echo-api" -ApiRevision "5"


New-AzApiManagementApiRevision -Context $context -ApiId "echo-api" -ApiRevision "5"
```

To polecenie tworzy poprawkę interfejsu `2` `echo-api` API interfejsu API.

### Przykład 2. Tworzenie poprawki interfejsu API z istniejącego interfejsu API i kopiowanie wszystkich operacji, tagów i zasad
```powershell
PS C:\>$context = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>New-AzApiManagementApiRevision -Context $context -ApiId "echo-api" -ApiRevision "5" -SourceApiRevision "1" -ServiceUrl "https://echoapi.cloudapp.net/rev4"
```
