# Set-AzApiManagementTenantAccess

## SYNOPSIS
Włączenie lub wyłączenie dostępu do dzierżawy.

## SKŁADNIA

```
Set-AzApiManagementTenantAccess -Context <PsApiManagementContext> -Enabled <Boolean> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzApiManagementTenantAccess** włącza lub wyłącza dostęp do dzierżawy.

## PRZYKŁADY

### Przykład 1. Włączanie dostępu do dzierżawy
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Set-AzApiManagementTenantAccess -Context $apimContext -Enabled $True
```

To polecenie umożliwia dostęp do dzierżawy w określonym kontekście.
