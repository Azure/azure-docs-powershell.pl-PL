# Get-AzApiManagementBackend

## SYNOPSIS
Poznaj szczegóły dotyczące bazy danych.

## SKŁADNIA

### ContextParameterSet (Domyślne)
```
Get-AzApiManagementBackend -Context <PsApiManagementContext> [-BackendId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzApiManagementBackend [-BackendId <String>] -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Poznaj szczegóły dotyczące bazy danych.

## PRZYKŁADY

### Przykład 1. Pobierz wszystkie bazy danych
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementBackend -Context $apimContext
```

Pobiera listę wszystkich zaplecza skonfigurowanych w usłudze zarządzania interfejsami API.

### Przykład 2. Uzyskiwanie bazy danych określonej przez identyfikator 123
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementBackend -Context $apimContext -backendId 123
```

Szczegółowe informacje dotyczące wskazanej bazy danych oznaczonej identyfikatorem "123"
