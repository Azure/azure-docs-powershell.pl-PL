# New-AzApiManagementHttpMessageDiagnostic

## SYNOPSIS
Tworzy wystąpienie narzędzia **PsApiManagementHttpMessageDiagnostic,** które jest ustawieniem diagnostycznym Http Message narzędzia Diagnostic

## SKŁADNIA

```
New-AzApiManagementHttpMessageDiagnostic [-HeadersToLog <String[]>] [-BodyBytesToLog <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **New-AzApiManagementHttpMessageDiagnostic** tworzy ustawienie diagnostyczne http wiadomości.

## PRZYKŁADY

### Przykład 1. Tworzenie podstawowego ustawienia diagnostycznego wiadomości HTTP
```powershell
PS C:\>  New-AzApiManagementHttpMessageDiagnostic -Headers 'Content-Type', 'UserAgent' -BodyBytes 100

Headers                   Body
-------                   ----
{Content-Type, UserAgent} Microsoft.Azure.Commands.ApiManagement.ServiceManagement.Models.PsApiManagementBodyDiagnosticSetting
```

Tworzenie ustawienia diagnostycznego wiadomości HTTP do dziennika `Content-Type` i `User-Agent` nagłówków wraz z 100 bajtami `body`
