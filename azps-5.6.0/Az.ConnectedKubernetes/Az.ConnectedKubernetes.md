# Moduł Az.ConnectedKubernetes
## Opis
Microsoft Azure PowerShell: polecenia cmdlet ConnectedKubernetes

## Az.ConnectedKubernetes Cmdlet
### [Get-AzConnectedKubernetes](Get-AzConnectedKubernetes.md)
Zwraca właściwości określonego połączonego klastra, takie jak nazwa, tożsamość, właściwości i dodatkowe szczegóły klastrów.

### [New-AzConnectedKubernetes](New-AzConnectedKubernetes.md)
Interfejs API do rejestrowania nowego klastru K8s, a tym samym tworzenia śledzonego zasobu w ARM

### [Remove-AzConnectedKubernetes](Remove-AzConnectedKubernetes.md)
Usuń połączony klaster, usuwając śledzony zasób w usłudze Azure Resource Manager (ARM).

### [Update-AzConnectedKubernetes](Update-AzConnectedKubernetes.md)
Interfejs API do aktualizowania niektórych właściwości połączonego zasobu klastrów.
