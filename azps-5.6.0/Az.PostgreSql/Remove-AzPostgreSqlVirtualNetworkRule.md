# Remove-AzPostgreSqlVirtualNetworkRule

## SYNOPSIS
Usuwa regułę sieci wirtualnej z podaną nazwą.

## SKŁADNIA

### Delete (Default)
```
Remove-AzPostgreSqlVirtualNetworkRule -Name <String> -ResourceGroupName <String> -ServerName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzPostgreSqlVirtualNetworkRule -InputObject <IPostgreSqlIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## OPIS
Usuwa regułę sieci wirtualnej z podaną nazwą.

## PRZYKŁADY

### Przykład 1. Usunięcie reguły sieci wirtualnej serwera PostgreSql według nazwy
```powershell
PS C:\> Remove-AzPostgreSqlVirtualNetworkRule -Name vnet -ResourceGroupName PostgreSqlTestRG -ServerName PostgreSqlTestServer

```

To polecenie cmdlet usuwa regułę wirtualnej sieci serwera PostgreSql według nazwy.

### Przykład 2. Usuwanie reguły sieci wirtualnej serwera PostgreSql przez tożsamość
```powershell
PS C:\> $ID = "/subscriptions/<SubscriptionId>/resourceGroups/PostgreSqlTestRG/providers/Microsoft.DBforPostgreSQL/servers/PostgreSqlTestServer/virtualNetworkRules/vnet"
PS C:\> Remove-AzPostgreSqlVirtualNetworkRule -InputObject $ID
 
```

Te polecenia cmdlet usuwają regułę wirtualnej sieci serwera PostgreSql za pomocą tożsamości.
