# Get-AzDataLakeAnalyticsJobPipeline

## SYNOPSIS
Pobiera potok zadania lub potoki zadania usługi Data Lake Analytics.

## SKŁADNIA

### GetAllInAccount (domyślne)
```
Get-AzDataLakeAnalyticsJobPipeline [-Account] <String> [-SubmittedAfter <DateTimeOffset>]
 [-SubmittedBefore <DateTimeOffset>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetBySpecificJobPipeline
```
Get-AzDataLakeAnalyticsJobPipeline [-Account] <String> [-PipelineId] <Guid> [-SubmittedAfter <DateTimeOffset>]
 [-SubmittedBefore <DateTimeOffset>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Narzędzie **Get-AzDataLakeAnalyticsJobPipeline** pobiera określony potok zadania usługi Azure Data Lake Analytics lub listę potoków.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie określonego potoku
```
PS C:\>Get-AzDataLakeAnalyticsJobPipeline -Account "contosoadla" -PipelineId 83cb7ad2-3523-4b82-b909-d478b0d8aea3
```

To polecenie pobiera określony potok o identyfikatorze "83cb7ad2-3523-4b82-b909-d478b0d8aea3" na koncie "contosoadla".

### Przykład 2. Uzyskiwanie listy wszystkich potoków na koncie
```
PS C:\>Get-AzDataLakeAnalyticsJobPipeline -AccountName "contosoadla"
```

To polecenie pobiera listę wszystkich potoków na koncie "contosoadla"
