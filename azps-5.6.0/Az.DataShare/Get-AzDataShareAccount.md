# Get-AzDataShareAccount

## SYNOPSIS
Pobiera informacje o kontach DataShare

## SKŁADNIA

### ByFieldsParameterSet (Domyślne)
```
Get-AzDataShareAccount [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceGroupParameterSet
```
Get-AzDataShareAccount -ResourceGroupName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzDataShareAccount -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzDataShareAccount** pobiera informacje o kontach usługi datashare w grupie subskrypcji/zasobów platformy Azure.
Jeśli określisz nazwę konta, to polecenie cmdlet pobiera informacje o tym koncie datshare.
Jeśli nie określisz nazwy, to polecenie cmdlet pobiera informacje o wszystkich kontach udostępniania danych w grupie subskrypcji/zasobów platformy Azure.
