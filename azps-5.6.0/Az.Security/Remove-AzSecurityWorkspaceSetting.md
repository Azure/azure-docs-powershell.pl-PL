# Remove-AzSecurityWorkspaceSetting

## SYNOPSIS
Usuwa ustawienie obszaru roboczego zabezpieczeń dla tej subskrypcji.

## SKŁADNIA

### SubscriptionLevelResource (Domyślne)
```
Remove-AzSecurityWorkspaceSetting -Name <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Remove-AzSecurityWorkspaceSetting -ResourceId <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzSecurityWorkspaceSetting -InputObject <PSSecurityWorkspaceSetting> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Usuwa ustawienie obszaru roboczego zabezpieczeń dla tej subskrypcji.
Ta akcja spowoduje, że nowo zainstalowani agenci zabezpieczeń będą raportować w domyślnym obszarze roboczym tej subskrypcji.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Remove-AzSecurityWorkspaceSetting -Name "default"
```

Usuwa ustawienie obszaru roboczego zabezpieczeń dla tej subskrypcji.
