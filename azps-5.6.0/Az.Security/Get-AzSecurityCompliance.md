# Get-AzSecurityCompliance

## SYNOPSIS
Uzyskiwanie z czasem zgodności zabezpieczeń subskrypcji

## SKŁADNIA

### SubscriptionScope (domyślne)
```
Get-AzSecurityCompliance [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SubscriptionLevelResource
```
Get-AzSecurityCompliance -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId
```
Get-AzSecurityCompliance -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Pobiera zgodność zabezpieczeń subskrypcji na podstawie bieżącego stosunku zasobów o dobrej kondycji i niebędące zabezpieczeniami dla tej subskrypcji.
Zgodność zabezpieczeń jest obliczana każdego dnia i zapisywana jest historia.
