# Remove-AzStreamAnalyticsFunction

## SYNOPSIS
Usuwa funkcję z zadania analizy strumienia.

## SKŁADNIA

```
Remove-AzStreamAnalyticsFunction [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzStreamAnalyticsFunction** usuwa funkcję asynchronicznie z zadania usługi Azure Stream Analytics.

## PRZYKŁADY

### Przykład 1. Usuwanie funkcji Analiza strumienia
```
PS C:\>Remove-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

To polecenie usuwa funkcję o nazwie ScoreTweet z zadania o nazwie StreamJob22.
