# Test-AzStreamAnalyticsOutput

## SYNOPSIS
Sprawdza stan połączenia danych wyjściowych.

## SKŁADNIA

```
Test-AzStreamAnalyticsOutput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Test-AzStreamAnalyticsOutput** sprawdza możliwość łączenia się z wyjściem przez usługę Stream Analytics.

## PRZYKŁADY

### Przykład 1. Testowanie stanu połączenia danych wyjściowych
```powershell
PS C:\>Test-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

To sprawdza stan połączenia wyjściowego danych wyjściowych w transmisję strumieniową WI-Fi.
