# Get-AzStreamAnalyticsOutput

## SYNOPSIS
Pobiera dane wyjściowe zdefiniowane w określonym zadaniu lub danych wyjściowych usługi Stream Analytics.

## SKŁADNIA

```
Get-AzStreamAnalyticsOutput [-JobName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzStreamAnalyticsOutput** wyświetla listę wszystkich wyników zdefiniowanych w określonym zadaniu analizy strumienia lub pobierających informacje o określonym wyniku.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie informacji o wynikach zadań
```powershell
PS C:\>Get-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob"
```

To polecenie zwraca informacje o wynikach zdefiniowanych w zadaniu StreamingJob.

### Przykład 2. Uzyskiwanie informacji o określonym wyniku zadania
```powershell
PS C:\>Get-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

To polecenie zwraca informacje o wynikach o nazwie Dane wyjściowe zdefiniowane w zadaniu StreamingJob.
