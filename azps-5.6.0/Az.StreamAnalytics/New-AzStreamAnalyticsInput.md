# New-AzStreamAnalyticsInput

## SYNOPSIS
Tworzy lub aktualizuje dane wejściowe zadania.

## SKŁADNIA

```
New-AzStreamAnalyticsInput [-JobName] <String> [[-Name] <String>] [-File] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzStreamAnalyticsInput** tworzy dane wejściowe w zadaniu Usługi Stream Analytics lub aktualizuje istniejące dane wejściowe.
Nazwę danych wejściowych można określić w pliku JSON lub w wierszu polecenia.
Jeśli oba te pola zostaną określone, nazwa w wierszu polecenia musi być dopasowana do nazwy w pliku.
Jeśli określisz dane wejściowe, które już istnieją i nie określisz parametru *Force,* polecenie cmdlet zapyta, czy zamienić istniejące dane wejściowe.
Jeśli określisz parametr *Force* i określisz istniejącą nazwę wprowadzania, dane wejściowe zostaną zastąpione bez potwierdzenia.

## PRZYKŁADY

### Przykład 1. Tworzenie danych wejściowych zadania z definicją z pliku
```powershell
PS C:\>New-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -File "C:\Input.json"
```

To polecenie umożliwia utworzenie danych wejściowych z Input.jspliku.
Jeśli istniejące dane wejściowe o nazwie określonej w pliku definicji danych wejściowych są już zdefiniowane, polecenie cmdlet zapyta, czy ma zostać zamienione.

### Przykład 2. Tworzenie danych wejściowych zadania
```powershell
PS C:\>New-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -File "C:\Input.json" -Name "EntryStream"
```

To polecenie tworzy nowe dane wejściowe dla zadania o nazwie EntryStream.
Jeśli dane wejściowe o tej nazwie są już zdefiniowane, polecenie cmdlet zapyta, czy ma zostać zamienione.

### Przykład 3. Zamienianie informacji o zadaniu na definicję z pliku
```powershell
PS C:\>New-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -File "C:\Input.json" -Name "EntryStream" -Force
```

To polecenie zastępuje definicję istniejącego źródła wprowadzania o nazwie EntryStream definicją z pliku bez potwierdzenia.
