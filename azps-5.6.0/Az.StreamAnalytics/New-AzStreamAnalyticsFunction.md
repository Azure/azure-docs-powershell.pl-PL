# New-AzStreamAnalyticsFunction

## SYNOPSIS
Tworzy lub zamienia funkcję w zadaniu Analiza strumienia.

## SKŁADNIA

```
New-AzStreamAnalyticsFunction [-JobName] <String> [[-Name] <String>] [-File] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzStreamAnalyticsFunction** tworzy funkcję w zadaniu Azure Stream Analytics lub zastępuje istniejącą funkcję.
Zdefiniuj funkcję w pliku JavaScript Object Notation (JSON).
Nazwę funkcji można określić za pomocą *parametru Name* lub w pliku json.
Jeśli określisz nazwę w obie strony, nazwy muszą być zgodne.
Aby zamienić istniejącą funkcję, określ nazwę istniejącej funkcji.

## PRZYKŁADY

### Przykład 1. Tworzenie funkcji Analiza strumienia
```
PS C:\>New-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob07" -File "C:\Function07.json"
```

To polecenie tworzy funkcję na podstawie Function07.jspliku.
Nazwa funkcji jest przechowywana w pliku json.

### Przykład 2. Tworzenie funkcji analizy strumienia o nazwie ScoreTweet
```
PS C:\>New-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -File "C:\Function22.json" -Name "ScoreTweet"
```

To polecenie tworzy funkcję w zadaniu o nazwie ScoreTweet.

### Przykład 3. Zamienianie funkcji Analiza strumienia
```
PS C:\>New-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -File "C:\Function22.json" -Name "ScoreTweet" -Force
```

To polecenie zastępuje definicję istniejącej funkcji o nazwie ScoreTweet definicją z tabeli Function22.jswł.
