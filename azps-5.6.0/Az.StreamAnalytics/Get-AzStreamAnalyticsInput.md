# Get-AzStreamAnalyticsInput

## SYNOPSIS
Pobiera informacje o zadaniach analizy strumieniowej.

## SKŁADNIA

```
Get-AzStreamAnalyticsInput [-JobName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzStreamAnalyticsInput** wyświetla listę wszystkich danych wejściowych zdefiniowanych w zadaniu analizy strumienia lub pobierających informacje o określonych danych wejściowych.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie informacji o danych wejściowych zdefiniowanych w zadaniu
```powershell
PS C:\>Get-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob"
```

To polecenie zwraca informacje o wszystkich danych wejściowych zdefiniowanych w zadaniu StreamingJob.

### Przykład 2. Uzyskiwanie informacji o konkretnym wejściu zdefiniowanym w zadaniu
```powershell
PS C:\>Get-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

To polecenie zwraca informacje o danych wejściowych o nazwie EntryStream zdefiniowanych w zadaniu StreamingJob.
