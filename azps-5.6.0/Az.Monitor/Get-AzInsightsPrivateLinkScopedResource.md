# Get-AzInsightsPrivateLinkScopedResource

## SYNOPSIS
Uzyskiwanie prywatnego zasobu z zakresem linków

## SKŁADNIA

### ByScopeParameterSet (domyślne)
```
Get-AzInsightsPrivateLinkScopedResource -ResourceGroupName <String> -ScopeName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByInputObjectParameterSet
```
Get-AzInsightsPrivateLinkScopedResource [-Name <String>] -InputObject <PSMonitorPrivateLinkScope>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzInsightsPrivateLinkScopedResource -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Uzyskiwanie lub lista dla prywatnego zasobu z zakresem linków, zasób o ograniczonych zakresach może być obszarem roboczym analizy dziennika lub składnikiem Szczegółowych informacji o aplikacji

## PRZYKŁADY

### Przykład 1
```powershell
Get-AzInsightsPrivateLinkScopedResource -ResourceGroupName "rg_name" -ScopeName "scope_name"
```

Zasób o zakresie listy w zakresie linków prywatnych "scope_name"

### Przykład 2
```powershell
Get-AzInsightsPrivateLinkScopedResource -ResourceGroupName "rg_name" -ScopeName "scope_name" -Name "scoped_resource_name"
```

Uzyskiwanie zasobu objętego zakresem w ramach prywatnego zakresu łączy "scope_name" o nazwie "scoped_resource_name"
