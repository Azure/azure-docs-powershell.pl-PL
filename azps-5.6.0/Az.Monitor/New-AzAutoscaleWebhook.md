# New-AzAutoscaleWebhook

## SYNOPSIS
Tworzy webhook autoskalowania.

## SKŁADNIA

```
New-AzAutoscaleWebhook [-ServiceUri] <String> [[-Property] <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzAutoscaleWebhook** tworzy autoskalowanie webhook.

## PRZYKŁADY

### Przykład 1

Tworzy webhook autoskalowania. (wygenerowane automatycznie)

```powershell <!-- Aladdin Generated Example --> 
New-AzAutoscaleWebhook -ServiceUri 'https://contoso.com'
```
