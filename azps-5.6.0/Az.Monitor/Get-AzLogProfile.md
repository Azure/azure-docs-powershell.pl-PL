# Get-AzLogProfile

## SYNOPSIS
Pobiera profil dziennika.

## SKŁADNIA

```
Get-AzLogProfile [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzLogProfile** pobiera profil dziennika.
