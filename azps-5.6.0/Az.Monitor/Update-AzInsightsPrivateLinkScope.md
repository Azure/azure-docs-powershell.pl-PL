# Update-AzInsightsPrivateLinkScope

## SYNOPSIS
Aktualizacja dla prywatnego zakresu łączy

## SKŁADNIA

### ByResourceNameParameterSet (Domyślny)
```
Update-AzInsightsPrivateLinkScope -ResourceGroupName <String> -Name <String> [-Tags <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Update-AzInsightsPrivateLinkScope -ResourceId <String> [-Tags <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObjectParameterSet
```
Update-AzInsightsPrivateLinkScope -InputObject <PSMonitorPrivateLinkScope> [-Tags <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Aktualizacja dla prywatnego zakresu łączy

## PRZYKŁADY

### Przykład 1
```powershell
Update-AzInsightsPrivateLinkScope -ResourceGroupName "rg_name" -Name "scope_name" -Tags "key:value"
```

zaktualizuj zakres łącza prywatnego o nazwie "scope_name" w grupie zasobów "rg_name", aby użyć tagu "klucz:wartość"

### Przykład 2
```powershell
Update-AzInsightsPrivateLinkScope -ResourceId "/subscriptions/{subscriptionId}/resourceGroups/rg_name/providers/microsoft.insights/privateLinkScopes/scope_name" -Tags "key:value"
```

Aktualizowanie zakresu łącza prywatnego przy użyciu identyfikatora zasobu w celu użycia tagu "klucz:wartość"

### Przykład 3
```powershell
Get-AzInsightsPrivateLinkScope -ResourceGroupName "rg_name" -Name "scope_name" | Update-AzInsightsPrivateLinkScope -Tags "key:value"
```

Aktualizowanie zakresu łącza prywatnego przy użyciu obiektu wejściowego zakresu łącza prywatnego w celu użycia tagu "klucz:wartość"
