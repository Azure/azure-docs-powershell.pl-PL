# New-AzStorageTableStoredAccessPolicy

## SYNOPSIS
Tworzy przechowywane zasady dostępu dla tabeli magazynu platformy Azure.

## SKŁADNIA

```
New-AzStorageTableStoredAccessPolicy [-Table] <String> [-Policy] <String> [-Permission <String>]
 [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet New-AzStorageTableStoredAccessPolicy** tworzy zasady dostępu przechowywanego dla tabeli magazynu platformy Azure.

## PRZYKŁADY

### Przykład 1. Tworzenie przechowywanych zasad dostępu w tabeli
```
PS C:\>New-AzStorageTableStoredAccessPolicy -Table "MyTable" -Policy "Policy02"
```

To polecenie tworzy zasady dostępu o nazwie Zasady02 w tabeli magazynu o nazwie Moja tabela.
