# Set-AzStorageQueueStoredAccessPolicy

## SYNOPSIS
Ustawia zasady dostępu przechowywanego dla kolejki magazynu platformy Azure.

## SKŁADNIA

```
Set-AzStorageQueueStoredAccessPolicy [-Queue] <String> [-Policy] <String> [-Permission <String>]
 [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-NoStartTime] [-NoExpiryTime] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzStorageQueueStoredAccessPolicy** ustawia zasady dostępu przechowywanego dla kolejki magazynu platformy Azure.

## PRZYKŁADY

### Przykład 1. Ustawianie przechowywanych zasad dostępu w kolejce z pełnymi uprawnieniami
```
PS C:\> Set-AzStorageQueueStoredAccessPolicy -Queue "MyQueue" -Policy "Policy07" -Permission arup
```

To polecenie ustawia zasady dostępu o nazwie Zasady07 dla kolejki magazynu o nazwie Moje kolejkowanie.
