# Enable-AzStorageDeleteRetentionPolicy

## SYNOPSIS
Włącz zasady przechowywania usuwania dla usługi obiektów blob magazynu platformy Azure.

## SKŁADNIA

```
Enable-AzStorageDeleteRetentionPolicy [-RetentionDays] <Int32> [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Enable-AzStorageDeleteRetentionPolicy** umożliwia usuwanie zasad przechowywania dla usługi obiektów blob magazynu platformy Azure.

## PRZYKŁADY

### Przykład 1. Włączanie zasad przechowywania usuwania dla usługi obiektów blob
```
C:\PS>Enable-AzStorageDeleteRetentionPolicy -RetentionDays 3
```

To polecenie umożliwia usuwanie zasad przechowywania dla usługi obiektów blob i ustawianie dni przechowywania usuniętych obiektów blob na 3.
