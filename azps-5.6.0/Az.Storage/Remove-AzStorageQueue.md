# Remove-AzStorageQueue

## SYNOPSIS
Usuwa kolejkę magazynu.

## SKŁADNIA

```
Remove-AzStorageQueue [-Name] <String> [-Force] [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzStorageQueue** usuwa kolejkę magazynu.

## PRZYKŁADY

### Przykład 1. Usuwanie kolejki magazynu według nazwy
```
PS C:\>Remove-AzStorageQueue "ContosoQueue01"
```

To polecenie usuwa kolejkę o nazwie ContosoQueue01.

### Przykład 2. Usuwanie wielu kolejek magazynu
```
PS C:\>Get-AzStorageQueue "Contoso*" | Remove-AzStorageQueue
```

To polecenie usuwa wszystkie kolejki z nazwami, które zaczynają się od nazwy Contoso.
