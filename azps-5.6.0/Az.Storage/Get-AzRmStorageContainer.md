# Get-AzRmStorageContainer

## SYNOPSIS
Pobiera lub wyświetla kontenery obiektów blob magazynu

## SKŁADNIA

### AccountName (Default)
```
Get-AzRmStorageContainer [-ResourceGroupName] <String> [-StorageAccountName] <String> [-Name <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### AccountObject
```
Get-AzRmStorageContainer -StorageAccount <PSStorageAccount> [-Name <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzRmStorageContainer** pobiera lub wyświetla kontenery obiektów blob magazynu

## PRZYKŁADY

### Przykład 1. Uzyskiwanie kontenera obiektów blob magazynu z nazwą konta magazynu i nazwą kontenera
```
PS C:\>Get-AzRmStorageContainer -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount" -ContainerName "myContainer"
```

To polecenie pobiera kontener obiektów blob magazynu z nazwą konta magazynu i nazwą kontenera.

### Przykład 2. Lista wszystkich kontenerów obiektów blob magazynu konta magazynu
```
PS C:\>Get-AzRmStorageContainer -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount"
```

To polecenie zawiera listę wszystkich kontenerów obiektów blob magazynu na koncie magazynu z nazwą konta magazynu.

### Przykład 3. Pobierz kontener obiektów blob magazynu z obiektem konta magazynu i nazwą kontenera.
```
PS C:\>$accountObject = Get-AzStorageAccount -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount"
PS C:\>Get-AzRmStorageContainer -StorageAccount $accountObject -ContainerName "myContainer"
```

To polecenie pobiera kontener obiektów blob magazynu z obiektem konta magazynu i nazwą kontenera.
