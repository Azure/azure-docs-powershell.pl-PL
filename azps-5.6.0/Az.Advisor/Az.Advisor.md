# Moduł Az.Advisor
## Opis
Polecenia umożliwiające interakcję z doradcą platformy Azure.

## Polecenia cmdlet az.advisor
### [Get-AzAdvisorRecommendation](Get-AzAdvisorRecommendation.md)
Otrzymuje listę zaleceń dotyczących usługi Azure Advisor.

### [Disable-AzAdvisorRecommendation](Disable-AzAdvisorRecommendation.md)
Wyłącz zalecenie doradcy platformy Azure.

### [Enable-AzAdvisorRecommendation](Enable-AzAdvisorRecommendation.md)
Włącza zalecenia doradcy platformy Azure.

### [Get-AzAdvisorConfiguration](Get-AzAdvisorConfiguration.md)
Uzyskaj konfiguracje doradcy platformy Azure dla danej subskrypcji lub grupy zasobów.

### [Set-AzAdvisorConfiguration](Set-AzAdvisorConfiguration.md)
Aktualizuje lub tworzy konfigurację doradcy platformy Azure.
