# Get-AzStorageSyncCloudEndpoint

## SYNOPSIS
To polecenie wyświetla listę wszystkich punktów końcowych chmury w obrębie danej grupy synchronizacji.

## SKŁADNIA

### StringParameterSet (domyślne)
```
Get-AzStorageSyncCloudEndpoint [-ResourceGroupName] <String> [-StorageSyncServiceName] <String>
 [-SyncGroupName] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ObjectParameterSet
```
Get-AzStorageSyncCloudEndpoint [-ParentObject] <PSSyncGroup> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ParentStringParameterSet
```
Get-AzStorageSyncCloudEndpoint [-ParentResourceId] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
To polecenie wyświetla listę wszystkich punktów końcowych chmury w obrębie danej grupy synchronizacji. Za jego pomocą można również wyświetlić listę atrybutów poszczególnych punktów końcowych chmury.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Get-AzStorageSyncCloudEndpoint -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName" -SyncGroupName "mySyncGroupName"
```

To polecenie pobiera wszystkie punkty końcowe chmury zawarte w określonej grupie synchronizacji. Określanie wartości -CloudEndpointName w celu zwrócenia określonej wartości.
