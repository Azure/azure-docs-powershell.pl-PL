# Get-AzStorageSyncService

## SYNOPSIS
To polecenie zawiera listę wszystkich usług synchronizacji przestrzeni dyskowej w obrębie danego zakresu grupy subskrypcji/zasobów.

## SKŁADNIA

### ParentStringParameterSet (domyślne)
```
Get-AzStorageSyncService [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### StringParameterSet
```
Get-AzStorageSyncService [-ResourceGroupName] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
To polecenie zawiera listę wszystkich usług synchronizacji przestrzeni dyskowej w obrębie danego zakresu grupy subskrypcji/zasobów. Za jego pomocą można również wyświetlić listę atrybutów poszczególnych usług synchronizacji przestrzeni dyskowej.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Get-AzStorageSyncService -ResourceGroupName "myResourceGroup"
```

To polecenie zawiera listę wszystkich zasobów usługi synchronizacji magazynu w obrębie danej grupy zasobów. Za jego pomocą można również wyświetlić listę atrybutów poszczególnych usług synchronizacji przestrzeni dyskowej. Specify -StorageSyncServiceName, aby zwrócić określoną.
