# Remove-AzOperationalInsightsStorageInsight

## SYNOPSIS
Usuwa szczegółowe informacje o przestrzeni dyskowej.

## SKŁADNIA

### ByWorkspaceName (Domyślna)
```
Remove-AzOperationalInsightsStorageInsight [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-Name] <String> [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByWorkspaceObject
```
Remove-AzOperationalInsightsStorageInsight [-Workspace] <PSWorkspace> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzOperationalInsightsStorageInsight** usuwa szczegółowe informacje o magazynie z obszaru roboczego.

## PRZYKŁADY

### Przykład 1. Usuwanie szczegółowych informacji o magazynie według nazwy
```
PS C:\>Remove-AzOperationalInsightsStorageInsight -ResourceGroupName "ContosoResourceGroup" -WorkspaceName "MyWorkspace" -Name "MyStorageInsight"
```

To polecenie usuwa analizę magazynu o nazwie MyStorageInsight z obszaru roboczego o nazwie MyWorkspace w określonej grupie zasobów.
Polecenie nie określa *parametru Force,* dlatego wyświetla monit o potwierdzenie przed usunięciem szczegółowych informacji o magazynie.

### Przykład 2. Usuwanie szczegółowych informacji o magazynie bez potwierdzenia
```
PS C:\>$Workspace = Get-AzOperationalInsightsWorkspace -ResourceGroupName "ContosoResourceGroup" -Name "MyWorkspace"

PS C:\>Remove-AzOperationalInsightsStorageInsight -Workspace $Workspace -Name "MyStorageInsight" -Force
```

Pierwsze polecenie używa Get-AzOperationalInsightsWorkspace cmdlet w celu uzyskania obszaru roboczego o nazwie MyWorkspace, a następnie zapisuje go w $Workspace danych. Drugie polecenie usuwa z magazynu szczegółowe informacje o magazynie o nazwie MyStorageInsight $Workspace bez monitowania o potwierdzenie.
