# Disable-AzOperationalInsightsLinuxPerformanceCollection

## SYNOPSIS
Zatrzymuje zbieranie liczników wydajności z komputerów z systemem Linux.

## SKŁADNIA

### ByWorkspaceName (Domyślna)
```
Disable-AzOperationalInsightsLinuxPerformanceCollection [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByWorkspaceObject
```
Disable-AzOperationalInsightsLinuxPerformanceCollection [-Workspace] <PSWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Disable-AzOperationalInsightsLinuxPerformanceCollection** zatrzymuje kolekcję liczników wydajności z połączonych komputerów z systemem Linux w obszarze roboczym.
