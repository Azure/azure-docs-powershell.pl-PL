# Remove-AzRoleDefinition

## SYNOPSIS
Usuwa niestandardową rolę w usłudze Azure RBAC.
Rola do usunięcia jest określana przy użyciu właściwości Identyfikator roli.
Usunięcie nie powiedzie się, jeśli istnieją istniejące przypisania ról wprowadzone do roli niestandardowej.

## SKŁADNIA

### RoleDefinitionIdParameterSet (domyślne)
```
Remove-AzRoleDefinition -Id <Guid> [-Scope <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RoleDefinitionNameParameterSet
```
Remove-AzRoleDefinition [-Name] <String> [-Scope <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzRoleDefinition -InputObject <PSRoleDefinition> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie Remove-AzRoleDefinition usuwa niestandardową rolę w usłudze Azure Role-Based Access Control.
Podaj parametr Id istniejącej roli niestandardowej, aby usunąć tę rolę niestandardową.
Domyślnie program Remove-AzRoleDefinition monit o potwierdzenie.
Aby pominąć monit, użyj parametru Force.
Jeśli istnieją istniejące przypisania ról do roli niestandardowej do usunięcia, usunięcie nie powiedzie się.

## PRZYKŁADY

### Przykład 1
```
Get-AzRoleDefinition -Name "Virtual Machine Operator" | Remove-AzRoleDefinition
```

### Przykład 2
```
Remove-AzRoleDefinition -Id "52a6cc13-ff92-47a8-a39b-2a8205c3087e"
```
