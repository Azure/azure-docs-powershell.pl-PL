# Register-AzResourceProvider

## SYNOPSIS
Rejestruje dostawcę zasobów.

## SKŁADNIA

```
Register-AzResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Register-AzResourceProvider** rejestruje dostawcę zasobów platformy Azure.

## PRZYKŁADY

### Przykład 1. Rejestrowanie dostawcy
```
PS C:\>Register-AzResourceProvider -ProviderNamespace Microsoft.Network
```

Zarejestruje to dostawcę witryny Microsoft.Network dla Twojego konta.
