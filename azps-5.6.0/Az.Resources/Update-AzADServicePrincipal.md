# Update-AzADServicePrincipal

## SYNOPSIS
Aktualizuje istniejącą jednostkę zabezpieczeń usługi Azure Active Directory.

## SKŁADNIA

### SpObjectIdWithDisplayNameParameterSet (Domyślne)
```
Update-AzADServicePrincipal -ObjectId <String> [-DisplayName <String>] [-Homepage <String>]
 [-IdentifierUri <String[]>] [-KeyCredential <KeyCredential[]>] [-PasswordCredential <PasswordCredential[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SpApplicationIdWithDisplayNameParameterSet
```
Update-AzADServicePrincipal -ApplicationId <Guid> [-Homepage <String>] [-IdentifierUri <String[]>]
 [-KeyCredential <KeyCredential[]>] [-PasswordCredential <PasswordCredential[]>]

```

### SPNWithDisplayNameParameterSet
```

 [-IdentifierUri <String[]>] [-KeyCredential <KeyCredential[]>] [-PasswordCredential <PasswordCredential[]>]

```

### InputObjectWithDisplayNameParameterSet
```

```

## OPIS
Aktualizuje istniejącą jednostkę zabezpieczeń usługi Azure Active Directory. Aby zaktualizować poświadczenia skojarzone z tym podmiotem zabezpieczeń usługi, użyj New-AzADSpCredential cmdlet. Aby zaktualizować właściwości skojarzone z aplikacją bazową, użyj Update-AzADApplication cmdlet.

## PRZYKŁADY

### Przykład 1. Aktualizowanie nazwy wyświetlanej podmiotu zabezpieczeń usługi

```powershell
PS C:\> Update-AzADServicePrincipal -ObjectId 784136ca-3ae2-4fdd-a388-89d793e7c780 -DisplayName MyNewDisplayName
```

Aktualizuje nazwę wyświetlaną podmiotu zabezpieczeń usługi o identyfikatorze obiektu "784136ca-3ae2-4fdd-a388-89d793e7c780" na "MyNewDisplayName".

### Przykład 2. Aktualizowanie nazwy wyświetlanej podmiotu zabezpieczeń usługi za pomocą funkcji pipingu

```powershell
PS C:\> Get-AzADServicePrincipal -ObjectId 784136ca-3ae2-4fdd-a388-89d793e7c780 | Update-AzADServicePrincipal -DisplayName MyNewDisplayName
```

Pobiera podmiot zabezpieczeń usługi o identyfikatorze obiektu "784136ca-3ae2-4fdd-a388-89d793e7c780" i potoki, które do polecenia cmdlet programu Update-AzADServicePrincipal są przekierowywane w celu zaktualizowania nazwy wyświetlanej podmiotu zabezpieczeń usługi na "MyNewDisplayName".

### Przykład 3

Aktualizuje istniejącą jednostkę zabezpieczeń usługi Azure Active Directory. (wygenerowane automatycznie)
