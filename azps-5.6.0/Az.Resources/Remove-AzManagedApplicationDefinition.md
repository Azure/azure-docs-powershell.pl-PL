# Remove-AzManagedApplicationDefinition

## SYNOPSIS
Usuwa zarządzaną definicję aplikacji

## SKŁADNIA

### RemoveByNameAndResourceGroup (Domyślna)
```
Remove-AzManagedApplicationDefinition -Name <String> -ResourceGroupName <String> [-Force]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### RemoveById
```
Remove-AzManagedApplicationDefinition -Id <String> [-Force] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzManagedApplicationDefinition** usuwa definicję aplikacji zarządzanej

## PRZYKŁADY

### Przykład 1. Usuwanie definicji aplikacji zarządzanej według identyfikatora zasobu
```
PS C:\>$ApplicationDefinition = Get-AzManagedApplicationDefinition -Name "myAppDef" -ResourceGroupName "myRG"
PS C:\>Remove-AzManagedApplicationDefinition -Id $ApplicationDefinition.ResourceId -Force
```

Pierwsze polecenie otrzymuje zarządzaną definicję aplikacji o nazwie myAppDef przy użyciu Get-AzManagedApplicationDefinition cmdlet.
Polecenie przechowuje je w $ApplicationDefinition zmienną.
Drugie polecenie usuwa definicję aplikacji zarządzanej zidentyfikowanych przez właściwość **ResourceId** $ApplicationDefinition.
