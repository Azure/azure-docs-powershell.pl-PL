# Remove-AzServiceFabricNode

## SYNOPSIS
Usuwanie węzłów z określonego typu węzła z klastrów.

## SKŁADNIA

```
Remove-AzServiceFabricNode -NumberOfNodesToRemove <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Użyj **funkcji Remove-AzServiceFabricNode,** aby usunąć węzły z określonego typu węzła z klastry. Usuwanie będzie kontynuowane tylko wtedy, gdy spełnia ono metrykę kondycji klastrów.

## PRZYKŁADY

### Przykład 1
```powershell
PS c:> Remove-AzServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeType 'nt1' -NumberOfNodesToRemove 2
```

To polecenie spowoduje usunięcie 2 węzłów z typu NodeType 'nt1'.
