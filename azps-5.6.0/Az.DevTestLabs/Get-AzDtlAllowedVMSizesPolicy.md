# Get-AzDtlAllowedVMSizesPolicy

## SYNOPSIS
Pobiera zasady dozwolonego rozmiaru maszyn wirtualnych w laboratorium DevTest Labs.

## SKŁADNIA

```
Get-AzDtlAllowedVMSizesPolicy [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzDtlAllowedVMSizesPolicy** pobiera zasady dozwolonych rozmiarów maszyn wirtualnych, które umożliwiają określenie listy dozwolonych rozmiarów maszyn wirtualnych w laboratorium.
Polecenie cmdlet zwraca włączony lub wyłączony stan zasad oraz listę wszystkich dozwolonych rozmiarów maszyn wirtualnych ustawionych w określonych zasadach.
