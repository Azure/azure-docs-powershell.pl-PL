# Set-AzDtlAutoShutdownPolicy

## SYNOPSIS
Ustawia zasady automatycznego zamykania laboratorium DevTest Labs.

## SKŁADNIA

### Włącz (domyślne)
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Enable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Wyłącz
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Disable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzDtlAutoShutdownPolicy** ustawia zasady automatycznego zamykania laboratorium, co spowoduje automatyczne zamknięcie wszystkich maszyn wirtualnych w laboratorium o określonej porze dnia.
Polecenie cmdlet używa określonej grupy zasobów i nazwy laboratorium do ustawienia zasad.

## PRZYKŁADY

### Przykład 1

Ustawia zasady automatycznego zamykania laboratorium DevTest Labs. (wygenerowane automatycznie)

```powershell <!-- Aladdin Generated Example --> 
Set-AzDtlAutoShutdownPolicy -Enable -LabName <String> -ResourceGroupName MyResourceGroup -Time <DateTime>
```
