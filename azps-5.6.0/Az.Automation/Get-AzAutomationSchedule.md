# Get-AzAutomationSchedule

## SYNOPSIS
Pobiera harmonogram automatyzacji.

## SKŁADNIA

### ByAll (Domyślne)
```
Get-AzAutomationSchedule [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzAutomationSchedule [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzAutomationSchedule** otrzymuje harmonogram automatyzacji azure.

## PRZYKŁADY

### Przykład 1. Uzyskiwanie harmonogramu
```
PS C:\>Get-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "DailySchedule08" -ResourceGroupName "ResourceGroup01"
```

To polecenie pobiera harmonogram o nazwie DziennySchedule08.
