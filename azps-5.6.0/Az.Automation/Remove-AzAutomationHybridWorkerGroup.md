# Remove-AzAutomationHybridWorkerGroup

## SYNOPSIS
Usuwa grupę pracowników hybrydowych z automatyzacji.

## SKŁADNIA

```
Remove-AzAutomationHybridWorkerGroup [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie Remove-AzAutomationHybridWorkerGroup cmdlet usuwa grupę pracowników hybrydowych z automatyzacji.

## PRZYKŁADY

### Przykład 1
To polecenie usuwa pracownika hybrydowego według imienia i nazwiska.

```powershell
PS C:\> Remove-AzAutomationHybridWorkerGroup -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "GroupName" `
                                                  -Force
```
