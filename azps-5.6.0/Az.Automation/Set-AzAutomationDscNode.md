# Set-AzAutomationDscNode

## SYNOPSIS
Modyfikuje konfigurację węzła, na który jest zamapowany węzeł DSC.

## SKŁADNIA

```
Set-AzAutomationDscNode -Id <Guid> -NodeConfigurationName <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie cmdlet **Set-AzAutomationDscNode** modyfikuje konfigurację węzła APS Desired State Configuration (DSC).
Usługa Azure Automation przechowuje konfigurację węzła DSC jako dokument konfiguracji formatu zarządzanego obiektu (MOF, Managed Object Format).

## PRZYKŁADY

### Przykład 1. Modyfikowanie mapowania konfiguracji węzła
```
PS C:\>Set-AzAutomationDscNode -NodeConfigurationName "Contoso.NodeConfiguration01" -ResourceGroupName "ResourceGroup01" -Id 064a8929-c98b-25e4-80hh-111c8a6067j8
```

To polecenie przypisuje konfigurację węzła o nazwie Contoso.NodeConfiguration01 do węzła o określonym identyfikatorze GUID.
