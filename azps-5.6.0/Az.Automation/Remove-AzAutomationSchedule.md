# Remove-AzAutomationSchedule

## SYNOPSIS
Usuwa harmonogram automatyzacji.

## SKŁADNIA

```
Remove-AzAutomationSchedule [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzAutomationSchedule** usuwa harmonogram z automatyzacji platformy Azure.

## PRZYKŁADY

### Przykład 1. Usuwanie harmonogramu
```
PS C:\>Remove-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -ResourceGroupName "ResourceGroup01"
```

To polecenie usuwa harmonogram o nazwie Harmonogram01 na koncie automatyzacji Contoso17 w grupie zasobów ResourceGroup01.
