# Set-AzAutomationSchedule

## SYNOPSIS
Modyfikuje harmonogram automatyzacji.

## SKŁADNIA

```
Set-AzAutomationSchedule [-Name] <String> [-IsEnabled <Boolean>] [-Description <String>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzAutomationSchedule** modyfikuje harmonogram w usłudze Azure Automation.

## PRZYKŁADY

### Przykład 1. Modyfikowanie harmonogramu
```
PS C:\>Set-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -Description "Automation Schedule" -ResourceGroupName "ResourceGroup01"
```

To polecenie modyfikuje opis harmonogramu o nazwie Harmonogram01.
