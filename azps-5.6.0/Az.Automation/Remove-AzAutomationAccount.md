# Remove-AzAutomationAccount

## SYNOPSIS
Usuwa konto automatyzacji.

## SKŁADNIA

```
Remove-AzAutomationAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Remove-AzAutomationAccount** usuwa konto usługi Azure Automation z grupy zasobów.
Aby uzyskać więcej informacji na temat kont automatyzacji, zobacz New-AzAutomationAccount cmdlet.

## PRZYKŁADY

### Przykład 1. Usuwanie konta automatyzacji
```
PS C:\>Remove-AzAutomationAccount -Name "ContosoAutomationAccount" -Force -ResourceGroupName "ResourceGroup01"
```

To polecenie usuwa konto automatyzacji o nazwie ContosoAutomationAccount bez monitowania o sprawdzanie poprawności użytkownika.
