# Remove-AzAutomationSourceControl

## SYNOPSIS
Usuwa kontrolkę źródła azure automation.

## SKŁADNIA

```
Remove-AzAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie Remove-AzAutomationSourceControl cmdlet usuwa kontrolkę źródłową z usługi Azure Automation.

## PRZYKŁADY

### Przykład 1
To polecenie usuwa kontrolkę źródła automatyzacji o nazwie VSTSNative z konta o nazwie devAccount.
To polecenie określa *parametr Force.* Dlatego nie jest wyświetlany monit o potwierdzenie.

```powershell
PS C:\> Remove-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
