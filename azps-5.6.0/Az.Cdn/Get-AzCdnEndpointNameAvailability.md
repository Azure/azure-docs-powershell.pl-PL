# Get-AzCdnEndpointNameAvailability

## SYNOPSIS
Pobiera stan dostępności punktu końcowego sieci CDN.

## SKŁADNIA

```
Get-AzCdnEndpointNameAvailability -EndpointName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzCdnEndpointNameAvailability** pobiera stan dostępności punktu końcowego usługi Azure Content Delivery Network (CDN).
