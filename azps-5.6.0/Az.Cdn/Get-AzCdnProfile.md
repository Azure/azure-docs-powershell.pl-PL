# Get-AzCdnProfile

## SYNOPSIS
Pobiera profil CDN.

## SKŁADNIA

```
Get-AzCdnProfile [-ProfileName <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Get-AzCdnProfile** pobiera profil usługi Azure Content Delivery Network (CDN) i powiązane z nim informacje.
