# Set-AzCdnEndpoint

## SYNOPSIS
Aktualizuje punkt końcowy sieci CDN.

## SKŁADNIA

```
Set-AzCdnEndpoint -CdnEndpoint <PSEndpoint> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzCdnEndpoint** aktualizuje punkt końcowy usługi Azure Content Delivery Network (CDN).
