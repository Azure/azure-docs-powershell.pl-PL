# New-AzContainerRegistry

## SYNOPSIS
Tworzy rejestr kontenera.

## SKŁADNIA

```
New-AzContainerRegistry [-ResourceGroupName] <String> [-Name] <String> [-Sku] <String> [-Location <String>]
 [-EnableAdminUser] [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie New-AzContainerRegistry cmdlet tworzy rejestr kontenera.

## PRZYKŁADY

### Przykład 1. Tworzenie rejestru kontenera przy użyciu nowego konta magazynu.
```powershell
PS C:\>New-AzContainerRegistry -ResourceGroupName "MyResourceGroup" -Name "MyRegistry" -Sku "Basic"

   Container registry location: eastus

Registry Name     Sku        LoginServer               CreationDate               Provisioni AdminUserE StorageAccountN
                                                                                  ngState    nabled     ame
-------------     ---        -----------               ------------               ---------- ---------- ---------------
myregistry        Premium    myregistry.azurecr.io     10/31/2017 6:49:31 PM      Succeeded  True
```

To polecenie tworzy rejestr kontenera z nowym kontem magazynu w grupie zasobów \` MyResourceGroup. \`

### Przykład 2. Tworzenie rejestru kontenera z włączonym użytkownikiem administratora.
```powershell
PS C:\>New-AzContainerRegistry -ResourceGroupName "MyResourceGroup" -Name "MyRegistry" -Sku "Basic" -EnableAdminUser

  Container registry location: eastus

Registry Name     Sku        LoginServer               CreationDate               Provisioni AdminUserE StorageAccountN
                                                                                  ngState    nabled     ame
-------------     ---        -----------               ------------               ---------- ---------- ---------------
myregistry        Premium    myregistry.azurecr.io     10/31/2017 6:49:31 PM      Succeeded  True
```

To polecenie tworzy rejestr kontenera z włączonym użytkownikiem administratora.
