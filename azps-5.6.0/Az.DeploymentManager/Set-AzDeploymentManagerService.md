# Set-AzDeploymentManagerService

## SYNOPSIS
Aktualizuje usługę.

## SKŁADNIA

```
Set-AzDeploymentManagerService [-InputObject] <PSServiceResource> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzDeploymentManagerService** aktualizuje usługę o określony obiekt usługi.
Polecenie cmdlet zwraca zaktualizowany obiekt usługi.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Set-AzDeploymentManagerService -InputObject $serviceObject
```

To polecenie aktualizuje usługę, której nazwa, nazwa topologii usługi i grupa zasobów są zgodne odpowiednio z właściwościami Name (Nazwa Usługi), ServiceTopologyName (Nazwa_usługi) i NazwaGrupy Zasobów ($serviceObject Nazwa Grupy Zasobów).
Usługa zostanie zaktualizowana do właściwości określonych w $serviceObject.
