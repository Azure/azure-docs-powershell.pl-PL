# Stop-AzDeploymentManagerRollout

## SYNOPSIS
Zatrzymuje rozsyłanie.

## SKŁADNIA

### Interakcyjna (domyślna)
```
Stop-AzDeploymentManagerRollout [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Stop-AzDeploymentManagerRollout [-ResourceId] <String> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Stop-AzDeploymentManagerRollout [-InputObject] <PSRollout> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Stop-AzDeploymentManagerRollout** zatrzymuje w toku wycofywanie i zwraca obiekt, który reprezentuje bieżący stan tego rzutowania.
Określ wycofanie według jego nazwy i nazwy grupy zasobów. Ewentualnie możesz podać obiekt Rollout lub Identyfikator Zasobu.

Pamiętaj, że po zatrzymaniu wycofywania nie można wznowić jego działania ani wznowić go ponownie. Możesz utworzyć tylko nowe rzutnie.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Stop-AzDeploymentManagerRollout -ResourceGroupName ContosoResourceGroup -Name ContosoRollout -SkipSucceeded
```

To polecenie zatrzymuje wykonanie o nazwie ContosoRollout w grupie ContosoResourceGroup. 

### Przykład 2. Zatrzymanie wycofywania przy użyciu identyfikatora zasobu
```powershell
PS C:\> Restart-AzDeploymentManagerRollout -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/rollouts/ContosoRollout"
```

To polecenie zatrzymuje wykonanie o nazwie ContosoRollout w grupie ContosoResourceGroup.

### Przykład 3. Zatrzymanie rzutowania przy użyciu obiektu rozsyłania.
```powershell
PS C:\> Get-AzDeploymentManagerRollout -InputObject $rolloutObject
```

To polecenie zatrzymuje rzutowanie, którego nazwa i grupa zasobów są zgodne odpowiednio z właściwościami Name (Nazwa) $rolloutObject ResourceGroupName (Nazwa grupy zasobów).
