# Set-AzDeploymentManagerServiceTopology

## SYNOPSIS
Aktualizuje topologię usługi.

## SKŁADNIA

```
Set-AzDeploymentManagerServiceTopology [-InputObject] <PSServiceTopologyResource>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie **cmdlet Set-AzDeploymentManagerServiceTopology** aktualizuje topologię usługi przy użyciu określonego obiektu topologii usługi.
Polecenie cmdlet zwraca obiekt topologii zaktualizowanej usługi.

## PRZYKŁADY

### Przykład 1
```powershell
PS C:\> Set-AzDeploymentManagerService -InputObject $serviceTopologyObject
```

To polecenie aktualizuje topologię usługi, której nazwa i grupa zasobów są zgodne odpowiednio z właściwościami Name (Nazwa) $serviceTopologyObject ResourceGroupName (Nazwa grupy zasobów).
To polecenie zwraca obiekt zaktualizowanej topologii usługi.
