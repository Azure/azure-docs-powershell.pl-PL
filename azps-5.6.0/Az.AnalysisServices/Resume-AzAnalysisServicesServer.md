# Resume-AzAnalysisServicesServer

## SYNOPSIS
Wznawia wystąpienie serwera usług Analysis Services

## SKŁADNIA

```
Resume-AzAnalysisServicesServer [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## OPIS
Polecenie Resume-AzAnalysisServicesServer cmdlet wznawia wystąpienie serwera usług Analysis Services

## PRZYKŁADY

### Przykład 1
```
PS C:\> Resume-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

To polecenie spowoduje wznowienie wstrzymanego serwera o nazwie serwer testowy w grupie testowej grupy zasobów.
