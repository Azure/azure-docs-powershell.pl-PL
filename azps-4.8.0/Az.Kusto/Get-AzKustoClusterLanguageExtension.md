# Get-AzKustoClusterLanguageExtension

## STRESZCZENIe
Zwraca listę rozszerzeń języków, które mogą być uruchamiane w ramach zapytań programu Keyword.

## POLECENIA

```
Get-AzKustoClusterLanguageExtension -ClusterName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Opis
Zwraca listę rozszerzeń języków, które mogą być uruchamiane w ramach zapytań programu Keyword.

## Przykłady

### Przykład 1: Lista wszystkich rozszerzeń językowych ustawionych dla klastra
```powershell
PS C:\> Get-AzKustoClusterLanguageExtension -ResourceGroupName testrg -ClusterName testnewkustocluster

Name
----
R
PYTHON
```

Powyższe polecenie zwraca listę rozszerzeń języków, które mogą być uruchamiane w ramach zapytań programu Keyword.
