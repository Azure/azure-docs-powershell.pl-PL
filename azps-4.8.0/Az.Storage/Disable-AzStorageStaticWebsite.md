# Disable-AzStorageStaticWebsite

## STRESZCZENIe
Wyłącz statyczną witrynę sieci Web dla konta usługi Azure Storage.

## POLECENIA

```
Disable-AzStorageStaticWebsite [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **disable-AzStorageStaticWebsite** wyłącza statyczną witrynę sieci Web dla konta usługi Azure Storage.

## Przykłady

### Przykład 1: wyłączanie statycznej witryny sieci Web dla konta usługi Azure Storage
```
C:\PS>Disable-AzStorageStaticWebsite
```

To polecenie wyłącza statyczną witrynę sieci Web dla konta usługi Azure Storage.
