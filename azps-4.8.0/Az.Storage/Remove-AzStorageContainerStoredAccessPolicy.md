# Remove-AzStorageContainerStoredAccessPolicy

## STRESZCZENIe
Usuwa zapisane zasady dostępu z kontenera magazynu platformy Azure.

## POLECENIA

```
Remove-AzStorageContainerStoredAccessPolicy [-Container] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzStorageContainerStoredAccessPolicy** usuwa zapisane zasady dostępu z kontenera magazynu platformy Azure.

## Przykłady

### Przykład 1: usuwanie zapisanych zasad dostępu z kontenera magazynu
```
PS C:\>Remove-AzStorageContainerStoredAccessPolicy -Container "MyContainer" -Policy "Policy03"
```

To polecenie usuwa zasady dostępu o nazwie Policy03 z kontenera przechowywanego o nazwie.
