## Opis
Polecenie cmdlet **Invoke-AzDataBoxEdgeDevice** wywołuje akcje skanowania, pobierania i instalowania aktualizacji na urządzeniu z danymi na krawędziach pola danych. Automatyczne skanowanie jest uruchamiane na urządzeniu codziennie, więc uruchomienie tego polecenia cmdlet może spowodować jawne wykonanie skanowania.

## Przykłady

### Przykład 1
```powershell
PS C:\> Invoke-AzDataBoxEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -ScanForUpdate
```

### Przykład 2
```powershell
PS C:\> Invoke-AzDataBoxEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -FetchUpdate
```

### Przykład 3
```powershell
PS C:\> Invoke-AzDataBoxEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -InstallUpdate
```
