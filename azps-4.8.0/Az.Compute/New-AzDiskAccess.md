# New-AzDiskAccess

## STRESZCZENIe
Tworzy zasób dostępu do dysku

## POLECENIA

```
New-AzDiskAccess [-ResourceGroupName] <String> [-Name] <String> [-Location] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **New-AzDiskAccess** tworzy zasób dostępu do dysku

## Przykłady

### Przykład 1
```
PS C:\> New-AzDiskAccess -ResourceGroupName "ResourceGroup01" -Name "DiskAccess01" -Location "NorthCentralUS"
```

To polecenie utworzy dostęp do dysku z podanymi właściwościami.
