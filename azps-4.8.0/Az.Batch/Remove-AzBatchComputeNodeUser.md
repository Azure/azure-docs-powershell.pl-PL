# Remove-AzBatchComputeNodeUser

## STRESZCZENIe
Usuwa konto użytkownika z węzła obliczeniowego partii.

## POLECENIA

```
Remove-AzBatchComputeNodeUser [-PoolId] <String> [-ComputeNodeId] <String> [-Name] <String>
 -BatchContext <BatchAccountContext> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzBatchComputeNodeUser** usuwa konto użytkownika z węzła obliczeniowego usługi Azure Batch.

## Przykłady

### Przykład 1: Usuwanie użytkownika z węzła obliczeniowego bez potwierdzenia
```
PS C:\>Remove-AzBatchComputeNodeUser -PoolId "Pool01" -ComputeNodeId "ComputeNode01" -Name "User14" -Force -BatchContext $Context
```

To polecenie usuwa użytkownika o nazwie User14 z węzła compute o nazwie ComputeNode01.
Węzeł obliczeniowy znajduje się w puli o nazwie Pool01.
To polecenie określa parametr *Force* .
Dlatego polecenie nie monituje o potwierdzenie.

W przypadku korzystania z uwierzytelniania za pomocą klucza udostępnionego domyślnie jest używany podstawowy klucz dostępu.
