# Remove-AzSqlServerCommunicationLink

## STRESZCZENIe
Umożliwia usunięcie łącza komunikacyjnego dla transakcji Elastic Database między dwoma serwerami.

## POLECENIA

```
Remove-AzSqlServerCommunicationLink [-LinkName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzSqlServerCommunicationLink** usuwa link komunikacji między serwerami dla transakcji Elastic Database w usłudze Azure SQL Database.

## Przykłady

### Przykład 1: Usuwanie łącza komunikacyjnego
```
PS C:\>Remove-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01"
```

To polecenie powoduje usunięcie linku komunikacyjnego serwer-do-serwera o nazwie Link01 na ContosoServer17.
