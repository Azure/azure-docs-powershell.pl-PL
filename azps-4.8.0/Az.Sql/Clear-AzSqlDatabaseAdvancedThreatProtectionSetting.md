# Clear-AzSqlDatabaseAdvancedThreatProtectionSetting

## STRESZCZENIe
Usuwa z bazy danych Zaawansowane ustawienia ochrony przed zagrożeniami.

## POLECENIA

```
Clear-AzSqlDatabaseAdvancedThreatProtectionSetting [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Clear-AzSqlDatabaseAdvancedThreatProtectionSetting** usuwa zaawansowane ustawienia ochrony przed zagrożeniami z bazy danych programu AzureAzure SQL.
Aby użyć tego polecenia cmdlet, określ parametry *ResourceGroupName* oraz *nazwa_serwera* w celu zidentyfikowania bazy danych, z której to polecenie cmdlet usunie ustawienia.

## Przykłady

### Przykład 1: usuwanie zaawansowanych ustawień ochrony przed zagrożeniami dla bazy danych
```
PS C:\>Clear-AzSqlDatabaseAdvancedThreatProtectionSetting -ResourceGroupName "ResourceGroup11" -ServerName "Server01" -DatabaseName "Database01"
```

To polecenie usuwa zaawansowane ustawienia ochrony przed zagrożeniami z bazy danych o nazwie Database01 na serwerze o nazwie Server01.
