# Get-AzSqlInstanceKeyVaultKey

## STRESZCZENIe
Pobiera klucze magazynu kluczy wystąpienia zarządzanego przez SQL.

## POLECENIA

### AddAzureRmSqlManagedInstanceKeyVaultKeyDefaultParameterSet (domyślny)
```
Get-AzSqlInstanceKeyVaultKey [[-KeyId] <String>] [-ResourceGroupName] <String> [-InstanceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyInputObjectParameterSet
```
Get-AzSqlInstanceKeyVaultKey [[-KeyId] <String>] [-Instance] <AzureSqlManagedInstanceModel>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyResourceIdParameterSet
```
Get-AzSqlInstanceKeyVaultKey [[-KeyId] <String>] [-InstanceResourceId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Get-AzSqlInstanceKeyVaultKey pobiera informacje o kluczach magazynu kluczy w wystąpieniu zarządzanym SQL. Możesz wyświetlić wszystkie klucze w zarządzanym wystąpieniu lub wyświetlić określony klucz, dostarczając KeyId.

## Przykłady

### Przykład 1: pobieranie wszystkich kluczy magazynu kluczy
```powershell
PS C:\> Get-AzSqlInstanceKeyVaultKey -ResourceGroupName 'ContosoResourceGroup' -InstanceName 'ContosoManagedInstanceName'

Thumbprint             : 6AB10000F99E1B6A22222F39E3F11CB5DC5A55A1
Type                   : AzureKeyVault
```

To polecenie pobiera wszystkie klucze magazynu kluczy w wystąpieniu zarządzanym SQL.

