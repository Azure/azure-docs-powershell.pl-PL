# Enable-AzOperationalInsightsLinuxPerformanceCollection

## STRESZCZENIe
Rozpoczyna zbieranie liczników wydajności z komputerów z systemem Linux.

## POLECENIA

### ByWorkspaceName (domyślny)
```
Enable-AzOperationalInsightsLinuxPerformanceCollection [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByWorkspaceObject
```
Enable-AzOperationalInsightsLinuxPerformanceCollection [-Workspace] <PSWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **enable-AzOperationalInsightsLinuxPerformanceCollection** rozpoczyna zbieranie liczników wydajności z podłączonych komputerów Linux w obszarze roboczym.
