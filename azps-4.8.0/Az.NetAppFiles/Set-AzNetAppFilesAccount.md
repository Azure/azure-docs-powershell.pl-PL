# Set-AzNetAppFilesAccount

## STRESZCZENIe
Umożliwia zaktualizowanie konta usługi Azure NetApp Files (ANF) przy użyciu nowego zestawu danych. Przydatne w przypadku usuwania skojarzonych z nimi aktywnych katalogów.

## POLECENIA

### ByFieldsParameterSet (domyślny)
```
Set-AzNetAppFilesAccount -ResourceGroupName <String> -Location <String> -Name <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByResourceActiveDirectory
```
Set-AzNetAppFilesAccount -Location <String> -Name <String> [-ActiveDirectory <PSNetAppFilesActiveDirectory[]>]
 [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzNetAppFilesAccount** modyfikuje konto usługi ANF.

## Przykłady

### Przykład 1: modyfikowanie konta ANF
```
PS C:\>Set-AzNetAppFilesAccount -ResourceGroupName "MyRG" -l "westus2" -Name "MyAnfAccount"

Output:

Location          : westus2
Id                : /subscriptions/subsId/resourceGroups/MyRG/providers/Microsoft.NetApp/netAppAccounts/MyAnfAccount
Name              : MyAnfAccount
Type              : Microsoft.NetApp/netAppAccounts
Tags              :
AccountId         : 9fa2ca6d-1e48-4439-30e3-7de056e44e5a
ActiveDirectories : {}
ProvisioningState : Succeeded
```

To polecenie wykonuje aktualizację na danym koncie. Brak usługi Active Directory oznacza, że zostanie ona usunięta z konta.
