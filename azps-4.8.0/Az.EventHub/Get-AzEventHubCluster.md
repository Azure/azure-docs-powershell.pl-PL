# Get-AzEventHubCluster

## STRESZCZENIe
Pobiera szczegóły pojedynczego klastra centrum zdarzeń lub listę klastrów centrum zdarzeń.

## POLECENIA

```
Get-AzEventHubCluster [-ResourceGroupName] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet Get-AzEventHubCluster zwraca informacje o klastrze centrum zdarzeń lub listę wszystkich klastrów centrum zdarzeń w danej grupie zasobów.
W przypadku podanej nazwy klastra zostaną zwrócone szczegółowe dane dotyczące pojedynczego klastra.
Jeśli nie podano nazwy klastra, zostanie zwrócona lista wszystkich klastrów w określonej grupie zasobów.

### Przykład 1
```powershell
PS C:\> Get-AzEventHubCluster -ResourceGroupName RSG-Cluster27651 -Name Eventhub-Cluster-5557

            ntHub/clusters/Eventhub-Cluster-5557
```
