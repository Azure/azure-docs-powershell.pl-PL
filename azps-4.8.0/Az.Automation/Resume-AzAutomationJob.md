# Resume-AzAutomationJob

## STRESZCZENIe
Wznawia zawieszone zadanie automatyzacji.

## POLECENIA

```
Resume-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Resume-AzAutomationJob** Wznawia zawieszone zadanie usługi Azure Automation.
Określ zawieszone zadanie.
Aby zawiesić zadanie, użyj polecenia cmdlet Suspend-AzAutomationJob.

## Przykłady

### Przykład 1: wznawianie zawieszonego zadania
```
PS C:\>Resume-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

To polecenie wznawia zadanie o określonym IDENTYFIKATORze.
