# Remove-AzAutomationSourceControl

## STRESZCZENIe
Usuwa kontrolę źródła usługi Azure Automation.

## POLECENIA

```
Remove-AzAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet Remove-AzAutomationSourceControl usuwa kontrolę źródła z usługi Azure Automation.

## Przykłady

### Przykład 1
To polecenie usuwa kontrolkę źródła automatyzacji o nazwie VSTSNative na koncie o nazwie devAccount.
To polecenie określa parametr *Force* . W związku z tym nie jest wyświetlany monit o potwierdzenie.

```powershell
PS C:\> Remove-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
