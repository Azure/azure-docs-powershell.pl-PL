## Opis
Polecenie cmdlet **Remove-AzSynapseRoleAssignment** trwale usuwa przypisanie roli Azure Synapse Analytics.

## Przykłady

### Przykład 1
```powershell
PS C:\> Remove-AzSynapseRoleAssignment -WorkspaceName ContosoWorkspace -RoleAssignmentId ContosoRoleAssignmentId
```

To polecenie usuwa przypisanie roli Azure Synapse Analytics o identyfikator przydziału roli.

### Przykład 2
```powershell
PS C:\> Remove-AzSynapseRoleAssignment -WorkspaceName ContosoWorkspace -RoleAssignmentName ContosoRole -SignInName ContosoName
```

To polecenie usuwa przydział roli usługi Azure Synapse Analytics z nazwą roli i nazwą główną użytkownika.

### Przykład 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Remove-AzSynapseRoleAssignment -RoleAssignmentId ContosoRoleAssignmentId
```

To polecenie usuwa przydział roli usługi Azure Synapse Analytics z identyfikatorem przypisania roli za pomocą potoku.
