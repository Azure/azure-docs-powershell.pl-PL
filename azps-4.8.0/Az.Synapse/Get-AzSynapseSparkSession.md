# Get-AzSynapseSparkSession

## STRESZCZENIe
Pobiera sesję Synapse Analytics Spark.

## POLECENIA

### GetByNameParameterSet (domyślny)
```
Get-AzSynapseSparkSession -WorkspaceName <String> -SparkPoolName <String> [-LivyId <Int32>] [-Name <String>]
 [-ApplicationId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByParentObjectParameterSet
```
Get-AzSynapseSparkSession -SparkPoolObject <PSSynapseSparkPool> [-LivyId <Int32>] [-Name <String>]
 [-ApplicationId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzSynapseSparkSession** pobiera informacje o sesji usługi Azure Synapse Analytics Spark.

## Przykłady

### Przykład 1
```powershell
PS C:\> Get-AzSynapseSparkSession -WorkspaceName ContosoWorkspace -SparkPoolName ContosoSparkPool
```

To polecenie pobiera wszystkie sesje platformy Spark w ramach określonej puli platformy Spark.

### Przykład 2
```powershell
PS C:\> Get-AzSynapseSparkSession -WorkspaceName ContosoWorkspace -SparkPoolName ContosoSparkPool -LivyId 1
```

To polecenie pobiera sesję programu Spark o określonym IDENTYFIKATORze livy.

### Przykład 3
```powershell
PS C:\> $pool = Get-AzSynapseSparkPool -WorkspaceName ContosoWorkspace -Name ContosoSparkPool
PS C:\> $pool | Get-AzSynapseSparkSession
```

To polecenie pobiera wszystkie sesje platformy Spark w określonej puli platformy Spark za pomocą potoku.
