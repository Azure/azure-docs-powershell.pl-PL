# Test-AzSynapseSparkPool

## STRESZCZENIe
Sprawdza, czy istnieje Pula usługi Synapse Analytics Spark.

## POLECENIA

### TestByNameParameterSet (domyślny)
```
Test-AzSynapseSparkPool [-ResourceGroupName <String>] -WorkspaceName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### TestByParentObjectParameterSet
```
Test-AzSynapseSparkPool -Name <String> -WorkspaceObject <PSSynapseWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **test-AzSynapseSparkPool** sprawdza, czy istnieje Pula Synapse Analytics Spark.

## Przykłady

### Przykład 1
```powershell
PS C:\> Test-AzSynapseSparkPool -WorkspaceName ContosoWorkspace -Name ContosoSparkPool
```

To polecenie sprawdza istnienie określonej puli platformy Spark.
