# Set-AzSynapseDataset

## STRESZCZENIe
Tworzenie lub aktualizowanie zestawu danych w obszarze roboczym.

## POLECENIA

### SetByName (domyślny)
```
Set-AzSynapseDataset -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapseDataset -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzSynapseDataset** umożliwia utworzenie zestawu danych lub zaktualizowanie istniejącego zestawu danych w obszarze roboczym.

## Przykłady

### Przykład 1
```powershell
PS C:\> Set-AzSynapseDataset -WorkspaceName ContosoWorkspace -Name ContosoDataset -DefinitionFile "C:\\samples\\Dataset.json"
```

To polecenie tworzy zestaw danych o nazwie ContosoDataset w obszarze roboczym o nazwie ContosoWorkspace.
Polecenie opiera się na zestawie danych na temat informacji zawartych w Dataset.jspliku.
