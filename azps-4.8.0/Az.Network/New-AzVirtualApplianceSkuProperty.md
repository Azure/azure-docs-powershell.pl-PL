# New-AzVirtualApplianceSkuProperty

## STRESZCZENIe
Zdefiniuj dla zasobu jednostkę SKU sieciowego urządzenia wirtualnego.

## POLECENIA

```
New-AzVirtualApplianceSkuProperty -VendorName <String> -BundledScaleUnit <String> -MarketPlaceVersion <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie New-AzVirtualApplianceSkuProperties definiuje jednostkę SKU dla zasobu sieciowego urządzenia wirtualnego.

## Przykłady

### Przykład 1
```powershell
PS C:\> $var=New-AzVirtualApplianceSkuProperty -VendorName "barracudasdwanrelease" -BundledScaleUnit 1 -MarketPlaceVersion 'latest'
```

Utwórz obiekt właściwości jednostki SKU urządzenia wirtualnego, który ma być wykorzystywany z poleceniem New-AzNetworkVirtualAppliance.
