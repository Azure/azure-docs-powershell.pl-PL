# New-AzApplicationGatewayFirewallCustomRule

## STRESZCZENIe
Tworzy nową regułę niestandardową dla zasad zapory Application Gateway.

## POLECENIA

```
New-AzApplicationGatewayFirewallCustomRule -Name <String> -Priority <Int32> -RuleType <String>
 -MatchCondition <PSApplicationGatewayFirewallCondition[]> -Action <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
**Nowy-AzApplicationGatewayFirewallCustomRule** tworzy regułę niestandardową dla zasad zapory.

## Przykłady

### Przykład 1
```powershell
PS C:\> $customRule = New-AzApplicationGatewayFirewallCustomRule -Name example-rule -Priority 1 -RuleType MatchRule -MatchCondition $condtion -Action Allow
```

Polecenie tworzy nową regułę niestandardową z nazwą przykładu — regułą, priorytetem 1, a typ reguły będzie MatchRule z warunkiem zdefiniowanym w zmiennej Condition, akcja będzie dozwolona. Nowa reguła dopasowania jest zapisywana w $customRule.
