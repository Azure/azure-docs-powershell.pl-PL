# New-AzFirewallHubIpAddress

## STRESZCZENIe
Adresy IP assoicated do zapory na wirtualnym koncentratorze

## POLECENIA

```
New-AzFirewallHubIpAddress [-PrivateIPAddress <String>] [-PublicIPs <PSAzureFirewallHubPublicIpAddresses>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Adresy IP assoicated zaporę w wirtualnym koncentratorze. Mogą to być adresy publiczne i prywatne

## Przykłady

### Przykład 1
```powershell
PS C:\> $fwpips = New-AzFirewallHubPublicIpAddress -Count 2
PS C:\> New-AzFirewallHubIpAddress -PublicIPs $fwpips
```

W tym przykładzie jest tworzony obiekt centralny adres IP z liczbą 2 publicznych adresów IP. Obiekt HubIPAddress jest ssociated z zaporą w koncentratorze wirtualnym.
