# Get-AzP2sVpnGateway

## STRESZCZENIe
Pobiera istniejące P2SVpnGateway w obszarze VirtualHub.

## POLECENIA

### ListBySubscriptionId (domyślny)
```
Get-AzP2sVpnGateway [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ListByResourceGroupName
```
Get-AzP2sVpnGateway [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzP2sVpnGateway** umożliwia otrzymywanie istniejących P2SVpnGateway w obszarze VirtualHub.
