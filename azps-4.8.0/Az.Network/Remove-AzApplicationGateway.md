# Remove-AzApplicationGateway

## STRESZCZENIe
Usuwa bramę aplikacji.

## POLECENIA

```
Remove-AzApplicationGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzApplicationGateway** usuwa bramę Application Gateway.

## Przykłady

### Przykład 1: usuwanie określonej bramy aplikacji
```
PS C:\>Remove-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
```

To polecenie usuwa bramkę Application Gateway o nazwie ApplicationGateway01 w grupie zasobów o nazwie ResourceGroup01.
