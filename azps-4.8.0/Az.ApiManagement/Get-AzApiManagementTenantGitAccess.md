# Get-AzApiManagementTenantGitAccess

## STRESZCZENIe
Pobiera konfigurację dostępu git dla dzierżawy.

## POLECENIA

```
Get-AzApiManagementTenantGitAccess -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzApiManagementTenantGitAccess** Pobiera konfigurację dostępu git dla dzierżawy.
Klucze nie zostaną uwzględnione w szczegółach wyników. Aby uzyskać klucz tajny klienta, użyj polecenie **Get-AzApiManagementTenantGitAccessSecret**.

## Przykłady

### Przykład 1: Pobieranie konfiguracji dostępu dzierżawy
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantGitAccess -Context $apimContext
```
