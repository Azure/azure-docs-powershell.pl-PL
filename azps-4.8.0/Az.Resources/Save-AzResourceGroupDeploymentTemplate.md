# Save-AzResourceGroupDeploymentTemplate

## STRESZCZENIe
Zapisuje szablon rozmieszczania grup zasobów do pliku.

## POLECENIA

```
Save-AzResourceGroupDeploymentTemplate -ResourceGroupName <String> -DeploymentName <String> [-Path <String>]
 [-Force] [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Save-AzResourceGroupDeploymentTemplate**  umożliwia zapisanie szablonu rozmieszczania grup zasobów w pliku JSON.

## Przykłady

### Przykład 1. Zapisz szablon wdrożenia
```powershell
PS C:\>Save-AzResourceGroupDeploymentTemplate -DeploymentName "TestDeployment" -ResourceGroupName "TestGroup"
```

To polecenie pobiera szablon wdrożenia z TestDeployment i zapisuje go jako plik JSON w bieżącym katalogu.

### Przykład 2

Zapisuje szablon rozmieszczania grup zasobów do pliku. (autogenerowana)

<!-- Aladdin Generated Example -->
```powershell
Save-AzResourceGroupDeploymentTemplate -DeploymentName 'TestDeployment' -Path <String> -ResourceGroupName 'TestGroup'
```
