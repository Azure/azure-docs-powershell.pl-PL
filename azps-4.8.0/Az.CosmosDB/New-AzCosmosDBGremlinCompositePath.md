# New-AzCosmosDBGremlinCompositePath

## STRESZCZENIe
Tworzy nowy obiekt typu PSCompositePath. Może on zostać przekazany jako wartość parametru w parametrze Set-AzCosmosDBGremlinGraph.

## POLECENIA

```
New-AzCosmosDBGremlinCompositePath [-Path <String>] [-Order <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Obiekt odpowiadający CompositePath API Gremlin.

## Przykłady

### Przykład 1
```powershell
PS C:\> New-AzCosmosDBGremlinCompositePath -Path "/abc" -Order Ascending

Path Order
---- -----
/abc Ascending
```
