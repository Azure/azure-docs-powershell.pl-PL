# Get-AzImportExportBitLockerKey

## STRESZCZENIe
Zwraca klucze funkcji BitLocker dla wszystkich dysków w określonym zadaniu.

## POLECENIA

```
Get-AzImportExportBitLockerKey -JobName <String> -ResourceGroupName <String> [-SubscriptionId <String[]>]
 [-AcceptLanguage <String>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## Opis
Zwraca klucze funkcji BitLocker dla wszystkich dysków w określonym zadaniu.

## Przykłady

### Przykład 1: wyświetlanie wszystkich kluczy funkcji BitLocker w określonym zadaniu ImportExport
```powershell
PS C:\> Get-AzImportExportBitLockerKey -JobName test-job -ResourceGroupName ImportTestRG 
BitLockerKey                                            DriveId
------------                                            -------
238810-662376-448998-450120-652806-203390-606320-483076 9CA995BA
```

To polecenie cmdlet wyświetla wszystkie klucze funkcji BitLocker określone w określonym zadaniu ImportExport.
