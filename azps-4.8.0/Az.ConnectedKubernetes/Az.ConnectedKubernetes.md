# AZ. ConnectedKubernetes, moduł
## Opis
Microsoft Azure PowerShell: polecenia cmdlet ConnectedKubernetes

## AZ. ConnectedKubernetes polecenia cmdlet
### [Get-AzConnectedKubernetes](Get-AzConnectedKubernetes.md)
Zwraca właściwości określonego klastra połączonego, w tym nazwę, tożsamość, właściwości i dodatkowe szczegóły klastra.

### [Nowe — AzConnectedKubernetes](New-AzConnectedKubernetes.md)
Interfejs API umożliwiający zarejestrowanie nowego klastra K8s i w związku z tym utworzenie prześledzonego zasobu w ARM

### [Remove-AzConnectedKubernetes](Remove-AzConnectedKubernetes.md)
Usuń podłączony klaster, usuwając prześledzony zasób w Menedżerze zasobów Azure (ARM).

### [Update-AzConnectedKubernetes](Update-AzConnectedKubernetes.md)
Interfejs API umożliwiający aktualizowanie określonych właściwości połączonego zasobu klastra
