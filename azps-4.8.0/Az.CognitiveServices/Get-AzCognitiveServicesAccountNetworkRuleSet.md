# Get-AzCognitiveServicesAccountNetworkRuleSet

## STRESZCZENIe
Uzyskiwanie właściwości NetworkRule konta usług poznawczych

## POLECENIA

```
Get-AzCognitiveServicesAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzCognitiveServicesAccountNetworkRuleSet** Pobiera właściwość NetworkRule konta usług poznawczych

## Przykłady

### Przykład 1: pobieranie właściwości NetworkRule dla określonego konta usług poznawczych
```
PS C:\> Get-AzCognitiveServicesAccountNetworkRuleSet  -ResourceGroupName "rg1" -Name "myaccount"
```

To polecenie pobiera właściwość NetworkRule określonego konta usług poznawczych.
