# Remove-AzureNetworkSecurityGroup

## STRESZCZENIe
Usuwa grupę zabezpieczeń sieci.

## POLECENIA

```
Remove-AzureNetworkSecurityGroup -Name <String> [-Force] [-PassThru] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureNetworkSecurityGroup** usuwa grupę zabezpieczeń sieci Azure z subskrypcji.
