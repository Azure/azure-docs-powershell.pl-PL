# Remove-AzureLocalNetworkGateway

## STRESZCZENIe
Usuwa bramę lokalnego sieci Azure.

## POLECENIA

```
Remove-AzureLocalNetworkGateway -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureLocalNetworkGateway** usuwa bramę lokalnego sieci Azure.
