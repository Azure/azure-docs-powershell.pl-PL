# Get-AzureVirtualNetworkGatewayKey

## STRESZCZENIe
Pobiera klucz bramy wirtualnej sieci Azure.

## POLECENIA

```
Get-AzureVirtualNetworkGatewayKey -GatewayId <String> -ConnectedEntityId <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureVirtualNetworkGatewayKey** Pobiera klucz bramy wirtualnej sieci Azure.
