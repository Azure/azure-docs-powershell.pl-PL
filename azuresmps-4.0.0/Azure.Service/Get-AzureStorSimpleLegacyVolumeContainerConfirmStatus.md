# Get-AzureStorSimpleLegacyVolumeContainerConfirmStatus

## STRESZCZENIe
Pobiera stan operacji Commit lub Rollback.

## POLECENIA

```
Get-AzureStorSimpleLegacyVolumeContainerConfirmStatus -LegacyConfigId <String>
 [-LegacyContainerNames <String[]>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureStorSimpleLegacyVolumeContainerConfirmStatus** Pobiera stan operacji Commit lub Rollback.
