# Reset-AzureVirtualNetworkGateway

## STRESZCZENIe
Resetuje bramę sieci wirtualnej.

## POLECENIA

```
Reset-AzureVirtualNetworkGateway -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Reset-AzureVirtualNetworkGateway** resetuje bramę sieci wirtualnej.
