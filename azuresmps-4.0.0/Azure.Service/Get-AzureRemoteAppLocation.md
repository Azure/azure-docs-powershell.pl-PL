# Get-AzureRemoteAppLocation

## STRESZCZENIe
Wyświetla listę lokalizacji obsługiwanych przez usługę Azure RemoteApp.

## POLECENIA

```
Get-AzureRemoteAppLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureRemoteAppLocation** zawiera listę lokalizacji obsługiwanych przez usługę Azure RemoteApp.
