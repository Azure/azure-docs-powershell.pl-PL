# Remove-AzureVNetConfig

## STRESZCZENIe
Usuwa konfigurację sieci z bieżącej subskrypcji platformy Azure.

## POLECENIA

```
Remove-AzureVNetConfig [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureVNetConfig** usuwa wszystkie ustawienia sieci wirtualnej z bieżącej subskrypcji platformy Azure.

## Przykłady

### Przykład 1: Usuwanie konfiguracji sieci wirtualnej z bieżącej subskrypcji
```
PS C:\> Remove-AzureVNetConfig
```

To polecenie umożliwia usunięcie konfiguracji sieci wirtualnej z bieżącej subskrypcji.
