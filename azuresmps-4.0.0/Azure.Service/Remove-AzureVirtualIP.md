# Remove-AzureVirtualIP

## STRESZCZENIe
Usuwa wirtualny adres IP z usługi w chmurze.

## POLECENIA

```
Remove-AzureVirtualIP [-ServiceName] <String> [-VirtualIPName] <String> [-Force] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureVirtualIP** usuwa wirtualny adres IP (VIP) z usługi platformy Azure.
Operacja kończy się powodzeniem tylko wtedy, gdy wirtualny adres IP nie ma skojarzonych z nim punktów końcowych.

## Przykłady

### Przykład 1: usuwanie wirtualnego adresu IP z usługi
```
PS C:\> Remove-AzureVirtualIP -VirtualIPName "Vip01" -ServiceName "ContosoService03"
```

To polecenie usuwa wirtualny adres IP z usługi.
