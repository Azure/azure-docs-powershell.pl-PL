# Get-AzureAffinityGroup

## STRESZCZENIe
Pobiera obiekt grupy koligacji platformy Azure.

## POLECENIA

```
Get-AzureAffinityGroup [[-Name] <String>] [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureAffinityGroup** pobiera grupę koligacji platformy Azure.
Obiekt grupy koligacji zawiera nazwę grupy koligacji, lokalizację, etykietę, opis oraz magazyn i usługi hostowane, które są częścią grupy koligacji.

## Przykłady

### Przykład 1: pobieranie właściwości grupy koligacji
```
PS C:\> Get-AzureAffinityGroup -Name "South01"
```

To polecenie pobiera właściwości grupy koligacji o nazwie South01.
