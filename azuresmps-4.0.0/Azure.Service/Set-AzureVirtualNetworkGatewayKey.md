# Set-AzureVirtualNetworkGatewayKey

## STRESZCZENIe
Ustawia klucz bramy sieci wirtualnej platformy Azure.

## POLECENIA

```
Set-AzureVirtualNetworkGatewayKey -GatewayId <String> -ConnectedEntityId <String> -SharedKey <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzureVirtualNetworkGatewayKey** ustawia klucz bramy wirtualnej sieci Azure.
