# Get-AzureVMPuppetExtension

## STRESZCZENIe
Pobiera rozszerzenie Puppet zastosowane na maszynie wirtualnej.

## POLECENIA

```
Get-AzureVMPuppetExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureVMPuppetExtension** pobiera rozszerzenie Puppet zastosowane na maszynie wirtualnej.

## Przykłady

### Przykład 1: uzyskiwanie rozszerzenia Puppet zastosowanego na maszynie wirtualnej
```
PS C:\> Get-AzureVMPuppetExtension -VM $VM
```

To polecenie pobiera rozszerzenie Puppet zastosowane na maszynie wirtualnej.
