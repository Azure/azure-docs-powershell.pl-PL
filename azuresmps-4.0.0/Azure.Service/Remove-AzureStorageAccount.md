# Remove-AzureStorageAccount

## STRESZCZENIe
Usuwa określone konto magazynu z subskrypcji.

## POLECENIA

```
Remove-AzureStorageAccount [-StorageAccountName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureStorageAccount** usuwa konto z subskrypcji platformy Azure.

## Przykłady

### Przykład 1: usuwanie konta magazynu
```
PS C:\> Remove-AzureStorageAccount -StorageAccountName "ContosoStore01"
```

To polecenie usuwa konto magazynu ContosoStore01 z określonego abonamentu.
