# Remove-AzureVMDiagnosticsExtension

## STRESZCZENIe
Usuwa rozszerzenie Diagnostyka platformy Azure z maszyny wirtualnej.

## POLECENIA

```
Remove-AzureVMDiagnosticsExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureVMDiagnosticsExtension** usuwa rozszerzenie Diagnostyka platformy Microsoft Azure z maszyny wirtualnej.
Musisz przekazać dane wyjściowe tego polecenia cmdlet do polecenia cmdlet **Update-AzureVM** .

## Przykłady

### Przykład 1: Usuwanie rozszerzenia Diagnostyka platformy Azure z maszyny wirtualnej
```
PS C:\> Remove-AzureVMDiagnosticsExtension -VM $VM | Update-AzureVM
```

To polecenie usuwa rozszerzenie Diagnostyka platformy Azure z maszyny wirtualnej.
