# Remove-AzureReservedIP

## STRESZCZENIe
Usuwa zastrzeżony adres IP według jego nazwy.

## POLECENIA

```
Remove-AzureReservedIP [-ReservedIPName] <String> [-Force] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureReservedIP** usuwa zastrzeżony adres IP według jego nazwy.

## Przykłady

### Przykład 1: usuwanie zastrzeżonego adresu IP według jego nazwy
```
PS C:\> Remove-AzureReservedIP -ReservedIPName $name
```

To polecenie usuwa zastrzeżony adres IP według jego nazwy.
