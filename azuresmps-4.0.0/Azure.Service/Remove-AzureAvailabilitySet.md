# Remove-AzureAvailabilitySet

## STRESZCZENIe
Usuwa zestaw dostępności z maszyny wirtualnej platformy Azure.

## POLECENIA

```
Remove-AzureAvailabilitySet -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureAvailabilitySet** umożliwia usunięcie zestawu dostępności z maszyny wirtualnej platformy Azure.
