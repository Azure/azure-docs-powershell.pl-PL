# Set-AzureWalkUpgradeDomain

## STRESZCZENIe
Przeprowadzi określoną domenę uaktualnienia.

## POLECENIA

```
Set-AzureWalkUpgradeDomain [-ServiceName] <String> [-Slot] <String> [-DomainNumber] <Int32>
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzureWalkUpgradeDomain** inicjuje rzeczywistą aktualizację wdrożenia platformy Azure.
Pakiet i konfiguracja uaktualnienia są ustawiane przy użyciu polecenia cmdlet **Set-AzureDeployment** z przełącznikiem-Upgrade.

## Przykłady

### Przykład 1. inicjowanie uaktualnienia wdrożenia produkcyjnego
```
PS C:\> Set-AzureWalkUpgradeDomain -ServiceName "MySvc1" -slot "Production" -UpgradeDomain 2
```

To polecenie inicjuje uaktualnienie domeny uaktualnienia 2 wdrożenia produkcyjnego usługi MySvc1.
