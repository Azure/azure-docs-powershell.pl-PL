# Get-AzureServiceExtension

## STRESZCZENIe
Umożliwia pobieranie rozszerzeń usługi w chmurze, które zostały zastosowane we wdrożeniu.

## POLECENIA

```
Get-AzureServiceExtension [[-ServiceName] <String>] [[-Slot] <String>] [[-ExtensionName] <String>]
 [[-ProviderNamespace] <String>] [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureServiceExtension** umożliwia pobieranie istniejących rozszerzeń usługi w chmurze, które zostaną zastosowane do wdrożenia.

## Przykłady

### Przykład 1: uzyskiwanie określonego rozszerzenia
```
PS C:\> Get-AzureServiceExtension -ServiceName $Svc -Slot "Production" -ExtensionName "RDP" -ProviderNamespace "Microsoft.Windows.Azure.Extensions"
```

To polecenie uzyskuje rozszerzenie usługi w chmurze o określonej nazwie i przestrzeni nazw.
