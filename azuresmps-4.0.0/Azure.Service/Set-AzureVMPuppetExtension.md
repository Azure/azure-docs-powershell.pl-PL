# Set-AzureVMPuppetExtension

## STRESZCZENIe
Ustawia rozszerzenie Puppet dla maszyny wirtualnej.

## POLECENIA

```
Set-AzureVMPuppetExtension [-PuppetMasterServer] <String> [[-Version] <String>] [-Disable]
 [[-ReferenceName] <String>] -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzureVMPuppetExtension** ustawia rozszerzenie Puppet maszyny wirtualnej.

## Przykłady

### Przykład 1: Ustawianie rozszerzenia Puppet dla maszyny wirtualnej
```
PS C:\> Set-AzureVMPuppetExtension -VM $VM
```

W tym przykładzie ustawiono rozszerzenie Puppet dla określonej maszyny wirtualnej jako przechowywane w zmiennej $VM.
