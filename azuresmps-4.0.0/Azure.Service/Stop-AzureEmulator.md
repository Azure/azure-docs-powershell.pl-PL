# Stop-AzureEmulator

## STRESZCZENIe
Zatrzymuje emulator obliczeniowy.

## POLECENIA

```
Stop-AzureEmulator [-PassThru] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Opis
W tym temacie opisano polecenie cmdlet w wersji 0.8.10 modułu PowerShell platformy Microsoft Azure.
Aby uzyskać wersję używanego modułu, należy wpisać w konsoli programu Azure PowerShell `(Get-Module -Name Azure).Version` .

Polecenie cmdlet **stop-AzureEmulator** zatrzymuje emulatora obliczeniowego platformy Azure.
Wszystkie usługi obecnie uruchomione w emulatorze zostaną usunięte.
