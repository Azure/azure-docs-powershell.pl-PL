# Get-AzureVMAccessExtension

## STRESZCZENIe
Pobiera rozszerzenie VMAccess zastosowane na maszynie wirtualnej.

## POLECENIA

```
Get-AzureVMAccessExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureVMAccessExtension** pobiera rozszerzenie VMAccess zastosowane na maszynie wirtualnej.

## Przykłady

### Przykład 1: uzyskiwanie rozszerzenia VMAccess dla maszyny wirtualnej
```
PS C:\> Get-AzureVMAccessExtension -VM $VM;
```

To polecenie uzyskuje rozszerzenie VMAccess dla maszyny wirtualnej.
