# Remove-AzureMediaServicesAccount

## STRESZCZENIe
Usuwa określone konto usługi Azure Media Services.

**Uwaga:**  Ta wersja jest przestarzała, sprawdź [nowszą wersję](https://docs.microsoft.com/powershell/module/azurerm.media/?view=azurermps-5.4.0#media_services).

## POLECENIA

```
Remove-AzureMediaServicesAccount -Name <String> [-Force] [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
W tym temacie opisano polecenie cmdlet w wersji 0.8.10 modułu PowerShell platformy Microsoft Azure.
Aby uzyskać wersję używanego modułu, należy wpisać w konsoli programu Azure PowerShell `(Get-Module -Name Azure).Version` .

Polecenie cmdlet **Remove-AzureMediaServicesAccount** usuwa konto usług multimedialnych.

## Przykłady

### Przykład 1: usuwanie konta usług multimedialnych
```
PS C:\> Remove-AzureMediaServicesAccount -Name "mediaservicesaccount" -Force
```
