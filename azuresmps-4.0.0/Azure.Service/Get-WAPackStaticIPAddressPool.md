# Get-WAPackStaticIPAddressPool

## STRESZCZENIe
Pobiera statyczne obiekty puli adresów IP.

## POLECENIA

### FromVMSubnetObject (domyślny)
```
Get-WAPackStaticIPAddressPool -VMSubnet <VMSubnet> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

### Odname
```
Get-WAPackStaticIPAddressPool -VMSubnet <VMSubnet> -Name <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Opis
Te tematy są przestarzałe i zostaną usunięte w przyszłości.
W tym temacie opisano polecenie cmdlet w wersji 0.8.1 modułu PowerShell platformy Microsoft Azure.
Aby sprawdzić używaną wersję modułu, należy wpisać ją w konsoli programu Azure PowerShell `(Get-Module -Name Azure).Version` .

Polecenie cmdlet **Get-WAPackStaticIPAddressPool** pobiera statyczne obiekty puli adresów IP.

## Przykłady

### Przykład 1: uzyskiwanie puli statycznych adresów IP z danej podsieci VMSubnet
```
PS C:\> $Subnet = Get-WAPackVMSubet -Name "ContosoVMSubnet01"
PS C:\> Get-WAPackStaticIPAddressPool -VMSubnet $Subnet -Name "ContosoStaticIPAddressPool01"
```

To polecenie pobiera statyczną pulę adresów IP o nazwie ContosoStaticIPAddressPool01 z określonej podsieci VMSubnet.

### Przykład 2: pobieranie wszystkich pul statycznych adresów IP z danej podsieci VMSubnet
```
PS C:\> $Subnet = Get-WAPackVMSubet -Name "ContosoVMSubnet01"
PS C:\> Get-WAPackStaticIPAddressPool -VMSubnet $Subnet
```

To polecenie pobiera wszystkie pule statyczne IP z określonego VMSubet.
