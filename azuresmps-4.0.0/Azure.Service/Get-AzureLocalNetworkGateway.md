# Get-AzureLocalNetworkGateway

## STRESZCZENIe
Pobiera lokalną bramę sieciową.

## POLECENIA

```
Get-AzureLocalNetworkGateway [-GatewayId <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureLocalNetworkGateway** pobiera bramę sieci lokalnej.
