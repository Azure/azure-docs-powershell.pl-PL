# Set-AzureStaticVNetIP

## STRESZCZENIe
Ustawia statyczne informacje o adresie IP VNet dla obiektu maszyny wirtualnej.

## POLECENIA

```
Set-AzureStaticVNetIP [-IPAddress] <String> -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzureStaticVNetIP** ustawia informacje o adresie IP statycznej sieci wirtualnej (wirtualnej) dla obiektu maszyny wirtualnej.

## Przykłady

### Przykład 1: Ustawianie adresu IP sieci wirtualnej skojarzonego z maszyną wirtualną
```
PS C:\> # Prerequisite: VNet has been set up with SubNet
          # Set-AzureVNetConfig -ConfigurationPath $vnetConfigPath;

          $vm = New-AzureVMConfig -Name $vmname -ImageName $img -InstanceSize $sz | Add-AzureProvisioningConfig -Windows -Password $pwd -AdminUsername $usr;
          $vm = Set-AzureSubNet -VM $vm -SubNetNames $sn;
          Set-AzureStaticVNetIP -IPAddress $ip -VM $vm;
          New-AzureVM -ServiceName $svc -VMs $vm -VNetName $vnetName -Location $loc;
```

Pierwsze polecenie ustawia ścieżkę konfiguracji sieci wirtualnej.

Drugie polecenie tworzy konfigurację maszyny wirtualnej.

Trzecie polecenie ustawia podsieć dla maszyny wirtualnej.

Czwarte polecenie ustawia adres IP maszyny wirtualnej.

Piąte polecenie tworzy maszynę wirtualną przy użyciu maszyny wirtualnej.
