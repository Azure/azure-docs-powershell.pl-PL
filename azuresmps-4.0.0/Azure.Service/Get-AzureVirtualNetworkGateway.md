# Get-AzureVirtualNetworkGateway

## STRESZCZENIe
Pobiera wirtualną bramę sieciową.

## POLECENIA

```
Get-AzureVirtualNetworkGateway [-GatewayId <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureVirtualNetworkGateway** pobiera bramę sieci wirtualnej platformy Azure.
