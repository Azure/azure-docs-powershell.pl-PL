# Remove-WAPackStaticIPAddressPool

## STRESZCZENIe
Usuwa obiekty statycznej puli adresów IP.

## POLECENIA

```
Remove-WAPackStaticIPAddressPool -StaticIPAddressPool <StaticIPAddressPool> [-PassThru] [-Force]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Opis
Te tematy są przestarzałe i zostaną usunięte w przyszłości.
W tym temacie opisano polecenie cmdlet w wersji 0.8.1 modułu PowerShell platformy Microsoft Azure.
Aby sprawdzić używaną wersję modułu, należy wpisać ją w konsoli programu Azure PowerShell `(Get-Module -Name Azure).Version` .

Polecenie cmdlet **Remove-WAPackStaticIPAddressPool** usuwa obiekty statyczne puli adresów IP.

## Przykłady

### Przykład 1: Usuwanie puli statycznych adresów IP
```
PS C:\> $StaticIPAddressPool = Get-WAPackStaticIPAddressPool -Name "ContosoStaticIPAddressPool01"
PS C:\> Remove-WAPackStaticIPAddressPool -StaticIPAddressPool $StaticIPAddressPool
```

Pierwsze polecenie pobiera statyczną pulę adresów IP o nazwie ContosoStaticIPAddressPool01 przy użyciu polecenia cmdlet **Get-WAPackStaticIPAddressPool** , a następnie zapisuje ten obiekt w zmiennej $StaticIPAddressPool.

Drugie polecenie usuwa statyczną pulę adresów IP przechowywaną w $StaticIPAddressPool.
Polecenie wyświetli monit o potwierdzenie.

### Przykład 2: usunięcie puli statycznych adresów IP bez potwierdzenia
```
PS C:\> $StaticIPAddressPool = Get-WAPackStaticIPAddressPool -Name "ContosoStaticIPAddressPool02"
PS C:\> Remove-WAPackStaticIPAddressPool -StaticIPAddressPool $StaticIPAddressPool -Force
```

Pierwsze polecenie pobiera statyczną pulę adresów IP o nazwie ContosoStaticIPAddressPool02 przy użyciu polecenia cmdlet **Get-WAPackStaticIPAddressPool** , a następnie zapisuje ten obiekt w zmiennej $ StaticIPAddressPool.

Drugie polecenie usuwa statyczną pulę adresów IP przechowywaną w $StaticIPAddressPool.
To polecenie zawiera parametr *Force* .
Polecenie nie wyświetla monitu o potwierdzenie.
