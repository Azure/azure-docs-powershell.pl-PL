# Remove-AzureVMDscExtension

## STRESZCZENIe
Usuwa rozszerzenie Azure DSC z maszyny wirtualnej.

## POLECENIA

```
Remove-AzureVMDscExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureVMDscExtension** usuwa rozszerzenie Azure DSC z maszyny wirtualnej.
Wyniki tego polecenia cmdlet muszą być przekazywane do polecenia cmdlet **Update-AzureVM** .

## Przykłady

### Przykład 1: Usuwanie rozszerzenia DSC z maszyny wirtualnej
```
PS C:\> Remove-AzureVMDscExtension -VM $VM | Update-AzureVM
```

To polecenie usuwa rozszerzenie Azure DSC z maszyny wirtualnej.
