# Save-AzureWebsiteLog

## STRESZCZENIe
Pobiera i zapisuje dzienniki określonej witryny sieci Web.

## POLECENIA

```
Save-AzureWebsiteLog [-Output <String>] [-PassThru] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Opis
W tym temacie opisano polecenie cmdlet w wersji 0.8.10 modułu PowerShell platformy Microsoft Azure.
Aby uzyskać wersję używanego modułu, należy wpisać w konsoli programu Azure PowerShell `(Get-Module -Name Azure).Version` .

Polecenie cmdlet **Save-AzureWebsiteLog** pobiera dzienniki określonej witryny sieci Web.

## Przykłady

### Przykład 1: pobieranie i zapisywanie dzienników witryny sieci Web
```
PS C:\> Save-AzureWebsiteLogs -Name mySite -Output .\logs.zip
```

W tym przykładzie dzienniki środowiska uruchomieniowego i wdrażania witryny Moja witryna witryny internetowej są pobierane do pliku logs.zip w bieżącym katalogu.
