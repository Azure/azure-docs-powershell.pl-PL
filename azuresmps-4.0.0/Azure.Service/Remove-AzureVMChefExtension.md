# Remove-AzureVMChefExtension

## STRESZCZENIe
Usuwa rozszerzenie Kuchmistrz zastosowane na maszynie wirtualnej.

## POLECENIA

```
Remove-AzureVMChefExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureVMChefExtension** usuwa rozszerzenie Kuchmistrz zastosowane na maszynie wirtualnej.

## Przykłady

### Przykład 1: Usuwanie rozszerzenia Kuchmistrz zastosowanego na określonej maszynie wirtualnej
```
PS C:\> Remove-AzureVMChefExtension -VM $VM;
```

To polecenie usuwa rozszerzenie Kuchmistrz zastosowane na maszynie wirtualnej.
