# Restore-AzureWebsiteDeployment

## STRESZCZENIe
Ponowne wdrażanie wcześniejszego wdrożenia witryny sieci Web na platformie Azure.

## POLECENIA

```
Restore-AzureWebsiteDeployment [-CommitId <String>] [-Force] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
W tym temacie opisano polecenie cmdlet w wersji 0.8.10 modułu PowerShell platformy Microsoft Azure.
Aby uzyskać wersję używanego modułu, należy wpisać w konsoli programu Azure PowerShell `(Get-Module -Name Azure).Version` .

Polecenie cmdlet **Restore-AzureWebsiteDeployment** powoduje ponowne wdrożenie wcześniejszego wdrożenia witryny sieci Web na platformie Azure.
Ten proces zastępuje bieżące wdrożenie w ramach wybranego wdrożenia.

## Przykłady

### Przykład 1: ponowne wdrażanie witryny
```
PS C:\> Restore-AzureWebsiteDeployment -Name "ContosoSite" -CommitId "f876543210"
```

To polecenie powoduje ponowne wdrożenie wdrożenia o IDENTYFIKATORze f876543210 dla witryny internetowej o nazwie ContosoSite.
