# Show-AzurePortal

## STRESZCZENIe
Wyświetlanie portalu zarządzania platformy Azure.

## POLECENIA

```
Show-AzurePortal [-Name <String>] [-Realm <String>] [-Environment <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Opis
W tym temacie opisano polecenie cmdlet w wersji 0.8.10 modułu PowerShell platformy Microsoft Azure.
Aby uzyskać wersję używanego modułu, należy wpisać w konsoli programu Azure PowerShell `(Get-Module -Name Azure).Version` .

Polecenie cmdlet **show-AzurePortal** umożliwia wyświetlenie portalu zarządzania platformy Azure.

## Przykłady

### Przykład 1: wyświetlanie informacji o witrynie sieci Web
```
PS C:\> Show-AzurePortal -Name mySite
```

W tym przykładzie zostanie otwarta przeglądarka w portalu Azure z wyświetlonymi informacjami o witrynie sieci Web o nazwie Moja witryna.
