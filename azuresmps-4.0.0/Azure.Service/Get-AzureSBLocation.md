# Get-AzureSBLocation

## STRESZCZENIe
Pobiera lokalizację magistrali usługowej.

## POLECENIA

```
Get-AzureSBLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Opis
W tym temacie opisano polecenie cmdlet w wersji 0.8.10 modułu PowerShell platformy Microsoft Azure.
Aby uzyskać wersję używanego modułu, należy wpisać w konsoli programu Azure PowerShell `(Get-Module -Name Azure).Version` .

Polecenie cmdlet **Get-AzureSBLocation** zwraca lokalizacje, z których usługa Service Bus jest dostępna.

[!INCLUDE [sb-deprecation.md](../include/sb-deprecation.md)]

## Przykłady

### Przykład 1: Uzyskiwanie lokalizacji usługi Service Bus
```
PS C:\> Get-AzureSBLocation
```

W tym przykładzie jest pobierana lokalizacja magistrali usług.
