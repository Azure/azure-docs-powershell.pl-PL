# Get-AzureWebsiteLocation

## STRESZCZENIe
Pobiera wszystkie dostępne lokalizacje witryn internetowych.

## POLECENIA

```
Get-AzureWebsiteLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Opis
W tym temacie opisano polecenie cmdlet w wersji 0.8.10 modułu PowerShell platformy Microsoft Azure.
Aby uzyskać wersję używanego modułu, należy wpisać w konsoli programu Azure PowerShell `(Get-Module -Name Azure).Version` .

Pobiera wszystkie dostępne lokalizacje witryn internetowych dla bieżącej subskrypcji.

## Przykłady

### Przykład 1: pobieranie wszystkich dostępnych lokalizacji
```
PS C:\> Get-AzureWebsiteLocations
```

W tym przykładzie są pobierane wszystkie dostępne lokalizacje witryn internetowych dla bieżącej subskrypcji.
