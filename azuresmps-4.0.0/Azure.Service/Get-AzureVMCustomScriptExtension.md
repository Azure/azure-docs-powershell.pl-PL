# Get-AzureVMCustomScriptExtension

## STRESZCZENIe
Pobiera informacje z niestandardowego rozszerzenia skryptu maszyny wirtualnej platformy Azure.

## POLECENIA

```
Get-AzureVMCustomScriptExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureVMCustomScriptExtension** pobiera informacje z niestandardowego rozszerzenia skryptu maszyny wirtualnej platformy Azure.

## Przykłady

### Przykład 1: pobieranie rozszerzenia skryptu maszyny wirtualnej platformy Azure
```
PS C:\> Get-AzureVMCustomScriptExtension -VM $VM;
```

To polecenie pobiera rozszerzenie skryptu maszyny wirtualnej platformy Azure przechowywane w zmiennej $VM.
