# Remove-AzureVMSqlServerExtension

## STRESZCZENIe
Usuwa rozszerzenie serwera SQL maszyny wirtualnej platformy Azure z obiektu maszyny wirtualnej.

## POLECENIA

```
Remove-AzureVMSqlServerExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureVMSqlServerExtension** usuwa rozszerzenie serwera SQL maszyny wirtualnej platformy Azure z obiektu maszyny wirtualnej.
