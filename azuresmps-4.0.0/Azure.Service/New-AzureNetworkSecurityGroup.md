# New-AzureNetworkSecurityGroup

## STRESZCZENIe
Tworzy grupę zabezpieczeń sieci Azure.

## POLECENIA

```
New-AzureNetworkSecurityGroup -Name <String> -Location <String> [-Label <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **New-AzureNetworkSecurityGroup** tworzy grupę zabezpieczeń sieci Azure w lokalizacji.
