# Remove-AzureStaticVNetIP

## STRESZCZENIe
Usuwa informacje o statycznym adresie IP sieci wirtualnej z obiektu maszyny wirtualnej (jeśli istnieje).

## POLECENIA

```
Remove-AzureStaticVNetIP -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureStaticVNetIP** usuwa informacje o adresie IP statycznej sieci wirtualnej (wirtualnej) z obiektu maszyny wirtualnej (jeśli istnieje).
