# Remove-AzureVMCustomScriptExtension

## STRESZCZENIe
Usuwa niestandardowe rozszerzenie skryptu z maszyny wirtualnej.

## POLECENIA

```
Remove-AzureVMCustomScriptExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureVMCustomScriptExtension** usuwa niestandardowe rozszerzenie skryptu z maszyny wirtualnej.

## Przykłady

### Przykład 1: Usuwanie niestandardowego rozszerzenia skryptu maszyny wirtualnej
```
PS C:\> Remove-AzureVMCustomScriptExtension -VM $VM;
```

To polecenie powoduje usunięcie niestandardowego rozszerzenia skryptu maszyny wirtualnej platformy Azure z określonej maszyny wirtualnej zapisanej w zmiennej $VM.
