# Get-WAPackCloudService

## STRESZCZENIe
Pobiera obiekty usługi w chmurze.

## POLECENIA

### Empty (domyślnie)
```
Get-WAPackCloudService [-Profile <AzureSMProfile>] [<CommonParameters>]
```

### Odname
```
Get-WAPackCloudService [-Name <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Opis
Te tematy są przestarzałe i zostaną usunięte w przyszłości.
W tym temacie opisano polecenie cmdlet w wersji 0.8.1 modułu PowerShell platformy Microsoft Azure.
Aby sprawdzić używaną wersję modułu, należy wpisać ją w konsoli programu Azure PowerShell `(Get-Module -Name Azure).Version` .

Polecenie cmdlet **Get-WAPackCloudService** Pobiera obiekty usługi w chmurze.
