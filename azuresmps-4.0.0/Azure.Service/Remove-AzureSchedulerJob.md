# Remove-AzureSchedulerJob

## STRESZCZENIe
Usuwa zadanie harmonogramu.

## POLECENIA

```
Remove-AzureSchedulerJob [-Force] [-Location <String>] -JobCollectionName <String> -JobName <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Opis
W tym temacie opisano polecenie cmdlet w wersji 0.8.10 modułu PowerShell platformy Microsoft Azure.
Aby uzyskać wersję używanego modułu, należy wpisać w konsoli programu Azure PowerShell `(Get-Module -Name Azure).Version` .

Polecenie cmdlet **Remove-AzureSchedulerJob** usuwa zadanie harmonogramu.

## Przykłady

### Przykład 1: usuwanie zadania harmonogramu
```
PS C:\> Remove-AzureSchedulerJob -Location "North Central US" -JobCollectionName "JobCollection01" -JobName "Job17"
```

To polecenie usuwa zadanie o nazwie Job17.
To zadanie jest częścią kolekcji zadań o nazwie JobCollection01 i znajduje się w określonej lokalizacji.
