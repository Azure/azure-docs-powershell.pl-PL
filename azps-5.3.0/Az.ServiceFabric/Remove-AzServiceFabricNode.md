# Remove-AzServiceFabricNode

## STRESZCZENIe
Usuwanie węzłów z określonego typu węzła z poziomu klastra.

## POLECENIA

```
Remove-AzServiceFabricNode -NumberOfNodesToRemove <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Za pomocą narzędzia **Remove-AzServiceFabricNode** można usuwać węzły z określonego typu węzła z poziomu klastra. Usuwanie jest kontynuowane tylko wtedy, gdy spełnia on metryki kondycji klastra.

## Przykłady

### Przykład 1
```powershell
PS c:> Remove-AzServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeType 'nt1' -NumberOfNodesToRemove 2
```

To polecenie usunie 2 węzły z NodeType ' NT1 '.
