# Update-AzRecoveryServicesVault

## STRESZCZENIe
Umożliwia zaktualizowanie Instalatora MSI do magazynu usług odzyskiwania.

## POLECENIA

```
Update-AzRecoveryServicesVault -ResourceGroupName <string> -Name <string> -IdentityType <MSIdentity>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
To polecenie cmdlet umożliwia dodanie lub usunięcie MSI z magazynu usług odzyskiwania. Użyj parametru-IdentityType param, aby przypisać tożsamość SystemAssigned do RSVault. Nie usuwaj MSI z magazynu przy użyciu opcji IdentityType.

## Przykłady

### Przykład 1
```powershell
PS C:\> Update-AzRecoveryServicesVault -ResourceGroupName "rgName" -Name "vaultName" -IdentityType SystemAssigned
```

To polecenie cmdlet umożliwia dodanie tożsamości SystemAssigned do magazynu usług odzyskiwania.
