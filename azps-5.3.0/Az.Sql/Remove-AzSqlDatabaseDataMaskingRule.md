# Remove-AzSqlDatabaseDataMaskingRule

## STRESZCZENIe
Usuwa regułę maskowania danych z bazy danych.

## POLECENIA

```
Remove-AzSqlDatabaseDataMaskingRule [-PassThru] [-Force] -SchemaName <String> -TableName <String>
 -ColumnName <String> [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzSqlDatabaseDataMaskingRule** usuwa określoną regułę maskowania danych z bazy danych SQL Azure.
Regułę maskowania danych można usunąć, korzystając z parametrów *ResourceGroupName*, *nazwa_serwera*, *DatabaseName* i *RuleID* w celu zidentyfikowania reguły, którą to polecenie cmdlet usuwa.
To polecenie cmdlet jest również obsługiwane przez usługę bazy danych SQL Server undatabase na platformie Azure.

## Przykłady

### Przykład 1: Usuwanie reguły maskowania danych bazy danych
```
PS C:\>Remove-AzSqlDatabaseDataMaskingRule -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01" -SchemaName "dbo" -TableName  "table1" -ColumnName "column1"
```

To polecenie usuwa nazwę reguły Rule01 zdefiniowaną dla Database01 bazy danych.
Baza danych znajduje się w witrynie Server01 i jest przypisana do ResourceGroup01 grupy zasobów.
