# Invoke-AzSqlElasticPoolFailover

## STRESZCZENIe
Praca awaryjna z elastyczną pulą.

## POLECENIA

```
Invoke-AzSqlElasticPoolFailover [-ElasticPoolName] <String> [-AsJob] [-PassThru] [-Force]
 [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Invoke-AzSqlElasticPoolFailover tryb pracy awaryjnej puli elastycznej. Po uruchomieniu tego polecenia cmdlet nastąpi przejście do trybu failover we wszystkich bazach danych w puli elastycznej.

## Przykłady

### Przykład 1
```powershell
PS C:\> Invoke-AzSqlElasticPoolFailover -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

To polecenie przejdzie w tryb pracy awaryjnej puli elastycznej o nazwie "ElasticPool01" na serwerze o nazwie "Server01".  Oznacza to, że przejęcie awaryjne będzie wykonywane we wszystkich bazach danych w puli elastycznej o nazwie "ElasticPool01".
