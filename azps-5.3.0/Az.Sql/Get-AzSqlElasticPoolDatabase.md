# Get-AzSqlElasticPoolDatabase

## STRESZCZENIe
Pobiera elastyczną bazę danych w puli elastycznej i jej wartości właściwości.

## POLECENIA

```
Get-AzSqlElasticPoolDatabase [-ElasticPoolName] <String> [-DatabaseName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzSqlElasticPoolDatabase** pobiera elastyczną bazę danych w puli elastycznej oraz wartości ich właściwości.
W usłudze Azure SQL Database można określić nazwę Elastic Database, aby wyświetlić wartości właściwości tylko tej bazy danych.

## Przykłady

### Przykład 1: pobieranie wszystkich baz danych w elastycznej puli
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

To polecenie pobiera wszystkie bazy danych z puli elastycznej o nazwie ElasticPool01.

### Przykład 2: pobieranie wszystkich baz danych w puli elastycznej przy użyciu filtrowania
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01" -DatabaseName "Database*"
```

To polecenie pobiera wszystkie bazy danych z puli elastycznej o nazwie ElasticPool01, która rozpoczyna się od "bazy danych".
