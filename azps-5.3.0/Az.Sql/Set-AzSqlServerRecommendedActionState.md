# Set-AzSqlServerRecommendedActionState

## STRESZCZENIe
Aktualizuje stan zalecanej akcji programu Azure SQL Server.

## POLECENIA

```
Set-AzSqlServerRecommendedActionState -RecommendedActionName <String> -State <RecommendedActionState>
 -ServerName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Stan aktualizacji polecenia cmdlet **Set-AzSqlServerRecommendedActionState** dla zalecanej akcji programu Azure SQL Server.
To polecenie cmdlet dotyczy, przywraca lub odrzuca zalecaną akcję na podstawie nowego stanu.
