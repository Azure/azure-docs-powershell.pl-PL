# Remove-AzSqlServerFirewallRule

## STRESZCZENIe
Usuwa regułę zapory z serwera bazy danych SQL.

## POLECENIA

```
Remove-AzSqlServerFirewallRule [-FirewallRuleName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzSqlServerFirewallRule** usuwa regułę zapory z określonego serwera bazy danych SQL Azure.

## Przykłady

### Przykład 1: Usuwanie reguły
```
PS C:\>Remove-AzSqlServerFirewallRule -FirewallRuleName "Rule01" -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

To polecenie powoduje usunięcie reguły zapory o nazwie Rule01 na serwerze o nazwie Server01.
