# Remove-AzKeyVaultManagedHsm

## STRESZCZENIe
Usuwa zarządzany element HSM.

## POLECENIA

### RemoveManagedHsmByName (domyślny)
```
Remove-AzKeyVaultManagedHsm [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveManagedHsmByInputObject
```
Remove-AzKeyVaultManagedHsm [-InputObject] <PSManagedHsm> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveManagedHsmByResourceId
```
Remove-AzKeyVaultManagedHsm [-ResourceId] <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzKeyVaultManagedHsm** umożliwia usunięcie określonego zarządzanego modułu HSM.
Usuwa również wszystkie klucze zawarte w tym wystąpieniu.
Pamiętaj, że chociaż określenie grupy zasobów jest opcjonalne dla tego polecenia cmdlet, należy zapewnić lepszą wydajność.

## Przykłady

### Przykład 1: usuwanie zarządzanego modułu HSM
```powershell
PS C:\> Remove-AzKeyVaultManagedHsm -HsmName 'myhsm' -Force

True
```

To polecenie powoduje usunięcie zarządzanego modułu HSM o nazwie myhsm z bieżącej subskrypcji.

### Przykład 2: usuwanie zarządzanego modułu HSM z określonej grupy zasobów
```powershell
PS C:\> Remove-AzKeyVaultManagedHsm -HsmName 'myhsm' -ResourceGroupName "myrg1" -PassThru

True
```

To polecenie powoduje usunięcie zarządzanego modułu HSM o nazwie myhsm z grupy zasobów o nazwie myrg1.
Jeśli nazwa grupy zasobów nie zostanie określona, polecenie cmdlet wyszukuje nazwa zarządzanego modułu HSM, który ma zostać usunięty z bieżącego abonamentu.
