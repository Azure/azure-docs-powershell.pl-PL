# New-AzCostManagementQueryComparisonExpressionObject

## STRESZCZENIe
Tworzenie obiektu w pamięci dla QueryComparisonExpression

## POLECENIA

```
New-AzCostManagementQueryComparisonExpressionObject -Name <String> -Operator <OperatorType> -Value <String[]>
 [<CommonParameters>]
```

## Opis
Tworzenie obiektu w pamięci dla QueryComparisonExpression

## Przykłady

### Przykład 1. Tworzenie obiektu wyrażenia porównawczego zapytania na potrzeby eksportu zarządzania kosztami
```powershell
PS C:\> New-AzCostManagementQueryComparisonExpressionObject -Name 'ResourceLocation' -Operator In -Value @('East US', 'West Europe')

Name             Operator Value
----             -------- -----
ResourceLocation In       {East US, West Europe}
```

To polecenie tworzy obiekt wyrażenia porównania zapytania na potrzeby eksportu zarządzania kosztami.
