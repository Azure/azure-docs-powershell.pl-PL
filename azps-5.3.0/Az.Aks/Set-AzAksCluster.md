# Set-AzAksCluster

## STRESZCZENIe
Zaktualizuj lub Utwórz zarządzany klaster Kubernetes.

## Opis
Zaktualizuj lub Utwórz zarządzany klaster Kubernetes.

## Przykłady

### Przykład 1
```powershell
PS C:\> Get-AzAks -ResourceGroupName group -Name myCluster | Set-AzAks -NodeCount 5
```

Ustaw liczbę węzłów w klastrze Kubernetes na 5.
