# AZ. Moduł komunikacji
## Opis
Microsoft Azure PowerShell: polecenia cmdlet komunikacji

## AZ. komunikaty cmdlet dotyczące komunikacji
### [Get-AzCommunicationService](Get-AzCommunicationService.md)
Pobierz CommunicationService i jej właściwości.

### [Get-AzCommunicationServiceKey](Get-AzCommunicationServiceKey.md)
Uzyskiwanie klawiszy dostępu do zasobu CommunicationService.

### [Nowe — AzCommunicationService](New-AzCommunicationService.md)
Utwórz nowe CommunicationService lub zaktualizuj istniejące CommunicationService.

### [Nowe — AzCommunicationServiceKey](New-AzCommunicationServiceKey.md)
Ponowne generowanie klucza dostępu CommunicationService.
PrimaryKey i SecondaryKey nie mogą być ponownie generowane w tym samym czasie.

### [Remove-AzCommunicationService](Remove-AzCommunicationService.md)
Operacja usuwania CommunicationService.

### [Set-AzCommunicationServiceNotificationHub](Set-AzCommunicationServiceNotificationHub.md)
Łączy centrum powiadomień platformy Azure z tą usługą komunikacyjną.

### [Update-AzCommunicationService](Update-AzCommunicationService.md)
Operacja aktualizowania istniejącego CommunicationService.
