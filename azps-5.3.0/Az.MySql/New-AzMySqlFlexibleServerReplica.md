# New-AzMySqlFlexibleServerReplica

## STRESZCZENIe
Tworzy serwer repliki dla serwera elastycznego MySQL

## POLECENIA

```
New-AzMySqlFlexibleServerReplica -Replica <String> -ResourceGroupName <String> -Master <IServerAutoGenerated>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## Opis
Tworzy nową replikę z istniejącej bazy danych.

## Przykłady

### Przykład 1. Tworzenie nowej repliki serwera MySql
```powershell
PS C:\> Get-AzMySqlFlexibleServer -ResourceGroupName PowershellMySqlTest -ServerName mysql-test | New-AzMySqlFlexibleServerReplica -Replica mysql-test-replica -ResourceGroupName PowershellMySqlTest

Name               Location AdministratorLogin Version StorageProfileStorageMb SkuName       SkuTier        
----               -------- ------------------ ------- ----------------------- -------       -------        
mysql-test-replica westus2   mysql_test         5.7     10240                   Standard_B1ms Burstable
```

To polecenie cmdlet powoduje utworzenie nowej repliki serwera MySql.

### Przykład 2: Tworzenie nowej repliki serwera MySql
```powershell
PS C:\> $mysql = Get-AzMySqlFlexibleServer -ResourceGroupName PowershellMySqlTest -ServerName mysql-test
PS C:\> New-AzMySqlFlexibleServerReplica -Master $mysql -Replica mysql-test-replica -ResourceGroupName PowershellMySqlTest

Name               Location AdministratorLogin Version StorageProfileStorageMb SkuName       SkuTier        
----               -------- ------------------ ------- ----------------------- -------       -------        
mysql-test-replica westus2   mysql_test         5.7     10240                   Standard_B1ms Burstable
```

To polecenie cmdlet z parametrem Master (inputobject) powoduje utworzenie nowej repliki serwera MySql.

ZŁOŻONE WŁAŚCIWOŚCI PARAMETRÓW

Aby utworzyć opisane poniżej parametry, Skonstruuj tablicę skrótów zawierającą odpowiednie właściwości. Aby uzyskać informacje na temat tabel skrótów, uruchom Get-Help about_Hash_Tables.


WZORZEC <IServerAutoGenerated> : obiekt serwera źródłowego, z którego ma zostać utworzona replika.
  - `Location <String>`: Lokalizacja geograficzna, w której mieszka zasób
  - `SkuName <String>`: Nazwa jednostki SKU, na przykład Standard_D32s_v3.
  - `SkuTier <SkuTier>`: Warstwa konkretnej jednostki SKU, na przykład GeneralPurpose.
  - `[Tag <ITrackedResourceTags>]`: Znaczniki zasobów.
    - `[(Any) <String>]`: Wskazuje, że do tego obiektu można dodać dowolną właściwość.
  - `[AdministratorLogin <String>]`: Nazwa logowania administratora serwera. Można określić tylko podczas tworzenia serwera (i jest on wymagany do utworzenia).
  - `[AdministratorLoginPassword <String>]`: Hasło logowania administratora (wymagane do utworzenia serwera).
  - `[AvailabilityZone <String>]`: informacje o strefie dostępności serwera.
  - `[CreateMode <CreateMode?>]`: Tryb tworzenia nowego serwera MySQL.
  - `[DelegatedSubnetArgumentSubnetArmResourceId <String>]`: Identyfikator zasobu poręczenia podsieci delegowanego.
  - `[HaEnabled <HaEnabledEnum?>]`: Włączanie lub niedostępności serwera.
  - `[IdentityType <ResourceIdentityType?>]`: Typ tożsamości.
  - `[InfrastructureEncryption <InfrastructureEncryptionEnum?>]`: Stan wskazujący, czy na serwerze włączono szyfrowanie infrastruktury.
  - `[MaintenanceWindowCustomWindow <String>]`: wskazuje, czy okno niestandardowe jest włączone, czy wyłączone.
  - `[MaintenanceWindowDayOfWeek <Int32?>]`: dzień tygodnia w oknie obsługi
  - `[MaintenanceWindowStartHour <Int32?>]`: godzina rozpoczęcia w oknie konserwacji
  - `[MaintenanceWindowStartMinute <Int32?>]`: rozpoczęcie minuty w oknie konserwacji
  - `[PropertiesTag <IServerPropertiesTags>]`: Metadane specyficzne dla aplikacji w postaci par klucz-wartość.
    - `[(Any) <String>]`: Wskazuje, że do tego obiektu można dodać dowolną właściwość.
  - `[ReplicationRole <String>]`: Rola replikacji.
  - `[RestorePointInTime <DateTime?>]`: Czas tworzenia punktu przywracania (format ISO8601), określający czas przywracania.
  - `[SourceServerId <String>]`: Identyfikator źródłowego serwera MySQL.
  - `[SslEnforcement <SslEnforcementEnum?>]`: Włączanie wymuszania protokołu SSL lub nie podczas nawiązywania połączenia z serwerem.
  - `[StorageProfileBackupRetentionDay <Int32?>]`: Dni przechowywania kopii zapasowej serwera.
  - `[StorageProfileStorageAutogrow <StorageAutogrow?>]`: Włączanie automatycznego zwiększania przestrzeni dyskowej.
  - `[StorageProfileStorageIop <Int32?>]`: IOPS magazynu dla serwera.
  - `[StorageProfileStorageMb <Int32?>]`: Maksymalna ilość miejsca do magazynowania dozwolona dla serwera.
  - `[Version <ServerVersion?>]`: Wersja serwera.
