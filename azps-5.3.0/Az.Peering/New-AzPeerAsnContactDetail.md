# New-AzPeerAsnContactDetail

## STRESZCZENIe
Tworzy szczegółowe dane kontaktowe w pamięci dla PeerAsn. 

## POLECENIA

```
New-AzPeerAsnContactDetail -Role <String> -Email <String> [-Phone <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Utwórz w pamięci PeerAsn szczegóły kontaktu.

## Przykłady

### Tworzenie szczegółów kontaktu i dodawanie do PeerAsn
```powershell
PS C:\> $nocContact = New-AzPeerAsnContactDetail -Role Noc -Email "noc@contoso.com" -Phone "+1 (887) 888-8088"
PS C:\> $customerContact = New-AzPeerAsnContactDetail -Role Noc -Email "noc@contoso.com" -Phone "+1 (887) 888-8088"
PS C:\> New-AzPeerAsn -Name $name -PeerName "Contoso Networks Limited" -PeerAsn 65000 -ContactDetail $nocContact,$customerContact
```

Rola i adres e-mail są wymagane. Telefon jest opcjonalny. Telefon obsługuje znaki +-() lub spacje. PeerAsn musi zawierać co najmniej jeden szczegół kontaktu typu "noc".
