# Set-AzApplicationGatewayPrivateLinkConfiguration

## STRESZCZENIe
Modyfikuje konfigurację PrivateLink dla bramy aplikacji.

## POLECENIA

```
Set-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 -IpConfiguration <PSApplicationGatewayPrivateLinkIpConfiguration[]> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzApplicationGatewayPrivateLinkConfiguration** modyfikuje konfigurację privateLink dla bramy aplikacji platformy Azure.

## Przykłady

### Przykład 1: Ustawianie konfiguracji PrivateLink
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway $AppGw -Name "privateLinkConfig01" -IpConfiguration $privateLinkIpConfiguration01
```

Pierwsze polecenie pobiera bramkę Application Gateway o nazwie ApplicationGateway01 należącej do grupy zasobów o nazwie ResourceGroup01 i zapisuje ją w zmiennej $AppGw.
Drugie polecenie ustawia konfigurację privateLink o nazwie privateLinkConfig01 w celu używania konfiguracji adresu IP przechowywanej w $privateLinkIpConfiguration 01
