# Get-AzNetworkUsage

## STRESZCZENIe
Wyświetlanie użycia sieci dla subskrypcji

## POLECENIA

```
Get-AzNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Get-AzNetworkUsage pobiera limity i bieżące użycie zasobów sieciowych.

## Przykłady

### Przykład 1
