# New-AzCosmosDBAccountKey

## STRESZCZENIe
Ponowne generowanie danego klucza konta CosmosDB.

## POLECENIA

### ByNameParameterSet (domyślny)
```
New-AzCosmosDBAccountKey -ResourceGroupName <String> -Name <String> [-KeyKind <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
New-AzCosmosDBAccountKey [-KeyKind <String>] -ResourceId <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByObjectParameterSet
```
New-AzCosmosDBAccountKey [-KeyKind <String>] -InputObject <PSDatabaseAccountGetResults> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Utwórz nowe konto usługi CosmosDB w danym przystawce zasobów.

## Przykłady

### Przykład 1
```powershell
PS C:\> New-AzCosmosDBAccountKey -ResourceGroupName rg -Name dbname
```

Nowe klucze są generowane dla konta o nazwie dbname (nazwa konta) w RG.

### Przykład 2

Ponowne generowanie danego klucza konta CosmosDB. (autogenerowana)
