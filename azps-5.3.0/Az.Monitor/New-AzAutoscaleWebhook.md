# New-AzAutoscaleWebhook

## STRESZCZENIe
Umożliwia utworzenie elementu webhook autoskalowania.

## POLECENIA

```
New-AzAutoscaleWebhook [-ServiceUri] <String> [[-Property] <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **New-AzAutoscaleWebhook** umożliwia utworzenie elementu webscale.

## Przykłady

### Przykład 1

Umożliwia utworzenie elementu webhook autoskalowania. (autogenerowana)

```powershell <!-- Aladdin Generated Example --> 
New-AzAutoscaleWebhook -ServiceUri 'https://contoso.com'
```
