# Get-AzCdnProfileSsoUrl

## STRESZCZENIe
Pobiera adres URL jednokrotnego podpisu w profilu usługi CDN.

## POLECENIA

### ByFieldsParameterSet (domyślny)
```
Get-AzCdnProfileSsoUrl -ProfileName <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByObjectParameterSet
```
Get-AzCdnProfileSsoUrl -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzCdnProfileSsoUrl** Pobiera adres URL rejestracji jednokrotnej profilu sieci dostarczania zawartości Azure (CDN, Content Delivery Network).
Ten adres URL umożliwia użytkownikom nawiązywanie połączenia z portalem dodatkowym i korzystanie z dodatkowych funkcji sieci CDN.
