# Unregister-AzProviderFeature

## STRESZCZENIe
Wyrejestrowuje funkcję dostawcy platformy Azure na koncie.

## POLECENIA

```
Unregister-AzProviderFeature -FeatureName <String> -ProviderNamespace <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Unregister-AzProviderFeature** umożliwia Wyrejestrowanie funkcji dostawcy platformy Azure na koncie.

## Przykłady

### Przykład 1. Wyrejestrowanie funkcji
```
PS C:\>Unregister-AzProviderFeature -FeatureName AllowApplicationSecurityGroups -ProviderNamespace Microsoft.Network
```

Spowoduje to Wyrejestrowanie funkcji AllowApplicationSecurityGroups dla usługi Microsoft. Network ze swojego konta.
