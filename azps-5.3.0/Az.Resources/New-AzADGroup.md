# New-AzADGroup

## STRESZCZENIe
Tworzy nową grupę usługi Active Directory.

## POLECENIA

```
New-AzADGroup -DisplayName <String> -MailNickname <String> [-Description <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Tworzy nową grupę usługi Active Directory. Poniżej przedstawiono wymagane uprawnienia:

- Wykres usługi Azure Active Directory
  - Directory. ReadWrite. All
- Program Microsoft Graph
  - Directory. ReadWrite. All
  - PrivilegedAccess.ReadWrite.AzureADGroup

## Przykłady

### Przykład 1. Tworzenie nowej grupy usługi AD

```powershell
PS C:\> New-AzADGroup -DisplayName "MyGroupDisplayName" -MailNickname "MyGroupNick"
```

Tworzy nową grupę usługi AD o nazwie "MyGroupDisplayName" i pseudonimie poczty "MyGroupNick".
