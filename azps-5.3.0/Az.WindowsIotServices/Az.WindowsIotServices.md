# AZ. WindowsIotServices, moduł
## Opis
Microsoft Azure PowerShell: polecenia cmdlet WindowsIotServices

## AZ. WindowsIotServices polecenia cmdlet
### [Get-AzWindowsIotServicesDevice](Get-AzWindowsIotServicesDevice.md)
Uzyskaj metadane niezwiązane z zabezpieczeniami usługi urządzeń z systemem Windows IoT.

### [Nowe — AzWindowsIotServicesDevice](New-AzWindowsIotServicesDevice.md)
Tworzenie lub aktualizowanie metadanych usługi urządzeń z systemem Windows IoT.
Typowym wzorcem do modyfikowania właściwości jest pobranie metadanych metadanych i zabezpieczeń usługi urządzenia IoT systemu Windows, a następnie połączenie ich z zmodyfikowanymi wartościami w nowej treści w celu zaktualizowania usługi urządzeń z systemem Windows IoT.

### [Remove-AzWindowsIotServicesDevice](Remove-AzWindowsIotServicesDevice.md)
Usuwanie usługi urządzeń z systemem Windows IoT.

### [Update-AzWindowsIotServicesDevice](Update-AzWindowsIotServicesDevice.md)
Aktualizuje metadane usługi urządzeń z systemem Windows IoT.
Typowym wzorcem do modyfikowania właściwości jest pobranie metadanych metadanych i zabezpieczeń usługi urządzenia IoT systemu Windows, a następnie połączenie ich z zmodyfikowanymi wartościami w nowej treści w celu zaktualizowania usługi urządzeń z systemem Windows IoT.
