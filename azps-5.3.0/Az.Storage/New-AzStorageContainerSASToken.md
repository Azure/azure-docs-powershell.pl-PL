# New-AzStorageContainerSASToken

## STRESZCZENIe
Generuje token SAS dla kontenera magazynu platformy Azure.

## POLECENIA

### SasPolicy
```
New-AzStorageContainerSASToken [-Name] <String> -Policy <String> [-Protocol <SharedAccessProtocol>]
 [-IPAddressOrRange <String>] [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-FullUri]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### SasPermission
```
New-AzStorageContainerSASToken [-Name] <String> [-Permission <String>] [-Protocol <SharedAccessProtocol>]
 [-IPAddressOrRange <String>] [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-FullUri]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **New-AzStorageContainerSASToken** generuje token dostępu współużytkowanego (SAS) dla kontenera magazynu platformy Azure.

## Przykłady

### Przykład 1. Generuj token SAS kontenera z uprawnieniem pełny kontener
```
PS C:\>New-AzStorageContainerSASToken -Name "Test" -Permission rwdl
```

W tym przykładzie Wygenerowano token SAS kontenera z uprawnieniem pełnego kontenera.

### Przykład 2: generowanie wielu tokenów SAS kontenera według rurociągu
```
PS C:\>Get-AzStorageContainer -Container test* | New-AzStorageContainerSASToken -Permission rwdl
```

W tym przykładzie Wygenerowano wiele tokenów SAS kontenera za pomocą potoku.

### Przykład 3: generowanie tokenu SAS kontenera z zasadami dostępu współużytkowanego
```
PS C:\>New-AzStorageContainerSASToken -Name "Test" -Policy "PolicyName"
```

W tym przykładzie Wygenerowano token SAS kontenera z zasadami dostępu współużytkowanego.

### Przykład 3: generowanie tokenu SAS kontenera tożsamości użytkownika z kontekstem magazynu na podstawie uwierzytelniania OAuth
```
PS C:\> $ctx = New-AzStorageContext -StorageAccountName $accountName -UseConnectedAccount
PS C:\> $StartTime = Get-Date
PS C:\> $EndTime = $startTime.AddDays(6)
PS C:\> New-AzStorageContainerSASToken -Name "ContainerName" -Permission rwd -StartTime $StartTime -ExpiryTime $EndTime -context $ctx
```

W tym przykładzie Wygenerowano token SAS kontenera tożsamości użytkownika z kontekstem magazynu na podstawie uwierzytelniania OAuth
