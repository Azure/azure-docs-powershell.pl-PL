# AZ. AppConfiguration, moduł
## Opis
Microsoft Azure PowerShell: polecenia cmdlet AppConfiguration

## AZ. AppConfiguration polecenia cmdlet
### [Get-AzAppConfigurationStore](Get-AzAppConfigurationStore.md)
Pobierz lub Wyświetl magazyny konfiguracji aplikacji.

### [Get-AzAppConfigurationStoreKey](Get-AzAppConfigurationStoreKey.md)
Wyświetla klucz dostępu do określonego magazynu konfiguracji.

### [Nowe — AzAppConfigurationStore](New-AzAppConfigurationStore.md)
Umożliwia utworzenie magazynu konfiguracji z określonymi parametrami.

### [Nowe — AzAppConfigurationStoreKey](New-AzAppConfigurationStoreKey.md)
Generuje klucz dostępu dla określonego magazynu konfiguracji.

### [Remove-AzAppConfigurationStore](Remove-AzAppConfigurationStore.md)
Usuwa magazyn konfiguracji.

### [Test — AzAppConfigurationStoreNameAvailability](Test-AzAppConfigurationStoreNameAvailability.md)
Sprawdza, czy nazwa magazynu konfiguracji jest dostępna do użycia.

### [Update-AzAppConfigurationStore](Update-AzAppConfigurationStore.md)
Umożliwia zaktualizowanie magazynu konfiguracji przy użyciu określonych parametrów.
