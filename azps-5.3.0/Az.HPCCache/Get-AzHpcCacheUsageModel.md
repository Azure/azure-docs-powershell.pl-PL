# Get-AzHpcCacheUsageModel

## STRESZCZENIe
Pobiera wszystkie usageModels dla obiektów docelowych magazynu systemu plików NFS.

## POLECENIA

```
Get-AzHpcCacheUsageModel [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzHpcCacheUsageModel** zwraca listę modeli użycia dla miejsca docelowego magazynu NFS.

## Przykłady

### Przykład 1
```powershell
PS C:\> Get-AzHpcCacheUsageModel
```
