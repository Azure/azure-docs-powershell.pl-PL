# Remove-AzHpcCacheStorageTarget

## STRESZCZENIe
Usuwa miejsce docelowe miejsca do magazynowania.

## POLECENIA

```
Remove-AzHpcCacheStorageTarget -ResourceGroupName <String> -CacheName <String> -Name <String> [-Force]
 [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzHpcCacheStorageTarget** usuwa miejsce docelowe miejsca do magazynowania z pamięci podręcznej Azure HPC.

## Przykłady

### Przykład 1
```powershell
PS C:\> Remove-AzHpcCacheStorageTarget -ResourceGroupName testRG -CacheName testCache -StorageTargetName testST
```
