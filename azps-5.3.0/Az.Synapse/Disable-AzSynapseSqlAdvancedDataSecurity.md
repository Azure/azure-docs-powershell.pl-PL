# Disable-AzSynapseSqlAdvancedDataSecurity

## STRESZCZENIe
Wyłącza zaawansowane zabezpieczenia danych w obszarze roboczym.

## POLECENIA

### DisableByNameParameterSet (domyślny)
```
Disable-AzSynapseSqlAdvancedDataSecurity [-ResourceGroupName <String>] -WorkspaceName <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DisableByInputObjectParameterSet
```
Disable-AzSynapseSqlAdvancedDataSecurity -InputObject <PSSynapseWorkspace> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DisableByResourceIdParameterSet
```
Disable-AzSynapseSqlAdvancedDataSecurity -ResourceId <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **disable-AzSynapseSqlAdvancedDataSecurity** wyłącza zaawansowane zabezpieczenia danych w obszarze roboczym.

## Przykłady

### Przykład 1
```powershell
PS C:\> Disable-AzSynapseSqlAdvancedDataSecurity -WorkspaceName ContosoWorkspace
```

To polecenie wyłącza zaawansowane zabezpieczenia danych w obszarze roboczym o nazwie ContosoWorkspace.

### Przykład 1
```powershell
PS C:\> Get-AzSynapseWorkspace -Name ContosoWorkspace | Disable-AzSynapseSqlAdvancedDataSecurity
```

To polecenie wyłącza zaawansowane zabezpieczenia danych w obszarze roboczym o nazwie ContosoWorkspace za pośrednictwem rurociągu.
