# New-AzRedisEnterpriseCacheDatabaseKey

## STRESZCZENIe
Generuje klucze dostępu bazy danych RedisEnterprise.

## POLECENIA

```
New-AzRedisEnterpriseCacheDatabaseKey -ClusterName <String> -ResourceGroupName <String>
 -KeyType <AccessKeyType> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Opis
Generuje klucze dostępu bazy danych RedisEnterprise.

## Przykłady

### Przykład 1: ponowne generowanie podstawowego klucza dostępu
```powershell
PS C:\> New-AzRedisEnterpriseCacheDatabaseKey -Name "MyCache" -ResourceGroupName "MyGroup" -KeyType "Primary"

PrimaryKey                                   SecondaryKey
----------                                   ------------
new-primary-key                              secondary-key

```

To polecenie generuje podstawowy klucz tajny (Primary Secret Access) służący do uwierzytelniania połączeń z bazą danych pamięci podręcznej usługi Redis Enterprise o nazwie moja pamięć podręczna.

### Przykład 2: ponowne generowanie pomocniczego klucza dostępu
```powershell
PS C:\> New-AzRedisEnterpriseCacheDatabaseKey -Name "MyCache" -ResourceGroupName "MyGroup" -KeyType "Secondary"

PrimaryKey                                   SecondaryKey
----------                                   ------------
primary-key                                  new-secondary-key

```

To polecenie powoduje wygenerowanie pomocniczego klucza tajnego dostępu do uwierzytelniania połączeń z bazą danych pamięci podręcznej usługi Redis Enterprise o nazwie moja pamięć podręczna.
