# Get-AzRedisEnterpriseCacheDatabaseKey

## STRESZCZENIe
Pobiera klucze dostępu do bazy danych programu RedisEnterprise.

## POLECENIA

```
Get-AzRedisEnterpriseCacheDatabaseKey -ClusterName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Opis
Pobiera klucze dostępu do bazy danych programu RedisEnterprise.

## Przykłady

### Przykład 1: uzyskiwanie kluczy dostępu do bazy danych
```powershell
PS C:\> Get-AzRedisEnterpriseCacheDatabaseKey -Name "MyCache" -ResourceGroupName "MyGroup"

PrimaryKey                                   SecondaryKey
----------                                   ------------
primary-key                                  secondary-key

```

To polecenie uzyskuje klucze dostępu Secret używane do uwierzytelniania połączeń z bazą danych pamięci podręcznej usługi Redis Enterprise o nazwie moja pamięć podręczna.
