# Get-AzSecurityAdaptiveApplicationControl

## STRESZCZENIe
Pobiera listę grup maszyn wirtualnych/serwerów związanych z aplikacją dla subskrypcji.

## POLECENIA

### SubscriptionScope (domyślny)
```
Get-AzSecurityAdaptiveApplicationControl [-SubscriptionId <String>] [-IncludePathRecommendation <Boolean>] [-Summary <Boolean>] 
[-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Kontrolki aplikacji adaptacyjnych są obliczane automatycznie przez usługę Azure Security Center, użyj tego polecenia cmdlet, aby uzyskać listę kontrolek aplikacji adaptacyjnych dotyczących zasobów w zakresie abonamentu.
