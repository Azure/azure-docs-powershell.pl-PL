# Set-AzSecurityAssessment

## STRESZCZENIe
Tworzenie lub aktualizowanie wyniku oceny zabezpieczeń dla zasobu

## POLECENIA

### SubscriptionLevelResource (domyślny)
```
Set-AzSecurityAssessment -Name <String> [-StatusCode <String>] [-StatusCause <String>]
 [-StatusDescription <String>]
 [-AdditionalData <System.Collections.Generic.Dictionary`2[System.String,System.String]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdLevelResource
```
Set-AzSecurityAssessment -Name <String> -AssessedResourceId <String> -StatusCode <String>
 [-StatusCause <String>] [-StatusDescription <String>]
 [-AdditionalData <System.Collections.Generic.Dictionary`2[System.String,System.String]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Tworzenie lub aktualizowanie wyniku oceny zabezpieczeń dla zasobu można użyć w celu zmiany stanu istniejącego wyniku lub dodania dodatkowych danych.
można użyć tylko w przypadku typów oceny "CustomerManaged" i tylko po utworzeniu dopasowanych metadanych oceny.

## Przykłady

### Przykład 1
```powershell
PS C:\> Set-AzSecurityAssessment -Name 4FB6C0A0-1137-42C7-A1C7-4BD37C91DE8D -StatusCode "Unhealthy"
```

Oznacza wynik subskrypcji jako "niezdrowy" dla oceny typu "4FB6C0A0-1137-42C7-A1C7-4BD37C91DE8D" — więcej szczegółów dotyczących typu oceny można znaleźć pod typem assessmentMetadata
