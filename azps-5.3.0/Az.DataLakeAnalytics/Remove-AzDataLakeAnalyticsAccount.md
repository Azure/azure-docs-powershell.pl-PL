# Remove-AzDataLakeAnalyticsAccount

## STRESZCZENIe
Usuwa konto Data Lake Analytics.

## POLECENIA

```
Remove-AzDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzDataLakeAnalyticsAccount** powoduje trwałe usunięcie konta usługi Azure Data Lake Analytics.

## Przykłady

### Przykład 1. Usuń konto
```
PS C:\>Remove-AzDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

To polecenie umożliwia usunięcie określonego konta usługi Data Lake Analytics.
