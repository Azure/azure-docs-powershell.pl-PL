# Remove-AzVMAccessExtension

## STRESZCZENIe
Usuwa rozszerzenie VMAccess z maszyny wirtualnej.

## POLECENIA

```
Remove-AzVMAccessExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Force] [-NoWait]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzVMAccessExtension** usuwa rozszerzenie maszyny wirtualnej dostępu do maszyny wirtualnej (VMAccess) z maszyny wirtualnej.

## Przykłady

### Przykład 1

Usuwa rozszerzenie VMAccess z maszyny wirtualnej. (autogenerowana)

```powershell <!-- Aladdin Generated Example --> 
Remove-AzVMAccessExtension -Name 'AgentPool01' -ResourceGroupName myresourcegroup -VMName 'VM01'
```
