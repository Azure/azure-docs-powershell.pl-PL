# Use-AzHDInsightCluster

## STRESZCZENIe
Umożliwia wybranie klastra do użycia z poleceniem cmdlet Invoke-RmAzureHDInsightHiveJob.

## POLECENIA

```
Use-AzHDInsightCluster [-ClusterName] <String> [-HttpCredential] <PSCredential> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **use-AzHDInsightCluster** umożliwia wybranie klastra usługi Azure HDInsight dla Invoke-AzHDInsightHiveJob polecenia cmdlet służącego do przesyłania zadań w usłudze Hive.

## Przykłady

### Przykład 1: Wybieranie klastra na potrzeby przesyłania zapytań dotyczących gałęzi
```
PS C:\># Cluster info
PS C:\>$clusterName = "your-hadoop-001"
PS C:\>$clusterCreds = Get-Credential

PS C:\>Use-AzHDInsightCluster `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

To polecenie umożliwia wybranie klastra do przesyłania zapytań dotyczących gałęzi.
