# Update-AzSentinelIncident

## STRESZCZENIe
Zaktualizuj zdarzenie.

## POLECENIA

### IncidentId (domyślny)
```
Update-AzSentinelIncident -ResourceGroupName <String> -WorkspaceName <String> -IncidentID <String>
 [-Classification <String>] [-ClassificationComment <String>] [-ClassificationReason <String>]
 [-Description <String>]
 [-Label <System.Collections.Generic.IList`1[Microsoft.Azure.Commands.SecurityInsights.Models.Incidents.PSSentinelIncidentLabel]>]
```

## Opis
Polecenie cmdlet **Update-AzSentinelIncident** aktualizuje zdarzenie w określonym obszarze roboczym.
Obiekt **incydentu** można przekazać jako parametr lub za pomocą operatora potoku lub można też określić parametry wymagane.
Za pomocą zmiennej *Confirm* i $ConfirmPreference programu Windows PowerShell można kontrolować, czy polecenie cmdlet monituje o potwierdzenie.

## Przykłady

### Przykład 1
```powershell
PS C:\> Update-AzSentinelIncident -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -IncidentId "MyIncidentId" -Severity High
```

Polecenie pobiera zdarzenie przez *IncidentId* i ustawia wartość właściwości *waga* na *wysoki*.  Pozostałe właściwości pozostają bez zmian.
