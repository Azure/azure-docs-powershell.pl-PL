# Invoke-AzsAzureBridgeProductDownload

## STRESZCZENIe
Pobiera produkt z usługi Azure Marketplace.

## POLECENIA

### Products_Download (domyślnie)
```
Invoke-AzsAzureBridgeProductDownload -Name <String> -ActivationName <String> -ResourceGroupName <String>
 [-AsJob] [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Zasobów
```
Invoke-AzsAzureBridgeProductDownload -ResourceId <String> [-AsJob] [-Force] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Pobiera produkt z usługi Azure Marketplace.

## Przykłady

### PRZYKŁAD 1
```
Invoke-AzsAzureBridgeProductDownload -ActivationName 'myActivation' -ProductName 'microsoft.docker-arm.1.1.0' -ResourceGroupName 'activationRG'
```

Pobierz produkt z usługi Azure Marketplace
