# Moduł migracji AzureRM.
## Opis
{{Moduł usługi Azure datamigration Service}}

## Polecenia cmdlet AzureRM. datamigration
### [Get-AzureRmDataMigrationProject](Get-AzureRmDataMigrationProject.md)
Pobiera właściwości projektu migracji bazy danych platformy Azure.

### [Get-AzureRmDataMigrationService](Get-AzureRmDataMigrationService.md)
Pobiera właściwości skojarzone z wystąpieniem usługi migracji bazy danych platformy Azure. 

### [Get-AzureRmDataMigrationTask](Get-AzureRmDataMigrationTask.md)
Pobiera obiekt PSProjectTask skojarzony z zadaniem migracji usługi migracji bazy danych platformy Azure.

### [New-AzureRmDataMigrationCommand (Get-AzureRmDataMigrationCommand.md)
Tworzy nowe polecenie, które ma zostać wykonane w ramach istniejącego zadania migracji usługi migracji bazy danych platformy Azure.

### [Nowe — AzureRmDataMigrationConnectionInfo](New-AzureRmDataMigrationConnectionInfo.md)
Tworzy nowy obiekt informacji o połączeniu określający typ serwera i nazwę dla połączenia.

### [Nowe — AzureRmDataMigrationDatabaseInfo](New-AzureRmDataMigrationDatabaseInfo.md)
Tworzy obiekt DatabaseInfo dla usługi migracji bazy danych platformy Azure, która określa źródło bazy danych do migracji.

### [Nowe — AzureRmDataMigrationFileShare](New-AzureRmDataMigrationFileShare.md)
Tworzy obiekt udostępnionego udziału w usłudze migracji bazy danych platformy Azure, który określa lokalny udział sieciowy do wykonania kopii zapasowych źródłowej bazy danych.

### [Nowe — AzureRmDataMigrationProject](New-AzureRmDataMigrationProject.md)
Tworzy nowy projekt usługi migracji bazy danych platformy Azure.

### [Nowe — AzureRmDataMigrationSelectedDB](New-AzureRmDataMigrationSelectedDB.md)
Tworzy obiekt wejściowy bazy danych, który zawiera informacje dotyczące źródłowej i docelowej bazy danych do migracji.

### [Nowe — AzureRmDataMigrationService](New-AzureRmDataMigrationService.md)
Tworzy nowe wystąpienie usługi migracji bazy danych platformy Azure.

### [Nowe — AzureRmDataMigrationSyncSelectedDB](New-AzureRmDataMigrationSyncSelectedDB.md)
Tworzy obiekt wejściowy bazy danych dla scenariuszy synchronizacji, w których znajdują się informacje dotyczące źródłowej i docelowej bazy danych do migracji.

### [Nowe — AzureRmDataMigrationTask](New-AzureRmDataMigrationTask.md)
Tworzy i rozpoczyna zadanie migracji danych w usłudze migracji bazy danych platformy Azure.

### [Remove-AzureRmDataMigrationProject](Remove-AzureRmDataMigrationProject.md)
Usuwa projekt usługi migracji bazy danych platformy Azure z platformy Azure.

### [Remove-AzureRmDataMigrationService](Remove-AzureRmDataMigrationService.md)
Usuwa wystąpienie usługi migracji bazy danych platformy Azure z platformy Azure.

### [Remove-AzureRmDataMigrationTask](Remove-AzureRmDataMigrationTask.md)
Usuwa zadanie usługi migracji bazy danych platformy Azure z platformy Azure.

### [Start — AzureRmDataMigrationService](Start-AzureRmDataMigrationService.md)
Powoduje uruchomienie wystąpienia usługi migracji bazy danych platformy Azure w stanie zatrzymanym. 

### [Zatrzymaj — AzureRmDataMigrationService](Stop-AzureRmDataMigrationService.md)
Zatrzymuje wystąpienie usługi migracji bazy danych platformy Azure, która jest w stanie uruchomienia.

### [Zatrzymaj — AzureRmDataMigrationTask](Stop-AzureRmDataMigrationTask.md)
Zatrzymuje zadanie usługi migracji bazy danych platformy Azure, które jest w stanie uruchomienia.
