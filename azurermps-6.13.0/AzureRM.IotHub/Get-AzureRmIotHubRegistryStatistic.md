# Get-AzureRmIotHubRegistryStatistic

## STRESZCZENIe
Pobiera RegistryStatistics dla IotHub.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Get-AzureRmIotHubRegistryStatistic [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Pobiera RegistryStatistics dla IotHub.
W tym artykule podano informacje o liczbie wszystkich urządzeń, włączonych i wyłączonych w IotHub.

## Przykłady

### Przykład 1 Pobierz RegistryStatistics
```
PS C:\> Get-AzureRmIotHubRegistryStatistic -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Pobiera RegistryStatictics dla IotHub o nazwie "myiothub".
