# Set-AzureRmVMBackupExtension

## STRESZCZENIe
Ustawia właściwości rozszerzenia kopii zapasowej na maszynie wirtualnej.

```
Set-AzureRmVMBackupExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Tag] <String>
```
