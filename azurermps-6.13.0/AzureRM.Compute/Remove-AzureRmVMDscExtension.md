# Remove-AzureRmVMDscExtension

## STRESZCZENIe
Usuwa obsługę rozszerzeń DSC z maszyny wirtualnej w grupie zasobów.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmVMDscExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmVMDscExtension** usuwa zażądaną obsługę rozszerzeń konfiguracji stanu z maszyny wirtualnej w grupie zasobów.

## Przykłady

### Przykład 1: Usuwanie rozszerzenia DSC
```
PS C:\> Remove-AzureRmVMDscExtension -ResourceGroupName "ResourceGroup001" -VMName "VM07" -Name "DSC"
```

To polecenie usuwa rozszerzenie o nazwie DSC na maszynie wirtualnej o nazwie VM07.
