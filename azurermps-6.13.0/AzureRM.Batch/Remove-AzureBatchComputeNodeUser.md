# Remove-AzureBatchComputeNodeUser

## STRESZCZENIe
Usuwa konto użytkownika z węzła obliczeniowego partii.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureBatchComputeNodeUser [-PoolId] <String> [-ComputeNodeId] <String> [-Name] <String>
 -BatchContext <BatchAccountContext> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureBatchComputeNodeUser** usuwa konto użytkownika z węzła obliczeniowego usługi Azure Batch.

## Przykłady

### Przykład 1: Usuwanie użytkownika z węzła obliczeniowego bez potwierdzenia
```
PS C:\>Remove-AzureBatchComputeNodeUser -PoolId "Pool01" -ComputeNodeId "ComputeNode01" -Name "User14" -Force -BatchContext $Context
```

To polecenie usuwa użytkownika o nazwie User14 z węzła compute o nazwie ComputeNode01.
Węzeł obliczeniowy znajduje się w puli o nazwie Pool01.
To polecenie określa parametr *Force* .
Dlatego polecenie nie monituje o potwierdzenie.
