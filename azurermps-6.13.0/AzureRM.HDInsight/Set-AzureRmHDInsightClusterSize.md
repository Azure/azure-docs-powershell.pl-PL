# Set-AzureRmHDInsightClusterSize

## STRESZCZENIe
Ustawia liczbę węzłów procesu roboczego w określonym klastrze.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Set-AzureRmHDInsightClusterSize [-ClusterName] <String> [-TargetInstanceCount] <Int32>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzureRmHDInsightClusterSize** określa liczbę węzłów procesu roboczego w określonym klastrze usługi Azure HDInsight.

## Przykłady

### Przykład 1: Ustawianie rozmiaru określonego klastra
```
PS C:\>Set-AzureRmHDInsightClusterSize -ClusterName "your-hadoop-001" -TargetInstanceCount 6
```

To polecenie ustawia rozmiar klastra o nazwie Your-Hadoop-001.
