# Remove-AzureRmPolicyRemediation

## STRESZCZENIe
Usuwa korygowanie zasad.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

### ByName (domyślny)
```
Remove-AzureRmPolicyRemediation -Name <String> [-Scope <String>] [-ManagementGroupName <String>]
 [-ResourceGroupName <String>] [-AllowStop] [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzureRmPolicyRemediation -ResourceId <String> [-AllowStop] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzureRmPolicyRemediation -InputObject <PSRemediation> [-AllowStop] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmPolicyRemediation** usuwa korygowanie zasad.

## Przykłady

### Przykład 1: usuwanie korygowania zasad w zakresie grupy zasobów
```
PS C:\> Remove-AzureRmPolicyRemediation -ResourceGroupName "myRG" -Name "remediation1"
```

To polecenie usuwa korygowanie o nazwie "remediation1" w grupie zasobów "myRG".

### Przykład 2: usuwanie korygowania grupy zarządzania za pomocą połączeń rurowych
```
PS C:\> $remediation = Get-AzureRmPolicyRemediation -ManagementGroupName "mg1" -Name "remediation1"
PS C:\> $remediation | Remove-AzureRmPolicyRemediation -Confirm
```

To polecenie usuwa korygowanie o nazwie "remediation1" z grupy zarządzania "MG1". Przed usunięciem zasobu zostanie wyświetlony monit o potwierdzenie.

### Przykład 3: Anulowanie i usuwanie korygowania zasad
```
PS C:\> Remove-AzureRmPolicyRemediation -ResourceGroupName "myRG" -Name "remediation1" -AllowStop
```

To polecenie usuwa korygowanie o nazwie "remediation1" w grupie zasobów "myRG". Jeśli korygowanie jest w toku, zostanie anulowane przed usunięciem.
