# New-AzureRmADGroup

## STRESZCZENIe
Tworzy nową grupę usługi Active Directory.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
New-AzureRmADGroup -DisplayName <String> -MailNickname <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Tworzy nową grupę usługi Active Directory.

## Przykłady

### Przykład 1 — Tworzenie nowej grupy usługi AD

```
PS C:\> New-AzureRmADGroup -DisplayName "MyGroupDisplayName" -MailNickname "myemail@domain.com"
```

Tworzy nową grupę usługi AD o nazwie "MyGroupDisplayName" i pseudonimie poczty " myemail@domain.com ".
