## STRESZCZENIe

## Opis
Polecenie cmdlet **New-AzureRmResourceLock** umożliwia utworzenie blokady zasobów.

## Przykłady

### Przykład 1. Tworzenie blokady zasobów w witrynie sieci Web
```
PS C:\>New-AzureRmResourceLock -LockLevel CanNotDelete -LockNotes "My lock notes" -LockName "ContosoSiteLock" -ResourceName "ContosoSite" -ResourceType "microsoft.web/sites"
```

To polecenie tworzy blokadę zasobu w witrynie sieci Web.
