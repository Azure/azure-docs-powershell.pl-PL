# Moduł wyszukiwania AzureRM.
## Opis
Ten temat zawiera tematy pomocy dotyczące apletów poleceń usługi Azure Search.

## AzureRM. Wyszukaj polecenia cmdlet
### [Get-AzureRmSearchAdminKeyPair](Get-AzureRmSearchAdminKeyPair.md)
Pobiera parę kluczy administratora usługi Azure Search.

### [Get-AzureRmSearchQueryKey](Get-AzureRmSearchQueryKey.md)
Pobiera klucze zapytania usługi Azure Search.

### [Get-AzureRmSearchService](Get-AzureRmSearchService.md)
Pobiera usługę Azure Search.

### [Nowe — AzureRmSearchAdminKey](New-AzureRmSearchAdminKey.md)
Generuje ponownie klucz administrator usługi Azure Search.

### [Nowe — AzureRmSearchQueryKey](New-AzureRmSearchQueryKey.md)
Utwórz nowy klucz kwerendy dla usługi Azure Search.

### [Nowe — AzureRmSearchService](New-AzureRmSearchService.md)
Tworzy usługę Azure Search.

### [Remove-AzureRmSearchQueryKey](Remove-AzureRmSearchQueryKey.md)
Usuń klucz zapytania z usługi Azure Search.

### [Remove-AzureRmSearchService](Remove-AzureRmSearchService.md)
Usuwanie usługi Azure Search.

### [Set-AzureRmSearchService](Set-AzureRmSearchService.md)
Aktualizowanie usługi Azure Search.
