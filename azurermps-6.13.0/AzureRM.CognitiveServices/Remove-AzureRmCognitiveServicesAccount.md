# Remove-AzureRmCognitiveServicesAccount

## STRESZCZENIe
Usuwa konto usług poznawczych.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmCognitiveServicesAccount** umożliwia usunięcie określonego konta usług poznawczych.

## Przykłady

### Przykład 1
To polecenie nie zwraca żadnych zmian.

```powershell
PS C:\> Remove-AzureRmCognitiveServicesAccount -ResourceGroupName cognitive-services-resource-group -name myluis
PS C:\>
```
