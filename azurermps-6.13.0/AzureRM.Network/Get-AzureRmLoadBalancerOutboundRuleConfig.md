# Get-AzureRmLoadBalancerOutboundRuleConfig

## STRESZCZENIe
Pobiera konfigurację reguły ruchu wychodzącego w usłudze równoważenia obciążenia.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Get-AzureRmLoadBalancerOutboundRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureRmLoadBalancerOutboundRuleConfig** Pobiera konfigurację reguły ruchu wychodzącego lub listę konfiguracji reguł ruchu wychodzącego w ramach modułu równoważenia obciążenia.

## Przykłady

### Przykład 1: Pobieranie konfiguracji reguły ruchu wychodzącego w usłudze równoważenia obciążenia
```powershell
PS C:\>$slb = Get-AzureRmLoadBalancer -ResourceGroupName "MyResourceGroup" -Name "MyLoadBalancer"
PS C:\>Get-AzureRmLoadBalancerOutboundRuleConfig -LoadBalancer $slb -Name "MyRule"
```

Pierwsze polecenie uzyskuje usługę równoważenia obciążenia o nazwie MyLoadBalancer, a następnie zapisuje ją w zmiennej $slb.
Drugie polecenie pobiera konfigurację reguły ruchu wychodzącego o nazwie Moja reguła skojarzona z tym modułem równoważenia obciążenia.
