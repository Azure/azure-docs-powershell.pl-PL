# Get-AzureRmApplicationGatewayHttpListenerCustomError

## STRESZCZENIe
Pobiera błędy niestandardowe od odbiornika http bramy Application Gateway.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Get-AzureRmApplicationGatewayHttpListenerCustomError [-StatusCode <String>]
 -HttpListener <PSApplicationGatewayHttpListener> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureRmApplicationGatewayCustomError** pobiera błędy niestandardowe z odbiornika http bramy Application Gateway.

## Przykłady

### Przykład 1. w przypadku odbiornika HTTP jest pobierany błąd niestandardowy
```powershell
PS C:\> $ce = Get-AzureRmApplicationGatewayCustomError -HttpListener $listener01 -StatusCode HttpStatus502
```

To polecenie pobiera i zwraca błąd niestandardowy kodu stanu HTTP 502 z odbiornika http $listener 01.

### Przykład 2: Pobiera listę wszystkich błędów niestandardowych w odbiorniku http
```powershell
PS C:\> $ces = Get-AzureRmApplicationGatewayCustomError -HttpListener $listener01
```

To polecenie pobiera i zwraca listę wszystkich błędów niestandardowych z odbiornika http $listener 01.
