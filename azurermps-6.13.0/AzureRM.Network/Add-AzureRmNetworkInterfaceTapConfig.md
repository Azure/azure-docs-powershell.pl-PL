# Add-AzureRmNetworkInterfaceTapConfig

## STRESZCZENIe
Tworzy zasób TapConfiguration skojarzony z NetworkInterface. Będzie to miało odwołanie do istniejącego zasobu VirtualNetworkTap.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

### SetByResource (domyślny)
```
Add-AzureRmNetworkInterfaceTapConfig -NetworkInterface <PSNetworkInterface> -Name <String>
 [-VirtualNetworkTap <PSVirtualNetworkTap>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### SetByResourceId
```
Add-AzureRmNetworkInterfaceTapConfig -NetworkInterface <PSNetworkInterface> -Name <String>
 [-VirtualNetworkTapId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Add-AzureRmNetworkInterfaceTapConfig** tworzy zasób TapConfiguration skojarzony z NetworkInterface. Będzie to miało odwołanie do istniejącego zasobu VirtualNetworkTap.

## Przykłady

### Przykład 1: Dodawanie TapConfiguration do danego NetworkInterface
```
PS C:\>Add-AzureRmNetworkInterfaceTapConfig -NetworkInterface $sourceNic -VirtualNetworkTap $vVirtualNetworkTap -Name 'myTapConfig'
```

Dodaj TapConfiguration do sourceNic. Ruch z maszyny wirtualnej sourceNic będzie dublowany na docelowej maszynie wirtualnej określonej w zasobie vVirtualNetworkTap.
