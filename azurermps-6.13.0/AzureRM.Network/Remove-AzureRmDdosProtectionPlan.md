# Remove-AzureRmDdosProtectionPlan

## STRESZCZENIe
Usuwa plan ochrony DDoS.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmDdosProtectionPlan -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Remove-AzureRmDdosProtectionPlan usuwa plan ochrony DDoS.

## Przykłady

### Przykład 1: Usuwanie pustego planu ochrony DDoS
```
D:\> Remove-AzureRmDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlan
```

W tym przypadku firma Microsoft usunie plan ochrony DDoS, jak określono.

### Przykład 2: usuwanie planu ochrony DDoS skojarzonego z siecią wirtualną
```
D:\> $vnet = Get-AzureRmVirtualNetwork -Name VnetName -ResourceGroupName ResourceGroupName
D:\> $vnet.DdosProtectionPlan = $null
```

Następnie utrzymujemy nowy stan, przełączając zmienną lokalną do **Set-AzureRmVirtualNetwork**. W tym momencie plan nie jest już skojarzony z siecią wirtualną.
Jeśli jest to Ostatnia skojarzona z planem, możemy usunąć plan ochrony DDoS przy użyciu polecenia Remove-AzureRmDdosProtectionPlan.
