# Remove-AzureRmAutomationVariable

## STRESZCZENIe
Usuwa zmienną automatyzacji.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmAutomationVariable [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmAutomationVariable** usuwa zmienną z usługi Azure Automation.

## Przykłady

### Przykład 1. Usuń zmienną
```
PS C:\>Remove-AzureRmAutomationVariable -AutomationAccountName "Contoso17" -Name "StringVariable22" -Force -ResourceGroupName "ResourceGroup01"
```

To polecenie usuwa zmienną o nazwie StringVariable22 na koncie automatyzacji o nazwie Contoso17.
To polecenie określa parametr *Force* .
W związku z tym nie jest wyświetlany monit o potwierdzenie.
