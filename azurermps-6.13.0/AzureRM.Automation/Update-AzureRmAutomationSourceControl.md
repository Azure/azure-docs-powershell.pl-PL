# Update-AzureRmAutomationSourceControl

## STRESZCZENIe
Aktualizuje kontrolę źródła usługi Azure Automation.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Update-AzureRmAutomationSourceControl -Name <String> [-AccessToken <SecureString>] [-FolderPath <String>]
 [-Branch <String>] [-Description <String>] [-AutoSync <Boolean>] [-PublishRunbook <Boolean>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Update-AzureRmAutomationSourceControl modyfikuje wartość pola w kontroli źródła w usłudze Azure Automation.

## Przykłady

### Przykład 1
To polecenie ustawia wartość FAŁSZ pola PublishRunbook dla kontroli źródła usługi Azure Automation o nazwie VSTSNative na koncie o nazwie devAccount.


```powershell
Update-AzureRmAutomationSourceControl -ResourceGroupName "rg1" `
                                      -AutomationAccountName "devAccount" `
                                      -Name "VSTSNative" `
                                      -PublishRunbook $false
```
