# Stop-AzureRmAutomationJob

## STRESZCZENIe
Zatrzymuje zadanie automatyzacji.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Stop-AzureRmAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **stop-AzureRmAutomationJob** zatrzymuje zadanie usługi Azure Automation.
Określanie uruchomionego zadania automatyzacji.

## Przykłady

### Przykład 1: Zatrzymywanie zadania
```
PS C:\>Stop-AzureRmAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

To polecenie zatrzymuje zadanie o określonym IDENTYFIKATORze.
