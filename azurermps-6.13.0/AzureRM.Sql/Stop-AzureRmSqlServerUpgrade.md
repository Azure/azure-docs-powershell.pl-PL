# Stop-AzureRmSqlServerUpgrade

## STRESZCZENIe
Zatrzymuje uaktualnienie serwera bazy danych SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Stop-AzureRmSqlServerUpgrade [-Force] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **stop-AzureRmSqlServerUpgrade** zatrzymuje uaktualnianie serwera bazy danych SQL Azure.

## Przykłady

### Przykład 1: zatrzymywanie uaktualnienia serwera
```
PS C:\>Stop-AzureRmSqlServerUpgrade -ResourceGroupName "ResourceGroup01" -ServerName "Server02"
```

To polecenie zatrzymuje żądanie uaktualnienia serwera o nazwie Server02 przypisanego do grupy zasobów o nazwie ResourceGroup01.
