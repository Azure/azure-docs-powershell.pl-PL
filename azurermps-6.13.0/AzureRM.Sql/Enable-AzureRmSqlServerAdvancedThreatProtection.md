# Enable-AzureRmSqlServerAdvancedThreatProtection

## STRESZCZENIe
Włączanie zaawansowanej ochrony przed zagrożeniami na serwerze.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Enable-AzureRmSqlServerAdvancedThreatProtection [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **enable-AzureRmSqlServerAdvancedThreatProtection** włącza zaawansowaną ochronę przed zagrożeniami na serwerze.

## Przykłady

### Przykład 1-Włączanie zaawansowanej ochrony przed zagrożeniami dla serwera
```powershell
PS C:\>  Enable-AzureRmSqlServerAdvancedThreatProtection `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### Przykład 2 — Włączanie zaawansowanej ochrony przed zagrożeniami serwera z zasobu serwera
```powershell
PS C:\>  Get-AzureRmSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Enable-AzureRmSqlServerAdvancedThreatProtection

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```
