# Remove-AzureRmSqlDatabaseFailoverGroup

## STRESZCZENIe
Usuwa grupę trybu failover bazy danych SQL Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmSqlDatabaseFailoverGroup [-ServerName] <String> [-FailoverGroupName] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
To polecenie usuwa grupę pracy awaryjnej o określonej nazwie, pozostawiając wszystkie bazy danych i relacje replikacji bez zmian. Punkt końcowy odbiornika zostanie wyrejestrowany z systemu DNS.
Aby wykonać polecenie, należy użyć głównego serwera grupy pracy awaryjnej.

## Przykłady

### Przykład 1
```
PS C:\> Remove-AzureRmSqlDatabaseFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg
```

Usuwanie grupy pracy awaryjnej.
