# Remove-AzureRmSqlSyncGroup

## STRESZCZENIe
Usuwa grupę usługi Azure SQL Database Sync.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmSqlSyncGroup [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmSqlSyncGroup** usuwa grupę synchronizacji bazy danych Azure SQL.

## Przykłady

### Przykład 1. Usuń grupę synchronizacji
```
PS C:\>Remove-AzureRmSqlSyncGroup -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "database01" -Name "syncGroup01"
```

To polecenie usuwa grupę usługi Azure SQL Database Sync o nazwie syncGroup01.
