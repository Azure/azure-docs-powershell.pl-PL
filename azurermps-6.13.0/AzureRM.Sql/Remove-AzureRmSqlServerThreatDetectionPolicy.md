# Remove-AzureRmSqlServerThreatDetectionPolicy

## STRESZCZENIe
Usuwa zasady wykrywania zagrożeń z serwera.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmSqlServerThreatDetectionPolicy [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Remove-AzureRmSqlServerThreatDetectionPolicy usuwa zasady wykrywania zagrożeń z programu Azure SQL Server.
Aby użyć tego polecenia cmdlet, określ parametry ResourceGroupName oraz nazwa_serwera w celu zidentyfikowania serwera, z którego to polecenie cmdlet usunie zasady.

## Przykłady

### Przykład 1: usuwanie zasad wykrywania zagrożeń dla bazy danych
```
PS C:\> Remove-AzureRmSqlServerThreatDetectionPolicy -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
```

To polecenie usuwa zasady wykrywania zagrożeń z serwera o nazwie Server01.
