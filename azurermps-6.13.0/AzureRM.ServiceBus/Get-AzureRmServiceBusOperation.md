# Get-AzureRmServiceBusOperation

## STRESZCZENIe
Lista obsługiwanych operacji ServiceBus

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Get-AzureRmServiceBusOperation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureRmServiceBusOperation** zawiera listę ServiceBus obsługiwanych operacji.

## Przykłady

### Przykład 1
```
PS C:\> Get-AzureRmServiceBusOperation
```

Lista obsługiwanych operacji ServiceBus
