# Get-AzureRmSecurityLocation

## STRESZCZENIe
Pobiera lokalizację, w której usługa Azure Security Center będzie automatycznie zapisywać dane dla określonej subskrypcji.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

### SubscriptionScope (domyślny)
```
Get-AzureRmSecurityLocation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SubscriptionLevelResource
```
Get-AzureRmSecurityLocation -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Zasobów
```
Get-AzureRmSecurityLocation -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
Usługa Azure Security Center automatycznie zdecyduje w lokalizacji, w której będą zapisywane niektóre dane.
Użyj tego polecenia cmdlet, aby wykryć lokalizację.

## Przykłady

### Przykład 1
```powershell
PS C:\> Get-AzureRmSecurityLocation
Id                                                                                                   Name
--                                                                                                   ----
/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869/providers/Microsoft.Security/locations/centralus centralus
```

Pobiera lokalizację, w której usługa Azure Security Center zapisuje obliczone dane zabezpieczeń.
