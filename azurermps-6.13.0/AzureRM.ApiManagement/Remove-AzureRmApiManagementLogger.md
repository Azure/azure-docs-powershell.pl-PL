# Remove-AzureRmApiManagementLogger

## STRESZCZENIe
Usuwa rejestratora zarządzania interfejsem API.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmApiManagementLogger -Context <PsApiManagementContext> -LoggerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmApiManagementLogger** usuwa **Rejestrator** zarządzania interfejsem API platformy Azure.

## Przykłady

### Przykład 1: usuwanie rejestratora
```
PS C:\>$apimContext = New-AzureRmApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzureRmApiManagementLogger -Context $apimContext -LoggerId "Logger123" -Force
```

To polecenie usuwa rejestratora o IDENTYFIKATORze Logger123.
