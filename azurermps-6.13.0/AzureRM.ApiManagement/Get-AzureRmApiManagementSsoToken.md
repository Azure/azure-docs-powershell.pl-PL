# Get-AzureRmApiManagementSsoToken

## STRESZCZENIe
Pobiera link z tokenem SSO do wdrożonego portalu zarządzania usługi zarządzania interfejsem API.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Get-AzureRmApiManagementSsoToken -ResourceGroupName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzureRmApiManagementSsoToken** zwraca link (adres URL), który zawiera token logowania jednokrotnego (SSO) do wdrożonego portalu zarządzania usługi zarządzania interfejsem API.

## Przykłady

### Przykład 1: uzyskiwanie tokenu SSO usługi zarządzania interfejsem API
```
PS C:\>Get-AzureRmApiManagementSsoToken -ResourceGroupName "Contoso" -Name "ContosoApi"
```

To polecenie pobiera token SSO usługi zarządzania interfejsem API.
