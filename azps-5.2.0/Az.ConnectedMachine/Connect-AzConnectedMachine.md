# Connect-AzConnectedMachine

## STRESZCZENIe
Interfejs API umożliwiający zarejestrowanie nowego komputera, a w związku z tym utworzenie prześledzonego zasobu w ARM

## POLECENIA

```
Connect-AzConnectedMachine [-ResourceGroupName] <String> [-Location] <String> [[-SubscriptionId] <String>]
 [[-Name] <String>] [[-DefaultProfile] <PSObject>] [[-PSSession] <PSSession[]>] [[-Tag] <Hashtable>]
 [[-Proxy] <Uri>] [<CommonParameters>]
```

## Opis
Interfejs API umożliwiający zarejestrowanie nowego komputera, a w związku z tym utworzenie prześledzonego zasobu w ARM

## Przykłady

### Przykład 1: znajduje się na pokładzie komputera, który jest podłączony do komputera.
```powershell
PS C:\> Connect-AzConnectedMachine -ResourceGroupName contoso-connected-machines -Name linux_eastus1_1 -Location eastus
```

Znajduje się na płycie, na którym znajduje się komputer, na którym jesteś podłączony.

```powershell

PS C:\> Connect-AzConnectedMachine -ResourceGroupName contoso-rg -Name win_eastus1_1 -Location eastus -PSSession $session
```

Na komputerze zdalnym jako urządzenie podłączone za pomocą funkcji komunikacji zdalnej programu PowerShell.
Uwaga: obecnie jest obsługiwany tylko system Windows jako element docelowy.
