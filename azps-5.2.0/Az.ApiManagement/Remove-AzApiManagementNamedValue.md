# Remove-AzApiManagementNamedValue

## STRESZCZENIe
Usuwa wartość zarządzania interfejsem API o nazwie.

## POLECENIA

```
Remove-AzApiManagementNamedValue -Context <PsApiManagementContext> -NamedValueId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzApiManagementNamedValue** usuwa przystawkę Zarządzanie interfejsem Azure API **o nazwie Value**.

## Przykłady

### Przykład 1. Usuń nazwaną wartość
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementNamedValue -Context $apimContext -NamedValueId "Property11" -PassThru
```

To polecenie usuwa nazwaną wartość o IDENTYFIKATORze Property11.
