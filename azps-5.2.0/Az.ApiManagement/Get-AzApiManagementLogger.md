# Get-AzApiManagementLogger

## STRESZCZENIe
Pobiera obiekty rejestratora zarządzania interfejsem API.

## POLECENIA

### GetAllLoggers (domyślny)
```
Get-AzApiManagementLogger -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByLoggerId
```
Get-AzApiManagementLogger -Context <PsApiManagementContext> -LoggerId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzApiManagementLogger** pobiera **rejestratora** zarządzania usługą Azure API lub wszystkich rejestratorów.

## Przykłady

### Przykład 1. Pobierz wszystkie rejestratory
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementLogger -Context $apimContext
```

To polecenie pobiera wszystkie rejestratory dla określonego kontekstu.

### Przykład 2: uzyskiwanie określonego rejestratora
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementLogger -Context $apimContext -LoggerId "Logger123"
```

To polecenie usuwa rejestratora o IDENTYFIKATORze Logger123.
