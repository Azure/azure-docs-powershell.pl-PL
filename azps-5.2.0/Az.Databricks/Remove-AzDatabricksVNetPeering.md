# Remove-AzDatabricksVNetPeering

## STRESZCZENIe
Usuwa obszar roboczy vNetPeering.

## POLECENIA

### Usuń (domyślnie)
```
Remove-AzDatabricksVNetPeering -Name <String> -ResourceGroupName <String> -WorkspaceName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzDatabricksVNetPeering -InputObject <IDatabricksIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Opis
Usuwa obszar roboczy vNetPeering.

## Przykłady

### Przykład 1: usuwanie komunikacji równorzędnej sieci datakostka według nazwy
```powershell
PS C:\> Remove-AzDatabricksVNetPeering -WorkspaceName databricks-test01 -ResourceGroupName lucas-manual-test -Name vnetpeering-t01

```

To polecenie usuwa elementy równorzędne wirtualnej z obiektów datakostka według nazwy

### Przykład 2: usuwanie komunikacji równorzędnej sieci datakostek według obiektów
```powershell
PS C:\> Get-AzDatabricksVNetPeering -ResourceGroupName lucas-manual-test -WorkspaceName databricks-test01 -PeeringName MyPeering-test01 | Remove-AzDatabricksVNetPeering

```

To polecenie usuwa elementy równorzędne wirtualnej z obiektów datakostka według obiektu
