# Remove-AzWcfRelay

## STRESZCZENIe
Usuwa WcfRelay z określonego obszaru nazw przekaźnika.

## POLECENIA

```
Remove-AzWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzWcfRelay** usuwa WcfRelay z określonego obszaru nazw Relay.

## Przykłady

### Przykład 1
```
PS C:\> Remove-AzWcfRelay -ResourceGroupName Default-ServiceBus-WestUS -NamespaceName TestNameSpace-Relay1 -Name TestWCFRelay1
```

Usuwa WcfRelay `TestWCFRelay1` z obszaru nazw `TestNameSpace-Relay1` .
