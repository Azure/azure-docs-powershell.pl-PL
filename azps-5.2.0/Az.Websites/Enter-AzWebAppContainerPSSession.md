# Enter-AzWebAppContainerPSSession

## STRESZCZENIe
Otwiera zdalną sesję programu PowerShell w kontenerze systemu Windows określonym w danej witrynie lub gnieździe i danej grupie zasobów.

## POLECENIA

### S1 (domyślnie)
```
Enter-AzWebAppContainerPSSession [-PassThru] [-Force] [[-SlotName] <String>] [-ResourceGroupName] <String>
 [-Name] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Enter-AzWebAppContainerPSSession [-PassThru] [-Force] [-WebApp] <PSSite>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
otwiera zdalną sesję programu PowerShell w kontenerze systemu Windows określonym w danej witrynie lub gnieździe i danej grupie zasobów.

## Przykłady

### Przykład 1
```
PS C:\> Enter-AzWebAppContainerPSSession -ResourceGroupName "Default-Web-WestUS" -Name "ContosoASP"
```

To polecenie otwiera zdalną sesję programu PowerShell w aplikacji kontenerowej systemu Windows ContosoASP
