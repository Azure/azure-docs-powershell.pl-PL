# Remove-AzDataLakeAnalyticsComputePolicy

## STRESZCZENIe
Usuwa określoną zasadę obliczania usługi Azure Data Lake Analytics

## POLECENIA

```
Remove-AzDataLakeAnalyticsComputePolicy [-ResourceGroupName <String>] [-Account] <String> [-Name] <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
**Element Remove-AzDataLakeAnalyticsComputePolicy** usuwa określoną zasadę obliczeniową usługi Azure Data Lake Analytics.

## Przykłady

### Przykład 1. Usuń zasady obliczania
```
PS C:\>Remove-AzDataLakeAnalyticsComputePolicy -Account "contosoadla" -Name myPolicy
```

To polecenie usuwa określoną zasadę obliczeniową z nazwą "Moja zasada" na koncie "contosoadla".
