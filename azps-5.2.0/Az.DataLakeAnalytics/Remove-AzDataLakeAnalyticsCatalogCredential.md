# Remove-AzDataLakeAnalyticsCatalogCredential

## STRESZCZENIe
Usuwa poświadczenia usługi Azure Data Lake Analytics.

## POLECENIA

```
Remove-AzDataLakeAnalyticsCatalogCredential [-Account] <String> [-DatabaseName] <String> [-Name] <String>
 [[-Password] <PSCredential>] [-Recurse] [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Remove-AzDataLakeAnalyticsCatalogCredential usuwa poświadczenia wykazu usługi Azure Data Lake Analytics.

## Przykłady

### Przykład 1: usuwanie poświadczenia
```
PS C:\> Remove-AzDataLakeAnalyticsCatalogCredential -AccountName "ContosoAdla" `
                      -DatabaseName "DatabaseName" `
                      -Name "CredName"
```

To polecenie usuwa określone poświadczenia wykazu usługi Data Lake Analytics.
