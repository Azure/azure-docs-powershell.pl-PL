# Set-AzDataLakeAnalyticsFirewallRule

## STRESZCZENIe
Umożliwia zaktualizowanie reguły zapory na koncie danych w usłudze Data Lake Analytics.

## POLECENIA

```
Set-AzDataLakeAnalyticsFirewallRule [-Account] <String> [-Name] <String> [[-StartIpAddress] <String>]
 [[-EndIpAddress] <String>] [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzDataLakeAnalyticsFirewallRule** umożliwia zaktualizowanie reguły zapory na koncie usługi Azure Data Lake Analytics.

## Przykłady

### Przykład 1: aktualizowanie reguły zapory
```
PS C:\>Set-AzDataLakeAnalyticsFirewallRule -Account "ContosoAdlAcct" -Name "My firewall rule" -StartIpAddress 127.0.0.1 -EndIpAddress 127.0.0.10
```

To polecenie powoduje zaktualizowanie reguły zapory o nazwie "Moja Reguła zapory" na koncie "ContosoAdlAcct" w celu uzyskania nowego zakresu adresów IP: 127.0.0.1-127.0.0.10
