# Unregister-AzResourceProvider

## STRESZCZENIe
Wyrejestrowuje dostawcę zasobów.

## POLECENIA

```
Unregister-AzResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Unregister-AzResourceProvider** wyrejestrowuje dostawcę zasobów platformy Azure.

## Przykłady

### Przykład 1: Wyrejestrowanie dostawcy zasobów z ProviderNamespace

```powershell
PS C:\>Unregister-AzResourceProvider -ProviderNamespace "Microsoft.support"
```

To polecenie wyrejestrowuje dostawcę zasobów "Microsoft. Support".
