# Remove-AzPolicySetDefinition

## STRESZCZENIe
Usuwa definicję zestawu zasad.

## POLECENIA

```

 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

```

```

## Opis
Polecenie cmdlet **Remove-AzPolicySetDefinition** usuwa definicję zasad.

## Przykłady

### Przykład 1: usuwanie definicji zestawu zasad według identyfikatora zasobu
```
PS C:\> $PolicySetDefinition = Get-AzPolicySetDefinition -ResourceId '/subscriptions/mySub/Microsoft.Authorization/policySetDefinitions/myPSSetDefinition'
PS C:\> Remove-AzPolicySetDefinition -Id $PolicySetDefinition.ResourceId -Force
```

Pierwsze polecenie pobiera definicję zestawu zasad za pomocą polecenia cmdlet Get-AzPolicySetDefinition.
Polecenie zapisuje je w zmiennej $PolicySetDefinition.
Drugie polecenie usuwa definicję zestawu zasad określoną przez właściwość **ResourceId** $PolicySetDefinition.
