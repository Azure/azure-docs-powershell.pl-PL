# New-AzCosmosDBCassandraClusterKey

## STRESZCZENIe
Tworzy nowy klucz klastra Cassandra CosmosDB.

## POLECENIA

```
New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
**Nowy-AzCosmosDBCassandraClusterKey** tworzy nowy klucz klastra CosmosDB Cassandra.

## Przykłady

### Przykład 1
```powershell
PS C:\> New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String>

Name   OrderBy
----   -------
{name}  Asc
```
