# AZ. CustomProviders, moduł
## Opis
Microsoft Azure PowerShell: polecenia cmdlet niestandardowych dostawców

## AZ. CustomProviders polecenia cmdlet
### [Get-AzCustomProvider](Get-AzCustomProvider.md)
Pobiera manifest dostawcy zasobów niestandardowych.

### [Get-AzCustomProviderAssociation](Get-AzCustomProviderAssociation.md)
Uzyskaj powiązanie.

### [Nowe — AzCustomProvider](New-AzCustomProvider.md)
Umożliwia utworzenie lub zaktualizowanie niestandardowego dostawcy zasobów.

### [Nowe — AzCustomProviderAssociation](New-AzCustomProviderAssociation.md)
Tworzenie lub aktualizowanie skojarzenia.

### [Remove-AzCustomProvider](Remove-AzCustomProvider.md)
Usuwa niestandardowego dostawcę zasobów.

### [Remove-AzCustomProviderAssociation](Remove-AzCustomProviderAssociation.md)
Usuwanie skojarzenia.

### [Update-AzCustomProvider](Update-AzCustomProvider.md)
Aktualizuje istniejącego niestandardowego dostawcę zasobów.
Jedyną wartością, którą można zaktualizować za pośrednictwem poprawki, są obecnie Tagi.
