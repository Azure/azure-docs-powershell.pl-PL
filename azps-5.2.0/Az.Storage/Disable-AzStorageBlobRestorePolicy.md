# Disable-AzStorageBlobRestorePolicy

## STRESZCZENIe
Wyłącza zasady przywracania obiektu BLOB na koncie magazynu.

## POLECENIA

### AccountName (domyślnie)
```
Disable-AzStorageBlobRestorePolicy [-ResourceGroupName] <String> [-StorageAccountName] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AccountName
```
Disable-AzStorageBlobRestorePolicy -StorageAccount <PSStorageAccount> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### BlobServicePropertiesResourceId
```
Disable-AzStorageBlobRestorePolicy [-ResourceId] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **disable-AzStorageBlobRestorePolicy** wyłącza zasady przywracania obiektu BLOB dla usługi Azure Storage BLOB.

## Przykłady

### Przykład 1: wyłącza zasady przywracania obiektu BLOB usługi Azure Storage BLOB na koncie magazynu
```powershell
PS C:\> Disable-AzStorageBlobRestorePolicy -ResourceGroupName "myresourcegoup" -StorageAccountName "mystorageaccount"
```

To polecenie wyłącza zasady przywracania obiektów BLOB usługi Azure Storage w ramach konta magazynu.
