# Get-AzCognitiveServicesAccountSku

## STRESZCZENIe
Pobiera dostępne jednostki SKU dla konta.

## POLECENIA

```
Get-AzCognitiveServicesAccountSku [-Type <String>] [-Location <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzCognitiveServicesAccountSku** Pobiera dostępne jednostki SKU dla konta usług poznawczych.
Jednostka SKU jest planem warstw dla konta.
Definiuje cenę, limit rozmów i stawkę za konto.
F0 SKU jest bezpłatną warstwą.
Płatne warstwy obejmują S0, S1, S2 itd.

## Przykłady

### Przykład 1
```powershell
PS C:\> (Get-AzCognitiveServicesAccountSku -Type 'TextAnalytics' -Location "westus").Value | Select-Object -E
xpandProperty Sku;

Name     Tier
----     ----
F0       Free
S0   Standard
```
