# Get-AzBlockchainMemberApiKey

## STRESZCZENIe
Wyświetla listę kluczy interfejsu API elementu członkowskiego Blockchain.

## POLECENIA

```
Get-AzBlockchainMemberApiKey -BlockchainMemberName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Opis
Wyświetla listę kluczy interfejsu API elementu członkowskiego Blockchain.

## Przykłady

### Przykład 1: Lista kluczy API Blockchain
```powershell
PS C:\> Get-AzBlockchainMemberApiKey -BlockchainMemberName dolauli001 -ResourceGroupName testgroup

KeyName Value
------- -----
key1    72_8u5HPZJxtZmtvm4Y4W9o-
key2    eu9kx94TKH506R0i4JhYBmsx
```

To polecenie wyświetla listę kluczy interfejsu API dla członka Blockchain.
