# Set-AzApplicationGatewayTrustedClientCertificate

## STRESZCZENIe
Modyfikuje łańcuch certyfikatów zaufanego urzędu certyfikacji klienta w bramie Application.

## POLECENIA

```
Set-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzApplicationGatewayTrustedClientCertificate** modyfikuje łańcuch certyfikatów zaufanego urzędu certyfikacji klienta w bramie Application.

## Przykłady

### Przykład 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Set-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name $certName --CertificateFile ".\clientCAUpdated.cer"
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Powyżej przykładowych scenariuszy pokazano, jak zaktualizować istniejący obiekt łańcucha certyfikatu zaufanego urzędu certyfikacji klienta. Pierwsze polecenie uzyskuje bramę Application Gateway i zapisuje ją w zmiennej $gw. Drugie polecenie modyfikuje istniejący obiekt łańcuchowy certyfikatu zaufanego urzędu certyfikacji klienta za pomocą nowego pliku łańcucha certyfikatów urzędu certyfikacji. W trzecim poleceniu jest aktualizowana Brama Application Gateway na platformie Azure.
