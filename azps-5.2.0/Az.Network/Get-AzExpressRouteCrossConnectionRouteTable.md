# Get-AzExpressRouteCrossConnectionRouteTable

## STRESZCZENIe
Pobiera tabelę tras z połączenia z ExpressRoute krzyżowego.

## POLECENIA

### SpecifyByParameterValues
```
Get-AzExpressRouteCrossConnectionRouteTable -ResourceGroupName <String> -CrossConnectionName <String>
 -PeeringType <String> -DevicePath <DevicePathEnum> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### SpecifyByReference
```
Get-AzExpressRouteCrossConnectionRouteTable -ExpressRouteCrossConnection <PSExpressRouteCrossConnection>
 -PeeringType <String> -DevicePath <DevicePathEnum> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzExpressRouteCrossConnectionRouteTable** pobiera szczegółową tabelę tras obwodu ExpressRoute. W tabeli Route zostaną wyświetlone wszystkie trasy lub można filtrować, aby były wyświetlane trasy dla określonego typu komunikacji równorzędnej. Za pomocą tabeli Route można sprawdzić poprawność konfiguracji i łączności równorzędnej.

## Przykłady

### Przykład 1. Wyświetlanie tabeli tras dla ścieżki podstawowej
```
Get-AzExpressRouteCrossConnectionRouteTable -ResourceGroupName $RG -ExpressRouteCrossConnectionName $CircuitName -DevicePath 'Primary'
```
