# New-AzFirewallThreatIntelWhitelist

## STRESZCZENIe
Tworzenie nowego whitelist analizy zagrożeń dla zapory platformy Azure

## POLECENIA

```
New-AzFirewallThreatIntelWhitelist [-FQDN <String[]>] [-IpAddress <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **New-AzFirewallThreatIntelWhitelist** powoduje utworzenie zagrożenia dla obiektu Intel whitelist, którego można użyć podczas tworzenia lub konfigurowania zapory systemu Azure.

## Przykłady

### Przykład 1
```powershell
PS C:\> New-AzFirewallThreatIntelWhitelist -IpAddress @("2.2.2.2", "3.3.3.3") -FQDN @("bing.com", "yammer.com")
```

W tym przykładzie jest tworzone zagrożenie, które Intel whitelist zawiera nazwę FQDN whitelist dwóch wpisów oraz adres IP whitelist z dwóch wpisów.
