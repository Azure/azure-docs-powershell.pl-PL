# Update-AzVirtualRouter

## STRESZCZENIe
Umożliwia zaktualizowanie routera wirtualnego. 

## POLECENIA

### VirtualRouterNameParameterSet (domyślny)
```
Update-AzVirtualRouter -ResourceGroupName <String> -RouterName <String> [-AllowBranchToBranchTraffic]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### VirtualRouterResourceIdParameterSet
```
Update-AzVirtualRouter [-AllowBranchToBranchTraffic] -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Umożliwia zaktualizowanie routera wirtualnego w celu włączenia lub wyłączenia gałęzi do ruchu w oddziale.

## Przykłady

### Przykład 1
```powershell
PS C:\> Update-AzVirtualRouter -ResourceGroupName $rgname -RouterName $virtualRouterName -AllowBranchToBranchTraffic
```

Umożliwia zaktualizowanie routera wirtualnego w celu zezwolenia na odgałęzienie do ruchu w oddziale

### Przykład 2
```powershell
PS C:\> Update-AzVirtualRouter -ResourceGroupName $rgname -RouterName $virtualRouterName
```

Umożliwia zaktualizowanie routera wirtualnego w celu zablokowania ruchu odgałęzienia
