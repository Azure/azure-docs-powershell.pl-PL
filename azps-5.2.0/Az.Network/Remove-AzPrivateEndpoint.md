# Remove-AzPrivateEndpoint

## STRESZCZENIe
Umożliwia usunięcie prywatnego punktu końcowego.

## POLECENIA

```
Remove-AzPrivateEndpoint -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzPrivateEndpoint** umożliwia usunięcie prywatnego punktu końcowego. 

## Przykłady

### Przykład 1
```
Remove-AzPrivateEndpoint -Name MyPrivateEndpoint1 -ResourceGroupName TestResourceGroup
```

W tym przykładzie jest usuwany prywatny punkt końcowy o nazwie MyPrivateEndpoint1.
