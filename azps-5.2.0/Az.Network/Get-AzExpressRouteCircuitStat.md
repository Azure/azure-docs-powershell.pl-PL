# Get-AzExpressRouteCircuitStat

## STRESZCZENIe
Pobiera statystykę użycia obwodu ExpressRoute.

## POLECENIA

```
Get-AzExpressRouteCircuitStat -ResourceGroupName <String> -ExpressRouteCircuitName <String>
 [-PeeringType <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzExpressRouteCircuitStat** pobiera statystykę ruchu dla obwodu ExpressRoute. Statystyka obejmuje liczbę bajtów wysłanych i odebranych w ramach tras podstawowych i pomocniczych.

## Przykłady

### Przykład 1: wyświetlanie statystyki ruchu dla elementu równorzędnego ExpressRoute
```
Get-AzExpressRouteCircuitStat -ResourceGroupName $RG -ExpressRouteCircuitName $CircuitName -PeeringType 'AzurePrivatePeering'
```
