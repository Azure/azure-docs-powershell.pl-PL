# New-AzP2sVpnGateway

## STRESZCZENIe
Utwórz nową P2SVpnGateway w obszarze VirtualHub, aby wskazać łączność witryny.

## Opis
Polecenie cmdlet **New-AzP2sVpnGateway** umożliwia utworzenie nowego P2SVpnGateway w obszarze VirtualHub na potrzeby punktu połączenia z witryną od klientów do usługi Azure VirtualWan.
