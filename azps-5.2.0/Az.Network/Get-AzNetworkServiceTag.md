# Get-AzNetworkServiceTag

## STRESZCZENIe
Pobiera listę zasobów informacji o znacznikach usług.

## POLECENIA

```
Get-AzNetworkServiceTag -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzNetworkServiceTag** pobiera listę zasobów informacji o znacznikach usług.

Pamiętaj, że określone informacje o regionie platformy Azure będą używane jako odwołania dla wersji (nie jako filtru opartego na lokalizacji). Na przykład, nawet jeśli określisz, `-Location eastus2` zostanie wykorzystana lista numerów seryjnych usług ze prefiksem we wszystkich regionach, które będą ograniczone do chmury, do której należy Twój abonament (tzn. Public, USA, Chiny lub Niemcy).
