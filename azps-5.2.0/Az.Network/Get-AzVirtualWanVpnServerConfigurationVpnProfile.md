# Get-AzVirtualWanVpnServerConfigurationVpnProfile

## STRESZCZENIe
Generuje i pobiera profil sieci VPN na poziomie VirtualWan-VpnServerConfiguration, aby wskazać konfigurację klienta witryny.

## POLECENIA

```
 -VpnServerConfiguration <PSVpnServerConfiguration> [-AuthenticationMethod <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzVirtualWanVpnServerConfigurationVpnProfile** umożliwia generowanie i pobieranie profilu sieci VPN na poziomie VirtualWan-VpnServerConfiguration, aby wskazać konfigurację klienta witryny. Jest to wymagane do prowadzenia łączności z witryną od punktu klienckiego lokacji do usługi Azure P2SVpnGateway.

## Przykłady

### Przykład 1
```powershell
PS C:\> Get-AzVirtualWanVpnServerConfigurationVpnProfile -Name WestUsVirtualWan -ResourceGroupName P2SCortexGATesting -VpnServerConfiguration $vpnServerConfig -AuthenticationMethod EAPTLS

             qsEAbWAcYiKfUHFxZzh2hwvtb49dfVtUDA%3D&st=2019-10-25T19%3A52%3A36Z&se=2019-10-25T20%3A52%3A36Z&sp=r&fileExtension=.zip
```

Powyższe polecenie wygeneruje i zwróci adres URL SAS dla klienta w celu pobrania profilu sieci VPN na poziomie VirtualWan-VpnServerConfiguration, aby wskazać konfigurację klienta witryny.
