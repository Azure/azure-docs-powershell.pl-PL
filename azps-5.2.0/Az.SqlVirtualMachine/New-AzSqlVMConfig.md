# New-AzSqlVMConfig

## STRESZCZENIe
Tworzy nową konfigurację dla maszyny wirtualnej SQL.

## POLECENIA

```
New-AzSqlVMConfig [-LicenseType] <String> [-Offer <String>] [-Sku <String>] [-SqlManagementType <String>]
 [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet New-AzSqlVMConfig tworzy nowy obiekt konfiguracji dla maszyny wirtualnej SQL.

## Przykłady

### Przykład 1
```powershell
PS C:\> $config = New-AzSqlVMConfig -LicenseType "PAYG"
PS C:\> New-AzSqlVM -ResourceGroupName "ResourceGroup01" -Name "vm" -SqlVM $config
Name ResourceGroupName  LicenseType Sku       Offer          SqlManagementType
---- -----------------  ----------- ---       -----          -----------------
vm   ResourceGroup01    PAYG        Developer SQL2017-WS2016 Full
```

Tworzy lokalny obiekt konfigurowalny maszyny wirtualnej SQL, którego można użyć w celu utworzenia maszyny wirtualnej Azure SQL.
