# Get-AzHpcCacheSku

## STRESZCZENIe
Pobiera wszystkie jednostki SKU dostępne w ramach abonamentu.

## POLECENIA

```
Get-AzHpcCacheSku [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzHpcCacheSku** zwraca listę jednostek SKU dostępnych w subskrypcji.

## Przykłady

### Przykład 1
```powershell
PS C:\> Get-AzHpcCacheSku
```
