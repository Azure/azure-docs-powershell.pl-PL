# Remove-AzBatchAccount

## STRESZCZENIe
Umożliwia usunięcie konta wsadowego.

## POLECENIA

```
Remove-AzBatchAccount [-AccountName] <String> [[-ResourceGroupName] <String>] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzBatchAccount** usuwa konto usługi Azure Batch.
To polecenie cmdlet wyświetla monit przed usunięciem konta, chyba że zostanie określony parametr *Force* .

## Przykłady

### Przykład 1. Usuń konto wsadowe
```
PS C:\>Remove-AzBatchAccount -AccountName "pfuller"
```

To polecenie usuwa konto wsadowe o nazwie pfuller.
To polecenie monituje o potwierdzenie przed usunięciem konta.
