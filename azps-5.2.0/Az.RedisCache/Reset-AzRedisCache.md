# Reset-AzRedisCache

## STRESZCZENIe
Ponownie uruchamia węzły pamięci podręcznej.

## POLECENIA

```
Reset-AzRedisCache [-ResourceGroupName <String>] -Name <String> -RebootType <String> [-ShardId <Int32>]
 [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Reset-AzRedisCache** powoduje ponowne uruchomienie węzłów wystąpienia pamięci podręcznej platformy Azure Redis.

## Przykłady

### Przykład 1. Uruchom ponownie oba węzły
```
PS C:\>Reset-AzRedisCache -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -RebootType "AllNodes" -Force
```

To polecenie powoduje ponowne uruchomienie obu węzłów w pamięci podręcznej o nazwie RedisCache06.
