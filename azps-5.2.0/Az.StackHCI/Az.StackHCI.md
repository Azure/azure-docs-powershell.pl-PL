# AZ. StackHCI, moduł
## Opis
Microsoft Azure PowerShell: polecenia cmdlet StackHCI

## AZ. StackHCI polecenia cmdlet
### [Rejestr — AzStackHCI](Register-AzStackHCI.md)
Register-AzStackHCI tworzy zasób usługi Microsoft. AzureStackHCI w chmurze reprezentujący klaster lokalny i rejestruje klaster lokalny na platformie Azure.

### [Test — AzStackHCIConnection](Test-AzStackHCIConnection.md)
Test-AzStackHCIConnection weryfikuje łączność z lokalnych węzłów klastrowanych z usługami platformy Azure wymaganymi przez stos HCL stosu platformy Azure.

### [Wyrejestrowanie — AzStackHCI](Unregister-AzStackHCI.md)
Unregister-AzStackHCI usuwa zasób usługi Microsoft. AzureStackHCI w chmurze reprezentujący klaster lokalny i wyrejestrowuje klaster lokalny na platformie Azure.
Zarejestrowane informacje dostępne w klastrze są używane do wyrejestrowywania klastra, jeśli nie są przekazywane żadne parametry.
