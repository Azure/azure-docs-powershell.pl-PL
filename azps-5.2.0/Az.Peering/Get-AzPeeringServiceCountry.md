# Get-AzPeeringServiceCountry

## STRESZCZENIe
Zawiera listę krajów, w których można korzystać z usługi peer.

## POLECENIA

```
Get-AzPeeringServiceCountry [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Zawiera listę krajów, w których można korzystać z usługi peer.

## Przykłady

### Przykład 1
```powershell
PS C:\> Get-AzPeeringServiceCountry
```

Lista krajów dostępnych dla usługi peer.
