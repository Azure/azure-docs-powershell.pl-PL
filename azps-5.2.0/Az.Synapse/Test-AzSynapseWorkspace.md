# Test-AzSynapseWorkspace

## STRESZCZENIe
Sprawdza, czy istnieje obszar roboczy analizy Synapse.

## POLECENIA

```
Test-AzSynapseWorkspace [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **test-AzSynapseWorkspace** sprawdza obecność obszaru roboczego analizy Synapse.

## Przykłady

### Przykład 1
```powershell
PS C:\> Test-AzSynapseWorkspace -Name ContosoWorkspace
```

To polecenie sprawdza, czy istnieje obszar roboczy analizy Synapse.
