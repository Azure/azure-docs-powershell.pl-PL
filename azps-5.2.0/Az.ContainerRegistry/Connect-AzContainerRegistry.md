# Connect-AzContainerRegistry

## STRESZCZENIe
Zaloguj się do rejestru kontenera platformy Azure.

## POLECENIA

### WithoutNameAndPasswordParameterSet (domyślny)
```
Connect-AzContainerRegistry -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### WithNameAndPasswordParameterSet
```
Connect-AzContainerRegistry -Name <String> -UserName <String> -Password <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Zaloguj się do rejestru kontenera platformy Azure.

## Przykłady

### Przykład 1
```powershell
PS C:\> Connect-AzContainerRegistry -Name $RegistryName
```

Zaloguj się do ACR bez poświadczeń, gdy zalogujesz się już do konta platformy Azure.

### Przykład 2
```powershell
PS C:\> Connect-AzContainerRegistry -Name $RegistryName -UserName $RegistryName -Password $AdminPassWord
```

Zaloguj się do ACR przy użyciu nazwy użytkownika/hasła administratora, gdy włączono administratora.

### Przykład 3
```powershell
PS C:\> Connect-AzContainerRegistry -Name $RegistryName -UserName $ServicePrincipal -Password $ServicePrincipalPassword
```

Zaloguj się do ACR przy użyciu identyfikatora aplikacji głównej usługi i hasła.
