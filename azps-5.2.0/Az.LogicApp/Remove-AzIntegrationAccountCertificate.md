# Remove-AzIntegrationAccountCertificate

## STRESZCZENIe
Usuwa certyfikat konta integracyjnego z grupy zasobów.

## POLECENIA

```
Remove-AzIntegrationAccountCertificate -ResourceGroupName <String> -Name <String> -CertificateName <String>
 [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzIntegrationAccountCertificate** usuwa certyfikat konta integracyjnego z grupy zasobów.
Określ nazwę konta integracji, nazwę grupy zasobów i nazwę certyfikatu.
Ten moduł obsługuje parametry dynamiczne.
Aby użyć parametru dynamicznego, wpisz go w poleceniu.
Aby wykryć nazwy parametrów dynamicznych, wpisz znak łącznika (-) po nazwie polecenia cmdlet, a następnie naciskaj klawisz Tab, aby przechodzić między dostępnymi parametrami.
W przypadku pominięcia wymaganego parametru szablonu polecenie cmdlet monituje o podanie wartości.

## Przykłady

### Przykład 1: Usuwanie certyfikatu konta integracji
```
PS C:\>Remove-AzIntegrationAccountCertificate -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -CertificateName "IntegrationAccountCertificate01"
```

To polecenie usuwa certyfikat konta integracyjnego o nazwie IntegrationAccount31.
