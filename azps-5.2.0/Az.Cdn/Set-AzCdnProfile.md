# Set-AzCdnProfile

## STRESZCZENIe
Aktualizuje profil sieci CDN.

## POLECENIA

```
Set-AzCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzCdnProfile** aktualizuje profil sieci dostarczania zawartości Azure (CDN, Content Delivery Network).
