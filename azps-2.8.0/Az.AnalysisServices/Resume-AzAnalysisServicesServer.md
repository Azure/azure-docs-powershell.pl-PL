# Resume-AzAnalysisServicesServer

## STRESZCZENIe
Wznawia działanie wystąpienia serwera usług Analysis Services

## POLECENIA

```
Resume-AzAnalysisServicesServer [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Resume-AzAnalysisServicesServer wznawia działanie wystąpienia serwera usług Analysis Services

## Przykłady

### Przykład 1
```
PS C:\> Resume-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

To polecenie spowoduje wznowienie wstrzymanego serwera o nazwie TestServer w ramach testera zasobów.
