# Clear-AzSqlServerAdvancedThreatProtectionSettings

## STRESZCZENIe
Usuwa zaawansowane ustawienia ochrony przed zagrożeniami z serwera.

## POLECENIA

```
Clear-AzSqlServerAdvancedThreatProtectionSettings [-PassThru] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet Clear-AzSqlServerAdvancedThreatProtectionSettings usuwa ustawienia zaawansowanego zabezpieczenia przed zagrożeniami z programu Azure SQL Server.
Aby użyć tego polecenia cmdlet, określ parametry ResourceGroupName oraz nazwa_serwera w celu zidentyfikowania serwera, z którego to polecenie cmdlet usunie ustawienia.

## Przykłady

### Przykład 1: usuwanie zaawansowanych ustawień ochrony przed zagrożeniami dla bazy danych
```
PS C:\> Clear-AzSqlServerAdvancedThreatProtectionSettings -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
```

To polecenie usuwa zaawansowane ustawienia ochrony przed zagrożeniami z serwera o nazwie Server01.
