# Set-AzSqlElasticPoolRecommendedActionState

## STRESZCZENIe
Aktualizuje stan zalecanej akcji puli elastycznej w usłudze Azure SQL.

## POLECENIA

```
Set-AzSqlElasticPoolRecommendedActionState -RecommendedActionName <String> -State <RecommendedActionState>
 -ServerName <String> -ElasticPoolName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Stan aktualizacji polecenia cmdlet **Set-AzSqlElasticPoolRecommendedActionState** w przypadku zalecanej akcji puli elastycznej w usłudze Azure SQL.
To polecenie cmdlet powoduje zastosowanie zalecanej akcji, która została przywrócona lub odrzucona na podstawie nowego stanu.
