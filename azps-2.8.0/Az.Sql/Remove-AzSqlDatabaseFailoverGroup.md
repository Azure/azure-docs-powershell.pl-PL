# Remove-AzSqlDatabaseFailoverGroup

## STRESZCZENIe
Usuwa grupę trybu failover bazy danych SQL Azure.

## POLECENIA

```
Remove-AzSqlDatabaseFailoverGroup [-ServerName] <String> [-FailoverGroupName] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
To polecenie usuwa grupę pracy awaryjnej o określonej nazwie, pozostawiając wszystkie bazy danych i relacje replikacji bez zmian. Punkt końcowy odbiornika zostanie wyrejestrowany z systemu DNS.
Aby wykonać polecenie, należy użyć głównego serwera grupy pracy awaryjnej.

## Przykłady

### Przykład 1
```
PS C:\> Remove-AzSqlDatabaseFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg
```

Usuwanie grupy pracy awaryjnej.
