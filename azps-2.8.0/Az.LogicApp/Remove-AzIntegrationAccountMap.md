# Remove-AzIntegrationAccountMap

## STRESZCZENIe
Usuwa mapę konta integracji.

## POLECENIA

```
Remove-AzIntegrationAccountMap -ResourceGroupName <String> -Name <String> -MapName <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzIntegrationAccountMap** umożliwia usunięcie mapy konta integracji z grupy zasobów.
Określ nazwę konta integracji, nazwę grupy zasobów i nazwę mapy.
Ten moduł obsługuje parametry dynamiczne.
Aby użyć parametru dynamicznego, wpisz go w poleceniu.
Aby wykryć nazwy parametrów dynamicznych, wpisz znak łącznika (-) po nazwie polecenia cmdlet, a następnie naciskaj klawisz Tab, aby przechodzić między dostępnymi parametrami.
W przypadku pominięcia wymaganego parametru szablonu polecenie cmdlet monituje o podanie wartości.

## Przykłady

### Przykład 1: usuwanie mapy konta integracji
```
PS C:\>Remove-AzIntegrationAccountMap -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -MapName "IntegrationAccountMap47"
```

To polecenie powoduje usunięcie mapy kont integracji o nazwie IntegrationAccountMap47.
