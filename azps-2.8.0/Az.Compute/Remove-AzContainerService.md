# Remove-AzContainerService

## STRESZCZENIe
Usuwa usługę kontenera.

## POLECENIA

```
Remove-AzContainerService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzContainerService** usuwa usługę kontenera z konta platformy Azure.

## Przykłady

### Przykład 1: Usuwanie usługi kontenera
```
PS C:\> Remove-AzContainerService -ResourceGroupName "ResourceGroup17" -Name "CSResourceGroup17"
```

To polecenie usuwa usługę kontenera o nazwie CSResourceGroup17.
