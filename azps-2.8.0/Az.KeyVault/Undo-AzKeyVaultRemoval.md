# Undo-AzKeyVaultRemoval

## STRESZCZENIe
Odkrycie usuniętego magazynu kluczy w stanie aktywnym.

## POLECENIA

### Domyślne (domyślnie)
```
Undo-AzKeyVaultRemoval [-VaultName] <String> [-ResourceGroupName] <String> [-Location] <String>
 [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Inputobject
```
Undo-AzKeyVaultRemoval [-InputObject] <PSDeletedKeyVault> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Undo-AzKeyVaultRemoval** powoduje odzyskanie uprzednio usuniętego magazynu kluczy. Odzyskany magazyn będzie aktywny po odzyskaniu
