# Remove-AzWebAppAccessRestrictionRule

## STRESZCZENIe
Usuwa regułę ograniczeń dostępu z aplikacji Azure Web App.

## POLECENIA

```
Remove-AzWebAppAccessRestrictionRule [-ResourceGroupName] <String> [-WebAppName] <String> -Name <String>
 [-TargetScmSite] [-SlotName <String>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzWebAppAccessRestrictionRule** usuwa regułę ograniczeń programu Access z aplikacji sieci Web platformy Azure

## Przykłady

### Przykład 1: Usuwanie reguły ograniczenia dostępu aplikacji sieci Web
```
PS C:\>Remove-AzWebAppAccessRestrictionRule -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" -Name IpRule
```

To polecenie usuwa regułę ograniczenia dostępu IpRule z aplikacji Azure Web App o nazwie ContosoSite należącej do grupy zasobów o nazwie Default-Web-Zachodnia.
