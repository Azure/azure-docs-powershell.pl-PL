# Remove-AzDdosProtectionPlan

## STRESZCZENIe
Usuwa plan ochrony DDoS.

## POLECENIA

```
Remove-AzDdosProtectionPlan -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Remove-AzDdosProtectionPlan usuwa plan ochrony DDoS.

## Przykłady

### Przykład 1: Usuwanie pustego planu ochrony DDoS
```
D:\> Remove-AzDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlan
```

W tym przypadku firma Microsoft usunie plan ochrony DDoS, jak określono.

### Przykład 2: usuwanie planu ochrony DDoS skojarzonego z siecią wirtualną
```
D:\> $vnet = Get-AzVirtualNetwork -Name VnetName -ResourceGroupName ResourceGroupName
D:\> $vnet.DdosProtectionPlan = $null
```

Następnie utrzymujemy nowy stan, przełączając zmienną lokalną do **Set-AzVirtualNetwork**. W tym momencie plan nie jest już skojarzony z siecią wirtualną.
Jeśli jest to Ostatnia skojarzona z planem, możemy usunąć plan ochrony DDoS przy użyciu polecenia Remove-AzDdosProtectionPlan.
