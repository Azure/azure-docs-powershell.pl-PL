# Remove-AzStorageQueue

## STRESZCZENIe
Usuwa kolejkę magazynu.

## POLECENIA

```
Remove-AzStorageQueue [-Name] <String> [-Force] [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzStorageQueue** umożliwia usunięcie kolejki magazynu.

## Przykłady

### Przykład 1: usuwanie kolejki magazynu według nazwy
```
PS C:\>Remove-AzStorageQueue "ContosoQueue01"
```

To polecenie umożliwia usunięcie kolejki o nazwie ContosoQueue01.

### Przykład 2: usuwanie wielu kolejek magazynowania
```
PS C:\>Get-AzStorageQueue "Contoso*" | Remove-AzStorageQueue
```

To polecenie usuwa wszystkie kolejki o nazwach rozpoczynających się od contoso.
