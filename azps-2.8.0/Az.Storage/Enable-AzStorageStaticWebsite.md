# Enable-AzStorageStaticWebsite

## STRESZCZENIe
Włącz statyczną witrynę sieci Web dla konta usługi Azure Storage.

## POLECENIA

```
Enable-AzStorageStaticWebsite [[-IndexDocument] <String>] [[-ErrorDocument404Path] <String>] [-PassThru]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **enable-AzStorageStaticWebsite** włącza statyczną witrynę sieci Web dla konta usługi Azure Storage.

## Przykłady

### Przykład 1: Włączanie statycznej witryny sieci Web dla konta usługi Azure Storage
```
C:\PS>Enable-AzStorageStaticWebsite -IndexDocument $indexdoc -ErrorDocument404Path $errordoc
```

To polecenie umożliwia włączenie statycznej witryny sieci Web dla konta usługi Azure Storage.
