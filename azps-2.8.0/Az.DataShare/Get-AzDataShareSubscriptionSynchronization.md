# Get-AzDataShareSubscriptionSynchronization

## STRESZCZENIe
Pobiera informacje o synchronizacji są uruchamiane w ramach akcji Udostępnij.

## POLECENIA

### ByFieldsParameterSet (domyślny)
```
Get-AzDataShareSubscriptionSynchronization -ResourceGroupName <String> -AccountName <String>
 -ShareSubscriptionName <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzDataShareSubscriptionSynchronization -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzDataShareSubscriptionSynchronization** zawiera informaiton o wszystkich synchronizacjach w subskrypcji udostępnionej w ramach konta udziału danych w danym klienta.

## Przykłady

### Przykład 1
```
PS C:\> Get-AzDataShareSubscriptionSynchronization -ResourceGroupName "ADS" -AccountName "WikiAds"  -ShareSubscriptionName "AdsShareSubscription"

durationMs        : 83660
endTime           : 7/10/2019 9:01:23 AM
message           :
startTime         : 7/10/2019 9:00:04 AM
status            : Succeeded
synchronizationId : b087e1a5-9144-4e1d-86d1-2a4adcf551d4
```

To polecenie udostępnia informacje o wszystkich synchronizacjach dla subskrypcji udziału AdsShareSubscription w obszarze WikiAds konta udostępniania danych.
