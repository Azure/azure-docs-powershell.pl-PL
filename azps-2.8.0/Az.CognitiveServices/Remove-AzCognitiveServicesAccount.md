# Remove-AzCognitiveServicesAccount

## STRESZCZENIe
Usuwa konto usług poznawczych.

## POLECENIA

```
Remove-AzCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzCognitiveServicesAccount** umożliwia usunięcie określonego konta usług poznawczych.

## Przykłady

### Przykład 1
To polecenie nie zwraca żadnych zmian.

```powershell
PS C:\> Remove-AzCognitiveServicesAccount -ResourceGroupName cognitive-services-resource-group -name myluis
PS C:\>
```
