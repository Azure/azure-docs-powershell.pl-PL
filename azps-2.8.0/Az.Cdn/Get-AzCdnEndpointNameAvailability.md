# Get-AzCdnEndpointNameAvailability

## STRESZCZENIe
Pobiera stan dostępności punktu końcowego sieci CDN.

## POLECENIA

```
Get-AzCdnEndpointNameAvailability -EndpointName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzCdnEndpointNameAvailability** Pobiera stan dostępności punktu końcowego sieci dostarczania zawartości Azure (CDN, Content Delivery Network).
