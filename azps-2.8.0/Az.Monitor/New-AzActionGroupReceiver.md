# New-AzActionGroupReceiver

## STRESZCZENIe
Tworzy nowy odbiorcę grupy akcji.

## POLECENIA

```
New-AzActionGroupReceiver -Name <String> [-UseCommonAlertSchema] [-EmailReceiver] -EmailAddress <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **New-AzActionGroupReceiver** umożliwia utworzenie nowego adresata grupy akcji w pamięci.

## Przykłady

### Przykład 1: Utwórz nowy odbiornik poczty E-mail w pamięci.
```
PS C:\>$emailReceiver = New-AzActionGroupReceiver -Name 'emailReceiver1' -EmailReceiver -EmailAddress 'user1@example.com'
```

To polecenie tworzy nowy odbiornik poczty E-mail w pamięci.

### Przykład 2: Utwórz nowy odbiornik SMS w pamięci.
```
PS C:\>$smsReceiver = New-AzActionGroupReceiver -Name 'smsReceiver1' -SmsReceiver -CountryCode '1' -PhoneNumber '5555555555'
```

To polecenie tworzy nowy odbiornik SMS w pamięci.

### Przykład 3: Tworzenie nowego odbiornika webhook w pamięci.
```
PS C:\>$webhookReceiver = New-AzActionGroupReceiver -Name 'webhookReceiver1' -WebhookReceiver -ServiceUri 'http://test.com'
```

To polecenie tworzy nowy odbiornik elementu webhook w pamięci.
