# Enable-AzHDInsightOperationsManagementSuite

## STRESZCZENIe
Włącza Pakiet Operations Management Suite (OMS) w klastrze usługi HDInsight i odpowiednie dzienniki zostaną wysłane do obszaru roboczego OMS określonego podczas włączania.

## POLECENIA

```
Enable-AzHDInsightOperationsManagementSuite [-Name] <String> [-WorkspaceId] <String> [-PrimaryKey] <String>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **enable-AzHDInsightOperationsManagementSuite** włącza Pakiet Operations Management Suite (OMS) w klastrze usługi Azure HDInsight.

## Przykłady

### Przykład 1
```
PS C:\> Enable-AzHDInsightOMS -Name testcluster -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

ErrorInfo  :

State      : Succeeded

RequestId  : 1417ad86-d055-48cd-9d68-a5c19a212a3a

StatusCode : OK
```

Pakiet Operations Management Suite (OMS) zostanie włączony w klastrze usługi HDInsight, a odpowiednie dzienniki zostaną wysłane do obszaru roboczego OMS o identyfikatorze 1d364e89-bb71-4503-aa3d-a23535aea7bd

### Przykład 2
```
PS C:\> Enable-AzHDInsightOMS -Name testcluster -ResourceGroupName testrg -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

ErrorInfo  :

State      : Succeeded

RequestId  : 1417ad86-d055-48cd-9d68-a5c19a212a3a

StatusCode : OK
```

Pakiet Operations Management Suite (OMS) zostanie włączony w klastrze usługi HDInsight, a odpowiednie dzienniki zostaną wysłane do obszaru roboczego OMS o identyfikatorze 1d364e89-bb71-4503-aa3d-a23535aea7bd
