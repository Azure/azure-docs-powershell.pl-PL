# New-AzServiceFabricApplicationType

## STRESZCZENIe
Utwórz nowy typ aplikacji sieci szkieletowej usług pod określoną grupą zasobów i klastrem.

## POLECENIA

```
New-AzServiceFabricApplicationType [-ResourceGroupName] <String> [-ClusterName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet tworzy nowy typ aplikacji sieci szkieletowej usług pod określoną grupą zasobów i klastrem.

## Przykłady

### Przykład 1
```powershell
PS C:\> $resourceGroupName = "testRG"
PS C:\> $clusterName = "testCluster"
PS C:\> $appTypeName = "testAppType"
PS C:\> $appType = New-AzServiceFabricApplicationType -ResourceGroupName $resourceGroupName -ClusterName $clusterName -Name $appTypeName -Verbose
```

W tym przykładzie zostanie utworzony nowy typ aplikacji "testAppType" w obszarze Grupa zasobów i klaster.
