# Remove-AzureRmPolicySetDefinition

## STRESZCZENIe
Usuwa definicję zestawu zasad.

## Opis
Polecenie cmdlet **Remove-AzureRmPolicySetDefinition** usuwa definicję zasad.

## Przykłady

### Przykład 1: usuwanie definicji zestawu zasad według identyfikatora zasobu
```
PS C:\> $PolicySetDefinition = Get-AzureRmPolicySetDefinition -ResourceId '/subscriptions/mySub/Microsoft.Authorization/policySetDefinitions/myPSSetDefinition'
PS C:\> Remove-AzureRmPolicySetDefinition -Id $PolicySetDefinition.ResourceId -Force
```

Pierwsze polecenie pobiera definicję zestawu zasad za pomocą polecenia cmdlet Get-AzureRmPolicySetDefinition.
Polecenie zapisuje je w zmiennej $PolicySetDefinition.
Drugie polecenie usuwa definicję zestawu zasad określoną przez właściwość **ResourceId** $PolicySetDefinition.
