# Get-AzureRmNetworkUsage

## STRESZCZENIe
Wyświetlanie użycia sieci dla subskrypcji

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Get-AzureRmNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Get-AzureRmNetworkUsage pobiera limity i bieżące użycie zasobów sieciowych.
