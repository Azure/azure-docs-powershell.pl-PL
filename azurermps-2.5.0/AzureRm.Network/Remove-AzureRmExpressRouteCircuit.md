# Remove-AzureRmExpressRouteCircuit

## STRESZCZENIe
Usuwa obwód ExpressRoute.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmExpressRouteCircuit -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmExpressRouteCircuit** usuwa obwód ExpressRoute.

## Przykłady

### Przykład 1: usuwanie obwodu ExpressRoute
```
Remove-AzureRmExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg
```

### Przykład 2: usuwanie obwodu ExpressRoute przy użyciu rurociągu
```
Get-AzureRmExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg | Remove-AzureRmExpressRouteCircuit
```
