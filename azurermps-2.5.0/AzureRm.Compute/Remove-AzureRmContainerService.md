# Remove-AzureRmContainerService

## STRESZCZENIe
Usuwa usługę kontenera.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## POLECENIA

```
Remove-AzureRmContainerService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzureRmContainerService** usuwa usługę kontenera z konta platformy Azure.

## Przykłady

### Przykład 1: Usuwanie usługi kontenera
```
PS C:\> Remove-AzureRmContainerService -ResourceGroupName "ResourceGroup17" -Name "CSResourceGroup17"
```

To polecenie usuwa usługę kontenera o nazwie CSResourceGroup17.
