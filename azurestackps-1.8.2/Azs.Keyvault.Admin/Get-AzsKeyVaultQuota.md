## STRESZCZENIe

## POLECENIA

```
Get-AzsKeyVaultQuota [[-Location] <String>] [<CommonParameters>]
```

## Opis
Wyświetlanie listy wszystkich obiektów przydziałów dla magazynu obiektów w lokalizacji.

## Przykłady
