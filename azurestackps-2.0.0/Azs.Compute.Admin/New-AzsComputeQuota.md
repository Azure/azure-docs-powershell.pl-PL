# New-AzsComputeQuota

## STRESZCZENIe
Umożliwia utworzenie lub zaktualizowanie przydziału obliczanego przy użyciu podanych parametrów przydziałów.

## POLECENIA

### Rozszerzenie (domyślne)
```
New-AzsComputeQuota -Name <String> [-Location <String>] [-SubscriptionId <String>]
 [-AvailabilitySetCount <Int32>] [-CoresCount <Int32>] [-Location1 <String>]
 [-PremiumManagedDiskAndSnapshotSize <Int32>] [-StandardManagedDiskAndSnapshotSize <Int32>]
 [-VirtualMachineCount <Int32>] [-VMScaleSetCount <Int32>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### Tworzonych
```
New-AzsComputeQuota -Name <String> -NewQuota <IQuota> [-Location <String>] [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Opis
Umożliwia utworzenie lub zaktualizowanie przydziału obliczanego przy użyciu podanych parametrów przydziałów.

## Przykłady

### Przykład 1. Tworzenie przydziału obliczanego z parametrami domyślnymi

```powershell
PS C:\> New-AzsComputeQuota -Name ExampleComputeQuotaWithDefaultParameters

                                     min/locations/local/quotas/ExampleComputeQuotaWithDefaultParameters
```

Wszystkie parametry, które nie zostały określone, zostaną ustawione na ich domyślny parametr, pokazane powyżej.


Dostosowywanie przydziału za pomocą parametrów. Wszystkie parametry, które nie zostały określone, będą miały wartość domyślną.
