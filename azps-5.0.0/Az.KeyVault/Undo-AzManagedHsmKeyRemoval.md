# Undo-AzManagedHsmKeyRemoval

## STRESZCZENIe
Przywraca usunięty klucz z zarządzanego modułu HSM do stanu aktywnego.

## POLECENIA

### Domyślne (domyślnie)
```
Undo-AzManagedHsmKeyRemoval [-HsmName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Inputobject
```
Undo-AzManagedHsmKeyRemoval [-InputObject] <PSDeletedKeyVaultKeyIdentityItem>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Undo-AzManagedHsmKeyRemoval** powoduje odzyskanie uprzednio usuniętego klucza.
Odzyskany klucz jest aktywny i będzie można go użyć do wszystkich zwykłych operacji kluczowych.
Aby można było wykonać tę operację, osoba dzwoniąca musi mieć uprawnienie "Odzyskaj".

## Przykłady

### Przykład 1
```powershell
PS C:\> Undo-AzManagedHsmKeyRemoval -HsmName testmhsm -Name testkey001

Vault/HSM Name : testmhsm
Name           : testkey001
Version        : 7cff8510da04433b98144a3e33ad2bae
Id             : https://testmhsm.managedhsm.azure.net:443/keys/testkey001/7cff8510da04433b98144a3e33ad2bae
Enabled        : True
Expires        :
Not Before     :
Created        : 10/14/2020 10:13:03 AM
Updated        : 10/14/2020 10:13:03 AM
Recovery Level : Recoverable+Purgeable
Tags           :
```

To polecenie odzyska klucz "testkey001", który został wcześniej usunięty, w stanie aktywnym i zdatnym do użycia.
