# Disable-AzHDInsightMonitoring

## STRESZCZENIe
Wyłącza monitorowanie w klastrze usługi HDInsight, a odpowiednie dzienniki będą przelewane do obszaru roboczego monitorowania określonego podczas włączania.

## POLECENIA

```
Disable-AzHDInsightMonitoring [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **disable-AzHDInsightMonitoring** wyłącza monitorowanie w klastrze usługi Azure HDInsight.

## Przykłady

### Przykład 1
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster

True
```

Monitorowanie zostanie wyłączone w klastrze usługi HDInsight, a odpowiednie dzienniki przestaną przepływ na obszar roboczy monitorowania.

### Przykład 2
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg

True
```

Monitorowanie zostanie wyłączone w klastrze usługi HDInsight, a odpowiednie dzienniki przestaną przepływ na obszar roboczy monitorowania.
