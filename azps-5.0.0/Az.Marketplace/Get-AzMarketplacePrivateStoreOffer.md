# Get-AzMarketplacePrivateStoreOffer

## STRESZCZENIe
Uzyskaj jeden lub więcej oferty prywatnego sklepu.

## POLECENIA

```
Get-AzMarketplacePrivateStoreOffer -PrivateStoreId <String> [-OfferId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Uzyskaj jeden lub więcej oferty prywatnego sklepu za pomocą planów publicznych + prywatnych, które zostały dodane w ramach zakresu dzierżawy. Jeśli jest wyświetlany identyfikator subskrypcji, uzyskaj jeden lub więcej ofert w magazynie prywatnym z planami prywatnymi tylko w zakresie abonamentu.
