# Get-AzBatchApplicationPackage

## STRESZCZENIe
Pobiera informacje o pakiecie aplikacji na koncie wsadowym.

## POLECENIA

```
Get-AzBatchApplicationPackage [-AccountName] <String> [-ResourceGroupName] <String> [-ApplicationName] <String>
 [[-ApplicationVersion] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzBatchApplicationPackage** pobiera informacje o pakiecie aplikacji na koncie usługi Azure Batch.

## Przykłady

### Przykład 1: uzyskiwanie szczegółowych informacji o pakiecie aplikacji na koncie wsadowym
```
PS C:\>Get-AzBatchApplicationPackage -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationName "Litware" -ApplicationVersion "1.0"
Format             : zip
State              : Active
Version            : 1.0
LastActivationTime : 13/05/2016 4:03:24 AM
StorageUrl         : https://contosobatch.blob.core.windows.net/app-test
StorageUrlExpiry   : 13/05/2016 8:04:44 AM
Id                 : litware
```

To polecenie pobiera szczegóły wersji 1,0 pakietu przykładowa.
