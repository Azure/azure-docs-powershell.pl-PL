# Get-AzRedisCachePatchSchedule

## STRESZCZENIe
Pobiera harmonogram poprawek.

## POLECENIA

```
Get-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzRedisCachePatchSchedule** pobiera harmonogram poprawek pamięci podręcznej w pamięci podręcznej platformy Azure Redis.

## Przykłady

### Przykład 1: pobieranie harmonogramu poprawek
```
PS C:\>Get-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06"
```

To polecenie pobiera harmonogram poprawek z pamięci podręcznej o nazwie RedisCache06.
