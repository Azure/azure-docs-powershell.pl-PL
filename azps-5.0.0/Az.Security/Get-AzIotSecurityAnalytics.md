# Get-AzIotSecurityAnalytics

## STRESZCZENIe
Uzyskaj analiza zabezpieczeń IoT

## POLECENIA

```
Get-AzIotSecurityAnalytics -ResourceGroupName <String> -SolutionName <String> [-Default]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet Get-AzIotSecurityAnalytics zwraca zestaw funkcji analizy zabezpieczeń konkretnego rozwiązania dotyczącego zabezpieczeń IoT
