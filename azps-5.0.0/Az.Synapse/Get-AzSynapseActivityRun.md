# Get-AzSynapseActivityRun

## STRESZCZENIe
Pobiera informacje o działaniach dotyczących uruchamiania rurociągu.

## POLECENIA

### GetByName (domyślny)
```
Get-AzSynapseActivityRun -WorkspaceName <String> -PipelineName <String> -PipelineRunId <String>
 -RunStartedAfter <DateTimeOffset> -RunStartedBefore <DateTimeOffset> [-ActivityName <String>]
 [-Status <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseActivityRun -WorkspaceObject <PSSynapseWorkspace> -PipelineName <String> -PipelineRunId <String>
 -RunStartedAfter <DateTimeOffset> -RunStartedBefore <DateTimeOffset> [-ActivityName <String>]
 [-Status <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzSynapseActivityRun** pobiera informacje o działaniu w obszarze roboczym dla określonego przebiegu procesu, który wystąpił w danym przedziale czasu. Ponadto możesz określić filtry dla nazwy działania i stanu uruchomienia.

## Przykłady

### Przykład 1
```powershell
PS C:\> Get-AzSynapseActivityRun -WorkspaceName ContosoWorkspace -PipelineName ContosoPipeline -PipelineRunId "f288712d-fb08-4cb8-96ef-82d3b9b30621" -RunStartedAfter [DateTimeOffset]"2018-09-01T21:00" -RunStartedBefore [DateTimeOffset]"2018-09-30T21:00"
```

To polecenie pobiera szczegóły dotyczące wszystkich działań uruchomionych w potoku o nazwie ContosoPipeline Uruchom z IDENTYFIKATORem "f288712d-FB08-4cb8-96ef-82d3b9b30621", który wystąpił między "2018-09-01T21:00" i "2018-09-30T21:00".

### Przykład 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseActivityRun -PipelineName ContosoPipeline -PipelineRunId "f288712d-fb08-4cb8-96ef-82d3b9b30621" -RunStartedAfter [DateTimeOffset]"2018-09-01T21:00" -RunStartedBefore [DateTimeOffset]"2018-09-30T21:00"
```

To polecenie pobiera szczegóły dotyczące wszystkich działań wykonywanych w potoku o nazwie ContosoPipeline Uruchom z IDENTYFIKATORem "f288712d-FB08-4cb8-96ef-82d3b9b30621", który wystąpił między "2018-09-01T21:00" i "2018-09-30T21:00" do rurociągu.
