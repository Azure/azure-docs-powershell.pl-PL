# Remove-AzPowerBIWorkspaceCollection

## STRESZCZENIe
Usuwa kolekcję obszarów roboczych usługi Power BI.

## POLECENIA

```
Remove-AzPowerBIWorkspaceCollection [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzPowerBIWorkspaceCollection** usuwa kolekcję obszarów roboczych usługi Power BI z subskrypcji i grupy zasobów platformy Azure.

## Przykłady

### Przykład 1: usuwanie kolekcji obszarów roboczych
```
PS C:\>Remove-AzPowerBIWorkspaceCollection -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11"
```

To polecenie usuwa kolekcję obszarów roboczych o nazwie WCN11 w określonej grupie zasobów.
