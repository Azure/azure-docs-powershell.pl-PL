# New-AzPeeringRegisteredAsn

## STRESZCZENIe
Tworzenie zarejestrowanego konta ASN dla komunikacji równorzędnej

## POLECENIA

### ByResourceGroupAndName (domyślny)
```
New-AzPeeringRegisteredAsn [-ResourceGroupName] <String> [-PeeringName] <String> [-Name] <String> -Asn <Int32>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Inputobject
```
New-AzPeeringRegisteredAsn -InputObject <PSPeering> [-Name] <String> -Asn <Int32> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
New-AzPeeringRegisteredAsn [-ResourceId] <String> [-Name] <String> -Asn <Int32> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Utwórz zarejestrowane zawiadomienie o wysyłce dla obiektów równorzędnych.

## Przykłady

### Uzyskiwanie komunikacji równorzędnej i tworzenie zarejestrowanego konta ASN
```powershell
PS C:\>$peering = Get-AzPeering -ResourceGroupName $resourceGroupName -Name $name
PS C:\>$peering | New-AzPeeringRegisteredAsn -Name $asnName -Asn $asn
```

Uzyskaj dostęp do sieci równorzędnej, do której chcesz dodać zarejestrowany ASN. Następnie Przekaż to unifiedgroup.

### Tworzenie zarejestrowanego konta ASN za pomocą resourceId równorzędnego
```powershell
PS C:\>New-AzPeeringRegisteredAsn -ResourceId $resourceId -Name $asnName -Asn $asn
```
