# Set-AzPeeringRegisteredPrefix

## STRESZCZENIe
Ustawia lub aktualizuje zarejestrowany prefiks na podstawie nadrzędnego zasobu równorzędnego.

## POLECENIA

### ByResourceGroupAndName (domyślny)
```
Set-AzPeeringRegisteredPrefix [-ResourceGroupName] <String> [-PeeringName] <String> [-Name] <String>
 -Prefix <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Inputobject
```
Set-AzPeeringRegisteredPrefix -InputObject <PSPeeringRegisteredPrefix> -Prefix <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Set-AzPeeringRegisteredPrefix [-ResourceId] <String> [-Name] <String> -Prefix <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Umożliwia zaktualizowanie zarejestrowanego prefiksu z poziomu nadrzędnego zasobu równorzędnego.

## Przykłady

### Przykład 1
```powershell
PS C:\> Set-AzPeeringRegisteredPrefix -ResourceId $resourceId -Prefix $newPrefix
```

Umożliwia zaktualizowanie prefiksu przez identyfikator zasobu.
