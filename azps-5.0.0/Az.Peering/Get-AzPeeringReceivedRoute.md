# Get-AzPeeringReceivedRoute

## STRESZCZENIe
Wyświetla listę odebranych tras dla komunikacji równorzędnej.

## POLECENIA

### ByResourceGroupAndName (domyślny)
```
Get-AzPeeringReceivedRoute [-ResourceGroupName] <String> -Name <String> [-Prefix <String>] [-AsPath <String>]
 [-OriginAsValidationState <String>] [-RPKIValidationState <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByResourceId
```
Get-AzPeeringReceivedRoute [-ResourceId] <String> [-Prefix <String>] [-AsPath <String>]
 [-OriginAsValidationState <String>] [-RPKIValidationState <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Opis
Wyświetla listę odebranych tras od komunikacji równorzędnej

## Przykłady

### Lista najlepszych 100 odebranych tras dla komunikacji równorzędnej
```powershell
PS C:\> Get-AzPeeringReceivedRoute -ResourceGroupName $resourceGroupName -Name $peeringName
```

Wyświetla listę wszystkich odebranych tras dla komunikacji równorzędnej

### Filtruj według ścieżki
```powershell
PS C:\> Get-AzPeeringReceivedRoute -ResourceGroupName $resourceGroupName -Name $peeringName -AsPath "1234 5674 9834"
```

Wyświetla listę wszystkich odebranych tras dla komunikacji równorzędnej z filtrem jako 

### Filtruj według RPKIValidationState
```powershell
PS C:\> Get-AzPeeringReceivedRoute -ResourceGroupName $resourceGroupName -Name $peeringName -RPKIValidationState "Valid"
```

Wyświetla listę wszystkich odebranych tras dla komunikacji równorzędnej z filtrem na RPKIValidationState

### Filtruj według OriginAsValidationState
```powershell
PS C:\> Get-AzPeeringReceivedRoute -ResourceGroupName $resourceGroupName -Name $peeringName -OriginAsValidationState "Valid"
```

Wyświetla listę wszystkich odebranych tras dla komunikacji równorzędnej z filtrem na OriginAsValidationState
