# Wait-AzDataLakeAnalyticsJob

## STRESZCZENIe
Oczekuje na ukończenie zadania.

## POLECENIA

```
Wait-AzDataLakeAnalyticsJob [-Account] <String> [-JobId] <Guid> [[-WaitIntervalInSeconds] <Int32>]
 [[-TimeoutInSeconds] <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **wait-AzDataLakeAnalyticsJob** oczekuje na ukończenie zadania usługi Azure Data Lake Analytics.

## Przykłady

### Przykład 1. oczekiwanie na ukończenie zadania
```
PS C:\>Wait-AzDataLakeAnalyticsJob -Account "ContosoAdlAccount" -JobId "a0a78d72-3fa8-4564-9b18-6becb3fda48a"
```

Poniższe polecenie czeka na zakończenie zadania o określonym IDENTYFIKATORze.
