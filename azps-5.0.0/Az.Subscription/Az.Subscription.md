## Opis
W tematach w tej sekcji opisano polecenia cmdlet usługi Azure PowerShell dotyczące subskrypcji platformy Azure w ramach usługi Azure Resource Manager (ARM). Polecenia cmdlet istnieją w obszarze nazw Microsoft. Azure. Commands. Subscription.

## AZ. polecenia cmdlet dotyczące subskrypcji
### [Get-AzSubscriptionAlias](Get-AzSubscriptionAlias.md)
Pobiera szczegóły aliasu subskrypcji

### [Nowe — AzSubscriptionAlias](New-AzSubscriptionAlias.md)
Tworzy nowy alias i abonament.

### [Remove-AzSubscriptionAlias](Remove-AzSubscriptionAlias.md)
Usuwa alias abonamentu.

### [Update-AzSubscription](Update-AzSubscription.md)
Aktualizuje subskrypcję platformy Azure
