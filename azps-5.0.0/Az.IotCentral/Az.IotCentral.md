# AZ. IotCentral, moduł
## Opis
Zapewnia zarządzanie tych dla aplikacji dla komputerów z Centrum IoT.

## AZ. IotCentral polecenia cmdlet
### [Get-AzIotCentralApp](Get-AzIotCentralApp.md)
Pobiera właściwości jednego lub kilku aplikacji centralnych IoT.

### [Nowe — AzIotCentralApp](New-AzIotCentralApp.md)
Tworzy nową aplikację centralną IoT.

### [Remove-AzIotCentralApp](Remove-AzIotCentralApp.md)
Usuwa aplikację centralną IoT.

### [Set-AzIotCentralApp](Set-AzIotCentralApp.md)
Aktualizuje metadane aplikacji Centrum IoT.
