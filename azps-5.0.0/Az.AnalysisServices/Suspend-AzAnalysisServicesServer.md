# Suspend-AzAnalysisServicesServer

## STRESZCZENIe
Wstrzymuje wystąpienie serwera usług Analysis Services

## POLECENIA

```
Suspend-AzAnalysisServicesServer [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie Suspend-AzAnalysisServicesServer polecenia zawiesza wystąpienie serwera usług Analysis Services

## Przykłady

### Przykład 1
```
PS C:\> Suspend-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

To polecenie spowoduje zawieszenie aktywnego serwera o nazwie TestServer w ramach testera zasobów.
