# Remove-AzStreamAnalyticsInput

## STRESZCZENIe
Usuwa dane wejściowe z zadania Stream Analytics.

## POLECENIA

```
Remove-AzStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzStreamAnalyticsInput** asynchronicznie usuwa dane wejściowe z zadania Stream Analytics na platformie Azure.

## Przykłady

### PRZYKŁAD 1. Usuń strumień wejściowy z zadania
```
PS C:\>Remove-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EventStream"
```

To polecenie usuwa EventStream wejściowy z StreamingJob.
