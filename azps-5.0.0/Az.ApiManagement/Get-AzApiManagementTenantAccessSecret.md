# Get-AzApiManagementTenantAccessSecret

## STRESZCZENIe
Pobiera klucze konfiguracji programu Access dla dzierżawy.

## POLECENIA

```
Get-AzApiManagementTenantAccessSecret -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Pobiera klucze konfiguracji programu Access dla dzierżawy.

## Przykłady

### Przykład 1: Uzyskaj klucze konfiguracji dostępu do dzierżawy
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantAccessSecret -Context $apimContext
```

To polecenie pobiera klucze konfiguracji dostępu do dzierżawy dla określonego kontekstu.
