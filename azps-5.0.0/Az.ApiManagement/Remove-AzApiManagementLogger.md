# Remove-AzApiManagementLogger

## STRESZCZENIe
Usuwa rejestratora zarządzania interfejsem API.

## POLECENIA

```
Remove-AzApiManagementLogger -Context <PsApiManagementContext> -LoggerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Remove-AzApiManagementLogger** usuwa **Rejestrator** zarządzania interfejsem API platformy Azure.

## Przykłady

### Przykład 1: usuwanie rejestratora
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementLogger -Context $apimContext -LoggerId "Logger123" -Force
```

To polecenie usuwa rejestratora o IDENTYFIKATORze Logger123.
