# Get-AzApiManagementOpenIdConnectProvider

## STRESZCZENIe
Pobiera dostawców z OpenID Connect.

## POLECENIA

### GetAllOpenIdConnectProviders (domyślny)
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByOpenIdConnectProviderId
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext> [-OpenIdConnectProviderId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByName
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzApiManagementOpenIdConnectProvider** Pobiera dostawców usługi OpenID Connect w usłudze Azure API Management.
ClientSecret nie zostaną uwzględnione w szczegółach wyników. Aby uzyskać klucz tajny klienta, użyj polecenie **Get-AzApiManagementOpenIdConnectProviderClientSecret**.

## Przykłady

### Przykład 1: Pobierz wszystkich dostawców
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext
```

To polecenie pobiera wszystkich dostawców usługi OpenID Connect dla określonego kontekstu.

### Przykład 2: Uzyskaj dostawcę przy użyciu identyfikatora
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext -OpenIdConnectProviderId "OICProvider01"
```

To polecenie uzyskuje dostawcę o IDENTYFIKATORze OICProvider01.

### Przykład 3: Uzyskaj dostawcę przy użyciu nazwy
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext -Name "Contoso OpenID Connect Provider"
```

To polecenie uzyskuje dostawcę o nazwie contoso OpenID Connect Provider.
