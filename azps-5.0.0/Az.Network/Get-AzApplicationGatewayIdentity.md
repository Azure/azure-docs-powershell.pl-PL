# Get-AzApplicationGatewayIdentity

## STRESZCZENIe
Uzyskaj tożsamość przypisaną do bramy aplikacji.

## POLECENIA

```
Get-AzApplicationGatewayIdentity -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzApplicationGatewayIdentity** pobiera tożsamość przypisaną bramie aplikacji.

## Przykłady

### Przykład 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $identity = Get-AzApplicationGatewayIdentity -ApplicationGateway $gw
```

W tym przykładzie pokazano, jak uzyskać tożsamość bramy aplikacji z bramy Application Gateway.
