# New-AzLoadBalancerProbeConfig

## STRESZCZENIe
Umożliwia utworzenie konfiguracji sondy dla modułu równoważenia obciążenia.

## POLECENIA

```
New-AzLoadBalancerProbeConfig -Name <String> [-Protocol <String>] -Port <Int32> -IntervalInSeconds <Int32>
 -ProbeCount <Int32> [-RequestPath <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **New-AzLoadBalancerProbeConfig** umożliwia utworzenie konfiguracji sondy modułu równoważenia obciążenia platformy Azure.

## Przykłady

### Przykład 1. Tworzenie konfiguracji sondy
```powershell
PS C:\>New-AzLoadBalancerProbeConfig -Name "MyProbe" -Protocol "http" -Port 80 -IntervalInSeconds 15 -ProbeCount 15
```

To polecenie umożliwia utworzenie konfiguracji sondy o nazwie Moja sonda przy użyciu protokołu HTTP.
Nowa sonda połączy się z usługą równoważenia obciążenia na porcie 80.

### Przykład 2

Umożliwia utworzenie konfiguracji sondy dla modułu równoważenia obciążenia. (autogenerowana)

<!-- Aladdin Generated Example -->
```powershell
New-AzLoadBalancerProbeConfig -IntervalInSeconds 15 -Name 'MyProbe' -Port 80 -ProbeCount 15 -Protocol 'http' -RequestPath 'healthcheck.aspx'
```
