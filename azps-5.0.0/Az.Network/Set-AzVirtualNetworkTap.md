# Set-AzVirtualNetworkTap

## STRESZCZENIe
Umożliwia zaktualizowanie naciśnięcia sieci wirtualnej.

## POLECENIA

```
Set-AzVirtualNetworkTap -VirtualNetworkTap <PSVirtualNetworkTap> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Set-AzVirtualNetworkTap** umożliwia zaktualizowanie naciśnięcia sieci wirtualnej.

## Przykłady

### Przykład 1: Konfigurowanie sieci wirtualnej
```
PS C:\>$vTap = Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
PS C:\>$vTap.DestinationNetworkInterfaceIPConfiguration = $newDestinationNic.IpConfigurations[0]
PS C:\>Set-AzVirtualNetworkTap -VirtualNetworkTap $vTap
```

Polecenie aktualizuje docelowy element IpConfiguration i aktualizuje naciśnięcie sieci wirtualnej.
W przypadku jakichkolwiek odwołujących się do niej konfiguracji, nie zostanie on uruchomiony jako dublowany do nowej aktualizacji docelowa Konfiguracja IP.
