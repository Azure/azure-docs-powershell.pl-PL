# Get-AzApplicationGatewayRewriteRuleSet

## STRESZCZENIe
Pobiera zestaw reguł ponownej edycji bramy aplikacji.

## POLECENIA

```
Get-AzApplicationGatewayRewriteRuleSet [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Pobiera zestaw reguł ponownej edycji bramy aplikacji.

## Przykłady

### Przykład 1: uzyskiwanie określonego zestawu reguł ponownej edycji
```
PS C:\> $AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Rule = Get-AzApplicationGatewayRewriteRuleSet -Name "RuleSet01" -ApplicationGateway $AppGW
```

Pierwsze polecenie uzyskuje bramkę Application Gateway o nazwie ApplicationGateway01 i zapisuje wynik w zmiennej o nazwie $AppGW.
Drugie polecenie pobiera zestaw reguł "ponowne wpisywanie o nazwie RuleSet01" od bramy aplikacji zapisanej w zmiennej o nazwie $AppGW.
