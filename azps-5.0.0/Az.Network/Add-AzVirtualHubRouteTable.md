# Add-AzVirtualHubRouteTable

## STRESZCZENIe
Tworzy zasób tabeli routingu centrum wirtualnego, który jest elementem podrzędnym VirtualHub.

## POLECENIA

```
Add-AzVirtualHubRouteTable -Name <String> -Route <PSVirtualHubRoute[]> -Connection <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Zasób tabela routingu koncentratora wirtualnego zawiera listę tras oraz listę połączeń, do których można dołączyć i które są używane do kierowania ruchu w wirtualnym koncentratorze.
