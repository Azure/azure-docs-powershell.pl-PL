# Get-AzVirtualNetworkTap

## STRESZCZENIe
Umożliwia wybranie sieci wirtualnej

## POLECENIA

### ListParameterSet (domyślny)
```
Get-AzVirtualNetworkTap [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzVirtualNetworkTap -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzVirtualNetworkTap umożliwia pobranie** wirtualnej sieci Azure w grupie zasobów.

## Przykłady

### Przykład 1: Uzyskaj sieć wirtualną naciśnij
```
PS C:\> Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
```

To polecenie pobiera VirtualNetwork, dla którego otrzymano odwołanie "VirtualTap1" w "ResourceGroup1".

### Przykład 2: pobieranie wszystkich naciśnięć sieci wirtualnej za pomocą funkcji filtrowania
```
PS C:\> Get-AzVirtualNetworkTap -Name "VirtualTap*"
```

To polecenie pobiera wszystkie VirtualNetwork, które rozpoczynają się od "VirtualTap".
