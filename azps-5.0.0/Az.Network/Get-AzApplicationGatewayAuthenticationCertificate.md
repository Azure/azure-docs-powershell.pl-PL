# Get-AzApplicationGatewayAuthenticationCertificate

## STRESZCZENIe
Pobiera certyfikat uwierzytelniania dla bramy aplikacji.

## POLECENIA

```
Get-AzApplicationGatewayAuthenticationCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Opis
Polecenie cmdlet **Get-AzApplicationGatewayAuthenticationCertificate** pobiera certyfikat uwierzytelniania dla bramy aplikacji platformy Azure.

## Przykłady

### Przykład 1: uzyskiwanie określonego certyfikatu uwierzytelniania
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $pool = Get-AzApplicationGatewayBackendAddressPool -Name "pool01" -ApplicationGateway $appgw
```

Pierwsze polecenie uzyskuje bramkę Application Gateway o nazwie appGwName i zapisuje ją w zmiennej $appgw.
Drugie polecenie pobiera certyfikat uwierzytelniania o nazwie pool01 i zapisuje go w zmiennej $pool.
