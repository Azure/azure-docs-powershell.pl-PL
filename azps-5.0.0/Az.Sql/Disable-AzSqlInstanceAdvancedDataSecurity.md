# Disable-AzSqlInstanceAdvancedDataSecurity

## STRESZCZENIe
Wyłącza zaawansowane zabezpieczenia danych w zarządzanym wystąpieniu.

## POLECENIA

```
Disable-AzSqlInstanceAdvancedDataSecurity [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Opis
Polecenie cmdlet **disable-AzSqlInstanceAdvancedDataSecurity** wyłącza zaawansowane zabezpieczenia danych w zarządzanym wystąpieniu.

## Przykłady

### Przykład 1-Wyłącz Zaawansowane bezpieczeństwo danych wystąpienia zarządzanego
```powershell
PS C:\>  Disable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```

### Przykład 2 — Wyłączenie zaawansowanego zabezpieczenia danych serwera z zasobu zarządzanego wystąpienia
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Disable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```
