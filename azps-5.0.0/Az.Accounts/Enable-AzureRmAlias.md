# Enable-AzureRmAlias

## STRESZCZENIe
Włącza aliasy prefiksów AzureRm dla modułów AZ.

## POLECENIA

```
Enable-AzureRmAlias [-Scope <String>] [-Module <String[]>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Opis
Włącza aliasy prefiksów AzureRm dla modułów AZ. Jeśli określono-moduł, na liście będą dostępne tylko moduły. W przeciwnym razie wszystkie aliasy AzureRm są włączone.

## Przykłady

### Przykład 1
```powershell
PS C:\> Enable-AzureRmAlias
```

Włącza wszystkie prefiksy AzureRm dla bieżącej sesji programu PowerShell.

### Przykład 2
```powershell
PS C:\> Enable-AzureRmAlias -Module Az.Accounts -Scope CurrentUser
```

Włącza aliasy AzureRm dla modułu AZ. Accounts dla bieżącego procesu i dla bieżącego użytkownika.
